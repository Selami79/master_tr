Angled Milling Heads and Milling Tool Locators (Mill-Turn)

Tool locators for turret use are divided into locators for turning tools and for milling tools. You can create milling tool locators for face milling, cross milling, or at an angular orientation. You can specify a fixed angle in the component properties, or read the orientation angle from the operation. This lets you create tilted plane, G68-style toolpaths with turret-only machines.

Loading tools into angled milling heads

The workflow for tools mounted in angled milling heads is different than for other tools. Follow these steps:

  1. While working in the toolpath settings, select the desired tool.

  2. Right-click on the tool and select Get angled head.

    Mastercam opens the Tool Setup Manager.

  3. From the Tool Locators window, select the angled head and drag it to the desired index position on the turret. If necessary, open a different component library that contains the angled head.

  4. If the selected index position does not match the tool number in the toolpath settings, you will need to edit the tool number.

    1. In the Tools in the machine group window, right-click on the tool for the operation and select Edit Tool.

    2. Edit the tool number to match the index position where the head is mounted.

  5. Load the tool by dragging it to the component tree and releasing it on the angled head.

  6. Click OK to return to your toolpath settings.

  7. To use the tool again in additional operations, simply select it like any other tool. You do not need to repeat the steps in Tool Setup Manager.

Another common approach is to use the Tool Setup Manager to load all of the tools before you begin programming your toolpaths. Once a tool has been mounted into an angled head, you can select it for an operation like any other tool.

Using angled milling heads in operations

When Mastercam detects that angled milling heads are available or when you load a tool into an angled milling head, Mastercam enables the Plane Rotation setup type.

Typically this setup type is not available for turret-only machines, but will be required whenever the milling head is oriented at any angle other than 0 or 90 degrees. The orientation of the head also filters the allowable axis combinations.

Angled heads can be created with a fixed angle, or you can configure them to get the angular orientation from the operation. This affects how you set the tool plane.

  • If the head is set to a fixed angle, you will typically want to ensure that the orientation of the tool plane matches the angled head. If it does not, Mastercam displays a warning message.

  • If the head is set to read the angle from the operation, you will be able to select a tool plane with your operation, or select the Derive from geometry option if your toolpath geometry supports it.

Tool numbering

Once a tool has been selected for an operation and mounted in an angled milling head, you cannot change the tool number. The only way to change the tool number is to select a new tool or delete the operation. The Assign tool numbers sequentially option in your machine group is not applied to tools mounted in angled milling heads.

Drill cycles

Drill cycles can only be output in plane rotation or tilted-plane (e.g. G68) mode. If tilted-plane machining or G68 mode is not supported by your machine or post, Mastercam will output longhand code instead of cycle output.

Scaling the spindle speed

Some live tooling heads are geared to increase, decrease, or reverse the programmed spindle speed. These settings are available in the component properties for the tool locator. The spindle speed adjustment will be applied automatically when you select the angled head.

Milling tool locators and the Holder page

For most Mill-Turn applications, if you load a tool in a milling tool locator there is no need to select or create a holder on the Holder page. Since Mastercam does not let you create the operation without a holder, many users define the holder as a very thin slice—for example, .001 inch. You can also use the Holder page to define a sleeve or collar for the tool.
