Selecting Mesh Facets

After you choose a mesh body on which to select facets, the Facet Selection flyout panel overlays the function panel. You can select facets using two methods:

  • Brush selects all facets within the area of the selection tool.

  • Single selects individual facets.

Brush Single

If you select Single, you do not need to set any additional options. You can individually select the facets of the mesh as needed.

If you select Brush, additional options for the boundary and the brush radius become available. The Method options determine which facets are selected within the brush radius. Inside selects facets beneath the cursor, as well as neighboring facets that are completely within the boundary of the brush radius. Intersect selects facets within and facets intersecting the boundary of the brush radius.

Inside Intersect
NOTE

You can click and drag your mouse to add more facets to your selection. You can also right-click to deselect individual facets, or right-click and drag your mouse to remove multiple facets from your selection.

Display toggles the display of the brush radius in the graphics window.

Display On Display Off

The Wrap around mesh option selects facets within and beyond the visible area of the selection brush. This means that you can select facets through the mesh that are within the brush radius.

Wrap around mesh off Wrap around mesh on

Use the Brush Radius scale to increase or decrease the brush size.

50% 100%

Reset deselects all mesh facets and you can select new ones.

Color Masking selects facets on the target mesh by color. Use Color Masking to limit the facets you can select on the mesh body to the options selected in the Colors list. The following options are available when Color Masking is active.

  • To select all facets of certain colors, click Select all facets by color. Mastercam selects all facets on the mesh that match the selections in the Colors list. Use this option when there are multiple facet colors on the selected mesh body and you want to select all facets of certain colors.

  • To deselect all facets selected using Color Masking, click Unselect all facets by color.

  • To modify the options in the Colors list, click Sample facets and follow the prompts in the graphics window. The Sample facets button allows you to precisely add or remove a facet color from the masking list if your mesh contains facets with subtle color differences.

NOTE

You can enable and disable Color Masking throughout the facet selection process. Facets selected before or after enabling Color Masking are not affected by Color Masking and its options.

Once you have finished selecting the facets, click OK in the flyout panel to return to the mesh function panel.
