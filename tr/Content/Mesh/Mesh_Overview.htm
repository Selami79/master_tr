Creating and Modifying Meshes

Mastercam supports the creation and modification of mesh entities. A mesh is a 3D object containing a collection of edges, faces, and vertices. All mesh functions can be found on the Mesh tab.

Creating Meshes

You can create new meshes in the following ways. These functions are found in the Simple and Create groups in the Mesh tab.

List of functions
Icon Name Description
Primitive Cylinder Creates a cylinder from a base point by dragging outward to set the radius, and then dragging up or down to set the height.
Primitive Block Creates a block from a base point by dragging outward to set the length and width, and then dragging up or down to set the height.
Primitive Sphere Creates a sphere from a base point and dragging outward to set the radius.
Primitive Cone Creates a cone from a base point by dragging outward to set the radius, and then dragging up or down to set the height.
Primitive Torus Creates a torus from a base point by dragging outward to set the major radius, and then dragging out the minor radius.
Meshes From Entities Extracts information from the selected solids, surfaces, or mesh entities to create a new mesh entity.

Importing Meshes

Mastercam supports opening and importing of other mesh file formats. This includes, but is not limited to:

  • StereoLithography files (*.stl)

  • Rhino 3D files (*.3dm)

  • AutoCAD files (*.dxf, *.dwg)

To open other file formats, select File, Open, Browse. In the Open dialog box, set the Type to the correct file format. Some file types include additional options, accessed by selecting Options in the Open dialog box.

Whether you are creating a new mesh or importing an existing one, the meshes must be manifold and contain no degenerate, flipped, or folded faces. To find and fix any of these issues, refer to the drop-down below or refer to the Check Mesh panel Help topic.

Modifying Meshes

After importing or creating a mesh, you can use the functions listed below to further refine and modify your mesh bodies. These functions are found in the Modify and Color groups of the Mesh tab.

NOTE

Run Check Mesh before using any of the Modify functions on the Mesh tab to ensure your mesh is error-free. If it is not and you attempt to use one of those functions, Mastercam reports an error and you cannot continue until the mesh is fixed.

For more information, see Fixing Mesh Errors.

List of functions
Icon Name Description
Offset Mesh Expands a mesh by a specified distance equally in all directions.

Smooth Free Edges

Cleans up free edges of a mesh entity. Use this function to prepare an edge or hole before using Fill Holes.

Fill Holes

Seals holes in a mesh entity.

Smooth Area

Smooths an entire mesh or a selected region of mesh facets.

Refine

Improves a mesh entity by adding more facets to detailed areas and reducing them in less detailed areas. Refine can also create uniform-size facets.

Decimation

Reduces the number of facets in a mesh entity. This may help to reduce the file size of the mesh entity.

Explode Mesh

Separates a complex mesh model into separate mesh bodies.

Modify Mesh Facets

Creates, removes, or repairs selected facets on a mesh body.

Trim to Plane

Trims a mesh entity to a selected plane.

Trim to Surface/Sheet

Trims a mesh entity to a selected surface or sheet.
Trim to Chain Trims a mesh entity to a selected chain.
Check Mesh Identifies errors on a selected mesh.
Clear All Resets all faces, features, and facets to their original color.
Modify Facet Color Changes the color of selected facets on a mesh model.
What other functions can I use with mesh entities?

You can also interact with mesh entities using the following functions. This not a comprehensive list.

Icon Name Tab Description
Modify at Intersection Wireframe Breaks, trims, or creates a point where wireframe entities intersect with a mesh.
Curve at Intersection Wireframe Creates curves at the intersections between meshes.
Curve Slice Along Curve Wireframe Creates wireframe slices through meshes along a selected chain.
Curve Slice by Plane Wireframe Creates curves on meshes.
Disassemble panel Model Prep Separates mesh bodies and organizes them in a specified plane.
Change Normal function Surfaces Reverses the normal direction of a mesh.
Orient Normals Surfaces Aligns the normals of neighboring facets within a selection.
