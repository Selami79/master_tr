Nesting Results dialog box

Use the Nesting Results dialog box to view the results from geometry or toolpath nesting before Mastercam actually creates the nested geometry or operations in the graphics window.

Parameters

Sheet (result)

Choose the sheet to view from the drop-down menu. Mastercam shows the results generated for each sheet in the nesting session. Sheets that have identical results are listed separately.

Utilization

Shows the percentage of the total sheet area being used by the results.

Results preview window

Shows the results on the selected sheet. Right-click in the window for details. Choose a part from the right-click menu to view it individually, or select Display all to view all parts simultaneously (default).

Horizontal/Vertical trim

Select to place a horizontal or vertical dashed line across the sheet. These lines indicate the location of cut-off operations on each sheet.

  • If you select the Trim function, you do not need to change these options. Mastercam automatically generates a saw toolpath operation and displays the cut-offs in the Results preview window.
  • If you leave the Trim function deselected, use these options to create toolpath geometry. You must manually select these lines and create your own toolpath operations to save the remainder of the sheet for future use.

Selecting Horizontal or Vertical trim changes the Total size, Utilization, and Total area amounts that are displayed in the Results dialog box.

White background

Select to display your results in the preview window on a white background.

Black part geometry

Select to apply a black outline to part geometry displayed in the preview window.

Line width

Select a line width from the drop-down list.

Total size/area

Shows the overall dimensions and area available on the sheet.

Nested size/area

Shows the overall dimensions and area used by the nested parts.

Details

Opens an ActiveReports dialog box in which you can set up the layout of the labels and reports.

Redo

Returns to the Nesting dialog box so that you can change parameters and regenerate the results.

Save

(Geometry nesting only) Saves the nesting session file (NSTX) to a new name.

Copy

Copies the nesting results to your clipboard.

Copy blank (no parts)

Copies the nesting result without the geometry to your clipboard.

Delete

Allows the removal of one or more selected parts from a sheet.

Sort

Toolpath nesting only. Opens the Selection dialog box. Allows you to select operations individually in the order that you want them output. Select a single operation or multiple operations. After you finish selecting, the Placement dialog box opens with additional options for outputting the selected operations before or after all the other operations or sheets.

Customize

(Geometry nesting only) Opens the Customize dialog box where you can change parameters, including the locking and deleting of selected parts, and regenerate the results.

Drag

Enables the Drag function in the Bump nesting dialog box. From there, you can move or copy selected parts to new positions on any sheet by dragging and/or rotating them.

Right-click menu options

Display all

Shows a thumbnail of all the results in the nesting session.
