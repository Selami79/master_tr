Nesting Configuration dialog box

Use the Nesting Configuration dialog box to set up preferences for Nesting. These options apply to both geometry and toolpath nesting.

Parameters

Load default sheet

Select to automatically load a rectangular sheet when you start a nesting session. The dimensions of the sheet are 96 x 48 inches or 2438 x 1218 mm.

Display group page

Enables the Groups tab in the Geometry and Toolpath Nesting dialog boxes. A nesting group is a collection of parts that form a complete set, like the parts of a table or cabinet.

Display results dialog

Automatically displays the Nesting Results dialog box where you can view the results from geometry or toolpath nesting before Mastercam actually creates the geometry or operations in the graphics window.

Higher resolution images in reports

Controls the resolution of images in Nesting reports.

  • Select for higher resolution images. Images in the Sheet/Result section of Nesting reports will have a maximum length or width of 2400 pixels. Images in the Part Information section will have a maximum length or width of 1200 pixels.
  • Deselect to create all images in Nesting reports at 600 x 300 pixels.

Place imported geometry on unique level

Tells Mastercam to place geometry automatically on a unique level named after the part file. If this option is off, Mastercam uses the part's existing levels. This option is on by default.

Place label on top of solid

Select to create labels on the top of the solid, instead of at Z zero.

Use part name as label

Tells Mastercam to use the part's name as the part's label in the sheet. The Add part labels option on the Parts tab must be enabled for part labels to appear.

Delete original chains

Deletes the chains that you selected when adding parts from the graphics window. When this checkbox is cleared, Mastercam leaves the chains in their original position in the graphics window. However, the nesting result may be positioned on top of the original chains depending on the location of the sheet origin.

Restore last session

(Geometry nesting only) Saves and reloads the settings from the previous session, even if you have not saved them as a nesting session. This option is useful when you are working with nesting settings to get the best sheet usage. You can exit Nesting, and the settings that you last used are automatically retrieved and loaded when you re-enter Nesting or restart Mastercam. For this option to be applied, there must be geometry present in the Part List when you click OK to exit the Nesting session.

Compress results

Select to compress (combine) duplicate results into the least number of sheets rather than setting each nesting result into a separate sheet.

Cycle colors starting with

Assigns a different color to the copies of each part. Enter the starting color by number or choose Color. Nesting assigns the color entered to Part 1 and cycles colors in numerical order.

Cycle levels starting with

Places the copies of each sheet and part on separate levels. For example, Sheet 1 can be placed on Level 1, Part 1 on Level 2, and so on. Nesting cycles through levels in numerical order starting with the level number entered.

Only unused levels

Places the sheet or part geometry on empty and unnamed levels starting with the level number entered. When deselected, Nesting merges the geometry with any geometry already on the level.

Color images in reports

Select to include and save color images in Nesting reports.

NOTE

Toolpaths will only display in color if Cycle colors starting with is enabled.

Image format

Choose an image format from the drop-down for the color images to be saved.

NOTE

Images are saved in C\Users\Public\Documents\Shared Mastercam 2025\common\reports\IMG.

Available image formats:

  • BMP
  • JPG
  • PNG
  • GIF

Save sheet scrap

Sheet scrap is material remaining from a sheet used in a previous nesting session. Select to automatically save a copy of the nested sheet in the scrap library (NSLX). Enter the name for the scrap library where you want Mastercam to place the sheet scraps.

Check for duplicates during save (slower)

Select to have Mastercam evaluate sheet scrap for duplicate shapes and sizes.

Maximum priority

Defines the highest maximum priority values for parts and sheets.

Accuracy

Determines the precision of the nest. The finer the accuracy, the longer the results may take to generate. Each of the predefined accuracy settings corresponds to a resolution value. Use a larger setting such as Coarse for bigger parts. Choosing User Defined enables the resolution field where you can enter the resolution for the results. A larger number creates a less precise result, but takes less time. A smaller number creates a more precise result, but takes more time.

Rectangular/TrueShape

  • Rectangular: Places a bounding box around the part and uses the box as the part boundary for nesting. The Rectangular nesting mode is available with Mastercam Router and Router 3D.
  • TrueShape: Nests parts according to each part's actual shape and other parameters. It allows access to sheet and part libraries, and extended functionality beyond Rectangular nesting.

Nest smaller parts in the holes of larger parts

(TrueShape nesting only) Nests parts in the holes of other parts before nesting parts on the sheet. Allows the usage of nestable material in the holes of parts.

Nest in holes before nesting on sheet

(TrueShape nesting only) Select to use any nestable material in the holes of parts before placing these parts on the sheet.

Nest to tool center/diameter

(Toolpath nesting only) These options contribute to the spacing between nested toolpaths. They are unavailable if the Optimize checkbox on the Parts tab of the Nesting dialog box is selected.

  • Nest to tool center: Use when nesting operations with Cutter Compensation set to Computer. Nesting to the tool center can nest parts so close that the tool will cut between two parts without wasting material. The image below depicts how nest to tool center is used. The equation used is:

    Nest to center = R1 + R2 + P-P

    1 Part 1
    2 Part 2
    3 Part to part distance (P-P)
    4 Tool number 2 radius (R2)
    5 Tool number 1 radius (R1)

    The image below shows two Contour operations nested to the tool center.

  • Nest to tool diameter: Use when nesting operations with Cutter Compensation set to Control to ensure enough space to accommodate the diameter of the tool.

    Nest to diameter = D1 + D2 + P-P

    1 Part 1
    2 Part 2
    3 Tool number 1 diameter (D1)
    4 Tool number 2 diameter(D2)
    5 Part to part distance (P-P)

    The image below shows two Contour operations nested to the tool diameter.

NOTE

Changing this parameter after creating a nesting operation does not affect the current nesting operation or the nesting configuration defaults file. Use one of the following methods to effectively modify the Nest to tool center / diameter setting for existing nesting operations:

  • Use the Tool and Holes checkboxes in the Select Operations dialog box. These options are available when you right-click and choose to Add Operations in the Parts tab.
  • Delete the current nesting operation, and then recreate it after saving new default parameters.

Automatically attach operation's geometry

(Toolpath nesting only) Automatically transforms the geometry originally used to create the operation (such as chains for pocketing or points/arcs for drilling).

Automatically attach geometry (search)

(Toolpath nesting only) Searches for and transforms any geometry it finds within the nesting boundary. For example, if you have chosen to nest a contour toolpath for a rectangle, this will transform anything inside the rectangle. This search finds geometry at all depths.

Create separate block drill operations

(Toolpath nesting only) Creates a new block drill operation for each sheet generated when nesting block drill toolpaths.

When you nest a block drill toolpath with this option selected, posting is turned off in the Toolpaths Manager for the nesting operation; and each new block drilling operation is inserted directly below the nesting result.

The block drill operations that are spawned from nesting block drill toolpaths are directly related to the nesting toolpath. To delete these operations, you need to select all the spawned operations plus the nested toolpath operation in Toolpaths Manager, and delete them all at once. Editing the parent nesting operation, automatically updates all spawned nesting operations.

Include rapid moves

(Toolpath nesting only) Select these options to expand the profile (outer boundary of the operation) to include rapid moves. Including rapid moves is especially advantageous when nesting drill operations.

  • All ops and Only drill ops save time by preventing Mastercam from profiling each drill entity. Mastercam draws the outer boundary around all the drill entities and the rapid moves.
  • None helps ensure that nestable material does not get included within a boundary because it is traversed by a rapid move.

Duplicate tool check during operation import

Opens the Duplicate Tool Check dialog box, where you select attributes used to test for duplicate tools.

Corner of sheet

Choose one of the four corners of the sheet to define the tool origin. Your selection should correspond to the way you set up the job. The toolpath coordinates that Mastercam outputs in the NC code are relative to the selected corner (tool origin). The selected tool origin displays in the Sheets tab preview area as a cyan-colored point in one of the upper or lower corners of the sheet.

Save as defaults

Select this option to save the nesting configuration settings to the nesting default file (.DEFX).
