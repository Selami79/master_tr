Parts: Geometry Nesting tab

Use the Geometry Nesting tab to set up part parameters for geometry nesting. Use the right-click menu in the Parts list to select the parts to be nested.

Follow these general steps when setting parts parameters:

  • Enter information about each part
  • Enter the number of parts
  • Increase the part yield

Parameters

Part list

Displays the parts included in the nesting session. Mastercam nests all of these parts on the sheets you specify in the Sheets tab. Right-click in the list box to add parts to the nesting session.

Right-click menu options

  • Add rectangular: Creates a rectangle to represent the part. Enter/Edit the dimensions of the rectangle by typing values in the Size fields.
  • Add chained parts: Returns you to the graphics window to chain part geometry.
  • Add chained cluster: Adds chained parts and maintains the position of the parts relative to each other. Nests the chained parts as a unit.
  • Disassemble solids: Returns you to the graphics window to nest solid bodies by disassembling the solid model.
  • Add solids: Returns you to the graphics window to select a solid body to nest.
  • Add parts from files: Retrieves geometry from another part file. This option has a limit of 1,000 characters for a single attempt, including the full path and file name. This is a limit of the operating system. If your selection contains too many characters, nothing is added to the Parts list and no error message displays. To add files totaling more than 1,000 characters with this option, you must make multiple selections.
  • Add clusters from files: Retrieves geometry from another part file, and groups it into a cluster.
  • Add clusters from folder: Retrieves geometry from one or more part files by allowing you to browse to and select an entire folder. When you select a folder, all supported files are added to the Parts list, and then grouped into a cluster. There is no character limitation with this option.
  • Add parts from folder: Retrieves geometry from one or more part files by allowing you to browse to and select an entire folder. When you select a folder, all supported files are added to the Parts list. There is no character limitation with this option.

  • Add options: Additional parameters for Mastercam to follow as it collects part geometry for a nesting session

    • Automatically attach geometry (search): When this option is selected, geometry that is enclosed by added geometry is attached to the nested parts. Note that the enclosed geometry does not appear in the Parts page, but appears when you view the nesting results.
    • Align with longest edge when disassembling solids: Arranges disassembled solids along the longest edge of the sheet.
    • Create a group when adding multiple parts: Creates a group from the parts you select. A nesting group is a collection of parts that form a complete set, for example, for a piece of furniture or a cabinet.
    • Ignore holes when adding parts: Nests only the outer boundaries of parts. Any geometry contained within the boundary keeps its position within the outer boundary and is copied and nested with it. When the Ignore holes option is on, Mastercam shows the inner boundaries in gray.
    • Scan for notes: Includes any note within the selected part's outer boundary. The note becomes part of the nested geometry.
  • Import from library: Imports existing part geometry from a nesting part library (NPL or NPLX file) or nesting session (NST or NSTX file) that you select.
  • Save to library: Saves any selected parts in the list to a nesting part library (NPLX file).
  • Attach geometry: Returns you to the graphics window to attach geometry to your nested parts.
  • Delete all: Removes all parts from the list.

Part preview

Displays the part that is highlighted in the Parts list. Shows the profile (outer boundaries), holes (inner boundaries), and any ignored chains (non-nestable geometry). Mastercam determines if any of the inner chains that it detects are holes, and outlines them in red. Right-click in the Part list window, and select Add options, Ignore holes when adding parts to change how Mastercam assesses inner chains.

Size

Shows the overall length and width of the part.

Minimum qty

Enter the number of copies of the selected part. Nesting creates as many copies as it can up to the minimum quantity. To make extra copies beyond the minimum quantity, use the Fill all sheets or Filler Qty fields.

Filler quantity

Enter the number of filler parts. Selecting this option limits filler parts to the area occupied by the nested minimum quantity. Nesting creates the minimum quantity of parts first, then creates filler parts starting with the part with the highest priority. If no priority is set, Nesting gives the largest part the highest priority and works down to the smallest part.

Step angle

Controls the degree of rotation Nesting can use when fitting parts on a sheet. For example, a 90 degree step angle allows Nesting to use 0, 90, 180, and 270 degree rotations. A Grain Direction set to Horizontal or Vertical disables this option.

Priority

Sets the order in which the parts are nested. By default, Nesting starts with the largest part first. Use the priority setting to force the nesting of a smaller part before a larger part, by giving the smaller part Priority 1 and the larger part Priority 2.

Label

Select Add part labels to enable the label field. Enter the text to be placed on all instances of the part or toolpath. When you select an Automatic label height, the text height defaults to 80% of the sheet-sheet distance.

Material

Enter a material in the field, or click Material to select a material from a material library.

Use bounding box

Places a bounding box around the extents of the part, and uses the box as the part boundary for nesting.

Common edge

Enable to position nested parts so that they share parallel edges. This allows a single cut to cut two edges simultaneously.

Grain direction

Use when you are nesting parts on material with a noticeable surface direction. Set the grain direction to match the orientation of the grain on the sheet. Mastercam rotates and positions the parts to match the grain, and disables the part step angle.

Grain Direction applies to all sheets in the nesting session. You cannot set it on a per sheet basis.

Mirror

Allows Nesting to flip the part on the X axis. You can choose to have only mirrored parts, both mirrored and non-mirrored parts, or only non-mirrored parts.

Part-part distance

Enter the amount of space between nested parts. Be sure to space the parts far enough apart to accommodate toolpaths.

Add part labels

Enables the label field on this tab so that you can have text placed on each part. Nesting places labels on parts along the X axis in the part's original orientation.

  • Height: Enter the height for the label text. All labels generated in a nesting session have the same height.
  • Automatic: Fits part labels to the part. When calculating the largest label size, Mastercam starts with the part dimension and works down until it finds the label extents that fit the part.

Optimized pairs

Creates close-fit pairs from parts in the nesting session. Once Mastercam creates the pair combination, it nests those parts in the pair combination. Optimized pairs is a method used to increase yield of parts on material.

Fill all sheets

Fills any unused material on all sheets in the nesting session. Nesting creates filler parts starting with the part with the highest priority. If no priority is set, Nesting gives the largest part the highest priority and works down to the smallest part.

Selecting this option disables Filler quantity.

Common Geometry Nesting Controls

New

Starts a new nesting session. Mastercam reloads the nesting defaults file (DEFX) including the default sheet, if one exists.

Open

Opens a nesting session file (NSTX).

Save

Saves sheets, parts, and any parameter settings to a nesting session file (NSTX).

Save as

Saves the nesting session file (NSTX) to a new name.

Restore

Restores the defaults file and current nesting session to factory settings.

Config

Opens the Nesting Configuration dialog box.

Customize Controls

Delete

Allows the removal of one or more selected parts from a sheet.

Lock

Prevents changes to the position of selected parts on the sheet. When selected, the part color toggles from purple (unlocked) to grey (locked).

Right-click menu: Dialog box

  • Save defaults: Saves all settings in the Sheets tab to the nesting defaults file (Nesting.DEFX or Nesting-MM.DEFX)
  • Load defaults: Retrieves Sheets tab settings from the nesting defaults file.
