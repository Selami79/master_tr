Managing Nesting Files

Mastercam provides several methods for streamlining your work when working with nesting. Click the drop-downs below to learn more.

Nesting defaults

The Nesting Configuration dialog box has an option for you to save its defaults separately from those of the other Nesting dialog box tabs. Using Save as defaults saves only the defaults on the Nesting Configuration dialog box, not the other nesting dialog box tabs.

Mastercam has a single nesting defaults file (NESTING.DEFX or NESTING-MM.DEFX) which you modify when you use the Save defaults option for each Nesting dialog box tab (right-click in the dialog box and choose Save defaults). You can return the DEFX file to the factory-set defaults by using the Restore button on the Nesting dialog box:

The Restore button overwrites your defaults with Mastercam's own nesting defaults. You cannot save your defaults in a different file with another name.

Nesting sessions

Nesting sessions save sheets, parts, and parameters for geometry nesting. Once you complete a nesting job, save the session by choosing the Save button at the bottom of the dialog box. Reopen the nesting session by using the Open button. Nesting sessions are saved with a NSTX extension. You can also import individual sheets or parts from a saved nesting session. Nesting sessions are only available for geometry nesting. Toolpath nesting saves parameters in an operation in the Toolpaths Manager.

On the Nesting Configuration dialog box, Select Restore last session to have Mastercam automatically load the geometry and settings from the previous nesting session.

Nesting part libraries

Nesting part libraries let you save and easily retrieve parts that you reuse regularly. Nesting part libraries have the NPLX extension. When you save a part to a library, Mastercam saves the part geometry, and settings for size, quantity, filler quantity, material, grain direction, and mirror.

To save a part to a library, right-click in the Parts list on the Parts tab and choose Save to library. Use Import from library in the same menu to bring a part into a nesting session from a library. To import more than one part from the same library, use [Shift+click] to select a range of parts or [Ctrl+click] to select separate parts.

Only geometry can be saved as parts in part libraries.

Nesting sheet libraries

Nesting sheet libraries save whole sheets and scrap sheets. Sheet libraries are saved with an NSLX extension. When you save a sheet to a library, Mastercam saves the sheet geometry and settings for size, material, origin, grain direction, and quantity.

To save a sheet in a library, right-click in the Sheets list on the Sheets tab and choose Save to library. Use Import from library in the same menu to bring a sheet into a nesting session from a library. To import more than one sheet from the same library, use [Shift+click] to select a range of sheets or [Ctrl+click] to select separate sheets.

A scrap sheet is the un-nested material left after a nesting session. To save a sheet in a sheet scrap library, select Save sheet scrap on the Nesting Configuration dialog box. Nesting automatically saves a copy of the nested sheet in the scrap library.

Nesting temporary files

Mastercam saves temporary files for nesting in \Mastercam 2025\Mastercam\common\TEMP, so that you can make them read-only if necessary.
