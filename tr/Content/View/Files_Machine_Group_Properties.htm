Files tab: Machine Group Properties - Lathe, Wire

Use the Files tab to specify the names and data paths of files needed by the operations in Lathe or Wire machine groups, and other important machine group properties. Click on a drop-down to read more about the individual sections of the tab:

Toolpath directory

Displays the path where the NC file will be written. Choose the following button to select a different directory:

When you post the files, you can create and name a separate NCI file, if desired. If you select operations from several machine groups, Mastercam creates a separate NC file for each group, according to each group's settings.

The default path and extension are set in the Files section of the control definition.

Working with files

Separate sections of this tab let you choose tool and operations libraries and toolpath defaults files. For each type of file, you can do the following:

  • Select a file by choosing Open next to the file name.
  • Edit the file by choosing Edit next to the file name.

Default paths and file names are read from the Files section of the control definition. If you select a different machine for this machine group, Mastercam replaces these files and paths with the values from the control definition for the new machine.

Working with the machine definition

The Files tab displays the name of the machine definition that was selected to create the machine group. You can choose a different machine in the same way as choosing other types of files using the Replace button.

If you select a new machine, Mastercam loads a copy of the machine definition in your part file. When you click Edit to view or make changes to the machine definition, you are only changing the local copy stored in your part file, not the machine definition stored on your hard drive. Selecting a new machine definition updates the other paths and file names with the defaults in the new control definition.

NOTE
  • The control and post are displayed for information purposes only. You can only change them by selecting a new machine definition.
  • If there are already operations in the machine group, be careful when selecting a different machine to ensure the new machine supports all the toolpath features that have been programmed.
Working with comments

If you wish, use the Group comment box to record comments that apply to all the operations in this group. Mastercam also lets you record operation-specific comments in the Toolpath parameters tab, or machine-specific comments in the machine definition.

  • Use the Output comments to NC file section to control which comments are processed by the post and added to the NC file. Comments that are not included in the NC file are still displayed in Mastercam.

  • Use the NC Output section of the control definition to set the defaults and enter the maximum number of characters allowed for comments.

Parameters

Group name

Enter the name of the machine group in this field. It will be displayed in the Toolpaths Manager.

Toolpath directory

Specifies where Mastercam writes the NC programs.

Select path (button)

Click to select a new default path for the selected file type. All files of this type found in the selected directory will be listed in the File drop-down in the Open dialog box.

Group comment

Enter a text comment for the group. You can output the comment to the NC file by choosing the option at the bottom of this tab.

Edit machine (button)

Opens the Machine Definition Manager to make edits to the machine. Changes you make are only saved to the local copy stored in your part file, not the master copy stored on your workstation. Not all Machine Definition Manager and Control Definition Manager functions are available.

Replace machine (button)

Click to select a different machine definition. When you replace the machine definition, Mastercam does the following:

  • Validates any operations that have already been created against the new machine definition. Mastercam alerts you if there are any incompatibilities.
  • Replaces the displayed post processor with the one used by the new machine definition.

Select file (button)

Click to choose a new file for the selected file type.

Edit item (button)

Click to open the file listed in the File field so you can edit it.

Tool Manager (button)

Opens the Tool Manager so you can edit a Lathe tool library.

Tool Manager (button )

Opens the Tool Manager so you can edit a Mill tool library.

NOTE

If you are running Mastercam Lathe, you must navigate to the proper folder to find your Mill tool libraries.

Output comments to NC file

Select which types of comments you want to include in the NC file. If you do not select the comment type, it will still be available for viewing within Mastercam.

How do I...?

How do I create or edit an operation defaults library (*.mcam-operations)?

Follow these steps to create a new operations library. This is a file with an .mcam-operations extension that contains commonly used toolpath operations that you can select and import into your current part file. Begin creating new operations files from a machine definition already loaded in the Toolpaths Manager.

  1. In the Toolpaths Manager, select Files under Properties.

  2. Select the Edit button to open the Edit Operation Library dialog box.

  3. Select Browse to open the Select Operation Defaults File dialog box.

  4. Right-click the operations file on which you want to base your new library, and choose Copy.

  5. Right-click in the window and choose Paste.

  6. Right-click the new copy of the operations file, choose Rename, enter the new file name, and press [Enter].

    NOTE

    When renaming your file, make sure not to change the file extension.

  7. Re-click the new file name. Make sure the new operations file name appears in the File name field.

  8. Click OK.

  9. If prompted to save changes to your operations library, click No.

  10. Edit the new operations library as necessary.

    NOTE

    You can also empty the new file and start with a blank file by choosing Delete all operations from the toolbar.

  11. When finished, click OK to save your changes and return to the Toolpaths Manager.

To edit an operations library, follow to step 4 and modify the operations in the list. You can do the following:

  • Edit the default parameters for a specific operation in the list by clicking Parameters.

  • Work in the operations list much like you can in the Toolpaths Manager. You can access many different options from the right-click menu, including adding, editing, and deleting operations. You can also perform other tasks similar to those in Toolpaths Manager such as selecting multiple operations, dragging and dropping operations within the list, and changing the names of operations.

Click OK when you are done. The operations library you just worked on is set as the current library for that machine group.

How do I create or edit toolpath default (*.mcam-defaults) files?

You can create new toolpath default files or edit existing ones. For more information about working with toolpath default files, refer to the Working with Toolpath Defaults Help topic.
