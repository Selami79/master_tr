Planes Manager

Use the Planes Manager to select, create, edit, and manage planes. The grid in the Planes Manager shows all of the planes defined in the current file. Refer to the parameters below for more information.

You can modify the Planes Manager. Mastercam remembers the settings from session to session.

  • Float, dock, collapse, or move the Planes Manager to another monitor.
  • Right-click the column header and select which columns to show or hide.
  • Reorder the columns by clicking and dragging a column header to a new location.
  • Click in a column header to sort that column alphabetically.
  • Use [Shift+click] or [Ctrl+click] to select multiple planes at a time.
  • Use the drop-down in the right-corner of the Planes Manager to change the font attributes and to reset their attributes to the default settings.

Create a new plane

Choose one of the following options to create a new plane using the New Plane function panel. You can name and define the plane and its origin, set a work offset, create a viewsheet, create a section view, and set other options.

  • From geometry: Create a new plane based on geometry you select in the graphics window.
  • From solid face: Create a new plane based on the solid face you select in the graphics window.
  • From Gview: Create a new plane based on the current view in the graphics window.
  • From entity normal: Create a new plane normal to the entity you select in the graphics window.
  • Relative to WCS: Create a new plane relative to the plane currently set as the active WCS. Select a plane from the list, Top, Front, Back, Bottom, Right, or Left.
  • Quick Cplane: Creates a new plane based on a solid face you select in the graphics window. This new plane is then set as the active Cplane.
  • Dynamic: Create a new plane using the Dynamic Gnomon to make a selection in the graphics window. When creating a plane using the Dynamic Gnomon, the Z axis of the new plane is oriented to the normal calculated from the entities. Additionally, the X axis is oriented along a wireframe entity's tangent.

Select a lathe plane

Select or create a new plane based on the Lathe coordinate system selected from the list. When working with a lathe machine, you can orient the construction plane to work in the radius (X/Z) or diameter (D/Z) coordinates.

Find a plane

Identify a plane in the grid based on the selected option.

  • From geometry: Prompts you to select geometry in the graphics window and then highlights the corresponding plane in the grid if one exists.
  • From plane: Select one of the following options:
    • All: Displays all plane gnomons and prompts you to select one in the graphics window.
    • Selected: Displays all gnomons for the planes selected in the Planes Manager and then prompts you to select one in the graphics window.
    • Displayed: Displays only the gnomons for planes with the Display column selected and prompts you to select from the graphics window.
  • Relative to WCS: Select Top, Front, Back, Bottom, Right, or Left, and a corresponding plane will be highlighted in the grid and indicated by a gnomon in the graphics window. If more than one plane is associated with the chosen plane, the Plane Selection dialog box displays.

Set your current WCS, construction plane, and tool plane with their origins to the selected plane

Sets your current WCS, construction plane, and tool plane with their origins to the plane you have selected in the Planes Manager.

Reset your current WCS, construction plane, and tool plane to their original state

Resets the planes to the state that existed when the file was opened.

Hide plane properties

Hides or displays the property controls below the planes grid. Use this control to maximize the space allotted to the grid, or minimize the space used by the manager.

Display options

Select one or more of the following display options:

  • List only planes associated with the selected plane: Shows only the currently selected plane and any planes that are linked to it.
  • Display plane info relative to WCS: Displays the names of the tool and construction planes relative to the WCS. For example, when this option is selected and your WCS and tool plane are set to the back side of the part, Mastercam displays the tool plane as Top, because the WCS rotates the coordinate system to align with the back plane. If this option is deselected, the tool plane displays as Back, which is the orientation relative to the default coordinate system.
  • Display origin values in world coordinates: Displays the Origin X, Origin Y, and Origin Z fields in world coordinates rather than in view coordinates (the default). This option controls the origin fields in the Planes Manager, the Planes (WCS) option in toolpath dialog boxes, and Plane Selection dialog box.
  • Contrast rows: Applies shading to every other row in the table.
  • Plane gnomon settings: Opens the Plane Gnomon Settings dialog box, allowing you to change the size of the plane's gnomon, or enable a translucent plane grid.

Follow rules

Select one or more rules. The state of follow rules are preserved between sessions.

  • Cplane/Tplane follows WCS: Always aligns the construction plane and tool plane to match the WCS.
  • Gview follows WCS: Always aligns the graphics view to match the WCS.
  • Cplane follows Gview: Always aligns the construction plane to match the graphics view. If the Tplane and the Cplane are set to the same plane, the Tplane will also follow the Gview.
  • Tplane follows Cplane: Aligns the Tplane with the Cplane when the construction plane has been changed by the user, or as the result of another follow rule. The Tplane can still be changed independent of Cplane.
  • Cplane = Top in Isometric Gview: Aligns the construction plane to the Top plane when the graphics view has been set to Isometric.

Section view

Toggles the display of the Section View, which allows you to see a section of the part by cutting a plane. This toggle is the same as the Section View toggle on the View tab. Use the drop-down menu to set which entities will be affected by the section view. You can also choose Show Caps, which fills in the section of the part with a solid cap.

Show gnomons

Select to toggle the display of plane gnomons. This toggle is the same as the Show Gnomons toggle on the View tab. To display a plane gnomon for a plane that is not the current WCS, Cplane, or Tplane, the Display column must be selected. Select one or more of the following options:

  • Selected planes: Always displays the gnomon for the currently selected plane in the graphics window whether or not the Planes Manager has focus. When this option is deselected, the gnomon for the current plane only displays when the Planes Manager has focus and is hidden when the focus switches to another function.

    The gnomon always displays at its origin. If your part is not drawn at the origin and the part is fit to the screen, you may have to zoom out to see the gnomon at the origin.

  • WCS: Displays a static plane gnomon at the WCS origin.
  • Cplane: Displays a static plane gnomon at the Cplane origin.
  • Tplane: Displays a static plane gnomon at the Tplane origin.
  • Display Cplane and Tplane in corners: Shows the Cplane gnomon in the upper-left corner of graphics window, and shows the Tplane gnomon in the upper-right.
  • Show plane names: Shows the name of the plane below its gnomon in the graphics area. Note that this does not apply to gnomons displayed for planes selected in the Planes Manager; those names will always be displayed.
  • Use origin colors: Uses the colors specified in the Colors page of the System Configuration dialog box to display the WCS, Tplane, and Cplane gnomons.
Plane columns and indicators

Name

Shows the name of the plane. A check mark indicates the plane is used somewhere in the file such as in an operation, a viewsheet, or your current construction plane (Cplane). To determine how the plane is used, right-click in the panels grid and select Report to view a plane usage summary.

Lathe users see a pre-defined plane, Lathe Z = World Z. Use it when your lathe part is drawn vertically, and its axis of rotation is the world Z-axis. Setting this plane as the WCS maps the axes to a standard lathe coordinate system.

G

Indicates the plane is used as the current Gview. Click in a row of the Gview column to assign that plane as the Gview. No G displays if the current Gview represents an undefined plane.

WCS

Indicates the plane is used as the current working coordinate system (WCS) plane. Click in a row of the WCS column to assign that plane as the WCS. The plane type (WCS) is also indicated with the plane name in the graphics window.

C

Indicates the plane is the current construction plane (Cplane). Click in a row of the C column to assign that plane as the Cplane. The plane type (C) is also indicated with the plane name in the graphics window.

T

Indicates the plane is the current tool plane (Tplane). Click in a row of the T column to assign that plane as the Tplane. The plane type (T) is also indicated with the plane name in the graphics window.

Offset

Indicates the number of the work offset associated with the plane. If you do not enter a Manual work offset, Automatic is selected by default and the field is blank.

Display

An X indicates that the plane's gnomon displays in the graphics window when Show Gnomons is toggled on. Click in the column for a plane to toggle the display status of the gnomon. If Show Gnomons is toggled off, the planes will not display even if the Display column is selected.

Section

An X indicates that a section view is created from the respective plane when the Section View is on.

Coordinate

Read-only. Indicates the coordinate mode (Mill, Lathe X, or Lathe D) of the plane. This column is hidden by default.

The View Axes Gnomon (in the lower left corner of the graphics window) will display with the selected coordinate system.

When Show Gnomons is toggled on and Display Cplane and Tplane in corner is selected, the Cplane coordinates will also display with the selected coordinate system.

Lock

An X indicates that the respective plane is locked from being edited or deleted. System planes cannot be locked. This column is hidden by default.

Link

An X indicates that the respective plane is a child plane in one or more linked plane relationships. System planes cannot be linked. This column is hidden by default.

Parameters

Work offset

Assigns a work offset value to the plane. When you use the plane for a Tplane and create a toolpath, Mastercam outputs the work offset when you post the operation.

  • Manual: Enables the value field so you can enter a work offset number. The default value is -1, which represents the Automatic setting. Alternately, click Get unique, and Mastercam searches existing named planes and planes stored in toolpaths, and assigns an unused, unique work offset value. Click Get unique again to assign the next unused work offset value. Subsequent clicking of Get unique toggles between the first unique work offset value and the second available value.
  • Automatic: Automatically assigns a matching work offset number during post processing (if one exists), or the next available number if no matching work offset number is found. Work offsets can exist assigned to planes, in a toolpath operation's Tplane, and in a transform operation's NCI. Mastercam searches for existing work offset numbers in that order. The value for Automatic is -1 and Automatic is the default setting.

If you duplicate a plane and only change the plane's name (keeping all other attributes the same), the Work offset value will be the same as the original plane.

Color

Assigns a color to the plane display of the gnomon.

X origin/Y origin/Z origin

Displays the origin of the plane from the main system origin in view coordinates. The coordinates along each axis are relative to the plane axes. Type new coordinates in the fields, or use the clipboard functions in the right-click menu to update origin values. Alternately, right-click in an Origin field and select X, Y, or Z coordinate of a point. Select a new origin in the graphics window.

NOTE

To display world coordinates, choose Display origin values in world coordinates in the Display options drop-down menu.

Select origin

Returns you to the graphics window to select a new origin for the currently selected plane.

Reset origin

Resets the selected plane’s origin coordinates to X0,Y0,Z0 and removes any existing associations with geometry.

Associative origin

Associates the plane origin to an entity and updates the origin if that entity moves. This checkbox is enabled upon plane creation if the plane can be associated to the geometry used to create the plane.

If the Cplane, Tplane, Gview, or WCS is assigned to that plane they will be updated as well.

Associative orientation

Indicates whether the plane's orientation is associated to geometry and if the orientation will update when that geometry is moved. This checkbox is enabled upon plane creation if the plane can be associated to the geometry used to create the plane.

If the Cplane, Tplane, Gview, or WCS is assigned to that plane they will be updated as well.

Planes cannot be set to Associative orientation after they are created. If you edit a plane and deselect Associative orientation, you will not be able to associate the plane and its geometry again.

Comment

Type a comment in this field, which will appear in the post processed NC code to communicate plane information to other users.

Right-click menu options

Edit

Opens the Edit Plane function panel so that you can modify the selected plane. You can only edit user-defined (named) planes. Main system planes (Top, Front, Back, Bottom, Right, Left, Isometric, Reverse Isometric, and Trimetric) cannot be edited.

Rename

Changes the name of a plane. You can only rename user-defined planes. You cannot name a plane Gview, Cplane, Tplane, or WCS; or rename the main system planes (Top, Front, etc.).

Delete

Select one of the following options:

  • All unused: Removes all planes that are not being used from the planes grid and part file. You cannot delete any of the main system planes.
  • Selected: Removes the selected plane from the planes grid and part file. You cannot delete any of the main system planes or any planes that are in use.

Lock/unlock

Locks or unlocks a user-defined plane. System planes cannot be locked/unlocked. Select one of the following options:

  • Selected: Locks any selected planes from being edited or deleted, and displays the lock icon in the Name column. If a plane is locked, certain dialog box parameters and right-click menu options are also disabled. Choose this option to unlock a previously locked plane.
  • Lock all: Locks all user-defined planes and displays the lock icon in the Name column.
  • Unlock all: Unlocks all previously locked planes.

Link/unlink

Links or unlinks a user-defined plane. System planes cannot be linked/unlinked. Select one of the following options:

  • Link selected: Opens the Plane Selection dialog box, where you can select a parent plane to link to the selected plane. Linked planes maintain a parent-child relationship. When the parent plane changes, the child planes linked to it will also change. If you change the orientation of the parent plane, the child planes move with it.
  • Unlink selected: Unlinks the selected child plane from the parent plane.
  • Unlink all: Unlinks all planes in the part file.

Rotate incremental

Opens the Rotate plane dialog box, where you change the orientation of the selected plane by rotating it by set values around each axis.

Fit origin

Changes the view in the graphics window to include both the part and the origin.

Display plane gnomons

Toggles Show Gnomon on and displays the plane based on the selected option:

  • All: Toggles the Display column on for all planes and displays the plane gnomons in the graphics window.
  • Selected: Toggles the Display column on for planes selected in the Planes Manager and displays the plane gnomons in the graphics window.
  • None: Toggles the Display column off and Show Gnomons off (if on) for all planes.

Coordinate display

Changes the coordinate display of created planes to Mill, Lathe X, or Lathe D. The coordinate mode will display in the Coordinate column, which is hidden by default.

The View Axes Gnomon (in the lower left corner of the graphics window) will display with the selected coordinate system. When Show Gnomons is toggled on and Display Cplane and Tplane in corner is selected, the Cplane coordinates will also display with the selected coordinate system.

Create relative

Opens the Create Relative Planes dialog box, where you can create named planes that are defined relative to the selected plane.

Duplicate

Creates a duplicate of the selected plane. The duplicated plane uses the same name as the selected plane with a -1 suffix (Plane-1). A duplicate plane made from a locked plane is unlocked when duplicated.

Report

Opens your default editor, which displays a report with detailed information about Selected, Used, or All planes.

Import

Imports planes from another part. Mastercam prompts you to select a file, and then displays all of its planes. Select the planes to be imported.

Reset sort

Returns the planes grid column sorting to the default sorting.

Plane gnomon right-click menu options

When a plane gnomon is displayed in the graphics window (when Show Gnomons and the Display column are toggled on) right-click on it to see a menu of Planes Manager functions.

How do I...

How do I understand the difference between relative planes, linked planes, and associative planes?

Relative planes, linked planes, and associative planes rely on a connection with other planes or geometry within the part file.

  • Relative planes are planes created in predefined spatial relationships relative to the selected plane.
    • For example, create PlaneA in the Planes Manager and right-click it to select Create relative. In the Create Relative Planes dialog box, select only Right side. Mastercam creates a plane that is a right-sided view of PlaneA.
    • Relative planes are linked by default. To unlink them, you can deselect Link in the Create Relative Planes dialog box.
  • Linked planes have a parent-child relationship; the selected plane is dependent on the plane to which you link it.
    • For example, create PlaneB and link it to PlaneA. When you move PlaneA, PlaneB will also move the amount necessary to retain its spatial relationship (the amount of transformation to get from PlaneB to PlaneA) to PlaneA.

    • Linked planes use icons to display their relationship in the Planes Manager. Parent planes display a chain link leading to a plus sign below the chain link. Child planes display a plus sign leading to a chain link beneath the plus sign.

  • Associative planes maintain a relationship with the geometry used to create them. When you create a new plane from geometry, entity normal, a solid face, or from the Dynamic Gnomon, Associative origin and Associative view are selected by default to maintain the predefined spatial relationship of the plane's view and origin relative to the geometry used to create the plane.
    • For example, create PlaneC from geometry, entity normal, a solid face, or from the Dynamic Gnomon. In the Advanced tab of the New Plane/Edit Plane function panel, Associative origin and Associative view are selected by default. If you transform the geometry you used to create the plane, the plane origin and view will transform to maintain their spatial relationship to the geometry.

    • If you break a plane origin's associativity, you can still reestablish it when selecting a new origin for the selected plane.

    • Broken orientation associativity cannot be reestablished.
    • Deselect Associative origin and Associative view when you are creating a plane from temporary guides or construction lines that you plan to delete.
How do I use work offsets?

In the parameters section of the Planes Manager, you can assign a work offset number to the currently selected plane.

  • Select Manual and type a 0 (zero) in the value field to assign the base offset in your control's numbering system (for example, G54 for Fanuc-style offsets), or higher integers to increment from the base, (for example, enter 2 for G56).

    NOTE

    These values are determined by your post processor.

  • Select Automatic to automatically assign a matching work offset number (if one exists), or the next available number if no matching work offset number is found.
  • When editing a plane, choose Get unique to have Mastercam review your model and assign an unused or new offset number.
How do I import planes from another part file?

You can drag-and-drop a Mastercam part file onto the Planes Manager to import planes including work offsets into the current part file. When the Plane Selection dialog box opens, select which planes to import and click OK.

How do I enable a translucent plane display?

If you want to enable a translucent plane display for your planes, select Display options, Plane gnomon settings from the Planes Manager.

Then, use the settings in the dialog box to show the plane, show the grid, change the axes size, and change the plane size. You can also set the opacity of the axes and the plane display.
