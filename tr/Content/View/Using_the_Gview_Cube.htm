Using the Gview Cube

Mastercam's traditional on-screen gnomon has been augmented with the Gview cube, a more graphical way to show plane orientation. The Gview cube, which is relative to the WCS by default, also features options to control the cube and thus the on-screen representation of your part. The following images shown the traditional gnomon on the left and the Gview cube on the right.

The cube has many interactive features. For example, double-clicking the cube fits the part to the screen (same as the Fit command). You can click a side, corner, or edge of the cube to switch to that view. Drag any part of the cube to change to a view, unconstrained by the preset views represented by the Gview cube's sides and edges.

The buttons next to the cube allow you to pan or rotate the view. The effect of dragging depends on whether you hold the button down and whether you press [Alt], [Ctrl], [Ctrl+Alt], as shown in the following lists.

NOTE
The default rotation amount is 15 degrees, but you can change this value in the Gview cube settings (see the context menu options below).

Arrow Buttons

  • Click: Pans the part up, down, left, or right

  • Click and hold: Continuously pans the part

  • [Alt]+click: Rotates the part up, down, left, or right

  • [Alt] and hold: Continuously rotates the part

Flip/Rotate Button

  • Click: Flips the part 180 degrees

  • [Shift]+click: Flips the part 90 degrees

  • [Shift+Alt]+click: Flips the part 90 degrees

  • [Ctrl]+click: Rotates the part counterclockwise about the center of the screen

  • [Ctrl+Alt]+click: Rotates the part clockwise about the center of the screen

Context Menu

Right-click the cube to display a context menu. The following list describes the menu selections.

  • Show Cube: Click this option to toggle the Gview cube on and off.

  • Relative to WCS: Sets the world view relative to the active WCS. For example, if your WCS and tool plane are set to the back side of the part and Relative to WCS is off, the tool plane is set as Back, which is the orientation relative to the default coordinate system. When Relative to WCS is selected, Mastercam sets the tool plane as Top. This is because the WCS rotates the coordinate system to align with the back plane. In the left-hand image below, the Gview and WCS are set to the Right plane and Relative to WCS is off. In the right-hand image below, the Gview and WCS are set to the Right plane and Relative to WCS is on.

  • Create dynamic plane: Lets you create a new plane using the Dynamic Gnomon. You can also select this function by clicking the cube’s XYZ axes.

  • Settings: Opens the Screen dialog box, where you can adjust the cube size, text size, opacity, arrow button pan/rotate, and arrow button rotation (degrees). The Screen settings page is also available in the System Configuration dialog box (File, Configuration), as shown in the following image.
