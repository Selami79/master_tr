Defining a Lathe Stock Model

Follow these steps to create a stock model for your lathe part. If you plan to use chuck jaws to hold your stock, you can create either the jaws or the stock model first. Choose the method that best meets your application needs:

  • If you create the jaws first, Mastercam automatically positions the stock in the jaws.
  • If you create the stock first, Mastercam gives you more control about where to locate it relative to your part. This is especially useful if your stock model is based on an imported solid. In this case, when you create the jaws, you can choose to automatically position them relative to the stock.
How do I define a stock model?
  1. If your part does not already have a machine group, select a machine definition from the Machine type group on the Machine tab.

  2. In the Toolpaths Manager, click Stock setup.

  3. Select the spindle where the stock will be located.

    4. NOTE

    You will only be able to select a sub-spindle if one has been added to the machine definition.

  4. Click Properties to either add a new piece of stock, or edit the stock model if one has already been created.

  5. Choose how you will create the geometry for the stock model.

  6. All stock models are 3D entities. You can choose from the following options:
    • Solid entity: Use a solid model from your Mastercam part file or an external CAD file. To use an external CAD file, typically you would import it into your Mastercam file and store it on a level.
    • Block: Create rectangular stock by entering its dimensions. Typically this is used for milling applications.
    • Cylinder: Create bar stock by entering its dimensions. This is the default selection and is suitable for most turning operations.
    • Extrude: Create a stock model by extruding a wireframe chain. For most turning applications, you would extrude a profile along the spindle axis.
    • Revolve: Create a stock model by rotating a wireframe chain. For most turning applications, you would revolve a profile around the spindle axis.
  7. If necessary, use the Position/Orientation on Machine tab to properly position the stock relative to your part. If you have already defined chuck jaws, you will not see this tab at all. Mastercam automatically uses the chuck jaws to position your stock.

    • For most typical turning applications, you will not use these options. The Stock is drawn in position on the machine option should be selected.

    • For Cylinder bar stock, you would typically use the Base point on the Geometry tab to draw the stock in the proper location.

  8. Preview your stock model and make sure it is properly positioned relative to your part geometry.
    • Click View component on the toolbar to see the stock as a stationary 3D solid model.
    • Click Preview Lathe Boundaries to see a silhouette of the boundary created as the stock revolves around the spindle.

  9. Click OK to return to the Stock setup tab. Once the stock has been added, the spindle label changes to (Defined).

NOTE

If the Use Machine Tree option is selected on the Stock setup tab, you can add stock by right-clicking on a set of chuck jaws and selecting Stock as shown below.

After you have created toolpaths to machine your stock, use the Lathe Stock View function to see solid-model representations of the machined stock after each operation.
