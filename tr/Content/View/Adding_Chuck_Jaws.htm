Adding Chuck Jaws (Lathe)

Follow these steps to define chuck jaws in Mastercam Lathe.

  1. In the Toolpaths Manager, click Stock setup.

  2. Select the spindle.

  3. Click the Properties button.
  4. Use the Geometry tab to decide how you want to represent the shape of the jaws. You can select a solid model, use chained wireframe geometry, or define it parametrically. Expand the following drop-downs to see the next steps. As you work, the Preview window updates to show the overall profile of the jaws.

    Although all three options are available for both Lathe and Mill-Turn, most Lathe users will choose Parametric and most Mill-Turn users will choose Solid entity, since Lathe does not support machine simulation.

    Defining your jaws with a wireframe profile

    You can model your jaws by selecting a chained profile of wireframe geometry. This technique is often used when your jaws have been machined to fit the shape of your part.

    Select Chain from the Geometry list, then click the Select Geometry button to select the geometry from the graphics window. The geometry should be a 2D profile drawn in the upper quadrant. No part of the profile should cross the spindle center line.

    You can create either rectangular or pie-shaped jaws with this method. See Geometry tab: Machine Component Manager (Chuck Jaws (Chain)) to learn more.

    Defining your component with a solid model

    You can model your collet or jaws by selecting a solid model. The model can be either a Mastercam solid on a level in your part file, or an external file in .STEP format.

    Select Solid entity from the Geometry list, then select the model.

    • In the picture below, click the left button to select a model from an external file.
    • Click the right button to select a model from the Solids Manager or the graphics window.

    Use the Align function panel to orient the model and align it with the chuck and spindle, then enter the other settings on this tab. See the following topics to learn more:

    • Align Chuck/Jaw/Collet panel
    • Geometry tab: Machine Component Manager (Collet (Solid))
    • Geometry tab: Machine Component Manager (Chuck Jaws (Solid))

    Defining your component parametrically

    Select Parametric from the Geometry list to define the collet or jaws by entering their dimensions directly in this dialog box. This lets you model them without creating or selecting any geometry.

    1. For jaws, choose either the Rectangular or Pie-style option and enter the related dimensions.
    2. Click the Steps: Initialize button to begin defining the jaw steps.

    3. Refine the dimensions for each step by entering the values directly in the grid. Use the other buttons to add or remove steps or invert the order.

    See Parameters tab: Machine Component Manager (Chuck Jaws) or Geometry tab: Machine Component Manager (Collet (Parametric)) to learn more.

  5. Click the Parameters tab to define the location and clamp position.

    1. Choose whether to clamp the part on the Inside diameter or Outside diameter.
    2. In the Reference section, click the Select Reference Point button and select a point on the profile. This will be the programmed point for chuck operations and identifies the clamping position.
    3. Use the Position options to define how to locate your part within the jaws.
      • Select From stock to let Mastercam calculate the diameter based on the stock boundaries. Enter a Grip length to define the position of the jaws reference position relative to the back face of the stock.
      • Deselect From stock to manually enter the coordinates. Since Z0 is the face of the chuck jaws in most applications, a negative Z value ensures that some stock is being clamped by the jaws.
  6. Preview the chuck jaws to make sure that they are positioned properly. Mastercam gives you several options:
    • Click Preview lathe boundaries to see a silhouette of the boundaries that are projected by the revolved jaws. This lets you easily compare the chuck jaws against 2D lathe part geometry.
    • Click the View this component's component group in the graphics window to see the chuck, jaws, and stock as a single assembly.

  7. Click OK to return to the Stock setup tab. Once the jaws have been added, the spindle label should say Defined.
