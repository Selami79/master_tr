Using the Lathe Z = World Z plane

Mastercam includes a special plane that you can use to machine lathe parts that are drawn with the Z-axis oriented vertically. Use it for lathe parts whose axis of rotation is the world Z-axis instead of the typical lathe Z-axis orientation. Open the Planes Manager in Mastercam Lathe to access this plane. By setting your WCS equal to this plane, you can create turning toolpaths within the typical lathe 2D coordinate system without transforming or rotating the geometry.

The following picture shows an example of such a part:

It is drawn in the standard world coordinate system with axes oriented as shown in the picture.

Mastercam cannot create its standard turning toolpaths with the axes oriented in this way. To machine this part properly, orient the coordinate system to the Lathe Z = World Z plane.

  1. Open the Planes Manager.

  2. Choose Cplane/Tplane follows WCS.

  3. Click Lathe Z = World Z to select it.

  4. Click in the WCS column to map the coordinate system to this plane.

  5. Click OK to close the Planes Manager.

You should see the part flipped on its side, like in the following example.

The axes should now be oriented as shown in the picture.

Mastercam can now create standard turning toolpaths for the part. You can also select Mastercam's lathe construction planes from the Planes Manager while in this plane. Click the topics below to learn more about specific applications.

Working with stock models

If you are using the Lathe Z = World Z plane to machine your part, make sure that you also select this plane when you create your stock model. Choose the Stock Setup tab in the Machine Group Properties dialog box, and click the button next to the stock plane name to select it. Then create your stock model normally.

Working with VTLs

Drawing the part vertically, with the Z-axis pointed upward, will NOT result in your being able to machine it properly on a vertical turret lathe. Instead, draw the part as you would for a normal lathe, using one of Mastercam's lathe construction planes, or if the part is already vertical, use the Lathe Z = World Z plane as described elsewhere in this topic. Then, select a vertical turret lathe machine definition. Mastercam will automatically map the coordinate system properly for a VTL.

Using a VTL and using the Lathe Z = World Z plane are two separate functions:

  • Use the Lathe Z = World Z plane to map the part geometry to a lathe coordinate system so that you can create proper lathe toolpaths.
  • Use the machine definition to place the toolpaths on a specific type of machine, whether it is a VTL, slant bed lathe, or other type of machine.
Setting Lathe Z = World Z as the default plane

If you think you will be using the Lathe Z = World Z plane frequently, you can choose to set it as the default plane for specific machines. In this way, whenever you select the machine definition, Mastercam sets the WCS equal to the Lathe Z = World Z plane.

To set this, follow these steps:

  1. Open the Machine Definition Manager.
  2. Access the General Machine Parameters dialog box.
  3. Select the Cplane, WCS, HTC/VTL tab.

  4. Choose Lathe Z = World Z in the Update WCS using group and save the machine definition.
