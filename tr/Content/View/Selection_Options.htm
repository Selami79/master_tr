Selection Options dialog box

Use the Selection Options dialog box to define the toolpath information (i.e., tool number, operation type, machine definition, etc.) that determines which operations are selected in the Toolpaths Manager.

Selecting most of the options in this dialog box activates Select (arrow button) which closes the dialog and lets you select entities from the graphics window. When you have completed selecting from the graphics window, press Enter to open the dialog box again. When you click OK, operations that match the criteria you chose are selected in the Toolpaths Manager.

The options in this dialog box are mutually exclusive. In other words, you can only choose one method of selection at a time.

Parameters

Geometry

Selects operations in Toolpaths Manager that match the geometry you chose in the graphics window.

Tool #

Selects operations in Toolpaths Manager that match the tool number information you enter.

Tool type

Selects operations in Toolpaths Manager that match the tool type information you enter.

Plane

Selects operations in Toolpaths Manager that match the plane information you enter.

Operation type

Selects operations in Toolpaths Manager that match the operation type information you enter.

NC file name

Selects operations in Toolpaths Manager that match the file name information you enter.

Work offset

Selects operations in Toolpaths Manager that match the work offset information you enter.

NOTE
  • If the operation you select, has no work offset associated with it, the Work offset field will be blank.
  • If your current Mastercam file has no operations with work offsets, the Work offset radio button, along with its field and Select button, will not be available.

Machine definition

Selects operations in Toolpaths Manager that match the machine definition information you enter.

Word match

Selects operations in Toolpaths Manager that match the Toolpath Parameters comment word you enter.

The Word match radio button activates both the Word match field and the Case sensitive checkbox. When you type a word into the field, Mastercam searches through the list of operations to locate each operation that has that word in the Comments field of its Toolpath Parameters tab. When you click OK, operations that have that word match will be selected in the Toolpaths Manager.

Check the Case sensitive checkbox if you want the search results to exactly match the word, including the upper- and lower-cased letter format. If there are no matches to the word you entered, the dialog box OK button will not be available.

Apply to

  • Selected operations: Applies settings in the Selection Options dialog box only to operations already selected in Toolpaths Manager.
  • All operations: Applies settings in the Selection Options dialog box to all operations in Toolpaths Manager.
