Toolpaths Manager Icons

Mastercam displays icons in the Toolpaths Manager operation list to give a visual representation of the operation information and status. Use the drop-downs below to learn more about these icons.

Operations List - Overlay Icons

Toolpaths Manager overlays the following icons on items inside of the operations list to provide information about the state of an operation. Multiple icons can overlay a single operation icon at once.

NOTE

Depending on the status of an operation, these icons may also display as stand-alone.

Icon

Name and Description

Dirty: This icon is superimposed when a toolpath or tool needs to be regenerated. A toolpath or tool is dirty when you have changed certain parameters of the underlying geometry.

Toolpaths can be regenerated by clicking the Regenerate buttons at the top of the Toolpaths Manager.

If a toolpath is locked and dirty, you must unlock it before regenerating it.

Locked: This icon is superimposed when a toolpath is locked. Lock a toolpath to prevent unintentional regeneration. Locking can be reversed by clicking the Lock button at the top of the Toolpaths Manager. If a toolpath is locked and dirty, you must unlock it before you can regenerate it.

Use the Lock button at the top of the Toolpaths Manager to lock and unlock the toolpath.

Posting off: This icon is superimposed when posting is turned off for the toolpath. Toggle posting on and off by clicking the Disable Posting button at the top of the Toolpaths Manager.

Use the Post button at the top of the Toolpaths Manager to toggle posting on and off.

Selected: This icon is superimposed when an operation is selected. You can toggle selected operations using the up and down arrow keys. See Toolpaths Manager Shortcut Keys for more information about arrow keys.

Disabled: This color is applied to icons when the display is toggled off. For example, a locked operation with the display toggled off will display a gray lock.

To learn more about displaying toolpaths, see Toolpath Display.
Operations List - Icons

Toolpaths Manager uses the icons shown below to provide information on the operations in a job.

Icon

 Name and Description

Machine group (Mill/Router, Lathe, Mill-Turn, Wire):Machining properties, toolpath groups, and toolpath operations indented under this icon are members of that group. Properties such as machine definition, posting information, sequence numbers, job setup, and NC file names are included under each machine group.

Properties: Machining properties (i.e., support files, tooling, stock) are indented under this heading.

Files: Opens the Files tab, where you can specify the names and data paths of files needed by operations in the parent toolpath group.

Tool settings (Mill/Router/Mill-Turn, Lathe): Opens the Tool Settings tab, where you can control how Mastercam assigns tool numbers, tool offset numbers, and default values for feeds, speeds, coolant, and other toolpath parameters.

Wire program settings: Opens the Program Settings tab, where you can set the program number and sequence numbers to be used when you post process the wirepaths to an NC file.

Stock setup (Mill, Lathe): Opens the Stock Setup tab, where you can set up a stock model to visualize toolpaths during backplot and verification.

Job Setup (Mill-Turn): Opens the Job Setup dialog box where you can set parameters for stock, stock handling, part geometry, chuck dimensions, pickoff values, and tool plane information. See Performing a Mill-Turn Job Setup for more information.

Toolpath group: Toolpath groups are subgroups that are “children” of machine groups and inherit machining properties and other important properties from their parent groups. These groups house toolpath operations as well as other subgroups. Mastercam supports unlimited subgroups.

Toolpath folder (Expanded, collapsed): Contains the operation components such as toolpath parameters, tool parameters, and geometry or solids. Double-click the icon to expand or contract the folder.

Toolpaths folder (gray): Along with turning gray when toolpath display is disabled, toolpath folders become gray if you have started to create additional toolpaths in a different machine group. Existing operations that are not supported by the active machine group are changed to gray.

Stock model operation: Contains the operation parameters. See Working with Stock Model Operations to learn more.

Toolpath parameters: Opens the appropriate Toolpath Parameters dialog box where you can select a tool, set feeds and speeds, and set other general toolpath parameters.

Subprogram: The toolpath contains one or more subprograms (NC program that is called from the main NC program to repeat code within an operation). Click to open the related dialog box for Subprograms.

Tool parameters (Mill/Router/Mill-Turn, Lathe): Click to open the Edit Tool/Define Tool dialog box, where you can define the tool and its parameters for the operation.

Geometry: Opens the geometry's appropriate editing interface such as the Chain Manager , Toolpath Hole Definition function panel, or Toolpath Surface Selection dialog box, where you can edit the geometry. For advanced multiaxis toolpaths, clicking this icon opens the Multiaxis dialog box (usually to the Toolpath Type page).

Geometry - Machining/Drive Surfaces: Opens the appropriate interface where you can change the selection of Machining/Drive Surfaces Geometry.

Geometry - Avoidance/Check Surfaces: Opens the appropriate interface where you can change the selection of Avoidance Geometry/Check Surfaces.

Geometry - Curves/Containment: Opens the Chain Manager to select curves for Project, Equal Scallop, or Blend toolpaths, or to select containment boundaries for surface toolpaths.

Start points: Identifies start points of surface geometry. Click to reselect a start point.

Flowline: Only for surface flowline toolpaths. Click to change the flowline parameters.

CAD file: Indicates a CAD file was selected for a surface toolpath. Click to select a different CAD file.

Solid: Shows that the toolpath group contains a solid, or a combination of solids and geometry. Click to open the appropriate dialog box where you can edit the solid.

Toolpath display: Toolpath display is on when the icon is dark. Click to open Classic Backplot.

Turning toolpath display on and off can be done by using shortcut keys (hotkeys). See Toolpath Display for additional information on turning the display of toolpaths on and off in the graphics window.

Processing a multi-threaded toolpath: Indicates that the toolpath is being processed in multi-threaded mode by Mastercam's Multi-Threading Manager (MTM). This processing mode allows Mastercam to split the programs that regenerate its largest, most complex toolpaths into smaller and separate program threads that execute simultaneously. See Multi-Threading Manager for more information.

Waiting to process a multi-threaded toolpath: Indicates that the dependent toolpath is waiting to be processed in multi-threaded mode by Mastercam's Multi-Threading Manager (MTM). Dependent threads require data from prior “parent” toolpath threads. Dependent threads cannot begin processing until the MTM completely finishes processing the parent thread, including translation of the binary NCI data. See Multi-Threading Manager for more information.

An error occurred during the multi-thread process: Indicates that an error has occurred while processing the toolpath in the multi-threaded mode by Mastercam's Multi-Threading Manager (MTM). This processing mode allows Mastercam to split the programs that regenerate its largest, most complex toolpaths into smaller and separate program threads that execute simultaneously. See Multi-Threading Manager for more information.

Toolpath NCI not restored: The NCI file has not been restored (via a toolpath regeneration) since the file was opened. By default, Mastercam restores all toolpath data upon opening a file. See Files page: System Configuration to change this option.

Trimmed toolpath: Toolpath has been trimmed. Click to open Classic Backplot. When you trim a toolpath, this icon replaces the Toolpath on icon. See Trimming Toolpaths for additional information.

Lathe stock update: Toggles on or off the Mastercam Lathe stock update, When enabled, Lathe provides feedback on stock shape as it is machined.

Batch: Operation has been marked “to Batch” on the Tool parameters tab in the Operation parameters dialog box. "Batch" means to set the operation to batch mode, to be processed separately from other operations using Mastercam's batch processing utility. (Not available in Mastercam Wire.) See Batch Processing Toolpath Operations for additional information.
