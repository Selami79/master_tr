Displaying a Tool and Coordinate Information

The Display group of the Mastercam ribbon allows you to customize the display of axes, gnomons, and a tool in the graphics window.

Select Show Axes, Show Gnomons, or Show Tool to turn it on and then use the drop-down to customize the display.

Your selections persist throughout Mastercam sessions.

Exploring Coordinate Information

Mastercam's on-screen status indicators show your work's orientation at a glance. The size of the on-screen indicators can be controlled on the Screen property page of the System Configuration dialog box.

Mastercam always displays the following information:

  • The scale and unit information in the lower-right corner shows whether you are working with an U.S. or metric part and the number of units per inch or millimeter.

  • The coordinate axes (the gnomon) in the lower-left corner shows the orientation of the current Gview. The text below the gnomon tells you the name of the current Gview, if it matches the orientation of a named plane, such as Top, Front, Isometric, etc. No name is shown if the current Gview orientation does not match the orientation of a named plane—for example, when you change the view using dynamic rotation.

  • Mastercam's status bar displays information about the WCS, Tplane, Cplane, and the coordinates of your mouse location in the graphics window.
Displaying coordinate axes

The Show Axes button on the View tab controls which coordinate axes and origin (World, WCS, Cplane, and/or Tplane) display in the graphics window. Each axis can be controlled individually. Click the drop-down to select which axes to display, and then use Show Axes or press [F9] to turn them on or off in the graphics window. If no axes are selected in the drop-down, clicking Show Axes turns them all on. The state of the button persists from between sessions.

Displaying additional gnomons

The Show Gnomons button on the View tab controls which gnomons (WCS, Cplane and/or Tplane) display in the graphics window. Each gnomon can be controlled individually. Click the drop-down to select which gnomon to display, and then use Show Gnomons or press [Alt+F9] to turn them on or off in the graphics window. If no gnomons are selected in the drop-down, clicking Show Gnomons turns them all on. The state of the button persists between sessions.

The Selected planes option always displays the gnomon for the currently selected plane in the graphics window whether or not the Planes Manager has focus. When this option is deselected, the gnomon for the current plane only displays when the Planes Manager has focus and is hidden when the focus switches to another function.

Choose Show plane names to display the name of the plane below its gnomon in the graphics area. Note that this does not apply to planes selected in the Planes Manager; those names are always displayed.

The Use origin colors option uses the colors specified in the Colors page of the System Configuration dialog box to display the WCS, Tplane, and Cplane gnomons.

  • If you have selected Display Cplane and Tplane in corners, the gnomon in the upper-left corner of the graphics window shows the Cplane orientation, and the gnomon in the upper-right corner of the graphics window shows the Tplane orientation,

  • The WCS gnomon is located at the WCS origin point and indicates the WCS orientation.

  • WCS, Cplane, and Tplane show the plane's gnomon at its origin point and indicates the plane's orientation.
Exploring Tool Display

The Show Tool function displays a mill, router, or lathe tool assembly in the graphics window. The tool tip is attached to your mouse cursor and displays with respect to the active tool plane. Use this function to view your tool assembly while part programming. You can quickly see if your tool or holder might collide with, or not fit into, your part.

For functions that support a display gnomon or green arrow, such as Create/Edit Plane, the tool detaches from your cursor and aligns with the blue Z axis of the gnomon or the green normal arrow.

Selecting and Displaying a Tool

You can set Mastercam to display either:

  • The tool of the active operation

  • A single tool for each machine group

To display the tool of the active operation:

  1. On the View tab, click the Show Tool drop-down.

  2. Select Display tool from selected operation.

  3. Toggle Show Tool on or press [F8].

  4. Select an operation in the Toolpaths Manager. Mastercam displays its assigned tool in the graphics window.

    NOTE

    If multiple operations are selected in the Toolpaths Manager, no tool displays in the graphics window.

To select and display a tool for a mill, router, or lathe machine group:

  1. Open the Tool Manager.

  2. Select the desired machine group from the drop-down. You can set one active display tool for each machine group.

    NOTE

    When active, the display tool appears for the machine group containing the red insertion arrow. Move the arrow between machine groups to switch display tools.

  3. In the upper tool list, right-click the desired tool and select Set display tool.

    4. An icon appears in the status column of the selected display tool.

  4. Click OK to exit the Tool Manager.

  5. Toggle Show Tool on or press [F8].

  6. If necessary, move the red insertion arrow to the Toolpaths Manager group for which you set the display tool.

    Mastercam displays the assigned tool in the graphics window.

    NOTE

    You may need to deselect Display tool from selected operation in the Show Tool drop-down to display the machine group display tool.

You can return to the Tool Manager at any point to select a different display tool for your machine group.

Customizing Tool Display in the Graphics Window

Mastercam allows you to customize how the tool displays in the graphics window.

Use the Show Tool drop-down to set the following options:

  • Display in front displays the designated tool in front of all other entities.

  • Snap to AutoCursor positions snaps the display tool to any active AutoCursor positions, as defined in the AutoCursor settings, while you are not currently in a function.

    NOTE

    You can adjust your AutoCursor settings on the Selection page of System Configuration. To learn more about AutoCursor, see AutoCursor Overview

Use the quick settings button in the Display group to customize your tool assembly display.

You can also access these settings in the Tools and Holders page of System Configuration.
