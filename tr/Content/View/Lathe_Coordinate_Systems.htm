Lathe Coordinate Systems

Traditional lathe turning applications use a very different coordinate system from milling applications. Instead of a 3D, XYZ space:

A lathe coordinate system for typical turning applications is 2D, with the tool axis perpendicular to the spindle (Z axis) instead of parallel to it:

Many lathes also include a Y axis that can be used for milling operations:

Mastercam includes a number of tools that you can use to create lathe parts and toolpaths for different types of lathes—including lathes with multiple spindles, multiple turrets, and live tooling—in a straightforward manner. These tools include machine definitions, construction planes, and the work coordinate systems. The following sections discuss techniques that you will typically use.

Lathe tool and machine definitions

Tool and machine definitions provide important information to Mastercam that helps it interpret your lathe coordinates.

  • Use the machine definition to tell Mastercam about the layout of the spindle and tool turrets. This is also where you tell Mastercam the direction and orientation of each of the machine axes. For sophisticated multi-turret and multi-spindle lathes, where there may be several X, Y, Z, A, B, C axes in the machine structure, create axis combinations to define which axes can work together for a given toolpath.
  • Use the tool definition to tell Mastercam how the tool is mounted in the turret and which spindle it is used for. When you select the tool (and axis combination, if applicable) for an operation, Mastercam will automatically read in the turret, spindle, and tool orientation.

Because the machine and axis orientation is set in the machine definition, you can use standard planes for creating with part geometry regardless of the machine configuration. When you create the NC code, the post can read machine-specific information from the machine definition.

Lathe Cplanes

Once your machine is set up, Mastercam includes special lathe construction planes that let you work in radius or diameter coordinates. For conventional 2D turning applications, use the Planes Manager to select the desired coordinate system. Choose to use either radius or diameter coordinates, then choose the desired X/D and Z directions.

This will set both the Cplane and Tplane; you do not need to change the WCS. For conventional 2D turning applications, you can leave the WCS as the system Top plane. If your lathe has a Y-axis, you can also use these planes to create 3D geometry.

You can configure your lathe machine definition to activate the desired coordinate system by default whenever you select it. Use the Cplane, WCS, VTL tab in the Machine Definition Manager.

Mastercam Lathe includes built-in C-axis toolpaths that automatically set the Tplane and Cplane for common milling setups.

Vertical turret lathes

Use the machine definition to tell Mastercam if you are using a vertical turret instead of a horizontal lathe. Once you make the selection in the machine definition, you can select the same lathe construction planes as before, and Mastercam will automatically display the axes correctly.

Mastercam automatically creates planes that map to standard lathe VTL axes when you select a machine definition for a VTL.

Hold your mouse over it to see how Mastercam automatically re-maps the axes when you select a VTL machine definition. When you re-select Isometric (WCS) for the Gview, Mastercam automatically displays the part oriented for a VTL.

Lathe parts drawn about the world Z-axis

You might get lathe parts which are drawn vertically, with the part's axis of rotation aligned with the world Z axis, like a mill coordinate system. Mastercam includes a special system-defined plane which transforms these axes to a typical lathe coordinate system without needing to transform or rotate the part geometry.

Display the Planes Manager, and then select Lathe Z = World Z. Click the WCS icon to align the work coordinate system with this plane, and choose OK. You can now choose the desired diameter or radius coordinate system from the Select lathe plane drop-down as described previously.

Rotating lathe tool axes, B-axis lathes

Mastercam includes support for B-axis lathes or multi-function machine tools. The tool is mounted on a rotary axis (typically B), which can be rotated to change the tool angle. Both Mastercam Lathe and Mastercam Mill-Turn support these machines.

For Mill operations, you typically select a different Tplane to generate the proper rotational output, but for turning operations in Lathe and Mill-Turn you enter the tool angle directly. Use the Tool Angle dialog box to enter the angle for each operation.
