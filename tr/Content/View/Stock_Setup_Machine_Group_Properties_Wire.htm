Stock Setup tab: Machine Group Properties - Wire

Use the Stock Setup tab to set up a stock model for Wire machine groups. Stock models to help you visualize your toolpaths more realistically. The stock model that you create here can be displayed with the part geometry when viewing the file or toolpaths, during backplot, or while verifying toolpaths. Click on each topic to read more about it.

Selecting the type of stock model

There are several ways to create stock models depending on your needs.

  • Choose Rectangular or Cylindrical to create simple stock models that are not defined by any actual geometry. All the dimensions are entered in this dialog box and no geometry is added to your part file. You can choose whether or not to display the stock on the screen.
  • Choose Solid/Mesh to use a solid model inside your current part file as the stock.
  • Choose File to use a solid model from an STL file as the stock model.
Setting the stock dimensions

Mastercam gives you several techniques for setting the stock dimensions. You can type the dimensions directly into the X, Y, and Z fields, or use the following methods when you don't know the exact dimensions:

  • Choose Select corners to return to the graphic window and select the two opposing corners of a 3D rectangle.
  • Choose Bounding box to have Mastercam calculate the furthest points in your part geometry.
  • Choose NCI extents to have Mastercam calculate the furthest points in your toolpaths, based on just the cutting motion.

Mastercam enters the stock dimensions in the fields. You can then edit them as needed.

Setting the stock plane

Select a stock plane to properly orient the stock model with respect to your part. You can align the stock model to any plane saved in your part file. When you choose a stock plane, the edges of the stock model will be parallel with the axes of the selected plane.

Select a stock plane when creating toolpaths in a work coordinate system (WCS) other than Top and want to align the stock model to your part. If you have several toolpaths in the machine group that use more than one WCS, Mastercam uses the stock plane to keep the stock model constant when the WCS for each operation changes. Mastercam stores the stock plane with the stock definition, independent of the WCS.

To select a stock plane, click Stock Plane and select the plane from the Plane Selection dialog box.

NOTE

If you change the stock plane, you might need to reset the stock origin.

Setting the stock origin

Use the stock origin to locate the stock relative to your part. First, you need to choose a point on the stock to use for the stock origin. In the dialog box, the cross shows you where the stock origin is:

The default position is the middle of the stock. Click on a corner of the part to set it as the stock origin. For example, router users often want the stock origin on the bottom of the part.

Then, enter the coordinates of this point in the Stock origin fields, or click Select to choose the point from the graphics window.

NOTE

Use Mastercam Simulator to perform gouge checking to ensure the placement is correct.

Parameters

Stock plane

Select a plane with which the stock model will be aligned. For example, if you will be machining the part in a work coordinate system (WCS) other than Top, you can select the same plane here so that the stock aligns properly with the geometry. Mastercam stores this plane separately from the WCS or tool plane used to create an operation, so even if you change the WCS or tool plane between toolpaths in the group, the stock will display in the plane that you select here. You can select any named plane stored in the part for the stock view.

Rectangular

Create a simple box-shaped stock model that is not defined by any actual geometry.

Cylindrical

Create a simple cylindrical stock model that is not defined by any actual geometry. Cylindrical stock models are defined by a length and radius. Select the axis of the cylinder's length.

Solid/Mesh

Use a solid model or a mesh entity inside your current part file as the stock. Choose Select to return to the graphics window, and select a solid or mesh entity.

File

Use a solid model from an STL file as the stock model. Select a file from the drop-list; or choose Select to navigate to another folder, and search for files.

Display

Displays the stock model in the graphics window.

Fit screen

Includes the stock boundary lines when you use the Fit to Screen function.

Wireframe

Choose to display the stock model as wireframe geometry. The geometry is not actually added to your part file, Mastercam simulates it based on the dimensions in this dialog box.

Shaded

Choose to display the stock as a shaded model. The model is not actually added to your part file, Mastercam simulates it based on the dimensions in this dialog box.

Stock origin

The stock origin is the X-Y-Z coordinate position of the point indicated by the cross in the picture of the stock model. Use it so Mastercam knows where your stock model is located relative to your part and coordinate system. You can either type in the X, Y, and Z coordinates directly in the fields, or choose Select to return to the graphics window and select a position.

X/Y/Z dimensions

Enter the dimensions of the stock model in these fields. Use the picture as a guide for determining which dimension goes in each field; it will change depending on whether you choose rectangular or cylindrical stock, and the orientation of the cylinder.

Select corners

Returns you to the graphics window to select two opposite points which represent the corners of the stock. This option also recalculates the stock origin based on the selected corners.

Bounding box

Returns you to the graphics window to select geometry that defines the limits of the bounding box.

NCI extents

Calculates the stock boundaries from the limits of the toolpaths in the group. Only cutting motion is considered, not rapid moves.

All entities

Choose to calculate stock based on all visible entities in the file.

All surfaces

Choose to calculate stock based on every visible surface in the file.

All solids

Choose to calculate stock based on every visible solid in the file.

Unselect all

Choose to remove all entities from the selection and return stock values to zero.
