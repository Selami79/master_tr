Levels Manager

Use the Levels Manager to manage the level settings for the current part file. With the Levels Manager, you can create a new level, set the main level, choose a level on which to create entities, select which levels are visible in the graphics window, and more. You can also use Cut, Copy, and Paste to add or remove entities between levels. Copy or cut one or more entities from the graphics window, or use the right-click menu to modify all entities on a level.

You can modify the Levels Manager function panel. Mastercam remembers the settings from session to session.

  • Float, dock, collapse, or move the Levels Manager to another monitor.
  • Right-click the column header and select which columns to show or hide.
  • Reorder the columns by clicking and dragging a column header to a new location.
  • Click in a column header to sort that column alphabetically.

Refer to the drop-downs below for more information.

Toolbar

Add a new level

Adds a new level to the part file.

Find level from geometry

Finds the level on which a selected entity is located.

Turn all levels on

Makes all levels visible. For more information, view the How do I use Turn all levels on and Turn all levels off? drop-down below.

Turn all levels off

Makes all but the main level invisible. For more information, view the How do I use Turn all levels on and Turn all levels off? drop-down below.

Reset all levels

Resets the level visibility settings to what they were when the file was loaded.

Hide level properties

Hides or displays the property controls below the levels grid. Use this control to maximize the space allotted to the grid, or minimize the space used by the manager.

Display options

Lets you choose from the following options:

  • Active level is always visible: While this option is active, you cannot turn off the main level.
  • Only active level is visible: Hides all but the main level.
  • Arrow keys set active level: Allows you to change the main level with the keyboard's up and down arrow keys.
  • Ignore blanked entities in count: Blanked entities will not appear in the Entities column.
  • Contrast rows: Applies shading to every other row in the table.
Property grid

To show or hide a property column in the Property grid, right-click any column header and select or unselect the property.

Number

Displays the number assigned to the level when it was created. Click in this column to set the main level.

Visible

Hides or shows entities for the selected levels, in the graphics window. An 'X' automatically appears in the Visible column for the selected main level and can be removed only by changing the main level or by deselecting the option Active level is always visible.

Name

Displays the level name, if one was provided when the level was created. To edit or add a name, double-click, and type the name you want to assign to the level.

Level set

Type the level set name. To edit or add a name, double-click, and type the level set name you want to assign to the level.

Entities

Displays the number of entities in the level. (Read-only.)

Parameters

Number

Displays the number assigned to the active level. To create a new level or to change the active level, enter a different number into the field.

Name

Displays the level name of the active level, if one was provided when the level was created. Enter a name into the field, to create or change the active level name.

Level set

Displays the level set name of the active level. Modify or add by typing the level set name you want to assign to the level.

Used

Displays only levels that contain entities in the current file.

Named

Displays only named levels.

Used or named

Displays only levels that contain entities in the current file and those that are named.

Range

Displays only levels that fall within the specified range.

Right-click menu options

Make active

Designates the selected level as the main level used in the graphics window.

Renumber level

Moves entities to an unused level while maintaining the level name, level set, and display status.

Select all entities

Selects all entities on the selected level.

Unselect all entities

Unselects all entities on the selected level.

Toggle visibility

Hides or shows entities of the selected level in the graphics window. You cannot toggle visibility of entities on the main level.

Cut entities

Removes geometry from a level and keeps it available for pasting. You can then use Paste entities to paste entities to a different level.

Copy entities

Copies geometry from a level. You can then use Paste entities to paste entities to a different level.

Paste entities

Places the cut or copied geometry into a level.

Delete entities

Removes geometry from a level.

Refresh levels list

Updates the levels list to reflect the current status.

Clear empty levels.

Removes all levels (or only selected levels) that contain no geometry. Disabled if Display is set to Range.

All level set on

Shows a set of levels based on their level set name. Only available when right-clicking an existing level.

All level set off

Hides a set of levels based on their level set name. Only available when right-clicking an existing level.

Report

Creates a text-based report of level details defined in the Levels Manager.

Import named levels

Loads a previously saved CSV file containing levels and level set names.

Export named levels

Saves all levels and level set names as a CSV file for reuse in other Mastercam files.

How do I import levels from another part file?

You can drag and drop a Mastercam part file onto the Levels Manager to import levels into the current part file.

How do I use Turn all levels on and Turn all levels off?

Turn all levels on and Turn all levels off are toggles that function as follows:

  • Select Turn all levels on the first time to make all levels visible. It remains toggled on and Turn all levels off toggles off.
  • Select Turn all levels on a second time to return all levels to their visible state prior to selecting the toggle. Turn all levels on toggles off.
  • Select Turn all levels off the first time to turn off the visible state of all levels, with the exception of the main level. It remains toggled on and Turn all levels on toggles off.
  • Select Turn all levels off a second time to return all levels to their visible state prior to selecting the toggle. Turn all levels off toggles off.
