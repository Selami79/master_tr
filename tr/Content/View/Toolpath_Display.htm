Toolpath Display

When you generate a toolpath, Mastercam draws the tool motion in the graphics window. Sometimes the display of multiple toolpaths can obscure the geometry and each other. You may find it simpler to turn off the toolpath display, which you can do using Toolpaths Manager options, and shortcut keys (hotkeys).

Toolpaths Manager display options

Use the following Toolpaths Manager options to display the toolpaths and/or associative geometry for only the operations you select in the Toolpaths Manager list:

Icon Name and Description
Only display selected toolpaths: Displays toolpaths only for the selected operations.
Toggle display on selected operations: Displays or hides selected operations in the graphics window. When a part contains a mixture of operations that are displayed or hidden, clicking Select all operations and then Toggle display on selected operations will hide all toolpaths in the graphics window. Clicking Toggle display on selected operations again shows all toolpaths.
Only display associative geometry: Displays all geometry associated with the selected operations. This includes geometry that was previously hidden or blanked using Display functions on the Home tab, and geometry on levels that were not marked as Visible in the Levels Manager.
Advanced Display: Toggles the toolpath display, allowing you to see the differences between individual toolpath entities such as entry motion and endpoints by modifying their color and display state.
NOTE
  • Toolpaths Manager display options override all other toolpath display states and settings.
  • Stock model operations are affected by Toolpaths Manager display options. Additionally, you can right-click in the Toolpaths Manager and choose Mill toolpaths, View stock model. Then use options in the Stock Model Viewer dialog box to limit the compare display to only selected components.

To toggle the display of all toolpaths in your graphics window, press the [Alt] key and the [T] key at the same time [Alt+T]. For this to work, you must be working in the graphics window, not in Toolpaths Manager.

This function does not change the display state of toolpaths (which individual toolpaths will display and which will not). It will only toggle the visibility of toolpaths on and off in the graphics window.

To change the display state of certain toolpaths, select one or more toolpath operations in Toolpaths Manager and press [T]. Mastercam turns off the toolpath display for the selected toolpaths. Press [T] again to turn the display back on. This is especially helpful when working on complex parts and you only want to view specific toolpaths.

The graphic below shows an operation with the toolpath display state on. Notice the dark toolpath folder and toolpath icon shown in the first image below. The second image shows what it looks like with the toolpath display state off. Notice that the toolpath folder and toolpath icon turns gray, indicating that the toolpath display state is off for that operation. When the display state is turned off for that operation and you press [Alt+T] in the graphics window, that toolpath will not be displayed.

NOTE
  • Toolpaths Manager display options override all other toolpath display states and settings.
  • Stock model operations are affected by Toolpaths Manager display options. Additionally, you can right-click in the Toolpaths Manager and choose Mill toolpaths, View stock model. Then use options in the Stock Model Viewer dialog box to limit the compare display to only selected components.
  • Toolpaths will not be displayed for operations marked dirty until they are regenerated.
