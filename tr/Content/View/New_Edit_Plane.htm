New Plane/Edit Plane Panels

Use the New Plane and Edit Plane function panels to create and edit planes. Once you define a plane, you can select it by its name and use it for other functions in your part file.

How do I create a plane from wireframe or surface geometry?

To create a plane using wireframe or surface geometry, follow these steps:

  1. Select From geometry from the Create a new plane drop-down in the Planes Manager.

  2. Select the necessary entities. Select either:
    1. Three points. These must be individual point entities and not locations based on other entities, like endpoints, midpoints, or intersections.
    2. Two intersecting lines.
    3. A flat 2D entity, such as an arc or flat surface. You cannot select a solid face.

  3. Typically there will be several possible axis orientations for the geometry that you have selected. Use the Select plane dialog box to cycle through them. Click OK when the desired orientation appears.

    The New Plane function panel displays.

  4. Enter a name for the plane.
  5. Set any options you need in the function panel.
  6. Click OK to create your plane.
How do I create a plane from a solid face?

To create a plane using a solid face, follow these steps:

  1. Select From solid face from the Create a new plane drop-down in the Planes Manager.

  2. Select a flat solid face. Use Select from back on the Selection Bar if you need to select a hidden face.

  3. Typically there will be several possible axis orientations for the geometry that you have selected. Use the Select plane dialog box to cycle through them. Click OK when the desired orientation appears.

    The New Plane function panel displays.

  4. Enter a name for the plane.
  5. Set any options you need in the function panel.
  6. Click OK to create your plane.
How do I create a plane normal to an entity?

To create a plane normal to an entity, follow these steps:

  1. Select From entity normal from the Create a new plane drop-down in the Planes Manager.

  2. Select the necessary flat entity. This entity will be parallel to the Z axis in the new plane.

  3. Typically there will be several possible axis orientations for the geometry that you have selected. Use the Select plane dialog box to cycle through them. Click OK when the desired orientation appears.

    The New Plane function panel displays.

  4. Enter a name for the plane.
  5. Set any options you need in the function panel.
  6. Click OK to create your plane.
How do I create a plane with the Dynamic Gnomon?

You can create a plane with the Dynamic Gnomon in the following way:

Planes Manager

  1. Select Dynamic from the Create a new plane drop-down in the Plane Manager.

    The New Plane function panel displays.

  2. Place the Dynamic Gnomon where you want the new plane origin to be in the graphics window. The Dynamic Gnomon functions allow you to make plane adjustments.
  3. Enter a name for the plane.
  4. After setting options in the function panel, click OK to create the plane.
NOTE

When creating or editing a dynamic plane, you can move the origin by using the Dynamic Gnomon or by entering coordinates in the function panel.

View Axes Gnomon

  1. Click the View Axes Gnomon in the lower left corner of the graphics window to activate the Dynamic Gnomon and open the New Plane function panel.

    You can also drag the View Axes Gnomon onto the graphics window.

  2. Make any plane adjustments as necessary.
  3. Enter a name for the plane.
  4. Set any options you need in the function panel.
  5. Click OK to create your plane.

The Dynamic Gnomon's orientation is based on the method used to access the New Plane function.

  • Using the View Axes Gnomon orients the Dynamic Gnomon to the current Gview.
  • Using the Planes Manager orients the Dynamic Gnomon to the current WCS.

For more information about the Dynamic Gnomon, refer to The Dynamic Gnomon Help topic.

How do I edit a plane?

To edit a plane, right-click a user-defined plane in the Planes Manager and select Edit. You cannot edit a system plane, such as Isometric or Top.

How do I create a viewsheet or section view from a plane?

To create a section view, select Section on the Basic tab of the New Plane or Edit Plane function panel to set that plane as a section view. See Section View for more information.

To create a viewsheet, select Create viewsheet on the Advanced tab. You can also save the current view, scaling, and level settings in the new viewsheet by selecting Save bookmark. See Using Viewsheets for more information.

What are associative planes? When should I use them?

Associative planes maintain a relationship with the geometry used to create them. When you create a new plane from an entity, Associative origin and Associative view are selected by default to maintain the predefined spatial relationship of the plane's view and origin relative to the geometry used to create the plane.

This option is available when creating a plane from geometry, entity normal, a solid face, or from the Dynamic Gnomon. In the Advanced tab of the New Plane/Edit Plane function panel, Associative origin and Associative view are selected by default. If you transform the geometry you used to create the plane, the plane origin and view transform to maintain their spatial relationship to the geometry.

If you break a plane origin's associativity, you can still reestablish it when selecting a new origin for the selected plane.

Broken orientation associativity cannot be reestablished.

Deselect Associative origin and Associative view when you are creating a plane from temporary guides or construction lines that you plan to delete.
