Tool Settings tab: Machine Group Properties - Mill-Turn, Lathe

Use the Tool settings tab to control how Mastercam assigns tool numbers, tool offset numbers, and default values for feeds, speeds, coolant, and other toolpath parameters for a Lathe or Mill-Turn machine.

Mastercam applies the Default program number option to any operations created after you set the program number. If you change the program number, only new operations get the new program number. To change the program number of existing operations, select an operation and right-click in the Toolpaths Manager. Choose Edit selected operations, Change program number.

Parameters

Default program number

The post processor will use this field if your machine tool requires a number for a program name.

From tool

Uses the feed rate, plunge rate, retract rate, and spindle speed directly from the tool definition.

From material

Calculates the feed rate, plunge rate, retract rate, and spindle speed based on the type of stock material.

From defaults

Uses the setting from the toolpath defaults file.

User defined

Select to have the default feeds and speeds come from values that you enter on this tab. After selecting this option, enter the default values for each type of feed rate and spindle speed. The values that you enter on this tab affect only the current machine group in the current part file. These values are not associative with operations after they have been created. This means that if you change these values, it will not affect the feeds and speeds for any operations that have already been created.

Adjust feed on arc move

Adjusts the current linear feed rate to fit arc geometry. The feed rate change occurs at the start point of the arc. The adjusted feed rate cannot fall below the minimum arc feed rate.

Minimum arc feed

Sets the slowest feed rate for areas in the toolpath where the tool approaches an arc with a radius equal to the radius of the tool. This option applies only when you select the Adjust feed on arc move option.

Assign tool numbers sequentially

Assigns the next available tool number for new tools created or selected from the tool library. If you select this option, Mastercam overwrites the tool numbers stored in the tool definition with a sequential number. Mastercam looks for the highest tool number in your current tool list and adds 1. If you do not select this option, Mastercam uses the number stored in the tool definition.

Warn of duplicate tool numbers

Informs you when duplicate tool numbers are entered and displays a description of the duplicate tool.

Warn on mill tool orientation conflict

Validates that the selected tool can be used in the orientation required for the operation. Mastercam checks if other operations use the tool and compares the previous tool orientation to the current one. If the orientations conflict (for example, the previous operation is facing and the current is cross milling), and the tool is not allowed to rotate (such as when mounted in a turret), an error displays.

Use tool's step, peck, coolant

Overrides the toolpath's default step, peck, and coolant parameters with information stored in the tool's definition.

Search tool library when entering a tool number

By selecting this option, you can re-select a tool that was used in an earlier operation by just entering its tool number in the Toolpath parameters tab.

Override defaults with modal values

When you select this option, the default values for any of the following toolpath parameters will be the value from the previous operation:

  • Clearance height
  • Retract height
  • Feed plane

These modal defaults will replace those found in the toolpath defaults file. Select each parameter for which you want to use a modal default.

Rapid moves

Sets the minimum distance between the tool and the stock, chuck, tailstock, and/or steady rest when making rapid moves. Mastercam displays a collision warning when creating the toolpath if a rapid move violates this boundary.

You can override this value for individual toolpaths by clicking the Stock Update button on the Toolpath Parameters tab.

Entry/Exit

Mastercam will maintain this distance between the tool and the stock, chuck, tailstock, and/or steady rest when entering or exiting the part using feed moves. Mastercam displays a collision warning when creating the toolpath if an entry or exit move violates this boundary.

You can override this value for individual toolpaths by clicking the Stock Update button on the Toolpath Parameters tab.

Sequence number

Enter the starting sequence number and the increment in the proper fields. The default values and specific format for these fields is determined by the NC Output section of the control definition.

Material

Shows the currently selected stock material.

Edit

Opens the Material Definition dialog box so you can edit the material definition. If you are working with a mill-turn machine, you will be able to edit the material parameters separately for mill and lathe, so that you can individually adjust the default feeds and speeds for each.

Select

Opens the Material List dialog box so you can select a material from a material library.
