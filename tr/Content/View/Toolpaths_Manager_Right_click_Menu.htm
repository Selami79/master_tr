Toolpaths Manager Right-click Menu

The Toolpaths Manager provides tools, functions, and commands that are accessible from a right-click menu. The following are choices from that menu. Click a drop-down menu below to view its description.

Mill toolpaths
  • Create a new stock model operation.
  • View the results of a stock model compare.
  • Export stock model compare results to an STL, 3MF, OBJ, or AMF file.
  • Convert stock model compare results to a mesh entity.
Milling toolpaths (Mill-Turn only)

Opens a submenu from which you can create Milling toolpaths within a Mill-Turn machine type. The toolpaths available are a subset of those available in Mastercam Mill.

Turning toolpaths (Mill-Turn only)

Opens a submenu from which you can create Turning toolpaths. The toolpaths available are a subset of those available in Mastercam Lathe.

Part Handling (Mill-Turn only)

Opens a submenu from which you can create Part Handling toolpaths. The toolpaths available handle pickoff, cutoff, stock movement, turret location, and spindle behavior.

Lathe toolpaths

Opens a submenu from which you can create and transform Lathe toolpaths.

Wire toolpaths

Opens a submenu from which you can create wirepaths.

Router toolpaths

Opens a submenu from which you can create, transform, trim, project, and nest Router toolpaths.

Edit selected operations

Opens a submenu for the selected operation. From this menu, you can:

  • Edit the operation's common parameters and NC file name.

  • Adjust the library origin.

  • Renumber tools, work offsets, and program numbers.

  • Reverse the selected operation or recalculate its feeds and speeds.

Groups

Opens a submenu from which you can create new machine and toolpath groups, as well as rename or delete existing groups.

Cut

Removes selected operations or groups from the list and places them on the clipboard to paste elsewhere. You can use Cut to delete operations and groups from a list.

Copy

Copies the selected operations or groups on the list to the clipboard to paste elsewhere.

Paste

Pastes cut or copied operations or groups into the list. Click the operation or group you want to paste the item ahead of.

Delete

Deletes the selected operation or group from the list.

Undelete

Undoes the last operation or group deleted from the list.

Undo

Allows you to undo changes made to an operation during a session of Mastercam. When you right-click an operation and choose Undo, a fly-out menu displays a list of the dates and times associated with the most recent changes to the operation that you can undo. Choose the date and time of the action that you want to undo.

Expand

Expands the tree structure for the selected machine group or toolpath group to show editable components such as parameters, tool definitions, and geometry.

Collapse

Collapses the tree structure for the selected machine group or toolpath group, hiding editable components such as parameters, tool definitions, and geometry.

Reports

Creates a text file with information about the selected operations or all operations. Mastercam opens the text file in your default text editor, where you can edit or save the report.

  • Select Operations list to create a report which includes the information displayed in the Toolpaths Manager.
  • Select Dependencies to create a report about which operations reference other operationsâ€”for example, transform, nesting, or pinch turn operations.
Operation Selection

Opens the Selection Options dialog box, where you can define operation parameters (i.e., tool number, operation type, machine definition, etc.) that determine which operations are selected.

Sort

Opens a submenu from which you can choose Unsort to unsort operations that have been previously sorted, or you can choose Sort, which opens the Sorting Options dialog box. The Sorting Options dialog box lets you set which methods (i.e., tool number, creation order, operation type, etc.) are used to sort the selected operations.

Import

Opens the Import Toolpath Operations dialog box, which lets you import saved operations to the current Mastercam file, with or without their geometry.

Export

Opens the Export Toolpath Operations dialog box, which lets you export operations from the current Mastercam file to an operations library, with or without their geometry.

Display options

Opens the Display Options dialog box, which lets you control the information that appears with each operation in Toolpaths Manager.

Setup sheet

Opens the ActiveReports Setup Sheet dialog box when the Setup sheet program in System Configuration, Reports is set to ActiveReports. This dialog box is used to generate an ActiveReports Setup Sheet. The fields in this dialog box provide for entry of general information, access to image and color usage, tool sorting options and views of operation for the ActiveReports Setup Sheet.

An Add Images button (located at the bottom of the dialog box) activates the Capture Images dialog box, allowing the addition of custom graphics. Clicking the OK in this dialog box opens the ActiveReports Setup Sheet Report, which allows addition of graphics and text in annotations, but does not provide for editing the report.

When the Setup sheet program in System Configuration, Reports is set to .SET, it opens using Code Expert, and a report is created that you can edit or print using the test/program editor.

Tool list

Creates an ActiveReport that lists tools used by any operation selected in the Toolpaths Manager.

Lathe stock preview

Opens the Lathe Stock Preview dialog box where you can display the machined stock, chuck, tailstock, and steady rest boundaries as solids or surfaces. This option is available only when your insert arrow is placed in a lathe machine group and when stock is defined for the job. For additional information on stock updates, please see Toolpath parameters tab (Lathe).

Lathe stock update

Choose this option to toggle whether the lathe stock updates or not as you work in an operation. This option is available only when your insert arrow is placed in a lathe machine group and when stock is defined for the job.

Turning stock preview (Mill-Turn only)

Opens the Turning Stock Preview dialog box where you can display the machined stock, chuck, tailstock, and steady rest boundaries as solids or surfaces. This option is available only when your insert arrow is placed in a Mill-Turn machine group and when stock is defined for the job.

Turning stock update (Mill-Turn only)

Choose this option to toggle whether or not Mill-Turn stock updates as you work in an operation. This option is available only when your insert arrow is placed in a Mill-Turn machine group and when stock is defined for the job.

Toolpath Editor

Opens the Toolpath Editor dialog box to modify an existing toolpath. You can:

  • Edit tool motion at specific point positions in the toolpath.
  • Add points to increase tool motion.
  • Move points to adjust tool motion.
  • Delete selected points, passes, cuts, and sections of a toolpath.
