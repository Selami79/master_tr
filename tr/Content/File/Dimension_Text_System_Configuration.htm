Dimension Text page: System Configuration/Drafting Options

Use the Dimension Text page to set properties for the text used in dimensions. These include:

  • Text height, width, and spacing
  • Lines and borders
  • Text path or direction
  • Point dimension text display
  • Text alignment and orientation
  • Font and style

You can see the effect of most of the settings in the large preview window as you make changes.

Select the Use factors option to link the size of the tolerance text, arrowheads, and witness lines to your Text height. If you do not select the Use factors option, define the default size of the arrowheads by entering the values directly in the Leaders/Witness page.

NOTE

This page is used when you are working in the System Configuration dialog box to set the default values for your dimensions, and also when you are working in the Drafting Options dialog box to edit an individual dimension.

Parameters

Text height

The height of the letters. This is the approximate height of a typical capital letter (for example, A) measured in the current units (inches or millimeters). Letters in some TrueType® fonts may not be created at exactly the height you specify because the font data embedded in the font may include additional space.

Tolerance height

Specifies the height of the text used in tolerance values, if they are included with the dimension.

When Use factors is selected, the Tolerance height is linked to the Text height, and will update whenever you change the Text height.

Expansion

Adjusts the width of the characters. The value that you enter here scales the characters in the original font by this amount.

Character spacing

Adjusts the amount of space between each line. The value that you enter here scales the line spacing in the original font by this amount.

Line spacing

Adjusts the amount of space between each line. The value that you enter here scales the line spacing in the original font by this amount.

Use factors

Links the size of arrowheads and other elements used for notes and dimensions to the Text height. Click the Factors button to enter the scaling values.

Factors

When Use factors is selected, opens a dialog box where you can specify the scaling values as a function of the text height. These are used to size arrowheads, witness lines, and tolerance text.

Bold

Makes each character thicker.

Italic

Makes the shape of each character slanted.

Underline

Draws a line under the text. This function does not apply if the text is aligned with a curve.

Filled

Creates solid letter forms instead of hollow outlines.

Display negative signs

Adds a negative sign (-) before dimension values that are less than zero, as defined by the ordinate group's origin.

Font

Select the default font or typeface for new notes.

Add

Click to select a TrueType® font and add it to the list.

Type

Select the desired format for dimensions for point entities:

  • 2D: Displays the X and Y coordinate values.
  • 2D label: Displays the X and Y coordinate values plus the axis labels.
  • 3D: Displays the X, Y, and Z coordinate values.
  • 3D label: Displays the X, Y, and Z coordinate values plus the axis labels.

Absolute

Displays absolute (or world) coordinates rather than coordinates relative to the dimension's view. Applies only to point dimensions.

Display in SmartMode

Allows the creation of point dimensions in Smart Drafting Dimensions, an intelligent, model drafting environment.

Text direction

Specifies the reading order of the text. Choose one of the following options:

  • Left: Organize the letters so that the reading order is from right to left.
  • Right: Organizes the letters so that the reading order is left-to-right.
  • Up: Organize the text in a vertical column reading from bottom to top.
  • Down: Organize the text in a vertical column reading from top to bottom.

Text orientation

Choose one of the following options to specify the placement of dimension text.

  • Aligned: Aligns dimension text parallel to leader lines.
  • Unidirectional: Displays dimension text horizontally, regardless of dimension orientation.
