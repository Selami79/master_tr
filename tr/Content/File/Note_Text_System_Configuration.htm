Note Text page: System Configuration

Use the Note Text page to set default properties for note text. These include:

  • Text height, width, and spacing
  • Lines and borders
  • Text direction
  • Font and style
  • Horizontal and vertical alignment
  • Mirroring
  • Angle, slant, and rotation

You can see the effect of most of the settings in the large preview window. The red "+" character indicates the anchor point that will be used to locate the text block or note.

Select the Use factors option to link the size of note/label arrowheads to your text dimensions. If you do not select the Use factors option, define the default size of the arrowheads by entering the values directly in the Leaders/Witness page.

Parameters

Text height

The height of the letters. This is the approximate height of a typical capital letter (for example, A) measured in the current units (inches or millimeters). Letters in some TrueType® fonts may not be created at exactly the height you specify because the font data embedded in the font may include additional space.

Expansion

Adjusts the width of the characters. The value that you enter here scales the characters in the original font by this amount.

Character spacing

Adjusts the amount of space between each line. The value that you enter here scales the line spacing in the original font by this amount.

Line angle

Rotates the note or block of text and sets the text at this angle.

Line spacing

Adjusts the amount of space between each line. The value that you enter here scales the line spacing in the original font by this amount.

Use factors

Links the size of arrowheads and other elements used for notes and dimensions to the Text height. Click the Factors button to enter the scaling values.

Factors

When Use factors is selected, opens a dialog box where you can specify the scaling values as a function of the text height. These are used to size arrowheads, witness lines, and tolerance text.

Bold

Makes each character thicker.

Italic

Makes the shape of each character slanted.

Underline

Draws a line under the text. This function does not apply if the text is aligned with a curve.

Filled

Creates solid letter forms instead of hollow outlines.

Outline box

Creates a line around the text block.

Font

Select the default font or typeface for new notes.

Add

Click to select a TrueType® font and add it to the list.

Justification

Specifies the horizontal alignment used for the text. Choose one of the following options:

  • Left: Aligns each line with the left edge of the text block.
  • Center: Spaces each line equally between with the left and right edges of the text block.
  • Right: Aligns each line with the right edge of the text block.

Text direction

Specifies the reading order of the text. Choose one of the following options:

  • Left: Organize the letters so that the reading order is from right to left.
  • Right: Organizes the letters so that the reading order is left-to-right.
  • Up: Organize the text in a vertical column reading from bottom to top.
  • Down: Organize the text in a vertical column reading from top to bottom.

Mirror

Choose one of the following options.

  • None: Does not mirror the text string about any axis.
  • X axis: Mirrors the text string about the X axis.
  • Y axis: Mirrors the text string about the Y axis.
  • Both: Mirrors the text string about both the X and Y axes.

Base position

Select a reference position on the text block that will be used to locate it. When you click a point or location to position the text block, the reference position that you select here will be located at the anchor location. The position that you select here is indicated in the preview window by a red "+" character.
