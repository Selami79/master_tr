Chaining page: System Configuration

Use the Chaining page to set default values that affect how you work with chains. After saving your settings, these values apply to all Mastercam sessions; that is, they are in effect each time you initialize Mastercam.

NOTE
  • To change default settings for chains you create during the current Mastercam session, use the Chaining options dialog box.
  • To set other parameters that affect screen display and entity selection, use the Screen property page.
Parameters

Use entity masking

Chains only entities that are the same type as the first entity chained. In Windows and Area chaining, chains only entities you set in the Chaining options dialog box.

Use color masking

Chains only entities that are assigned the same color as the first entity chained. In Windows and Area chaining, chains only entities that match the color set in the Chaining options dialog box.

Use level masking

Chains only entities that are on the same level as the first entity chained. In Windows and Area chaining, chains only entities that are on the level that you set in the Chaining options dialog box.

Open chains

Lets you select only open-chain entities for chaining, unless Closed chains is also selected. Applies to Window and Polygon selection.

Closed chains

Lets you select only closed-chain entities for chaining, unless Open chains is also selected. Applies to Window and Polygon selection.

Section angle

Selecting this checkbox lets you enter an angle value in the text box. Chaining stops when Mastercam encounters an angle that exceeds this value. Applies to Chain, Window, Polygon, and Area selection.

Line style

The line style to use when displaying chains.

Line width

The line width to use when displaying chains.

Small / Medium / Large

Specifies the size of chaining arrows. Choose the size that works best for the geometry in the graphics window.

Translucent

Specifies translucent chaining arrows.

Default to last used

When selected, sets the Chaining dialog box default to the last chaining mode you selected. Deselect to always use the default chaining mode you set on this page.

Cplane

Chains only entities that are parallel to the current construction plane and at the same Z depth as the first chained entity. When you use the Window, Area, or Polygon methods to chain wireframe entities, Cplane chains only entities that are on the current construction plane and at the current Z depth. See Solids Mode: Chaining for more information about solids chaining and Cplane chaining.

Mastercam uses the Plane tolerance setting in the Chaining options dialog box to determine whether or not an entity lies in the construction plane.

3D

Chains entities defined in the X, Y, and Z axes simultaneously. Recognizes co-planar as well as non-planar branches. At branch points, Mastercam prompts you to select the next entity in the chain.

Selection method

Choose a default wireframe selection method from the drop-down list.

  • Chain: Creates a chain by clicking on an entity in the graphics window.

  • Chain (partial): Use your mouse to define the start and the end of an open chain. Mastercam creates an open chain consisting of all adjacent entities between these two points.

  • Window: Selects multiple chains by drawing a rectangle around them. Mastercam orders the chains based on the setting you choose. The selection boundaries are determined by your selection in the Entity selection settings drop-down menu below.

  • Polygon: Selects chains within a multi-sided, irregular window that you draw around the entities.

  • Area: Selects all chains within a closed boundary with a single mouse click. When using this chaining method, you can turn Infinite nesting in area chains (in the Chaining options dialog box) on or off, giving you two methods for chaining closed boundaries.

  • Vector: Creates chains from entities that intersect a vector line, extending chains in both directions until a branch point is encountered.

  • Single: Creates a chain out of a single line, arc, or spline. Useful for created ruled surfaces that require two opposing entities for the surface boundaries. You can also use this setting to chain surface edges.

  • Point: Collects points as chained entities. Use this method to chain a point to serve as a plunge point or as a move-to point in a toolpath. Points are sometimes used when chaining entities for surfaces.

Entity selection settings

This drop-down menu provides options that modify Window and Polygon chaining. Select any of the following:

  • In: Chains all entities within the window.

  • Out: Chains entities that lie outside of the window.

  • In +: Chains entities inside and that intersect the window.

  • Out +: Chains entities that lie outside and that intersect the window.

  • Intersect: Chains entities that intersect the window.

Wait

Works only with Partial chaining. Allows the same entity endpoint to be selected so that you can chain a single or partial chain in both directions.

Edge selection method

Choose a default preference for selecting solid edges.

  • None: Mastercam does not initially select an edge selection method in the Chaining dialog box.

  • Edges: Creates a chain from the selected edge. Use [Shift+click] to select tangent solid edges. When in Cplane mode, only edges that are parallel to the Cplane are chained, and the chain will propagate to include adjacent edges at the same depth relative to the Cplane.

  • Loop: Creates a chain from the selected loop. Hovering over an edge will highlight the loop associated with it. If an edge is shared, move your mouse to highlight the other available loops before selecting. When in Cplane mode, only loops that are parallel to the current construction plane are chained.

  • Partial loop: Creates an open chain from a loop. In the graphics window, click the entity where you want to start the chain, and then click where you want to end the chain. Mastercam creates an open chain consisting of all adjacent entities between these two points.

Face selection method

Choose a default preference for selecting solid faces.

  • None: Mastercam does not initially select a face selection method in the Chaining dialog box.

  • Face: Creates a chain from the selected face. When in Cplane mode, double-clicking or triple-clicking will only select faces that are parallel to the Cplane and at the same depth as the face that was selected. Window select will only select faces that are in the window, parallel to the current Cplane, and at the application's current Z depth.

  • Outer open edges: Chains, in a clockwise direction, the outer open edges of a selected face.

  • Outer shared edges: Chains, in a counterclockwise direction, the outer shared edges of a selected face.

  • Cavities: Selects all cavities on a face. This selection method creates counterclockwise chains by default. These chains can be individually edited in Chain Manager as individual loops.

  • Bosses: Selects all bosses on a face. This selection method creates clockwise chains by default. These chains can be individually edited in Chain Manager as individual loops.

Cavities

Choose a cavity selection default from the drop-down list.

  • All: Chains all cavities on a selected face.

  • Blind: Chains only blind cavities on a face. Blind cavities terminate partway through the stock.

  • Through: Chains only through cavities on a face. Through cavities pass completely through the stock.

CW / CCW

Sets default chain direction to CW (clockwise) or CCW (counterclockwise) regardless of where you position the cursor when selecting the chain.

Use cursor position

Sets chaining direction relative to the position of the cursor when you select an entity to be chained, overriding the default chain direction. Selecting this option gives you more control over the direction of closed chains.

Open chains

Specifies a search direction for open chains. These options apply only to Window and Polygon chaining. The search point is set by the first corner of the window or polygon.

  • One way: Selects the next chain that is closest to the start point of the previous chain.
  • Zigzag: Selects the next chain that is closest to the endpoint of the previous chain.

Plane mask in chaining

Sets Cplane chaining as the startup default for general selection.

Ignore depths in 3D chaining

Chains entities regardless of their Z depth, when making general chaining selections while in 3D construction mode. The endpoints of the entities must be within the chaining tolerance in the XY plane.

Break closest entity to thread point

When checked, establishes a wirepath start point by breaking the entity closest to the thread point at a position perpendicular to it. If a perpendicular position cannot be established, this setting is ignored.

Sorting

Choose one of the following sort options from the drop-down list:

  • Next closest: Sorts chains starting with the closest chain to the search point. Mastercam selects an entire grouping of chains before moving to the next grouping.
  • Inside to outside: Sorts chains so that innermost chains are processed together, then the next outer chains, and so forth.
  • Outside to inside: Sorts chains so that outermost chains are processed together before the next inner chains, and so forth.
  • Optimize inside to outside: Processes one set of nested chains from the inner to the outer chain, then processes the next set of nested chains from inner to outer, and so forth.
  • Optimize outside to inside: Processes one set of nested chains from the outer to the inner chain, then processes the next set of nested chains from outer to inner, and so forth.

Infinite nesting in area chains

Applies only to Area chaining. Use Area chaining to chain entities within a closed boundary by clicking once within the boundary. When this option is enabled, Mastercam chains all entities within a closed boundary, even if they lie within other closed boundaries. When disabled, Mastercam chains only the outer and inner boundary closest to the point you entered to define the area chain and any curves or points between the two. Nested geometry is any geometry that lies completely within a closed boundary.

Reverse inner chains

Swaps the chaining direction when chains are surrounded by an outer boundary. This chaining option is useful for geometry that consists of inside and outside shapes. It provides an automated method of reversing the cutter compensation from the inside shape to the outside shape.

Ignore depths (in 3D mode)

Defines the default setting for the Ignore depths (in 3D mode) field in the Chaining options dialog box. When chaining in 3D mode with this option on, Mastercam chains entities regardless of their Z depth, as long as the endpoints of the entities are within the chaining tolerance in the XY plane.

Allow surface edges in single mode

If you have selected Single in the Chaining dialog box, this option lets you chain surface edges.

Break entities in Dynamic

When enabled, breaks an entity at the position of the start and end point set by the Dynamic Start/End options in the Chaining dialog box. Dynamic Start/End sets a chain start or end point by dragging it to any position on an entity instead of restricting it to an entity endpoint.

Prompt for window start point

Used only with Window chaining. When you select this checkbox, Mastercam will prompt you to "Sketch approximate start point" after you finish defining the selection window. The point you choose in the graphics window is used as the start point of the first open chain. Mastercam then assigns start and end points for all remaining open chains in the selection.

Deselect this option if you want Mastercam to order the chains based on the position of the first corner of the window, which defines the search point.

Set start of chain from point entities

Forces the chain start point to a point that you create on an entity in the chain. When enabled, this option lets you click anywhere on the entity to chain it, but places the chain start point at the point entity. When disabled, the chain starts at the point closest to the cursor position.

Sync mode

A set of methods Mastercam uses to align chains. Select one of the following methods from the drop-down list:

  • None: Synchronizes the chains by dividing them each into an even number of points. Certain surfaces and toolpaths require more precision than this option provides.
  • By entity : Matches the chains by the endpoints of each entity. Requires both chains to have the same number of entities.
  • By branch: Matches the chains at branch points. Can be used for most chain synchronization.
  • By node: Matches two or more splines by the node points of each spline. Each spline must have the same number of node points. Applies only to parametric splines.
  • By point: Matches the chains by point entities on the endpoints of each entity. You need to have created the points where you want the chains to sync.
  • Manual: Matches the chains of user-defined areas.
  • Manual/density : Matches the chains that you specify and allows you to assign a density for each chain. If an area has small radii, use a higher density (such as 2) for a better finish.
