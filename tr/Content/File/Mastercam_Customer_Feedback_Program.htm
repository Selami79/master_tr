Mastercam Customer Feedback Program

Mastercam includes an optional data feedback tool titled Mastercam Customer Feedback Program. The program is voluntary. If you choose to turn this functionality on, it runs during every Mastercam session, collecting and sending usage information to Mastercam (CNC). This data is used only for quality and product improvement purposes and helps give CNC an idea of the relative value of various functionality within its products, along with hardware and software trends in the user community. The data does not include user model data or any information that could be used to recreate models. This program cannot/will not collect data for the purpose of reverse engineering.

The Mastercam Customer Feedback Program at no time collects data that is not relative to Mastercam. No sensitive, personal, or otherwise private data is gathered. All information collected in this program is strictly confidential.

The Mastercam Customer Feedback Program dialog box appears when you first open Mastercam after an install. To participate in the program, you must "opt-in" by choosing Join the Mastercam customer feedback program. After you have selected the option, click OK to begin participating.

Types of data collected

Listed below are some of the types of data that this tool collects if you choose to participate:

  • The Mastercam build number
  • Your SIM number
  • The name of the function you called and how you accessed it (tab command, hotkeys, menu)

The list above is not comprehensive. These are examples of the types of data CNC collects.

How CNC uses the information

The combined data gathered from Mastercam Customer Feedback Program participants is used by CNC to answer the following types of questions:

  • What Mastercam functions are used the most, and which are used the least?
  • How often and when is Mastercam Help used?
  • Which file translators are used, and how often?
  • Which Add-Ins are used and how often?
  • What commands are used sequentially or in conjunction with each other?

You can also open the dialog box at any time to change your setting by selecting File, Community and choosing Customer Feedback.

NOTE
  • Participating in this program will not impact your processing time or computer performance.
  • If you choose not to participate, you will be reminded again after a six-month period.
  • If you are running the Mastercam Customer Feedback Program, the program closes when you close Mastercam.
  • The data gathered by this program is sent to Mastercam when you shut down Mastercam, or after a specific number of transactions.
  • If you do not have an Internet connection, the data will be saved to your local drive. The next time you open Mastercam, the program will attempt to send the data. If the data cannot be sent over a period of five days, it is automatically purged from your system.
  • You can change your participation at any time, and any number of times during a Mastercam session. For example, you can let the program collect data up until a certain point in your work, turn it off for a period of time, and then turn it back on. If you choose not to participate, you will be reminded again after a six-month period.
