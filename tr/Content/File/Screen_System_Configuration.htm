Screen page: System Configuration

Use the Screen page to do the following:

  • Control how Mastercam looks and operates
  • Change the size of gnomons and the Gview cube, as well as text and opacity of some on-screen controls
  • Manage how Mastercam displays and lets you select certain entities
  • Specify how Mastercam handles error messages generated during toolpath operations
Parameters

Report toolpath error messages to

Specifies how Mastercam error messages appear. The options are as follows:

  • Screen and log file: Directs error messages to both the screen (in the form of message dialogs) and to the event log.
  • Log file only: Directs error messages only to the log. This setting stops Mastercam from interrupting functions with error dialogs.

Number of functions in the MRU menu

Sets the maximum number of items displayed in the Recent Functions panel. Once the maximum number is reached, least recently used items are removed as new ones are added.

Include immediate mode functions

Sets the default display of immediate mode functions in the Recent Functions panel. When selected, displays immediate mode functions in Recent Functions panel. When deselected, does not display those functions.

Number of items in the MRU data field

Specifies the number of values to remember in numerical data fields. Click in a field in the function panel and press the down arrow on your keyboard to access the MRU values. Not all panels and fields support MRU functions.

Draw stop/info/regs symbols

Displays wire symbols that show positions within a wirepath.

Draw thread/cut symbols

Displays wire symbols that show positions on wire geometry.

Corner Gnomon size and a Display Text Size

The corner gnomon size adjusts the size of the gnomons in the corner and the gnomon labels.

The display text size adjusts on-screen text at the bottom of the screen.

Cube text size, cube/gnomon size, cube opacity

Adjusts the text size, cube size, and opacity of the Gview cube. Use the slider to adjust to the desired size and opacity.

Overlay Controls Opacity

Controls the opacity of the on-screen Selection Bar, Gview Cube, and Quick Mask controls, as well as the Gnomon/Geometry switch that displays when you perform Transform and Model Prep functions. The Gview Cube has its own Cube Opacity slider and the Overlay Control handles the Status Bar and Quick Mask icons. (See below)

Temporary midpoints delay

When using AutoCursor, controls how many seconds before midpoints are displayed when hovering over an entity. The default is 1 second.

Toolpaths

Displays toolpaths in the graphics window.

Level name

Displays the active level name in the lower-middle area of the graphics window.

Unit/scale

Displays the unit/scale information in the lower-right portion of the graphics window.
