Toolpaths page: System Configuration

Use the Toolpaths page to set default parameters that control how toolpaths are created, maintained, and displayed. These include:

  • How machining, avoidance, and tool containment boundaries are selected for Surface Rough, Surface Finish, and 3D High Speed toolpaths.
  • Criteria for keeping or deleting regen files.
  • Buffering size (for toolpath algorithms that require large amounts of computer resources).
  • Prompting for a CAD file during toolpath creation.
  • Options for associativity warnings.
  • Display of surface/plane intersections.
  • Activating multi-threaded toolpath processing for 3D High Speed toolpaths and some Multiaxis toolpaths.
  • Tools when importing operations.
NOTE

You can use the Toolpaths Manager to change tool display settings for an operation by clicking Parameters and then choosing Tool display on the Toolpath parameters tab for tab-style toolpaths. This does not apply to tree-style toolpaths. Tool display parameters affect how Mastercam displays toolpaths as it generates them. It does not affect Classic Backplot or Mastercam Simulator.

Parameters

Suppress associativity warning messages

Turns off associativity warnings. These are displayed when an action you are about to take will affect associated geometry or parameters.

Use existing tools when importing operations

Uses tools already in the tool list that match tools required by an imported operation without asking. Deselect to always ask first.

Get defaults from previous operations

Mastercam pulls operation parameters from the previous operation of the same type. If no compatible operation is found, the parameters are pulled from the defaults library. When deselected, the new operation parameters are always pulled from the defaults library.

Include comments

When selected, adds the comments from the most recent operation to subsequent operations of the same type.

Display surface/plane intersection

When Mastercam creates a surface toolpath, it creates "slices" of the surfaces based on parameters such as Maximum stepover. Select to view the slices Mastercam creates as it calculates the toolpath.

Automatically calculate HST defaults

Updates cut parameters and linking parameters when a new tool is selected. This only applies to 3D High Speed toolpaths. When deselected, cut parameters and linking parameters will not be updated until you click Recalculate values on the Tool page.

Enable Multi-threading

Activates multi-threaded toolpath processing. Mastercam divides programs that generate its largest, most complex toolpaths into smaller and separate program threads that execute simultaneously.

When you generate multi-threaded toolpaths, you can continue working in Mastercam once the toolpaths have been sent to the Multi-Threading Manager (MTM). You do not have to wait for the toolpath generation to complete before starting another Mastercam function. Mastercam continues to process these toolpaths in the background.

Lock feed rates

Locks your manual feeds and speeds. Deselect to allow Mastercam to reset the values when you edit the toolpath. These reset values can be based on the tool definition in the tool library, the selected material, or from values you defined in the Tool Setting tab of the Machine Group Properties dialog box or the Tool page of the Machine Group Setup function panel.

If you want to reset the values for your toolpath speeds and feeds for a single operation to their default settings, use the Re-initialize speeds & feeds right-click option. This option is available while you are in the tool list in your operation's toolpath dialog box.

Generate toolpath for new operations

When creating a new toolpath operation, Generate toolpath is selected by default.

Generate toolpath for existing operations

When editing an existing toolpath operation, Generate toolpath is selected by default.

NOTE

This does not apply to toolpath operations in files from previous Mastercam versions.

Wire

When selected, the Chain Manager displays immediately after you select a Contour wirepath.

Machining

Specifies how you want to select machining regions (surfaces, meshes, or solids) for a Surface Rough, Surface Finish, or 3D High Speed toolpath.

  • None: You are not prompted to select any machining regions during creation. To select machining regions, click Select on the Surface parameters tab for Surface Rough and Surface Finish toolpaths. This option is not applicable to 3D high speed toolpaths.
  • Prompt: Prompts you to select machining regions during toolpath creation.
  • All: Mastercam selects all surfaces, meshes, and solids in the graphics window for the machining regions.

CAD file

Specifies whether or not to prompt for a CAD file to import geometry into a Surface Rough or Surface Finish toolpath. Select one of the following options:

  • None: You are not prompted to select a CAD file during creation. To select a CAD file, click Select on the on the Surface parameters tab for Surface Rough and Surface Finish toolpaths. This option does not apply to 3D High Hpeed toolpaths.

  • Prompt: Prompts you to select a CAD file during toolpath creation.

    Using the Prompt option provides faster processing time. Mastercam extracts the information it needs to create the toolpath from the file, so that you do not need to retrieve the CAD file into Mastercam, which frees memory for toolpath creation.

Avoidance

Specifies how you want to select avoidance regions (surfaces, meshes, or solids) for a Surface Rough, Surface Finish, or 3D High Speed toolpath.

  • None: You are not prompted to select avoidance regions during creation. To select avoidance region, click Select on the on the Surface parameters tab for Surface Rough and Surface Finish toolpaths. This option is not applicable to 3D high speed toolpaths.
  • Prompt: Prompts you to select avoidance regions during toolpath creation.
  • Unselected: Uses any surfaces, meshes, or solids not previously selected (for machining regions) as avoidance.

Since not every toolpath supports avoidance geometry, Mastercam will not be able to generate them if you choose Unselected.

Containment

Specifies how to select the tool containment boundary for Surface Rough, Surface Finish, or 3D high speed toolpath.

  • None: Turns off prompting for a tool containment boundary during toolpath creation. To select a containment boundary click Select on the Surface parameters tab for Surface Rough and Surface Finish toolpaths. For 3D high speed toolpaths, click Select next to Boundary chains on the Toolpath Control page.
  • Prompt: Prompts you to select a tool containment boundary during toolpath creation.

Show toolpath/surface selection dialog

Automatically displays the Toolpath/surface selection dialog box when you create a Surface Rough or Surface Finish toolpath.

Delete regen files on exit

Deletes regen files that meet the deletion criteria when you exit Mastercam.

Keep regen files larger than (MB)

Keeps regen files that are larger than the specified size (megabytes).

Keep regen files not older than (days)

Keeps regen files that are newer than the specified number of days.

Keep regen files that took more time to create than (minutes)

Keeps regen files that took more time to create than the number of minutes specified here.

Percentage of physical memory

Sets an approximate maximum amount of memory that Mastercam can use, on average, while making lengthy toolpath calculations. This setting indicates a memory buffer size limit expressed as a percentage of your computer's physical memory.

This setting lets you optimize Mastercam's use of your computer's memory and disk space for toolpath algorithms that require a large amount of computer resources. For computers with small amounts of memory or those running multiple Mastercam sessions or other programs, you may wish to reduce the buffer size to use slightly less memory and slightly more disk space.

For computers with large amounts of memory or those running only Mastercam, you may wish to increase the buffer size to use slightly more memory and slightly less disk space.
