Page Setup dialog box

Use the Page Setup dialog box to set the defaults for print options and line width attributes. You can modify the printout and preview window using on-screen controls. Settings persist across Mastercam sessions.

The print preview shows the current viewport, which is whatever you see in the Mastercam graphics window. The wireframe or shaded print mode is based on the current display mode in Mastercam.

Unshaded Art base surfaces will not print.

On-screen Controls
  • Use your mouse wheel to zoom in and out in the preview window. Zoom is disabled when Keep scale is selected in Page Setup.
  • Click the left mouse button and drag to pan the geometry in the preview window.
  • Grab and drag the page preview margins to adjust them. The image scales around the center of the page as you drag. You can also enter values in the Margins fields.
Parameters

Margins

Sets the minimum distance between each edge of the page and the page content. You can set individual margins for Left, Right, Top, and Bottom. You can also adjust margins by dragging them in the preview window.

Line width

  • Use entity: Print using each entity's native line width.
  • Uniform line width: Sets a uniform line width for all printed entities. Type a number between 1 and 5 pixels in the text box. If your printer has a high density (600 dpi or more), you may need to use a larger value to produce a noticeable difference.
  • Color/Width map: Select to enable the Color/Width map button, which opens the Linewidth Attributes dialog box, where you can map colors to specific line widths for printing.

Options

  • Color: Select to support a color printer. This default setting may be overridden by your printer's driver properties.
  • Name/Date: Print the file name and date header on each page.
  • Screen information: Includes visible screen information such as plane name, coordinate axes, scaling information, and so on. This option works in conjunction with those on the Printing page and Screen page of the System Configuration dialog box.

Scaling

  • Fit/Center: Centers the geometry within the margins of the preview window.
  • Keep scale: This option works in conjunction with Fit/Center. If you have selected Keep scale, the geometry is centered but not fit to the preview window. If you have deselected Keep scale, the geometry is centered and fit to the preview window.
  • Scale: You can enter a value in the text box between 0.00001 and 10000.0 to resize the entities in the preview window.
  • Dashed line scaling: Specifies the proportion of the dashed lines on the screen to those in the printout. Type a percentage in the % of screen size box. This option works in conjunction with those on the Printing page and Screen page of the System Configuration dialog box.
