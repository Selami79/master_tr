Mastercam Shortcut Keys

Mastercam provides both predetermined shortcut keys (hotkeys) and the ability to custom assign your own shortcut keys through the Customize Ribbon page of the Options dialog box (choose File, Options, Customize Ribbon, and then click Customize).

The following list contains most of the default Mastercam shortcut keys:
