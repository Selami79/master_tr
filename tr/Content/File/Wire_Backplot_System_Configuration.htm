Wire Backplot page: System Configuration

Use the Wire Backplot page to set parameters that affect how the wire moves and appears on the screen during Classic Backplot.

You can set some general settings such as the step speed/increments, the screen cleanup, and the simulated wirepath display. You can also set up the motion colors and color loop options.

NOTE
  • The parameters you choose are for display purposes only and do not affect the part when it is machined.
  • To change Wire backplot settings during the current Mastercam session, backplot an operation, then, in the Backplot Options dialog box, choose Options.
Parameters

Fast steps/sec:

Sets the speed for "turbo" mode. Turbo mode occurs when you click on the Step forward or Step back buttons on the playback bar and hold the left mouse button down.

Endpoints

Moves the tool from endpoint to endpoint along the toolpath while backplotting.

Interpolate

Moves the tool the specified distance between points along the toolpath while backplotting. You must enter a Step increment value if you choose this option.

Step increment

Sets the distance that the tool moves between points. Enable by choosing Interpolate.

Each pass

Erases the backplot display from the graphics window after each pass.

Each operation

Erases the backplot display from the graphics window after each operation.

Fit wirepath

Displays the entire toolpath in the graphics window when you begin a backplot. Does not apply when restarting a backplot.

Show wire compensation vectors

Displays wirepath compensation vectors during backplot. The vector color can be set in the Backplot options or as a default in System Configuration.

Simulate wire compensation

Displays a simulation of the compensated wirepath when compensation is applied in the control. (Compensation type is set on the second tab of the Wire parameters dialog box for applicable wirepath types.) This simulation occurs when G41/42 codes are found in the wirepath. In the simulation, Backplot draws the wirepath as a centerline, while the compensated path represents the edge.

By default, the compensated path is shown in gray, unless Color Loop is active.

Color (Wire)

Sets the color of the wire display during backplot. Enter a number for the color or choose the Color button to select a color.

Plain (Wire Guide)

Displays a wireframe representation of the wire guide during backplot.

Shaded (Wire Guide)

Displays a solid representation of the wire guide during backplot.

Color (Wire Guide)

Sets the backplot display color of the wire guides. Enter a number for the color or choose the Color button to select a color.

Material (Wire Guide)

Enabled when Shaded is selected. Displays a solid representation of the wire guide during backplot. Also enables the Material option.

Active

Determines the opacity of the wire and wire guides as Mastercam performs the backplot. The lower the opacity, the more the geometry and wirepaths show through the wire and guides. When opacity is set to its highest value of 1.0, the wire and the wire guides are fully opaque, appearing solid. When opacity is set to its lowest value of 0.01, the wire and guides are translucent, appearing nearly transparent.

Color loop

Select to cycle through a series of six colors, one for each wirepath pass or operation, when backplotting. Select from the following two choices:

  • By pass: Select to use a different color each time Mastercam backplots a different pass.
  • By operation: Select to use a different color each time Mastercam backplots a different operation.

Rapid

Choose to display rapid moves during the backplot in the color you select. Enter the number of the color to display rapid moves during backplotting or choose Color to select a color from the Select color dialog box. The color you choose for rapid moves is shown in the box to the right of the color number.

Linear feed (XY Motion)

Choose to display XY linear feed moves during the backplot in the color you select. Enter the number of the color to display the XY linear feed moves during backplotting or choose Color to select a color from the Select color dialog box. The color you choose for XY linear feed moves is shown in the box to the right of the color number.

Choose to display XY clockwise arc feed moves during the backplot in the color you select. Enter the number of the color to display the XY clockwise arc feed moves during backplotting or choose the Color button to select a color from the Select color dialog box. The color you choose for XY clockwise arc feed moves is shown in the box to the right of the color number.

Enter the number of the color to display the XY counterclockwise arc feed during backplotting or choose Color to select a color from the Select color dialog box. The color you choose for XY counterclockwise arc feed moves is shown in the box to the right of the color number.

Enter the number of the color to display the UV linear feed moves during backplotting or choose Color to select a color from the Select color dialog box. The color you choose for UV linear feed moves is shown in the box to the right of the color number.

CW arc feed (UV Motion)

The color you choose for XY clockwise arc feed moves is shown in the box to the right of the color number.
