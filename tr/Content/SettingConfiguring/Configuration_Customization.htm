Configuring and Customizing Mastercam

Within Mastercam, you can use the System Configuration dialog box (File, Configuration), the Options dialog box (File, Options) and the Mastercam Advanced Configuration dialog box (Advanced Configuration from the Mastercam 2025 Start menu) to configure how Mastercam looks and acts under certain conditions.

System Configuration

Use the System Configuration dialog box to review and set Mastercam startup default values and to manage the configuration files that store these values. Mastercam settings are stored in a configuration file (*.config), which typically resides in a CONFIG directory under the main Mastercam directory. When you restart Mastercam or initialize operations (for example, open a new file), Mastercam restores the default settings that have been saved in the Mastercam configuration file.

There are standard configuration files for both U.S. and metric units. You can also create custom configuration files to meet specific workflow requirements, and merge configuration files. Although there can be multiple configuration files, each containing different values, only one is current and active at a time.

NOTE

Configuration files may be stored and accessed from any directory you choose, even on a network. However, multiple Mastercam installations can share a configuration file only if the paths stored in the *.config file are valid for each installation.

Setting default values

The System Configuration dialog box consists of property pages that give you access to groups of related default values and settings. To access a property page, click its name in the list of configuration topics on the left. When you change a setting on a page, a check mark displays in front of its name so that you can track where you have made changes during an editing session.

Managing configuration files

Changes made to configuration settings are enacted only for the current session unless you save them to a configuration file. In addition to saving your settings in a configuration file, you can create new configuration files, merge configuration files, and change the current configuration file using functions along the bottom left of the dialog box.

Mastercam Options

Use the Options dialog box to customize Mastercam's interface and appearance.

Customizing ribbon tabs and menus

Mastercam makes it easy for you to customize the interface so that the functions that are most important to you are always easily available.

  • Customize the Quick Access Toolbar.
  • Add your own tabs and commands to the ribbon.
  • Edit the graphics window context menu.
Creating custom keyboard shortcuts

Mastercam provides both predetermined shortcut keys (hotkeys) and the ability to create your own shortcut keys through the Options dialog box.

Once you map a commonly used function to a custom keyboard shortcut, you can use that shortcut to open the function rather than select a function from a ribbon with the mouse.

The combinations of key sequences you can use are listed below. “Key” is an keyboard function key F1 through F12.

  • Alt+Key
  • Ctrl+Key
  • Ctrl+Alt+Key
  • Shift+Alt+Key
  • Shift+Ctrl+Key
  • Shift+Ctrl+Alt+Key
NOTE

You cannot map functions to key sequences using a single alphanumeric key or one in combination with a [Shift] key. These keys are reserved for use by Mastercam default keyboard shortcuts. For example, many of the single alphanumeric keys or [Shift+Key] sequences are used with data entry, function panel, and dialog box shortcut keys.

The Mastercam Advanced Configuration dialog box lets you control several advanced options and features. For more information about the specific options and how they affect Mastercam, refer to the Mastercam Advanced Configuration dialog box Help topic.
