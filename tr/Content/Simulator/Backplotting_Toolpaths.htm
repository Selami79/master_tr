Backplotting Toolpaths with Mastercam Simulator

NOTE

This procedure is for Mastercam Simulator's Backplot mode. For Classic Backplot, refer to Backplotting with Classic Backplot.

Backplotting shows the path the tools take to cut your part. This display lets you spot errors in the program before you machine the part. You can backplot your toolpath using Mastercam Simulator by following the steps below.

  1. Toolpath your part.

  2. Select Simulator options on the Toolpaths Manager or Files under Properties in the Toolpaths Manager depending on your machine type.

  3. Set any of the necessary options. Click OK once you are satisfied.
  4. Select the toolpaths you want to backplot, and select Verify selected operations on the Toolpaths Manager. A window shows the progress as your NCI data loads. You do not need to wait for the progress bar to be complete to start your backplot.

  5. On the Home tab of Mastercam Simulator, select Backplot.

  6. Use the playback bar along the bottom of the window to move forward and backward. You can also adjust the Precision slider. When you are done, keep the Mastercam Simulator window open. You can move this window to secondary monitor.
  7. If necessary, adjust your toolpaths in Mastercam.
  8. Click Verify selected operations on the Toolpaths Manager again to update the toolpaths in the Mastercam Simulator window. Mastercam updates only the changes in the toolpaths.
NOTE

If you select dirty operations for backplotting, the Warning: Dirty Operations Selected dialog box displays. In this dialog box, you can specify whether to regenerate all dirty operations before simulation or to proceed with the dirty operations as is.
