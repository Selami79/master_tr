Simulator Options dialog box

Use the Simulator Options dialog box to set options before running Mastercam Simulator for wire or lathe machines. To set simulator options for mill or router machines, refer to the Machine Group Setup panel: Mill/Router Help topic.

NOTE
This topic does not apply to Mill-Turn.
Components

This page contains options for stock, stock boundaries, and fixtures.

Stock setup

Uses stock boundary information from the Stock Setup page to determine stock boundaries.

Scan toolpaths

Determines stock boundaries from tool movements.

Use stock setup values

Uses stock boundary information from the Stock Setup page to determine stock boundaries.

Pick stock corners

Returns you to the graphics window to pick the corners of the stock from the current geometry.

Box

Displays the stock in a rectangular box shape.

Cylinder

Displays the stock in a cylindrical shape.

Cylinder axis

Sets the center axis of the cylinder to the X, Y, or Z axis.

Cylinder diameter

Sets the cylindrical stock diameter.

Center on axis

Rotates the stock and toolpath around the selected center axis.

Solid/Mesh

Uses a solid model or mesh in the current part file as the stock. Click Select to return to the graphics window to select the solid or mesh.

File

Use a solid model from an STL file as the stock. Select a file from the drop-down list or choose Select to navigate to another folder and search for files.

Stock model

Uses a stock model operation from the drop-down. Mastercam lists all stock model operations defined in the current part file.

Fixtures

Enables the use of fixtures in Mastercam Simulator. Fixtures can be used for collision checking.

Levels

Sets the levels on which the fixtures are located.

Solids/Meshes

Uses solids, surfaces, or meshes in the current part file as the fixtures. Click Select to return to the graphics window to select the solids, meshes, or surfaces.

File

Uses a selected file as the fixture shape.

Restore settings

Resets settings in this tab to their default values.

Data

This page provides options for compensation and tolerances.

Cutter compensation in control

Allows you to view simulated cutter compensation during the verification process, if you selected cutter compensation in control in the toolpath.

True thread mode

Simulates accurate threads. Deselect to display threads as concentric circles and grooves, which allows Mastercam Simulator to run faster but less accurately.

Curve tolerance

Sets the tolerance for tool simulation when following a curve.

STL tolerance

Sets tolerance for saved STL files. This option also affects your tool display in Mastercam Simulator. A larger value displays a coarser tool.

Workpiece tolerance

Adjusts the quality of the workpiece displayed during simulation. A smaller tolerance value produces a smoother workpiece. You can also use this option to match the workpiece tolerance to the stock tolerance when comparing your verification results.

Restore settings

Resets settings in this tab to their default values.

Simulation

This page provides options for the machine and posting.

Machine

Select the machine to be simulated. Note that the only machines available are those in the \Shared Mastercam 2025\machine simulation\machsim folder. If no machine is selected, the machine set in the Simulation tab of the Machine Simulation dialog box will be used.

NOTE

For more information on adding a new machine, refer to the Adding a New Machine Help topic or contact your local Mastercam Reseller.

Post settings

Opens the Machine Simulation dialog box, where you can set up post options before running Mastercam Simulator.

Use external post

Uses an external post instead of a generic post when simulating toolpaths. By using your own posts, you can view a more realistic simulation. The selected Machine must contain a *.pst file.

NOTE

To modify your post, you must have a license to do so. Contact your Mastercam Reseller for more information.

Position

Allows you to transform the geometry a specified amount on each axis. You can enter the X, Y, and Z translation values manually or by selecting from the screen. Set to Automatic to allow Mastercam to handle the position.

Transform into WCS

Positions the workpiece to match the Working Coordinate System settings entered in the operation's Planes or Toolpath Coordinate System dialog boxes.

Arc curve tolerance

Enter the tolerance for arc curves when in Simulation.

Tool Center Point Control

Enables a linear interpolation around the tool tip position or around the rotational spindle head. Use this option when there are large jumps in the rotational axis or singularities in the code that cause faults.

Maximum distance

Sets the maximum segment length to create interpolated points between calculated moves.

Maximum angle change

Sets the maximum angle difference for interpolated moves between calculated moves.

Restore settings

Resets settings in this tab to their default values.
