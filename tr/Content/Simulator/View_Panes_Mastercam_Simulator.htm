View Panes: Mastercam Simulator

Mastercam Simulator provides various ways to analyze your simulation. By using the panes located on the View tab in the Show group, you can view the collision report, view more detailed information on the toolpath, compare stock, and use the axis control. Click the drop-downs below to learn more about a specific pane.

Report

The Report pane lists collisions, proximity alerts, tool orientation, and axis overruns found in the current simulation. Proximity alerts and axis overruns only display in Simulation mode. Tool orientation only displays when simulating a mill-turn machine using the Sync Manager. Collisions display in all Mastercam Simulator modes.

You can filter the generated report by using the icons at the top of pane so you can see only collisions, proximity alerts, axis overruns, or a combination. You can also use the Save Report button to export the results to a CSV file. The generated report includes the moves, the event, the objects that caused the event, and the range.

NOTE

Modify the collision checking, proximity alerts, and axis overrun options in the Options dialog box, accessed by selecting File, Options in Mastercam Simulator.

Move Info

The Move Info pane displays information about the section of the current toolpath in the simulation. This information includes the Operation Name, Operation Number, Feed Length, Total Time, Spindle Speed, Coolant, and many more.

Move List
NOTE

This pane is only available for Mill-Turn.

The Move List pane shows the NC code of the current simulation. The displayed values match the absolute machine coordinates, such as X, Y, and Z (in the case of 3-axis machining) and A, B, C, and so on in the case of 4- or 5-axis machining.

  • Use the slider bar on the right side of the tab to scroll through the moves. Select a block to move the tool to that position.
  • Overrun Axis Limits will display with an exclamation mark and collisions will display with a red cross
  • Right-click the columns and select Settings to set the number of significant digits and to show text colors based on toolpath analysis options.
Axis Control

Axis Control is only available when using Simulation mode inside Mastercam Simulator.

The Axis Control pane manually moves the linear and rotational axes of the machine. Use the sliders and buttons to move the axes, spindle/turret, and the jog amount, similar to how you would on an actual machine. This is useful when checking if the machine limits or collision checking has been set correctly. Use Reset All to return all options to their default setting.

Icon Increment Amount
Small Increment: Increases or decreases the Jog by 1.
Medium Increment: Increases or decreases the Jog by 10.
Large Increment: Increases or decreases the Jog by 100.
Relative: Sets the axis values relative to the home position.
Machine

The Machine pane contains the machine definition of the current simulation. The display shows the hierarchical representation of the machine and how machine parts are defined.

The icons shown in the Machine pane are described as follows:

Icon Description
Translation axis
Tool object
Rotation axis
Workpiece object
Axis transformations
Stock object
Geometry objects
Initial stock object
Toolpath objects
Collision definition sets
Compare

Uses the entered Stock to leave and tolerance options to compare your verification or simulation results to the workpiece or a selected STL file to check accuracy. The part model is then colored according to the amount of the stock, if any, left on the model above or below tolerances that are specified in the Compare dialog box.

For more information on comparing Verify and Simulation results, see Comparing and Analyzing Verification/Simulation Results.

Timeline Zoom

The Timeline Zoom pane allows you to zoom into the simulation timeline to see more or less detail. This is especially helpful when using Bookmarks that may be close together. Use the Timeline Zoom slider, on the bottom left corner of Mastercam Simulator, to zoom in and out of the timeline.

The Timeline, on the bottom right corner of Mastercam Simulator, displays the total timeline of the simulation. The highlighted section displays the section you are currently zoomed into.

If you toggle Timeline Zoom off, the simulation will return to its default setting. If you toggle Timeline Zoom back on, it will remember your zoom setting, and apply it based on the current location of the simulation.

Toolpath Analysis

Toolpath Analysis is only available when using Simulation mode in Mastercam Simulator.

The Toolpath Analysis pane analyzes changes to toolpaths by applying color schemes. You can choose from the following options:

  • Operation: Colors each toolpath differently.
  • Tool: Colors toolpaths when different tools are being used.
  • Feed Rate: Colors toolpaths by feed rate and rapid moves.
  • Segment Length: Colors toolpaths based on the length of the segments.
