Options dialog box: Mastercam Simulator

Use the Options dialog box to customize Mastercam Simulator's appearance and functionality.

If you make changes in the Options dialog box, those changes only persist for the current Mastercam session. To make changes to your defaults that persist between sessions, see Setting Mastercam Simulator's Default Appearance and Functionality.

General page

Overlay control opacity

Controls the opacity of the playback bar. Use the slider, or enter a percentage.

Number of moves before / Number of moves after

Enter a number of moves before and after to control how much of the toolpath displays when you select Segment on the Backplot or Verify tab.

Collision tolerance

Determines how close two objects can get before being considered a collision. View reported collisions in the Report pane.

Proximity alert distance

Only available in Simulation mode. Determines how close objects can get before reporting an alert. View reported alerts in the Report pane.

Simulation engine

Select either the 3-axis or 5-axis simulation engine to see alternate results for multiaxis motion. The 3-axis engine is used when the selected toolpaths and the selected tool plane is set to the WCS Top plane, and if the toolpaths are not using 5-axis motion (G11).

Always use 5-axis engine

Defaults to the 5-axis engine each time you use Mastercam Simulator.

Interpolation step

Specifies the distance between points when Interpolate Mode is on.

Adaptive quality

Processes and displays the result in a lower resolution and, once complete, continues to process at a high resolution, resulting in highly defined areas, especially in rounded areas.

Enable auto start

Causes Mastercam Simulator to play your verify session when the window opens and rough data has been transferred into Mastercam Simulator.

Graphics page

  • Deslect the Use Mastercam colors to set your own colors for Mastercam SImulator.

  • Click the drop-down arrow next to any component to select a different color for that component.

  • Use the Opacity slider to set the translucency for each component.

  • Since Multiaxis toolpaths create a tool vector for every position in the toolpath, shortening the 4-5 Axis Tool Vector Length can simplify display. The length is for display only and does not change the tool vector length set in the Multiaxis toolpath parameters.
