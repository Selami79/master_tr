Comparing and Analyzing Verification/Simulation Results

You can compare your verification or simulation results to the workpiece or a selected STL file (a 3D model file) to check accuracy by using Compare or Smart Measure to analyze the simulation.

Using Compare

When using Compare, Mastercam Simulator colors the part model according to the amount of stock left on the model beyond the tolerances you specify in the Compare dialog box.

  1. Run through your toolpath verification or simulation.
  2. Click Compare on the Verify or Simulation contextual tab.

  3. Click Refresh on the Compare dialog box to run the comparison. The verification results are compared to the workpiece by default.

You can also set the following options in Compare

  • Click on any Start or End field to manually enter a new value for each comparison level.
  • Select colors for each level of remaining stock.

  • Click Show stock or Show model to change the comparison display.

  • Enter a Stock to leave value to compare your verification results to an offset version of the workpiece or STL file. Click Refresh to update the comparison results.

  • Select an STL file for comparison (instead of the workpiece) by clicking the drop-down box and clicking Browse.

  • Click Options to adjust the increment between the colors. Enter a Tolerance in the Compare Options dialog box and click OK to update the comparison results.

Using Smart Measure

Use the Smart Measure function to analyze and measure points and dimensions in Mastercam Simulator. Smart Measure is located on the contextual tabs for Verify and Simulation mode.

NOTE

If 3-axis is selected as the Simulation engine (set in the Mastercam Simulator Options dialog box) , you only have access to Point and Distance. To access all Smart Measure functionality, Simulation engine must be set to 5-axis.

Use the Label Display drop-down to toggle the display of pop-ups and points in the graphics window. Depending on the type of Dimension selected, the pop-up displays a combination of the following:

Label Description
Distance (X, Y, Z) Distance between selected points in X, Y, and Z.
Distance (3D) Distance between the selected points in three-dimensional space.
Relative to Indicates the relative operation number.
Point 1 The location of the first selected point.
Point 2 The location of the second selected point.
Deviation

Indicates the shortest distance between the cut geometry and the underlying workpiece geometry.

NOTE

Workpiece geometry is geometry that is not defined as a stock model or a fixture, and is visible upon opening Mastercam Simulator.

Collision Indicates if a collision is detected or not.
Tool number Indicates the last tool used to machine the selected point.

Distance

Measures the linear distance between two placed points, two selected features, or the X, Y, and Z distances between two placed points.

Curve

Measures the radius of a curved edge or face.

Point

Measures the X, Y, and Z location of a point.

Angular

Measures the angle between features.

When you toggle Smart Measure off, the points and labels still display in Mastercam Simulator. However, you cannot add any new points. You can also select and drag the pop-up windows so the information is not in the way of your part.

To remove any individual measurements, click its associated pop-up window. Smart Measure must be toggled on. To remove all created measurements, select Clear All.
