Using Mastercam Simulator

The backplot, verify and simulation functions in Mastercam Simulator allow you to check your toolpath results before sending them to your machine tool.

  • Backplotting shows the path the tools take to cut your part.
  • Verifying your toolpaths allows you to use solid models to simulate part machining against a selected stock definition.
  • Simulating your toolpath allows you to prove out Mill, Mill-Turn, or Router toolpaths, by showing you the machine on which the toolpath will be cut.
NOTE

If you select dirty operations before launching Mastercam Simulator, the Warning: Dirty Operations Selected dialog box displays. In this dialog box, you can specify whether to regenerate all dirty operations before verifying or to proceed with the dirty operations as is.

Backplotting with Mastercam Simulator

Backplotting shows the path the tools take to cut your part and helps you spot errors in the program. As you move through the backplot, information about the current toolpath motion displays in the Move List.

The following procedure shows how to backplot with Mastercam Simulator.

  1. In Mastercam's Toolpaths Manager, select the operations you want to backplot.

  2. Select Verify selected operations.

    Mastercam Simulator opens.

  3. In Mastercam Simulator, click Backplot.

    Mastercam Simulator switches to Backplot mode.

  4. Click the Play button to start backplotting.

After backplotting, you can use the options on the Backplot contextual tab to change your view.

  • Click Quick Verify to display a shaded path the width of the tool for Mill, Router, and Wire toolpaths. For Lathe toolpaths, this option displays the lathe tool removing the cut material.
  • Use Restrict Drawing and Stop Restrict Drawing to isolate and zoom in on particular toolpath sections. Restrict Drawing removes all currently backplotted toolpaths from the graphics window. The remaining toolpaths then become the only ones displayed. You can click this button repeatedly to focus on specific backplot results. Stop Restrict Drawing displays all backplotted toolpaths.
  • Click anywhere along the backplotted toolpath to jump to that section of the toolpath.
Verifying with Mastercam Simulator

Verifying your toolpaths allows you to use solid models to simulate part machining against a selected stock definition. The result represents the surface finish, and shows collisions, if any exist. You can identify and correct program errors before they reach the shop floor.

You can verify the selected operations, add operations to the current verification session, or verify the selected machine group.

NOTE

For 2D and 3D Mill toolpaths, you can make it easier to differentiate between toolpath entities such as entry motion, transition motion, vectors, and endpoints, by changing their color and display style in the Colors, Advanced Toolpath Display page in System Configuration.

The following procedure shows how to verify with Mastercam Simulator.

  1. In Mastercam's Toolpaths Manager, select the operations you want to backplot.

  2. Select Simulator options (wire or lathe machines) or Files under Properties (mill or router machines).

  3. Set any of the necessary options. Click OK once you are satisfied.

  4. Select Verify selected operations in the Toolpaths Manager. A window shows the progress as your NCI data loads. You do not need to wait for the progress bar to be complete to start your verification.

  5. Click the Play button to start verifying.

You can also use the Verify selected operations drop-down to choose Add to current verification.

There are restrictions when adding an operation to a current session.

  • Only operations positioned below the selected operations in the Toolpaths Manager can be added to the current session.

  • Operations that depend on a previous operation cannot be added unless the parent operation is also selected.

You can further refine your results by using the options discussed in the Refining Your Verification/Simulation View Help topic. You can compare your verification or simulation results to the workpiece or a selected STL file by following the procedures in the Comparing and Analyzing Verification/Simulation Results Help topic.

Simulating with Mastercam Simulator (Mill-Turn)

Simulating your toolpath proves out Mill-Turn toolpaths by allowing you to see the machine that it will be cut on. It helps detect collisions between your stock, tools, holders, and any machine components before sending any code to your machine tool. Simulation inside Mastercam Simulator is a light-weight version of the stand-alone Machine Simulation.

The following procedure shows how to simulate with Mastercam Simulator.

  1. Create your toolpaths in Mastercam.
  2. Click the Post selected operations button.

    Mastercam opens Sync Manager.

  3. Create any required syncs and set Sync Manager options.

  4. Click the Launch button in the Sync Manager.

You can further refine your results by using the options discussed in the Refining Your Verification/Simulation View Help topic. You can compare your verification or simulation results to the workpiece or a selected STL file by following the procedures in the Comparing and Analyzing Verification/Simulation Results Help topic.

Simulating with Mastercam Simulator (Mill or Router)

Simulating your toolpath proves out Mill or Router toolpaths by allowing you to see the machine that it will be cut on. It helps detect collisions between your stock, tools, holders, and any machine components before sending any code to your machine tool. Simulation inside Mastercam Simulator is a light-weight version of the stand-alone Machine Simulation.

The following procedure shows how to simulate with Mastercam Simulator.

  1. In Mastercam, select Files under the machine group in the Toolpaths Manager.

    The Machine Group Setup function panel displays.

  2. Select the Simulation page.
  3. Select the machine configuration from the Machine drop-down list. This list may vary if you have added machines to Mastercam.
  4. Set the Position, Post settings, and Arc curve tolerance if necessary.
  5. Click OK to save your changes and exit the panel.

  6. Select Verify selected operations on the Toolpaths Manager.

    Mastercam Simulator displays.

  7. On the Home tab, select Simulation.

  8. Click the Play button to start simulating.

You can further refine your results by using the options discussed in the Refining Your Verification/Simulation View Help topic. You can compare your verification or simulation results to the workpiece or a selected STL file by following the procedures in the Comparing and Analyzing Verification/Simulation Results Help topic.

Once Mastercam Simulator is launched, you can use additional features to improve your experience. The available functions depend on the current Mode of Mastercam Simulator. Click the links below to learn more.

  • Capture/Replay: Move through the toolpath motion to see material being added and removed.

  • Saving Stock as an STL file: Save your final verification or simulation results as an STL file.

  • Using Bookmarks: Set bookmarks to refer back to various points of the simulation.

  • Creating a Presentation: Save a stand-alone executable that can be used to show a simulation on any computer.

  • Saving a Recording: Save a video of your simulation session in .mp4 format.

  • Setting Stop Conditions : Pause your simulation at specific points.

  • Setting up Collision Checking: Detect collision between your stock, tools, and machine components during the playback.
