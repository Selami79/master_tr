Custom Texture dialog box

Use the Custom Texture dialog box to apply a custom textures operation to closed contours you select in the graphics window. You can apply the texture to the inside and outside of one or more chained boundaries, as well as between nested chains.

A custom texture combines a base pattern file with a tiled texture style, such as a grid or tessellation. Base pattern files are Art Surface Library files (.asl) that were created from selected areas of Art models using the Copy Art Surface Area to Library function.

In the Custom texture parameters dialog box, you choose a pre-defined texture style that is already associated with an .asl file.

Select Browse to use a different base pattern file with the selected texture style. Other options in this dialog box let you control the spacing, rotation, scaling, and randomization of your custom texture.

After modifying custom texture parameters to achieve a specific pattern, simply save your settings to a preset file so that you can use them again.

All of these parameters work together to produce the custom textures Art surface operation. Try using the default settings for a standard Texture Style first (either using the default .asl file or replacing it with your own), and view the resulting effect. Then use the Art Manager to edit the operation's parameters, and regenerate the Art model. Repeat until you get the desired result.

Parameters

Switch to advanced parameters

Opens the Advanced Parameters dialog box, where you can review and set additional options for the Art surface operation.

Presets

  • Opens a list of options to:
  • Automatically load the most recently used parameters.
  • Save your current dialog box settings to a file (.aps) in your Shared Mastercam 2025 folder.
  • Load a set of previously saved parameters by selecting it from the list.
  • Delete a set of saved parameters.
  • Reset all dialog box parameters to their initial default values.

Preset files are stored locally in a fixed directory structure (..\Art\Presets) in your Shared Mastercam 2025 folder.

NOTE

Networked colleagues can access your preset from your Shared Mastercam 2025 folder. If you send your preset files to Mastercam Art users who do not have access to your network, make sure to let the recipients know the correct folder where they should paste the file in their Mastercam installation. Your recipients can use Windows Explorer copy and paste functions to add your files to their local machine. Their file structure must match yours.

Texture style

A list of predefined custom textures; each one is paired with a default Art Surface Library file (.asl) to create the pattern illustrated by the associated icon. Choose a texture from the list that best approximates the type of pattern you want to use or modify. Then edit the default parameters or replace the base pattern file as necessary to create your own design.

NOTE
  • Use the Resize Texture % parameter to adapt a custom texture to the correct scale.
  • Use options in the Presets field to save the parameters that define your texture so you can easily reuse it.

Base pattern file (Read only)

Displays the path and filename of the Art Surface Library file (.asl) used in the custom texture. Use the Browse button to select a different .asl file. For more information on creating .asl files, see Working with Art Surface Library files.

Browse (.asl)

Click to access the Open dialog box where you can choose a new Art Surface Library file (.asl). The selected file is used as the base pattern file for the Art surface operation.

Scale XYZ (base pattern element)

Use these fields to scale the base pattern element independently in the selected axis by the percentage you enter.

Best practice: The sample Art Surface Library files (.asl) used as default base pattern files were created in a standard  1X1 / 25.4mm cube. When creating your own base pattern files, create them at a consistent size. This makes them easier to scale and space in the Create Textures Custom Art Surface Operation function and achieve the desired results.

Spacing XY

Sets the spacing between the base elements in an array, from center to center in the selected axis.

NOTE

To achieve no overlap of the ellipses when defining a handcrafted texture, set the spacing in XY equal to or greater than the specified Radius XY parameters.

Resize texture %

Provides a quick way to proportion your texture by adjusting the overall size of the element used in the texture (base pattern file or ellipse) by the percentage you enter.  Say you select the Texture Style Stone and the stones are too big or too small for your application. Simply adjust the percentage in this field, and your texture is resized accordingly.

Rotation angle (texture / pattern)

Rotates the entire texture / pattern about the center by the specified angle. When used in a Pattern on Contour Art surface operation, the element (.asl file) is rotated by this angle before it is placed in the pattern.

If you enter a positive number, the rotation is in a counterclockwise direction; negative numbers rotate the texture / pattern in a clockwise direction. By selecting the Relative checkbox, the base element will rotate as well.

Relative

(Applied only when a rotation angle is entered.) Select to rotate the individual base elements in the texture / pattern by the specified rotation angle. When used in a Pattern on Contour operation, the element (.asl file) rotates and stays normal to the contour.

Randomize placement

Mastercam Art offers 100 different numbers that generate the random placement of base pattern elements. Any number, from 1 to 100, that is entered into the field will create a different, randomized pattern that will not change upon regeneration. Entering zero into the field will disable the parameter.

NOTE
  • The size of the number (between 1 and 100) has no effect on the amount of randomization. For example, entering either 5 or 10 will a produce unique result, but both numbers will initialize the same amount of randomization.

  • If you randomize and there is too much spacing in the resulting pattern, adjust the Spacing X and Y parameters to smaller values. Or create another texture on top of the existing texture using the same boundary. Use the same parameters but enter a different number in the Randomize Placement field. The new texture will fill in the original texture in different areas. Another way to fill blank areas is to manually paste the base element into the texture. Use the Paste from Library command to select the element from the Art surface library, then scale, rotate, and place it into position. If you are creating the texture on a flat plane, use the application style Blend to keep the base pattern elements from stacking upon one another.

Base height

Adds a vertical extension / foundation along the Z axis to the start of the cross section when creating the Art surface operation. The base height raises or lowers the surface by the specified amount, creating a vertical wall. Enter a positive number to raise the Art surface, or enter a negative number to lower it. The following example use the same cross section shape to illustrate the effects of the base height parameter.

Base pattern variation

Use this function to alter a custom texture by randomly scaling each base pattern element within a controlled range. Choose a scaling type to control the variation of the pattern. Different scaling types allow or restrict variations along the X, Y, and Z axes.

Parameters are expressed as a percentage (0% to 1000%) of the original values. The dimensions of each pattern element will vary up to the percentage entered in the field.

Scaling type: Off

The Variation X, Y, and Z fields are disabled, so there is no variation in scale between base pattern elements.

Scaling type: Scale XYZ

Varies the scale of base pattern's elements up to the percentage entered. All axes are scaled equally, maintaining the aspect ratio of each element.

Scaling type: Scale XY, Z locked

Varies the scale of base pattern's elements up to the percentage entered. The X and Y axes are scaled equally, while the value of Z remains unchanged.

Scaling type: Scale XYZ independent

Varies the scale of base pattern's elements up to the percentage entered. Each axis (X, Y, and Z) is scaled independently.

Variation XY

Varies the spacing of the base pattern elements in X and Y up to the percentage entered.  If you enter a value of 25, then each element in the base pattern could be translated up to 25 percent with each placement.

Rotation variation

Varies the rotation of the base pattern elements up to the percentage entered.  If you enter a value of 25 (degrees of rotation), then each element in the base pattern could be rotated up to 25 degrees with each placement.

Shift texture XY

Shift the entire texture independently in the X and Y axis. You can move the texture up by the distance of the base element plus the spacing amount. This parameter can help you align the texture with another design element.

Application style

Defines how a new surface operation shape interacts with existing shapes and with the Art base surface (ABS). The icons for each application style option illustrate how the Art surface operation is created (stacked) in relation to the existing Art model. The blue ellipse represents an existing surface. The red bar represents the new surface that will be applied and demonstrates what the specified application style will do when applied to the blue ellipse. The red bar extends beyond the blue ellipse to illustrate how the shape will react to a flat ABS.

NOTE

You can edit the Art surface operation at any time to change the application style and other parameters--including the geometry used to define the Art surface boundaries. Simply regenerate the Art model to view the results. You can also use Mastercam Art's Undo and Redo options to restore or discard changes to the Art model.

Add

Adds the red bar to the Art model and then "grows" on top of the blue ellipse, following its shape. Note the intersection of the red and blue shape.

Sub

Subtracts the red bar from the Art model, and cuts the red shape out of the ellipse, following its shape.

Blend

Adds the red bar to the Art model. When it intersects the blue ellipse, the highest parts of each shape are displayed, resulting in a blend.

Add cut

Adds the red bar to the Art model and ignores the blue ellipse, cutting through it.

Add cut trim

Adds the red bar to the Art model, cuts through the blue ellipse, and then trims the red shape to the boundary of the ellipse.

Blend edge

Adds the red bar to the Art model, cuts through the blue ellipse, trims the red bar to the boundary of the ellipse, and then blends the edge of the red bar to the shape of the blue ellipse. With this function, you can also control the length of the blend.

Sub cut

Subtracts the red bar from the Art model and ignores the blue ellipse, cutting through it.

Inverse

Applies the inverse of the operation by flipping the operation in the Z axis inside the selected drive contour.

Mask

Isolates the selected elements. In the field graphic, the red bar was masked, and the blue base shape would not be affected in the operation.

NOTE

Mask elements on an Art surface operation to add a texture to masked elements while avoiding non-masked elements.
