Handcrafted dialog box

Use the Handcrafted dialog box to apply a handcrafted textures operation to closed contours you select in the graphics window. You can apply the texture to the inside and outside of one or more chained boundaries, as well as between nested chains.

All handcrafted textures are built from arrayed ellipses that you modify using the fields in this dialog box. The ellipses can be stretched, overlapped, randomized, and offset to make many new and interesting textures.

You can choose from a list of standard pre-defined texture styles, including those resembling stones, hand-carving, spheres, elongated ellipses, cords in a vertical or horizontal orientation, and others.

Or, you can create your own texture by modifying an existing texture style to achieve a specific pattern, and then saving your settings to a preset file.

All of these parameters work together to produce the handcrafted textures Art surface operation. Try using the default settings for a standard Texture Style first, and view the resulting effect. Then use the Art Manager to edit the operation's parameters, and regenerate the Art model. Repeat until you get the desired result.

Parameters

Presets

  • Opens a list of options to: Automatically load the most recently used parameters.
  • Save your current dialog box settings to a file (.aps) in your Shared Mastercam 2025 folder.
  • Load a set of previously saved parameters by selecting it from the list.
  • Delete a set of saved parameters.
  • Reset all dialog box parameters to their initial default values.

Preset files are stored locally in a fixed directory structure (..\Art\Presets) in your Shared Mastercam 2025 folder.

NOTE

Networked colleagues can access your preset from your Shared Mastercam 2025 folder. If you send your preset files to Mastercam Art users who do not have access to your network, make sure to let the recipients know the correct folder where they should paste the file in their Mastercam installation. Your recipients can use Windows Explorer copy and paste functions to add your files to their local machine. Their file structure must match yours.

Texture style

A list of predefined handcrafted textures you can choose to create the pattern illustrated by the associated icon. Choose a texture from the list that best approximates the type of pattern you want to use or modify. Then edit the default parameters as necessary to create your own design.

NOTE
  • Use the Resize Texture % parameter to adapt a custom texture to the correct scale.
  • Use options in the Preset field to save the parameters that define your texture so you can easily reuse it.

Radius XYZ

Adjusts the base ellipse radius in the selected axis. X is horizontal, Y is vertical, and Z controls the height. All of these controls interrelate.

Spacing XY

Sets the spacing between the base elements in an array, from center to center in the selected axis.

NOTE

To achieve no overlap of the ellipses when defining a handcrafted texture, set the spacing in XY equal to or greater than the specified Radius XY parameters.

Ellipse variation XY

Determines the random variation of ellipse size. Entering .05 allows each new ellipse that Mastercam Art creates to vary up to .05 from the base element ellipse you defined using other parameters in this dialog box.

Resize texture %

Provides a quick way to proportion your texture by adjusting the overall size of the element used in the texture (base pattern file or ellipse) by the percentage you enter.  Say you select the Texture Style Stone and the stones are too big or too small for your application. Simply adjust the percentage in this field, and your texture is resized accordingly.

Randomize size

Mastercam Art offers 100 different numbers that generate the random placement of differently-sized base pattern ellipses. Any number, from 1 to 100, entered into the field will create a individual, randomized texture that will not change upon regeneration. Entering zero into the field will cause the texture to reorganize each time you generate.

NOTE
  • The size of the number has no effect on the amount of randomization. For example, entering either 5 or 10 will a produce unique result, but both numbers will initialize the same amount of randomization.
  • If you randomize and there is too much spacing in the resulting pattern, adjust the Spacing X and Y parameters to smaller values. Or create another texture on top of the existing texture using the same boundary. Use the same parameters but enter a different number in the Randomize Size field. The new texture will fill in the original texture in different areas. If you are creating the texture on a flat plane, use the application style Blend to keep the base pattern elements from stacking upon one another.

Base height (cross section)

Adds a vertical extension / foundation along the Z axis to the start of the cross section when creating the Art surface operation. The base height raises or lowers the surface by the specified amount, creating a vertical wall. Enter a positive number to raise the Art surface, or enter a negative number to lower it. The following example use the same cross section shape to illustrate the effects of the base height parameter.

Smoothing style / Adjust shape

Blends the filleted surface edges using the selected style. The cross-sectional shapes indicate the style of smoothing to apply. When the cross section meets in the middle of opposing boundaries, it can create a ridge. Use this parameter to change the look of the ridge.

Top flat

Reserved for future development.

Build shape to: Inside/Outside

Creates the new Art surface operation on the side of the chained boundary you choose (Inside or Outside). The default is Inside. The red circle in the icons represents the geometry boundaries.

Application style

Defines how a new surface operation shape interacts with existing shapes and with the Art base surface (ABS). The icons for each application style option illustrate how the Art surface operation is created (stacked) in relation to the existing Art model. The blue ellipse represents an existing surface. The red bar represents the new surface that will be applied and demonstrates what the specified application style will do when applied to the blue ellipse. The red bar extends beyond the blue ellipse to illustrate how the shape will react to a flat ABS.

NOTE

You can edit the Art surface operation at any time to change the application style and other parameters--including the geometry used to define the Art surface boundaries. Simply regenerate the Art model to view the results. You can also use Mastercam Art's Undo and Redo options to restore or discard changes to the Art model.

Add

Adds the red bar to the Art model and then "grows" on top of the blue ellipse, following its shape. Note the intersection of the red and blue shape.

Sub

Subtracts the red bar from the Art model, and cuts the red shape out of the ellipse, following its shape.

Blend

Adds the red bar to the Art model. When it intersects the blue ellipse, the highest parts of each shape are displayed, resulting in a blend.

Add cut

Adds the red bar to the Art model and ignores the blue ellipse, cutting through it.

Add cut trim

Adds the red bar to the Art model, cuts through the blue ellipse, and then trims the red shape to the boundary of the ellipse.

Blend edge

Adds the red bar to the Art model, cuts through the blue ellipse, trims the red bar to the boundary of the ellipse, and then blends the edge of the red bar to the shape of the blue ellipse. With this function, you can also control the length of the blend.

Sub cut

Subtracts the red bar from the Art model and ignores the blue ellipse, cutting through it.

Inverse

Applies the inverse of the operation by flipping the operation in the Z axis inside the selected drive contour.

Mask

Isolates the selected elements. In the field graphic, the red bar was masked, and the blue base shape would not be affected in the operation.

NOTE

Mask elements on an Art surface operation to add a texture to masked elements while avoiding non-masked elements.
