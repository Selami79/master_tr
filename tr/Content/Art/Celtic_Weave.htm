Celtic Weave dialog box

Use the Celtic Weave dialog box to create an over and under blending of selected closed and open contours that intersect, producing an Art surface operation that resembles a Celtic-style knot or weave.

Each chain you select can have multiple intersections and multiple chains intersecting at the same point.

Use the options in this dialog box to define a cross section and set other parameters for basic and advanced fields. You can adjust the intersections of the contours, and select the weave style you want to apply. Advanced fields allow you to control the height and length of the "knots" created at each intersection, and define end conditions for open contours.

Parameters

Switch to advanced parameters

Opens the Advanced Parameters dialog box, where you can review and set additional options for the Art surface operation.

Presets

Opens a list of options to:

  • Automatically load the most recently used parameters.
  • Save your current dialog box settings to a file (.aps) in your Shared Mastercam 2025 folder.
  • Load a set of previously saved parameters by selecting it from the list.
  • Delete a set of saved parameters.
  • Reset all dialog box parameters to their initial default values.

Preset files are stored locally in a fixed directory structure (..\Art\Presets) in your Shared Mastercam 2025 folder.

NOTE

Networked colleagues can access your preset from your Shared Mastercam 2025 folder. If you send your preset files to Mastercam Art users who do not have access to your network, make sure to let the recipients know the correct folder where they should paste the file in their Mastercam installation. Your recipients can use Windows Explorer copy and paste functions to add your files to their local machine. Their file structure must match yours.

Cross section window

  • Displays the dynamic 2D cross sectional shape used to generate the Art surface. To define the shape:
  • Click and drag one of the red node points. Using the nodes to change the cross section shape updates cross section fields with the corresponding values. You can get the look you want without entering specific parameters.
  • To define an arc cross section, click the center node point to change the radius. You can also click the node points on the end to position them.
  • To define a Bezier curve, click and drag the node points to position the curve's endpoints. Then, change the angle and length of the control points to further define the shape of the curve.
  • Choose Dynamic Cross Section and select a predefined, customizable cross section from the drop-down list. These shapes include Line, Concave Arc (default), Convex Arc, Concave Parabola, Convex Parabola, and Bezier Curve.
  • Click Select geometry, and use the cursor in the Mastercam graphics window to select a user-created cross section, ridge profile, or ridge path. Unless you are creating a surface using the Sculpt Organic surface operation, user-defined geometry should be 2D and created in the same plane as the Art base surface.
  • Enter values in the cross section fields. The cross section window display changes, based on the parameters you enter.
  • From the Cross Section Library list, choose Load cross section from catalog to load a cross section from a library of saved Art cross section files (.acs). When you have defined a cross section you want to reuse, choose Cross Section Library, Save cross section to catalog, and save it to the library.

Dynamic cross section list

Allows you to select a predefined, customizable cross section from the drop-down list. These shapes include Line, Concave Arc (default), Convex Arc, Concave Parabola, Convex Parabola, and Bezier Curve.

Select geometry

Choose this option and use the cursor in the Mastercam graphics window to select a user-created cross section, ridge profile, or ridge path.

Unless you are creating a surface using the Sculpt Organic surface operation, user-defined geometry should be 2D and created in the same plane as the Art base surface. The selected geometry displays in the cross section window. You can then use the fields to change its shape and size.

Load / Store cross section

To load a cross section from a library of saved Art cross section files (.acs), choose Cross Section Library, Load cross section from catalog. Use options in the Open dialog box to choose the cross section.

When you have defined a cross section you want to reuse, choose Cross Section Library, Save cross section to catalog. Use options in the Save as dialog box to save the Art cross section library file to a selected location.

Zoom cross section window

Choose one of the following options to change the cross section window display:

  • Zoom In
  • Zoom Out
  • Zoom to Fit
NOTE

Zoom to Fit is particularly useful if the cross section scales outside of the viewing window, such as when you enter large values in the cross section fields.

Height (line)

Available only when you choose to define a line-based cross section. Sets the height of the line in the Y axis, as represented in the Cross Section Window by the top, right-most node. If you modify this field, the Angle field also changes. Use the Height and Angle fields to define a precise cross reference shape.

Top flat

Reserved for future development.

Angle (line)

Available only when you choose to define a line-based cross section. Sets the angle between the line and the X axis, as represented in the Cross Section Window by the top, right-most node. Use the Height and Angle fields to define a precise cross reference shape.

Base height

Adds a vertical extension / foundation along the Z axis to the start of the cross section when creating the Art surface operation. The base height raises or lowers the surface by the specified amount, creating a vertical wall. Enter a positive number to raise the Art surface, or enter a negative number to lower it. The following example use the same cross section shape to illustrate the effects of the base height parameter.

Radius (arc)

Available only when you choose to define an arc-based cross section. Sets the radius of the circle on which the arc-shaped cross section is based, as represented in the Cross Section Window by the center node. Use the Radius, Start Angle, and End Angle fields to define a precise cross reference shape.

Start angle (arc)

Available only when you define an arc-based cross section. Sets the polar angle of the arc cross section start point, as measured from a theoretical line through the center point of the arc. The bottom, leftmost node represents this field in the Cross Section Window. You can use the node to adjust the cross-sectional shape inward. Use the Radius, Start Angle, and End Angle fields to define a precise cross reference shape.

End angle (arc)

Available only when you choose to define an arc-based cross section. Sets the polar angle of the arc cross section end point, as measured from a theoretical line through the center point of the arc. The top, rightmost node represents this field in the Cross Section Window. You can use the node to adjust the cross-sectional shape inward. Use the Radius, Start Angle, and End Angle fields to define a precise cross reference shape.

Width (parabola)

Available only when you choose to define a parabola-based cross section. Sets the distance of the parabola-shaped cross section in the X axis, as represented in the Cross Section Window by the bottom, left-most node. Use the Width and Height fields to define a precise cross reference shape.

Height (parabola)

Available only when you choose to define a parabola-based cross section. Sets the distance of the parabola-shaped cross section in the Y axis, as represented in the Cross Section Window by the top, right-most node. Use the Width and Height fields to define a precise cross reference shape.

X Y coordinates (Bezier nodes 1 and 2)

Click and drag the red node points or enter values into the parameter fields to position the Bezier curve's endpoints.

  • Node 1 is the endpoint on the X axis. Node 2 lies on the Y axis.
  • Change the angle and length of the control points to further define the shape of the curve.

Angle 1, Angle 2 (Bezier control points)

Defines the polar angle from the X axis to a node's control point. Rotating the control point changes the angle of the curve.

  • Angle 1 measures the angle between the X axis and the control point attached to Node 1.
  • Angle 2 measures the angle between the X axis and the control point attached to Node 2.

Length 1, Length 2 (Bezier control points)

The distance from a node's endpoint to its control point. Stretching or compressing this distance changes the height of the curve.

  • Length 1 measures from Node 1 to its attached control point.
  • Length 2 measures from Node 2 to its attached control point.

Resize (Bezier curve)

Scales the cross section by the percentage you enter.

NOTE

Regenerating the surface resets the altered cross section to 100%.

X / Y Scaling

Available only when you load a user-defined cross section (.acs file), or select geometry in the Mastercam graphics window to define the cross section shape. Sets the scaling percentage in the X or Y axis. To keep the cross section proportional to its original shape, set both X and Y scaling factors to the same value.

Weave style

Defines the crossover pattern created for intersecting contours in the Art surface operation, beginning with the selected chaining direction, and, for multiple chains, starting with the first chain in the selected chaining order. Choose one of the following options:

  • Over -Under: Alternates an "over and under" weave pattern. At the first intersection, the contour that begins the weave pattern is raised over the "knot" formed by the intersection. At the next intersection, the same contour is lowered under the knot, and so on.
  • Under-Over:  Alternates an "under and over" weave pattern. At the first intersection, the contour that begins the weave pattern is lowered under the "knot" formed by the intersection. At the next intersection, the same contour is raised over the knot, and so on.
  • Over: Raises the chain that begins the weave pattern over each "knot" that is formed at all intersections.
  • Under: Lowers the chain that begins the weave pattern under each "knot" that is formed at all intersections.
NOTE

If you set Adjust Knot to No crossover in the Advanced Parameters - Celtic Weave dialog box, the weave style you specify here is not used.

Weave adjust shape

Blends the filleted surface edges using the selected style. The cross-sectional shapes indicate the style of smoothing to apply. When the cross section meets in the middle of opposing boundaries, it can create a ridge. Use Weave Adjust Shape to change the look of the ridge.

Application Style

Defines how a new surface operation shape interacts with existing shapes and with the Art base surface (ABS). The icons for each application style option illustrate how the Art surface operation is created (stacked) in relation to the existing Art model. The blue ellipse represents an existing surface. The red bar represents the new surface that will be applied and demonstrates what the specified application style will do when applied to the blue ellipse. The red bar extends beyond the blue ellipse to illustrate how the shape will react to a flat ABS.

NOTE

You can edit the Art surface operation at any time to change the application style and other parameters--including the geometry used to define the Art surface boundaries. Simply regenerate the Art model to view the results. You can also use Mastercam Art's Undo and Redo options to restore or discard changes to the Art model.

Add

Adds the red bar to the Art model and then "grows" on top of the blue ellipse, following its shape. Note the intersection of the red and blue shape.

Sub

Subtracts the red bar from the Art model, and cuts the red shape out of the ellipse, following its shape.

Blend

Adds the red bar to the Art model. When it intersects the blue ellipse, the highest parts of each shape are displayed, resulting in a blend.

Add cut

Adds the red bar to the Art model and ignores the blue ellipse, cutting through it.

Add cut trim

Adds the red bar to the Art model, cuts through the blue ellipse, and then trims the red shape to the boundary of the ellipse.

Blend edge

Adds the red bar to the Art model, cuts through the blue ellipse, trims the red bar to the boundary of the ellipse, and then blends the edge of the red bar to the shape of the blue ellipse. With this function, you can also control the length of the blend.

Sub cut

Subtracts the red bar from the Art model and ignores the blue ellipse, cutting through it.

Inverse

Applies the inverse of the operation by flipping the operation in the Z axis inside the selected drive contour.

Mask

Isolates the selected elements. In the field graphic, the red bar was masked, and the blue base shape would not be affected in the operation.

NOTE

Mask elements on an Art surface operation to add a texture to masked elements while avoiding non-masked elements.
