Art Manager Right-Click Menus

Different right-click menus display, based on where you right-click in Art Manager history tree. This topic introduces each right-click menu and its functions.

Art model right-click menu

Right-click on the Art model—the topmost level in the history tree, and choose one of the following options.

  • Enable all: Art base surfaces and Art surface operations that have been previously suppressed in the Art model. This function makes it easy to quickly restore the entire Art model.

  • Rebuild all: Rebuilds the entire Art model in the graphics window display using the current enable/disable settings in the Art Manager history tree.

    If you choose to regenerate an Art Model, and then press [Esc] to abort the regeneration before it has run to completion, the Art model display in the graphics window will also be incomplete. Use Rebuild All to restore the entire Art model display.

    NOTE

    This option does not regenerate operations in the Art model; it simply rebuilds the display.

  • Auto regen: Regenerates all operations each time you edit an element in the Art Manager history tree. Deselect Auto Regen to make multiple edits to your Art model without having to wait for each operation to regenerate individually.

Art base surface right-click menu

To access this menu, right-click on an Art base surface in the history tree. The following list show all options from this menu. Click an option to view more information.

  • Statistics: Displays the current extents of the selected Art model.

  • Delete: Marks the selected Art operation, or Art base surface and all of its operations, for deletion. The deletion is not processed until you regenerate or rebuild the Art model.

    NOTE

    Use Undo to restore deleted operations.

  • Suppress: Acts as a toggle to add or remove a selected operation or Art base surface from the history tree. Mastercam Art ignores all suppressed base surfaces and operations when regenerating the model. Suppressed Art base surfaces and operations are also removed from the graphics window display. If your Art model contains multiple Art base surfaces or operations, use this option to simplify the graphics view display. Suppressing operations and Art base surfaces can significantly reduce regeneration time.

  • Rename: Lets you enter a unique name for the selected Art base surface. A descriptive name makes it easier to find a specific Art base surface in the Art Manager history tree.

  • Attributes: Opens the Change Art Base Surface Parameters dialog box where you can view and edit the base surface resolution, surface size, orientation, and other basic components.

  • Toolpath Art Base Surface: Reads all the data in the selected Art base surface and opens the Toolpath Art Base Surface dialog box. Use this dialog box to define toolpath parameters and create toolpaths for all operations in the Art base surface.

  • Export: Choose an output format for the selected Art base surface. You can set export parameters for STL, DXF, or 256 grayscale image (BMP, JPG, TIF) file types. You can also choose to export the Art base surface as a Mastercam surface.

  • Regen: Regenerates all changed Art base surfaces and Art surface operations in the Art model using the current parameters and completely rebuilds the Art model. You must regenerate elements that are marked "dirty" and "deleted" to see the resulting changes in the Art model.

  • Auto Regen: Regenerates all operations each time you edit an element in the Art Manager history tree. Deselect Auto Regen to make multiple edits to your Art model without having to wait for each operation to regenerate individually.

Art surface operation right-click menu

  • The Delete and Suppress functions that appear in the Art base surface right-click menu list above are also available at the Art surface operation level. Refer to those descriptions for more information.
  • The Rename function is enhanced for Art surface operations (ASO). At this level, clicking on this function opens the Name the Operation dialog box. Use the Rename dialog box to distinguish the ASO in the Art Manager tree with a unique name, specific font color, and bolding.

Additional Art surface operation options

  • To return to the graphics window and re-select the boundary chains that drive an ASO, double-click Geometry in the Art history tree, or right-click and choose Replace contours.
  • To modify the parameters used to create an ASO, double-click Parameters in the Art history tree, or right-click and choose Change Parameters.
