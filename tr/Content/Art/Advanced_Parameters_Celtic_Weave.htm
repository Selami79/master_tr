Advanced Parameters - Celtic Weave dialog box

Use the Advanced Parameters - Celtic Weave dialog box to set parameters that further define the Celtic Weave Art surface operation, including:

  • Crossover parameters that control the height and length of the "knots" created at each intersection in the weave pattern.
  • Conditions that define the ends of each open contour in the Art surface operation, including the length, style and shape.
Parameters

Cancel and return to base parameters

Returns all fields in the Advanced parameters dialog box to their default values and closes the dialog box. The other values you set for the Art surface operation are unchanged. Only the Advanced parameters dialog box is reset. You can also click Cancel to reject the changes and exit the dialog box.

To save the settings you defined in the Advanced parameters dialog box and exit, click OK.

Crossover parameters

These parameters further define the raised "knot" created by the intersection of contours. You can set the overall length of the crossover segment, and the length of its topmost peak. You can also raise the peak height by a specified percentage of the height of the Celtic weave Art base surface operation.

Adjust knot

Choose one of the following options to define crossover parameters.

  • Auto Smooth: Mastercam Art calculates the total length of the crossover segment, based on the Peak Length and Height Factor you specify. This option also uniformly smooths the transition to and from the intersecting contours in the weave.
  • User Defined: Allows you to manually enter the total length of the crossover segment, and set other crossover parameters.
  • No crossover: Does not create overlapping crossover segments for intersecting contours. This option deactivates all crossover parameters in the dialog box.
  • Position Peak: Used only for Celtic weave Art surface operations performed on a single contour that may, or may not intersect. Allows you to specify a position where the peak of the "knot" should appear in the operation, regardless of the points of intersection.
  • Center Peak: Used only for Celtic weave Art surface operations performed on a single contour that may, or may not intersect. Choose this option to position the peak of the "knot" created by the operation in the exact center of the selected contour, regardless of the points of intersection.
NOTE

Position Peak is the only Adjust Knot method that allows you to specify the XY and Select fields.

XY (crossover length)

Available only when Adjust Knot method is set to Position Peak. Allows you to specify a position where the peak of the "knot" should appear in the Celtic weave surface operation, regardless of the point of intersection. You can enter XY coordinates, or use the Select button to choose a position in the graphics window. If you choose a position that does not lie along the drive contour, Mastercam Art places the knot in the nearest position along the contour, relative to the position you choose.

Total length

Defines the total length of the crossover segment. At the intersection of one or more contours, this is the total distance that the topmost contour crosses over the "under-crossing" contour.

Peak length (crossover)

Defines the distance of the apex (peak) of the crossover segment. At the intersection of one or more contours, this is the distance of the highest point at which the topmost contour crosses over the "under-crossing" contour.

NOTE

The peak length value you enter must be less than the specified Total Length of the crossover.

Height factor

Raises the peaks of the "knot" by a specified percentage of the maximum height of the Celtic weave Art base surface operation. You cannot enter a value that is less than 100%. Entering 100% effectively deactivates this option.

End condition (weave)

These parameters apply only to open contours. Use them to set the appearance of the end of each open contour used in the base surface operation.

Length (end condition)

Defines the distance along which the end condition Shape is applied to the Celtic weave Art surface operation. This distance is measured in a reverse direction from the start and end points of the contours you used to create the operation.

Style (end condition)

Choose one of the following options to specify where to apply the end condition parameters.

  • Apply to Both the start and end of open contours.
  • Apply to only the Start or End of the contours (based on chaining direction).
  • Do not apply an end condition style (None).

End shape

Blends the filleted surface edges using the selected style. The cross-sectional shapes indicate the style of smoothing to apply. When the cross section meets in the middle of opposing boundaries, it can create a ridge. Use End Shape to change the look of the ends.
