Wrap Simulation dialog box

Use the Wrap Simulation dialog box to transform the active Art base surface by wrapping it horizontally or vertically. You can define the wrap radius, angle, and direction. The Art base surface can wrap to the top / outside or bottom / inside of the defined wrap radius.

This function helps you visualize the Art base surface as a round or cylindrical shape and check the simulation for design issues.

Click the examples below to view examples of an Art base surface wrap that simulates a ring and a tube.

Example 1: Ring

This example shows an Art base surface ring design in top view before the wrap operation, and the wrap simulation is shown below in top and isometric views.

Example 2: Tube

In this example, the Art base surface wraps around the X axis, making it a tube. A smaller radius would completely close the tube.

NOTE

Use this function only to visualize the finished product, not to prepare the part for machining. To create the appropriate toolpaths, leave the Art base surface model flat in top view. Then choose axis substitution in the Toolpath Art Base Surface dialog box, Toolpath Parameters tab.

Parameters

Presets

Opens a list of options to:

  • Automatically load the most recently used parameters.
  • Save your current dialog box settings to a file (.aps) in your Shared Mastercam 2025 folder.
  • Load a set of previously saved parameters by selecting it from the list.
  • Delete a set of saved parameters.
  • Reset all dialog box parameters to their initial default values.

Preset files are stored locally in a fixed directory structure (..\Art\Presets) in your Shared Mastercam 2025 folder.

NOTE

Networked colleagues can access your preset from your Shared Mastercam 2025 folder. If you send your preset files to Mastercam Art users who do not have access to your network, make sure to let the recipients know the correct folder where they should paste the file in their Mastercam installation. Your recipients can use Windows Explorer copy and paste functions to add your files to their local machine. Their file structure must match yours.

Wrap radius

Sets the radius that the Art model is wrapped around.

Wrap direction

Determines whether wrapping occurs around the X or Y axis. Choose Horizontal to wrap the Art model around the X axis, or Vertical to wrap it around the Y axis.

Start angle

Determines where the relief model begins wrapping. The default is 180, which is 12:00 on a circle. This setting wraps the Art model, starting at the center.

Wrap sign

Determines whether the Art model wraps to the top / outside or to the bottom / inside of the defined wrap radius.
