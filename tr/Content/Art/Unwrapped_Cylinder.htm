Unwrapped Cylinder dialog box

Use the Unwrapped Cylinder dialog box to create or modify an unwrapped cylindrical Art base surface (ABS). This function is ideal for cylinders, such as a ring, that you work with as a flat surface and then cut on a rotary axis. Use this function to unwrap a specified cylinder into a new flat ABS.

Define the ABS by entering a wrapped diameter, an unwrapped length, or a ring size. All of the fields in the Cylinder group are associative and update when one is modified.

From this dialog box, you can assign a specific level, resolution, and shading quality to the ABS. You can also define the total height/depth of the Art model and the number of degrees that the ABS will rotate around the Z axis. Selecting the three options in the Create Geometry section will draw a rectangular boundary, toolpath margins, and/or a centerline onto your ABS. Clicking on the button at the bottom of the dialog accesses Art's Shading Setting dialog box where you can apply more effects to the ABS.

The ABS you create using this function is a 2D surface that you can manipulate by adding Art surface operations. An ABS defines the extents of the surface area you can shape with Art surface operations. Once you define the extents of the Art base surface, Mastercam Art sets a recommended resolution and displays the total file size of the part.

NOTE
  • While designing, combine a lower resolution with a higher shading quality to get a clearer view of the Art model without a decrease in regeneration speed. The file size displayed in this dialog box will reflect any changes in resolution. When the design is complete and you are ready to create toolpaths, use the Art Manager to increase the Art base surface resolution and regenerate all of its operations.
  • If you choose to work with ring sizes, Mastercam defaults to metric values in the Cylinder fields. If the Mastercam configuration is U.S., the MM checkbox becomes active and is selected.
  • In Mastercam Art, you use Art surface operations to modify a single surface. With other Mastercam products, such as Mastercam Router, you patch individual surface types together to form a whole.
Parameters

Diameter

Enter the outside diameter of the cylinder that defines the new Art base surface in its wrapped state. The Diameter, Length and Ring size fields are associative. If you modify one of these fields, the others update automatically.

Length

This value is calculated from the value entered in the Diameter or Ring size fields and sets the length of the cylinder / ring Art base surface in its unwrapped state. You can also manually enter a value in this field. The Diameter, Length and Ring size fields are associative. If you modify one of these fields, the others update automatically.

Finger size

Choose from a list of standard jewelry ring sizes to define the diameter and length of the Art base surface. The Diameter, Length, and Ring size fields are associative.

  • If you modify one of these fields, the others update automatically.
  • If you select a standard ring size from the drop-down list, and then modify the Diameter or Length field, Mastercam sets the Ring size to Custom.
NOTE

If the Mastercam configuration is U.S., the MM field is available. Select this field to convert all values in the Cylinder fields to metric, the most common unit of measure for defining ring sizes in jewelry.

Band height

Enter the vertical extent of the cylinder / ring Art base surface.

Margins

You cannot exceed the boundaries of an Art Base Surface (ABS) when you are machining it. However, to avoid unfinished edges, it is desirable to cut past the top and bottom edges of the cylinder / ring.

To do this, use the Margins field to extend the ABS on either side by the specified amount, creating a slightly larger ABS than the part requires. The margin lets the toolpath flow off the edge of the specified cylinder, allowing the face of the cylinder to be machined as well as the edges.

NOTE

When using the Margins field, you can also select the Create Geometry Boundary checkbox. This selection creates geometry that frames the specified cylinder. You can then see where the cylinder ends and the margins begin.

Z Coordinate

Sets the Z depth / height of the Art base surface (ABS) along the Z axis. Enter a value or use Select to choose a position in the graphics window. Note: The Z depth of an ABS is usually set to zero.

Select Z

Use the cursor to set the Z depth / height of the Art base surface (ABS) along the Z axis by choosing a position in the graphics window. Note: The Z depth of an ABS is usually set to zero.

Resolution/File size

Sets the number of points per square inch or millimeter (metric units) in your Art model. Mastercam Art selects a default resolution based on the size of the Art base surface (ABS).

A value of 200 defines the Art model as 200x200, or 40,000 points per square inch / millimeter. Increase resolution to improve the definition and display of the shaded Art model. Mastercam Art displays the change in your part's total file size based on what you enter in this field.

The resolution of the Art model is dynamic, which means you can change it at any time. Reducing the resolution can significantly increase your productivity. A lower resolution decreases the file size, so the Art model regenerates faster. Resolution becomes very significant when you have many or complex Art surface operations and Art base surfaces in an Art model.

Rotation

Sets the number of degrees to rotate the Art base surface around the Z axis on the XY plane.

Shading quality

Controls the sharpness/preciseness of shading without affecting file size or performance. Increasing shading quality will improve the display of details, such as fine textures or leaf veining.

To increase the quality and see more detail, make the value smaller (for example, change 0.01 to 0.005).

Level

Displays the level the Art base surface is on. Click to open the Levels Manager and choose a different level.

Z Limit

Select the checkbox to activate this option. The Z Limit option constrains the total height / depth of the Art model along the Z axis by the value you enter in the field.

If the Art model Z value exceeds the specified limit, either in a positive or negative direction, Mastercam displays a warning message so you can make the necessary adjustments.

Deselect the checkbox if you are not limited to a certain stock size or if no restrictions are placed on the overall height / depth of the Art model. When you clear the checkbox, Mastercam Art ignores the value that displays in the Z Limit field.

NOTE

At the bottom of the Art Manager tab, the total Art model height in the Z axis dynamically displays. If a Z Limit constraint is applied to the Art model, the remaining height before reaching the maximum is also calculated and displayed.

Margins

Creates two geometric line entities. These entities are offset outward from the defined Art base surface extents by the distance you defined in the Cylinder, Margins field.

Centerline

Creates a geometric line entity along the Art base surface centerline.

Boundary

If selected, creates rectangular geometry that frames the perimeter of the Art base surface.

Art model shading

Opens the Art Shading Setting dialog box where you can define the color, opacity, and reflection quality (material and power) of the Art base surface. Additionally, you can use three methods to crop the Art base surface.
