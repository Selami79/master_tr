Art Base Surface Toolpath Parameters tab

Use this tab to define the toolpath cutting method, stepover, and cutting direction. For some cutting methods, you can also define the toolpath type (roughing / finishing), and choose to select toolpath containment boundaries.

Use the Depth Cuts, Approach, Filter, and Advanced buttons to set additional toolpath parameters. To activate these options, first select the checkbox next to the option. Then click the option to access the appropriate dialog box (not all options display for all cutting methods).

Parameters
NOTE

Not all options are available for all Cutting methods.

Art Base Surface Cutting Methods

Stepover Percent

Sets the distance the cutter will overlap each pass of the cutter, relative to the size of the tool (50% of a .125 tool would be .0625). The Stepover percent and Stepover distance fields are associative; changing one field changes the other.

Stepover Distance

Defines the cutter overlap between passes by the specified distance. The Stepover percent and Stepover distance fields are associative; changing one field changes the other.

Spiral Resolution

Enabled when Cutting method is set to Spiral - Ellipse to edge or Spiral - Ellipse to corner. Adjusts the smoothness of the spiral by percentage. If you enter a large value in this field, the toolpath may contain flats or facets. Lower values create smoother curves.

Corner Radius

Enabled when Cutting method is set to Spiral - Circular contained or Spiral - Rectangular rounded corners. Sets the radii on all corners of the toolpath. Enter 0 (zero) for sharp corners. Enter a larger number for rounded corners.

Roughing / Finishing

Choose the type of toolpath to create. Rough toolpaths quickly remove large amounts of stock in preparation for a finish pass. When you choose Roughing, the Stock to leave in Z parameter is added to the Art base Surface Toolpath Parameters tab. Typically, a finish toolpath follows a roughing toolpath, but this is not required. You can create a finish toolpath even if no roughing operation has been created.

End in Point

Enabled when Cutting method is set to Spiral - Follow boundary contained. Select this option to have the spiral end/begin in the center on a point, rather than on a line segment. End in Point eliminates irregular line segments at the center of the spiral.

Finish Pass

Select to add a final pass that traces the containment boundary and removes any scallops.

True Spiral

Select to cut a spiral path without any walkover segments.

Seam Overlap

Enabled when Cutting method is set to Cylinder Milling Spiral. Determines the distance the tool overlaps the end of each pass when cutting a cylindrical Art base surface before repositioning and making the next cut. Mastercam uses the diameter of the selected tool as the default value. Enter a positive value here to ensure that Mastercam does not leave unwanted scallops or cusps in the transition area along the seam of the cylinder / ring.

Front to Back / Back to Front

Select to machine from front to back or back to front.

ZigZag

Cuts in one direction, and then reverses the path and comes back. This allows the tool to stay down on the surface at all times, and move back and forth across the part.

OneWay

Forces the tool to cut the part in one direction, retract / come up, go back to the beginning, and cut again in the specified direction.

Outside-In

If you select this option, the toolpath spirals from the outer boundary of the surface to the center (inside).

NOTE

Outside-in might leave an imperfection on soft materials, because the last small column of material cut can snap off and leave a tear mark.

Inside-Out

If you select this option, the toolpath spirals from the center (inside) to the boundary (outer edge) of the surface.

Climb

Enabled when Cutting method is set to Parallel - Spiral contained or Parallel - Contained with edge finish. Cuts the part in one direction with the tool rotating in the same direction as the tool motion.

Conventional

Enabled when Cutting method is set to Parallel - Spiral contained or Parallel - Contained with edge finish. Cuts the part in one direction with the tool rotating in the opposite direction of the tool motion.

Parts vs Cutter / Toolpath Tolerance Optimization

Enabled when Cutting method is set to Parallel - Spiral contained or Parallel - Contained with edge finish. Advanced parameter. The default value (50) is appropriate for most models. Use the appropriate lowest value. A larger value results in less optimizations, a smaller tolerance for the toolpath, and increases the file size.

Machining Angle

Enabled when Cutting method is set to Parallel - Given angle or Parallel - Contained with edge finish. Sets the parallel toolpath to the angle you enter here. Enter the angle by typing the value, or by clicking one of the predefined angle buttons.

Stock to leave in Z

Enabled when Roughing is selected. Defines the amount of stock in the Z axis that remains on the part when the operation is finished. Typically, this would be removed by a separate finish operation.

Tool containment

Specifies whether the selected cutting method requires a containment boundary. A red cross in the checkbox indicates that the field is not needed and you cannot select it. Otherwise, you can select this option to access the Chaining dialog box. From there, choose the containment boundaries (closed chains or curves) that will limit tool motion. Available for the following Cutting methods:

  • Parallel: Contained, Spiral Contained, Contained with edge finish, Horizontal contained
  • Spiral: Rectangular contained, Circular contained, Follow boundary contained

Depth Cuts

Select the checkbox to use the settings in the Depth cuts dialog box to divide the total depth of the Art model into multiple depth cuts. Click the button to open the dialog box. Selecting depth cuts activates the Stock to leave in Z parameter in the Art Base Surface Toolpath Parameters tab.

Approach

Select the checkbox to use the Contour ABS Edge dialog box setting to control the number of cuts needed to make a finish cut around the part's boundary. Click the button to open the dialog box.

Filter

Select the checkbox to filter the toolpath using the parameters you define in the Filter settings dialog box. Click the button to open the dialog box. When you filter a toolpath, Mastercam replaces tiny toolpath moves that lie, within a specified tolerance, in a straight line with longer lines or arcs.

Advanced

Select the checkbox to apply the parameters you define in the Advanced Setting dialog box to the toolpath. These include setting a Z plunge angle, retracting at top height, accelerated roughing, and smoothing. Click the button to open the dialog box.
