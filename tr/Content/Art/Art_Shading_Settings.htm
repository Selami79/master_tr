Art Shading Settings dialog box

Mastercam Art's Shading Settings greatly simplify the presentation of different material concepts to your customers. The variables set in this dialog are dynamic and immediately visible. They do not require you to regenerate the Art model. Moreover, you can quickly and easily change the look of an Art base surface as many times as you wish.

Use the parameters in this dialog box to define the color, opacity, surface material, and reflectivity of the Art base surface (ABS). From here you can also determine how much of the ABS plane is displayed in the graphics window.

The following examples show how Art's shading settings affect the look of a part.

   

Mastercam Art also provides a library of bitmaps and sample styles that mimic the look of many frequently used materials.

Parameters

Presets

Opens a list of options to:

  • Automatically load the most recently used parameters.
  • Save your current dialog box settings to a file (.aps) in your Shared Mastercam 2025 folder.
  • Load a set of previously saved parameters by selecting it from the list.
  • Delete a set of saved parameters.
  • Reset all dialog box parameters to their initial default values.

Preset files are stored locally in a fixed directory structure (..\Art\Presets) in your Shared Mastercam 2025 folder.

NOTE

Networked colleagues can access your preset from your Shared Mastercam 2025 folder. If you send your preset files to Mastercam Art users who do not have access to your network, make sure to let the recipients know the correct folder where they should paste the file in their Mastercam installation. Your recipients can use Windows Explorer copy and paste functions to add your files to their local machine. Their file structure must match yours.

Styles

A list of predefined surface styles. Each style is associated with a set of parameters (color, opacity, reflection control, and material) to produce the effect of a specific surface type.

Choose a style from the drop-down list that best approximates the surface you want the ABS to display. Customize the style by editing the default parameters or by replacing the bitmaps as necessary.

NOTE

To use your own bitmap (reflection or material), create a square bitmap file with any image-editing software. The number of pixels on each side of the square must be a power of two. For example, 16 x 16, 64 x 64, or 128 x 128 are acceptable. Then place it within the Mastercam installation folder in Art/Art Reflectivity Bitmaps or Art/Art Material Bitmaps. The bitmaps that come standard with your Art installation are 256 colors, and 512 x512 pixels.

Color

Displays the color of the Art base surface. Click to open the Colors dialog box and choose a different color.

The ABS color greatly influences the look of the Art model as it interacts with the Reflection Bitmap.

Opacity power

Changes the transparency of the Art model.

Entering a higher value in this field makes the Art model more opaque. A lower value makes the Art model more transparent, and the underlying geometry easier to select.

Reflection bitmap

Choose the image that will be reflected by the Art base surface from the drop-down list.

Reflection bitmaps are images that are stored locally in your Mastercam installation folder in the Art/Art Reflectivity Bitmaps folder. Use Windows Explorer's copy and paste functions to add reflection images to this folder.

NOTE
  • To use your own reflection bitmap, create a square bitmap file with any image-editing software. The number of pixels on each side of the square must be a power of two. For example, 16 x 16, 64 x 64, or 128 x 128 are acceptable. Then place it in the Art/Art Reflectivity Bitmaps folder. The reflectivity bitmaps that come standard with your Art installation are 256 colors, and 512 x512 pixels.
  • When working with reflection materials on multiple PCs, make sure that the bitmaps are available and in the Art Reflectivity Bitmaps folder on all PCs.

Browse (Reflection Bitmaps)

Click to select the directory that contains the bitmap image you would like the Art base surface to reflect. By default, this button opens the Art/Art Reflectivity Bitmaps folder in your local Mastercam installation directory.

File name

(Read only) Displays the path and filename of the bitmap file (.bmp) used as the Material Bitmap. Choose a bitmap from the drop-down list to select a different image to map to the Art base surface.

Material power

Changes the opacity of the material bitmap.

Entering a higher value in this field makes the material bitmap more opaque. A lower value makes the image more transparent, causing it to have a softer appearance.

Tile

Select to map multiple copies the material bitmap image over the Art base surface according to the dimensions you specify.

Deselect to stretch a single copy of the image over the entire Art base surface.

Maintain aspect ratio

Select to make the Tile width and Tile height fields associative so that entering a value one field changes both values. This keeps the image height and width proportionate to each other.

Deselect to independently set the Tile width and Tile height, which may stretch or skew the image.

Tile width

Available only if Tile is selected.

Enter the maximum number of inches / mm used to create each tile along the X axis. The Tile width and Tile columns fields are associative. Changing one updates the other.

Tile columns

Available only if Tile is selected. Enter the number of columns of tiles to create along the X axis. The Tile width and Tile columns fields are associative. Changing one updates the other.

X Offset

Moves the material bitmap along the X axis by the value you enter into the field.

Tile height

Available only if Tile is selected.

Enter the maximum number of inches / mm used to create each tile along the Y axis. The Tile height and Tile rows fields are associative. Changing one updates the other.

Tile rows

Available only if Tile is selected.

Enter the number of rows of tiles to create along the Y axis. The Tile height and Tile rows fields are associative. Changing one updates the other.

Y Offset

Moves the material bitmap along the Y axis by the value you enter into the field.

Rotation

Rotates the material bitmap image around the center by the specified angle.

Crop base plane

Trims the surface plane of the Art model but leaves all Art surface operations unchanged.

Crop above

Trims the Art base surface above the specified Z value.

Crop below

Trims the Art base surface below the specified Z value.
