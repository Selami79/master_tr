Mastercam Art Overview

Mastercam brings its power and experience to artistic relief design and cutting with Art. You can bring your 2D sketches, clip art, photos, and CAD files to life by crafting them on screen and cutting them with specialized toolpaths. With Art, you can:

  • Work with files including JPG, TIFF, and raster-based images—and create a 3D relief based on your preferences.
  • Create 3D sculpted models from 2D artwork.
  • Create complex organic surfaces with a few simple clicks.
  • Add, subtract, intersect, and blend multiple organic shapes.
General Art workflow

Below is a general outline process of creating and machining an application using Art.

Importing and refining the 2D art design.

First, you need to import a 2D design. There are many methods available to bring 2D artwork into Mastercam.

  • Use Raster to Vector to convert clip art to 2D geometry.

  • Use the Design Library function to import geometry.

  • Simply open a third-party file in Mastercam and use Mastercam's built-in converts to import the geometry.
  • Use Mastercam's CAD functions to create the geometry you want to work with.

Building a 3D Model

Mastercam Art provides a number of functions you can use to create an Art base surface. Once you have an Art base surface, create the Art surface operations that "puff up" the 2D geometry.

Creating toolpaths

Next, choose a Mill or Router machine definition from the Machine tab. Then create an Art toolpath by selecting the Art tab, and the Surface function.

Use this function to:

  • Select a cutting tool and set feeds, speeds, and other general toolpath parameters
  • Enter values for different cutting parameters and compensation options
  • Define the toolpath cutting method, stepover, cutting direction, and tool containment boundaries

NOTE

For more information about how to machine an art-based surface, refer to the Machining Art Base Surfaces Help topic.

In the Toolpaths Manager, use the Machine Group Setup function panel to set up other aspects of the job, such as tool settings and stock setup.

Simulating the machining

When you finish defining the Art model, Use Classic Backplot or Mastercam Simulator to simulate the tool motion used to cut the part and to view a solid model of the machining process.

Simulating shows the path the tools take to cut your part. This display lets you spot errors in the program before you machine the part. As you backplot toolpaths, Mastercam displays the current X, Y, and Z coordinates in the left side of the Status Bar (lower left corner of the screen).

Outputting NC code

Use the Toolpaths Manager Post function to produce NC output for your machine. Post processing, or posting, is the process by which the toolpaths in your Mastercam part files are converted to a format that can be understood by your machine tool's control (for example, G-codes). A special program called a post processor, or post, reads your Mastercam file and writes the appropriate NC code. Upload the code to your CNC machine.

Machining the part

Set up your CNC machine, clamp the part securely, and machine the part. In a production setting, you may need to make custom jigs and fixtures first.

Terms and concepts

When working with Mastercam Art, you need to understand a few key terms and concepts.

Art model refers to the entire sculpted model.

An Art base surface (ABS) is the first surface you create when creating an Art model. It is a 2D or 3D surface based on a mesh of optimized triangles that is unique to Mastercam Art. You can choose to display the ABS in the graphics window as a triangulated grid or as fully shaded. Everything in the Art model is created on the ABS, so you must begin your work by defining it.

  • To create a 2D ABS, use the Rectangular function. If you plan to cut the part on a rotary axis, you can also use the Unwrapped Cylinder function.
  • To create a rectangular 3D ABS, use the From File function, and import a graphic image file.

Use Art surface operations to manipulate the ABS by adding details or features. Art surface operations use geometry as contour boundaries and are built using cross sections that you define and adjust.

For example, an Organic Art surface applies the cross section by reaching from opposing boundaries to the center. It scales and trims itself as the boundaries become closer or farther apart. Art surface operations can be stacked (added or subtracted from each other) to create sculptural models.
