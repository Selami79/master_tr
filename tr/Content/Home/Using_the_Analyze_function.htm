Using the Analyze Functions

Use the Analyze functions, located on the Home tab, to view entity properties and edit some or all of the data, based on the entity type. To analyze entities, use general selection methods, and select one or more entities before or after choosing the Analyze function. You can analyze all entity types, including:

  • Lines, arcs, points, and splines (parametric and NURBS)
  • All surfaces, meshes, and solids
  • Drafting dimensions

Analyze functions—including Position, Contour, Dynamic, and Distance (between points)—report position coordinates in relation to the Cplane origin.

Click a drop-down below for information on analyzing multiple entities, editing the analyze properties entity attributes fields, or working with associativity. For details on entity properties fields other than attributes, see the specific entity type dialog box topic (for example, Line properties dialog box).

NOTE

Analyze Deviation uses a color-coded "heat map" in the graphics window to display the differences between entities. Deviation ranges, as well as maximum positive and negative deviations, are represented by colored dots. Hover over any of these dots in the graphics window to view its specific deviation. Edit the options within the function panel to change the display.

Selecting multiple entities

When you select more than one entity to analyze, a single properties dialog box appears for the first entity type. The left and right arrow buttons in the dialog box indicate that you are analyzing more than one entity, and allow you to cycle through the previous or next entity in the selection group.

The sequence in which the properties dialog boxes appear is determined by the selection method you use. If you pre-select multiple entities, Mastercam sorts and displays the entities in an ascending numeric sequence based on the entity number (a database property automatically assigned to all entities). If you select entities after choosing the Analyze function, the order in which you select the entities determines the sequence.

Viewing and editing entity attributes

Entity attribute fields for level, color, line style and width, or point style appear in all analyze properties dialog boxes. Use these fields to modify the attributes of a single entity, or all selected entities.

  • To assign a new level, type the level number in the field or click the levels button to open the Levels dialog box.
  • To modify the entity color, type the color ID number in the field or click the color palette to open the Colors dialog box.
  • To modify the line style or width, select one from the drop-down list.
  • To apply attribute changes to all selected entities, click Propagate Attribute Changes. This button is only available when multiple entities are selected.
Working with associativity

The Association button on the properties dialog boxes lets you view associations to the selected entities. These associations include solids, meshes, surfaces, toolpaths, planes, dimensions, 3D annotations, or groups.

Additionally, if you are deleting an entity that is associative, the Deleting a referenced entity dialog box will warn you. You can use the Show associative entities drop-down to view the associations.

NOTE

To define startup default Mastercam attribute parameters, such as line and point styles, or line width parameters, use the CAD Settings page in the System Configuration dialog box.
