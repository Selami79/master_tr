Create Boundary Add-In

Use this Add-In to create boundaries based on surfaces, solids, or solid faces. You can create boundaries that represent not just the surfaces, but the area that would be machined with any of several cutting strategies. Use this Add-In to create containment boundaries for surface toolpaths, or to preview and analyze the effects of different machining scenarios without having to repeatedly generate the toolpath. Follow these steps:

  1. After selecting the Add-In, you are prompted to select the surfaces, solids, or solid faces to be analyzed. Click Select to select different surfaces while the Add-In is running.
  2. Select one of four boundary strategies.
  3. Set boundary parameters in the bottom half of the dialog box. These include the size of the tool used to calculate the machined area.
  4. Click Apply to generate the boundary geometry.
  5. Edit settings, and click Apply to generate new boundaries representing different scenarios. While the function is active, you can change entity attributes—such as color and level—so that different boundaries are distinct.

You can generate the following types of boundaries:

Silhouette boundary

This projects the outer extent of the selected surfaces as a 2D boundary at the current Z depth in the current Cplane. No tool is involved in the calculations, and the cutter dimensions are not active.

Shallow area

Mastercam calculates the portion of the surfaces that falls between the specified angle limits, and offsets it by the tool dimensions. Each point on the boundary corresponds to the compensation point of the tool at the angle limit.

Theoretical rest area

This boundary uses the dimensions of two tools:

  • The first tool is a reference tool that Mastercam uses to calculate the theoretical rest area.
  • Mastercam then calculates how much of this area would be machined by the second tool.

The boundary represents the machined area.

Cutter contact area

This boundary shows how much of the surfaces would be machined by the cutter. Configure the machining area by choosing whether to limit the boundary to the tool center or the contact point.

NOTE

This boundary type is not effective for vertical or near-vertical walls. Typically, you should use this on surface angles less than 80 degrees.

XY Thickness and Z Thickness

Use these settings to move the tool away from the surfaces by a specified amount. The XY Thickness moves the tool away from walls and vertical surfaces, and the Z Thickness moves the tool away from horizontal surfaces. For sloping surfaces, Mastercam interpolates a value between the two.

Option Settings Outcome

XY Thickness and Z Thickness are both zero.

For finishing operations, these values should be set to zero. The calculations are based on the dimensions of the cutter as entered in this dialog box, with no offset.

XY Thickness and Z Thickness have the same positive value.

For roughing and semi-finishing operations, you should set these values greater than zero. The calculations are based on a modified cutter, the surface of which is offset to be larger than the entered dimensions. This simulates leaving material on the part.

XY Thickness and Z Thickness have the same negative value.

In special circumstances—such as when making electrodes with a spark gap—you can set the thickness to a value less than zero. The cutter removes material at a level below the designated surface. The calculations are based on a modified cutter, offset to be smaller than the entered dimensions.

XY Thickness and Z Thickness are different.

The thickness in the XY plane may be set to one value and the thickness in the Z direction to another. This is often useful for operations near steep walls. The calculations are based on a modified cutter, offset by different amounts in the XY plane and Z directions. This leaves material on the part.
NOTE
  • Typically, Mastercam should be in 3D mode when creating the boundaries. 2D mode is useful for creating tool containment boundaries.
  • Click Undo to remove boundaries that have been created with Apply. Undo is active while the CreateBoundary function is running.
  • Most of the fields store recently used values so you can easily recall them.
Parameters

Select button

Returns to the graphic window so you can edit the set of selected surfaces.

Silhouette boundary

Projects the outer extent of the selected surfaces as a 2D boundary at the current Z depth in the current Cplane.

XY offset

After calculating the silhouette boundary, Mastercam offsets it by an additional amount that you specify here.

Z offset

After the silhouette boundary is created, Mastercam translates it by this amount in the Z axis.

Shallow area

Calculates the portion of the surfaces that falls between the specified angle limits, and offsets it by the tool dimensions.

Start angle

Enter the lower angle limit.

End angle

Enter the upper angle limit.

Theoretical rest area

Creates a boundary based on the dimensions of two tools. The first tool is a reference tool that Mastercam uses to calculate the theoretical rest area. Mastercam then calculates how much of this area would be machined by the second tool and displays its boundary.

Reference tool diameter

Enter the diameter of a reference roughing tool. Mastercam uses this to calculate the initial rest area.

Reference tool radius

Enter the corner radius of a reference roughing tool. Mastercam uses this to calculate the initial rest area.

Cutter contact area

Shows how much of the surfaces would be machined by the cutter whose dimensions are entered below.

Adjust to

With Adjust To: Center, Mastercam limits the tool so that its center point stays on the boundary. With Adjust To: Contact Pt, Mastercam limits the tool so that its contact point stays on the boundary. This has the effect of increasing the machining area for convex parts or bosses, and decreasing the machining area for concave parts or pockets.

Resolution

Represents the granularity of the boundary. A small value results in a more detailed boundary, but it takes longer to calculate.

Tool diameter

Enter the diameter of the theoretical tool that will machine the selected surfaces.

Corner radius

Enter the corner radius of the theoretical tool that will machine the selected surfaces.

XY thickness

Keeps the tool away from walls and vertical surfaces by the specified distance. Use this when you want to see the effect of a roughing operation or leftover stock.

Z thickness

Keeps the tool away from floors and horizontal surfaces by the specified distance. Use this when you want to see the effect of a roughing operation or leftover stock.

Offset

After calculating the boundary, Mastercam offsets it by an additional amount that you specify here. If you see jagged edges on the calculated boundary, sometimes adding a small amount with this field can result in a smoother boundary. This Offset works differently than the XY Thickness. The XY Thickness is applied to the tool diameter before the boundary is calculated. The Offset is added to the boundary after it is calculated.
