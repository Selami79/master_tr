Control Definition Compare Add-In

Use this Add-In to compare two control definitions and highlight the differences. You can run it in either of two modes:

  • Compare all the settings except for post text.
  • Compare post text.

You can also use this utility to compare a control definition to its default values to see which values are different.

Loading the control definitions

Loading the control definitions requires two tasks:

  • Select the .mcam-control files that contain the control definitions.
  • Since each .mcam-control file can contain multiple control definitions, you then need to select the desired control definition from each .mcam-control file.

To select the .mcam-control files, click the Folder button next to each File field. You can also click the small arrow in the File field to select recently used .mcam-control files.

You will then be prompted to select a control definition. Each control definition is identified by the name of the post with which it is associated.

  • You can compare two control definitions from the same .mcam-control file. For example, if a .mcam-control contains control definitions for several different posts, you can use this Add-In to quickly see the differences. Simply select the same .mcam-control file in each File field, and then select each control definition as desired.
  • You can compare control definitions from different versions of Mastercam. This can be useful when upgrading to a new version of Mastercam to make sure that nothing unexpected has changed.
  • You can compare a control definition to its default settings to see if any settings are different. The default values are stored in the control definition Default setting for control type for each product.

Comparing the data

Each control definition opens in a "tree" format. Topics that contain differences are highlighted. Expand each element to see the individual settings.

  • Click the Post Text button to toggle between comparing post text entries in the post and values from Control Definition Manager topics.

    NOTE

    Not all posts contain post text. When you select the control definition, Mastercam tells you if the associated post does not have a post text section. Contact your post developer if you think this might be an error.

  • Use the All, Differences, and Collapse buttons to control the tree display.
    • Click All to expand all the elements in the tree and see all of the values.
    • Click Differences to expand only those topics that contain differences and highlight the values.
    • Click Collapse to close all of the elements in the tree.
  • Click Next and Previous to navigate from one difference to the next.
  • Click Configuration to change the color used to highlight the elements in the tree. See Control Definition Compare: Configuration dialog box to learn more.
Parameters

File

The name of the .mcam-control file that contains the control definition. Choose a recently used .mcam-control file from the list or click the folder icon to select a new one. After you select the .control file, you will be prompted to select the control definition inside it.

Definition

The name of the post (.pst or .mcpost file) that is associated with the selected control definition.

Control type

The product associated with the selected control definition. This can be Mill, Router, Lathe, Wire, or Mill/Turn. Note that Mill/Turn control definitions use MP posts that run with Mastercam Lathe, not posts for the Mastercam Mill-Turn product.

Post text

You can use this Add-In to compare either the post text associated with the control definitions or the values from the different topics in the Control Definition Manager.

  • Click this button to view and compare the post text entries.
  • Click again to compare the settings from the Control Definition Manager topics.

All

Expands all the topics in the tree and displays their values.

Collapse

Collapses all the topics in the tree.

Differences

Highlights only the topics that contain differences between the two control definitions and displays their values.

Next

Go to and highlight the next difference.

Previous

Go to and highlight the previous difference.

Configuration

Choose the colors used to display different types of data.
