MPBin License Entry dialog box

Use the MPBin License Entry dialog box to manage the list of license numbers and/or customer IDs that will be associated with an encrypted post. You can enter license numbers from HASPs, NetHASPs, and software licenses. Customer IDs can only be used with software licenses though, not HASPs or NetHASPs.

Once you build the list, do any of the following:

  • Click the OK button. This will update the list of license numbers and customer IDs for the current session and close this dialog box. You can now bin the selected post to the updated list of licenses and customers.
  • Click the Apply button. This will update the list of license numbers and customer IDs for the current session and keep this dialog box open. You can now switch between the MPBin Utility dialog box and this one, updating the list of licenses and/or selecting different posts as desired.
Entering license numbers and customer IDs

You can build your list of license numbers and/or customer IDs by entering them directly in the window.

  1. Click the Add new ID button.

    You can also right-lick in the window and select Add new ID.

  2. Click in the Type column and select the type of number—License ID or Customer ID.

  3. Click in the ID column and enter the value.

  4. Repeat steps 1–3 as desired to add new numbers to the list. You can include both types of numbers in the same list.
NOTE

Click the License number button to add the license number of your workstation to the list.

Using license numbers and customer IDs from a text file

You can store a list of license numbers and customer IDs in a text file, which lets you easily recall them for future encryption jobs. Each line in the file should have two comma-delimited values, where the first value is 0 for a license number or 1 for a customer ID. For example:

0, 12345
1, 67890

  • If the line contains a single value, it will be processed as a license number.

  • The default extension for the file is .sim, but you can use any extension you like.

Follow these steps to load a list of numbers from a file.

  1. Click the Select button.

  2. Select the desired file and click Open.
  3. When the file name appears in the dialog box, click Load. Any entries that are already in the window will be overwritten.
  4. If desired, add new entries to the list.
    1. Click the Add new ID button to create each new entry.

    2. Click Save to save the updated list to the file.

Parameters

License numbers

Displays the list of licenses that will be allowed to run the encrypted posts. To enter new numbers, press the [Down] arrow to move the cursor to a new line.

Text file for license numbers

Displays the name of the file which contains the list of license numbers. Click the file Select button to select a different file.

Load

Loads the numbers from the current license file.

Sort

Sorts the list of license numbers by the ID value.

Save

Saves the current list of license numbers to the file displayed in the Text file for license numbers field.

License number

Loads the license number that is being used by your Mastercam workstation.
