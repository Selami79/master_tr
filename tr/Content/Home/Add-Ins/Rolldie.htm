Rolldie Add-In

The Rolldie Add-In creates toolpaths around a rotary axis. The toolpath can be rounded or squared on outside sharp corners. When a tapered tool is used, the tool can run up and down the crease formed at the intersection of two walls at an inside sharp corner.

The chained entities selected to create the toolpath can be in the flat or already rolled. Typically this toolpath is used to generate 5-axis tool vectors around a rotary axis, but there is an option to produce a 3-axis toolpath from flat input. This is typically selected for tapered tools.

In addition to calculating the toolpath, Mastercam also creates the rolled geometry. If you wish, set the geometry color and level before running the Add-In to organize it.

Follow these steps to run the Add-In:

  1. Open your part file, and select the Rolldie Add-In.
  2. Chain the contours from which the toolpath is to be generated.
  3. Mastercam displays the Toolpath parameters tab, where you can select a tool.
  4. If you are creating a 5-axis rolled toolpath and the part is not already rolled, select Rotary axis. In the Rotary Axis dialog box, select Axis substitution. Set the diameter, rotation direction, and which axis is to be replaced.
  5. Select the Contour parameters tab and enter your cutting parameters.
  6. Set Depth to a negative amount and choose Incremental. This sets the depth of the tool tip. If you use a tapered tool, the toolpath is offset so that the edge of the tapered tool runs on the wrapped or flat chain, resulting in the depth that you have entered.
  7. Select OK when done.
  8. Mastercam displays the Rolldie dialog box. Set your RollDie options, and click OK to generate the toolpath.
Parameters

Chordal deviation

Controls the precision with which Mastercam calculates the toolpath.

Show roll

Shows the rolled part.

Corners

Tells Mastercam whether to create Round or Square corners. Typically you select square corners when you use a tapered tool. Do not select Square if your part has inside corners, and you are using a non-tapered tool.

Tool vector length

Controls the toolpath display by determining the length of the tool axis at each tool position. Also used as the vector length in the NCI file. For most tools, use 1 inch or 25 mm as the tool vector length. Entering a smaller value provides less onscreen display of the toolpath. When you like the toolpath display, change the tool vector length to a larger value to create a more accurate NCI file (this does not apply to RollDie operations).

Show taper

If you are using a tapered tool, tell Mastercam how you want to represent the taper. Select Lines to show a red line at each tool vector from the tip of the tool to the point on the rolled chain that the tool vector was generated from, giving a line along the taper angle of the tool. Select Surfaces to build surfaces that model the part being cut.

3-axis tool path

Select this option to produce a 3-axis toolpath. This is useful when a taper tool is used to run up and down the crease on inside sharp corners.

Z=Radius

Flat entities at Z = 0.0 roll around the chosen axis at the diameter that you enter in the Rotary Axis dialog box. If the entities are at a positive Z height, select Z = Radius to automatically set their rolled radius equal to their Z height.

Auto reverse

Select this to use an open chain as the center line of a slot or blade. The toolpath is generated in the selected chain direction, around the end, back down the other side, and around the start.

Slot / Slot width

Select Slot and enter a Slot width to use the chained contour as the center line of a slot. Mastercam offsets the tool from the center line to produce a slot of the given width around the chained contour. If you enter a Stock to leave amount, the leftover stock will be on the slot wall and towards the chained contour.

Prerolled

Use these options to tell Mastercam whether the geometry is flat or already rolled. Select No when the chained contours are flat, or select About X or About Y to indicate the axis around which the chained entities are rolled.
