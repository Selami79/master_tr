MD CD PST Rename Add-In

Use the MD CD PST Rename Add-In to rename any or all of the files associated with a machine definition:

  • The machine definition itself (.mcam-mmd, .mcam-lmd, .mcam-rmd, or .mcam-wmd file)
  • The control definition file (.mcam-control file)
  • The post (.pst or.mcpost file). If the post has been encrypted (or "binned"), Mastercam will automatically detect and rename the associated .psb or .mcbpost file.

When you use this Add-In to rename the files, Mastercam automatically updates the internal references to each file that are stored in the machine and control definition. If you rename the files yourself without using this Add-In, it is likely that these references will be broken.

NOTE

Run this Add-In only when no other data is loaded in Mastercam. This includes geometry, toolpaths, machine groups, and tool definitions. If the Add-In detects that the Mastercam database is not empty, it displays a warning message. If you choose to ignore that message and click OK to continue, all data in the part file is lost. To clear your database, save your current work, select Design from the Machine tab, and click File, New.

Follow this outline to rename your machine definition, control definition, or post.

  1. If necessary, create a clean Mastercam session.
    1. Save your current work.
    2. Select Design from the Machine tab.

    3. Click File, New.
  2. Open the Add-In. Press [Alt+C] and select MD_CD_PST_Rename.dll.
  3. Click the Configuration button to select back-up and logging options. Backing up your original files before making any changes is a recommended best practice. See MD CD PST Rename: Configuration dialog box to learn more.
  4. Select the folder that contains the desired machine definition. Click the arrow in the Folder field to select a recently used folder or click the button next to the field to select a new one.

  5. Select the machine definition.
    1. Click either the Mill, Lathe, Router, or Wire button to filter the available machine definitions by Mastercam product.
    2. Select the desired machine definition from the list.

    Mastercam automatically loads the name of the .mcam-control file associated with the machine definition, along with the posts (.pst or .mcpost file). Depending on how the control definitions are configured, there might be more than one post.

  6. Enter the new names for each file in the Rename to fields.

    • You do not need to rename all of the files. For any files that you do not wish to rename, simply leave the original name in the field.
    • You cannot change the file extension. For example, you cannot rename a post file with a.pst extension to a .mcpost file.
  7. Click the Apply button to rename the files and leave the Add-In open to continue the session, or click OK to rename the files and close the Add-In.
Parameters

Folder

The folder from which you can select machine definitions. Choose a recently used folder from the list or click the button next to the list to select a new one.

Mill / Lathe / Router / Wire buttons

Click the button to filter the machine definitions in the selected folder by Mastercam product.

Machine

Select a machine definition from the list. The list displays all machine definitions of the selected product type that are available in the selected folder.

Control

Displays the name of the control definition file (.mcam-control file) that is associated with the selected machine definition.

Post Files

Displays all of the posts for which a control definition exists in the selected .mcam-control file. Each .mcam-control file might have control definitions for several posts.

Rename to

Enter the new file name. You cannot change the file extension, only the name. For example, you cannot rename a .pst file to a .mcpost file.

Log

View changes that have been made recently. Click the Configuration button to disable logging or change the location where the log file is written.

Configuration

Configure backup and logging options.
