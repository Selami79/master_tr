Chaining Geometry for a Coons Surface

When you chain the geometry for a Coons surface, Mastercam prompts you to select the chains according to the layout of the patches. Coons patch boundaries are laid out in a grid of along and across curves. Along curves lie in rows that are parallel to the stepping direction. Across curves lie in columns that are perpendicular to the stepping direction. Mastercam asks you to select all the along curves for the entire surface, followed by the across curves.

The following picture shows how multiple patches form a 3-by-3 grid that defines the Coons surface. The curve segment which borders each patch is a separate chain.

  1. Start point
  2. Across columns
  3. Along rows

To create this Coons surface, click on the individual chains according to the numbers in the picture. (Note that you must be in single chaining mode.) First select the along curves for each row, then select the across curves for each column. When selecting each curve, make sure that the chaining directions are consistent.
