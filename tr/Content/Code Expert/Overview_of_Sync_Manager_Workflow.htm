Overview of Sync Manager Workflow

Because Sync Manager and Mastercam run in separate application windows, it is important to follow the proper workflow to ensure that they are properly coordinated. Follow this outline to work on your part in Sync Manager.

To work in Sync Manager:
  1. Open a Mill-Turn part in Mastercam.

  2. Click Post on the Toolpaths Manager. Mastercam generates the .iof file and opens it in Sync Manager.

  3. While you work in Sync Manager, leave the Mastercam window open with your part loaded.
  4. If you make changes in Sync Manager, press [Ctrl+S] (or click Save) to save the syncs and other changed values back to your Mastercam part file.
  5. Save the Mastercam part file from inside Mastercam. If you do not save your Mastercam part file, the data from your Sync Manager session will be lost when you exit Mastercam.
NOTE

The recommended best practice is to not open .iof files directly from inside Code Expert, but always from Mastercam.

To check Sync Manager status:

While you are working in Sync Manager, icons in the document's tab keep you alerted whether or not you are properly coordinated with Mastercam:

  • A green icon means that your Sync Manager session and your Mastercam session are properly coordinated. This means that if you have created syncs or changed other settings, the changes have been written back to your Mastercam part file.

  • A yellow icon means that your Sync Manager session and your Mastercam session are not coordinated. This means you have edited your machining job in Sync Manager (such as creating a sync) but have not yet saved the data back to Mastercam.

  • A red icon means that your Sync Manager session is not connected to a Mastercam session. This means that Mastercam has been closed or a different part has been loaded in Mastercam. If you create syncs or do any other work, you will not be able to save your work back to your part file.

  • A lock means you have made changes to the part in Mastercam, and so Sync Manager functionality is disabled. You cannot create syncs or change other toolpath settings. Click G1 again in Mastercam to regenerate the .iof file and restore Sync Manager functionality.

    If you make changes to your part in Sync Manager, you will not be able to go back to your part in Mastercam until you save your changes or close Sync Manager.
