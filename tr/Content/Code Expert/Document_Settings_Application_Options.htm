Document Settings page: Application Options

Use Document Settings page to configure Code Expert's document settings. Click a drop-down below to learn more.

Display
  • Line modifications: Displays a color code to the left of a modified line. Yellow indicates a change that can be undone, and green indicates a change that's been saved.
  • Whitespace: Displays space characters as dots and tabs as arrows.
  • Show visual glyphs for word wrap: Displays a symbol where word wrapping occurs. (The Word wrap option must be on.)
  • Whitespace trim mode:
    • None: No whitespace trimming.
    • Trailing whitespace when pasted: Trims trailing whitespace for pasted text.
    • Trailing whitespace when text replaced: Trims trailing whitespace when the entire document contents is replaced.
Margins
  • Indicator: Displays a vertical bar for indicator glyphs.
  • Line numbers: Displays line numbers to the left of the document.
  • Ruler: Displays a character ruler across the top of the document.
  • Selection: Displays a vertical bar for line-modification colors. Click and drag your mouse in the bar to perform line-based text selection.
Settings
  • Auto convert tabs to spaces: Converts all tab characters to the number of spaces specified in the Tabs/Tab size option.
  • Blank line when no selection: Specifies whether cutting or copying from a blank line replaces the Clipboard contents.
  • Drag with HTML: Specifies whether the Clipboard stores cut, copied, or dragged text as HTML, which retains syntax highlighting.
  • Drag with RTF: Specifies whether the Clipboard stores cut, copied, or dragged text as RTF, which retains syntax highlighting.
  • Scroll past document end: Enables the document to scroll a full page beyond the last line.
  • Scroll to caret on select all: Specifies whether the Select All command scrolls the window to the last selected line.
  • Wordwrap mode: If set to Word, breaks lines that are wider than the editor window. Text after the break continues on the next line.
  • Default encoding: Specifies the default encoding to use when opening text documents or creating new ones.
Scrollbars

These options specify whether to display the horizontal and vertical scroll bars.

Current Line

Choose Enable current line highlighting to automatically highlight the line on which the cursor is located.
