Viewing Sync Points in Multi-Stream Views

Code Expert gives you several tools for finding sync points or wait codes in a multi-stream document. These include toolbar buttons, connection lines, and colored highlighting.

NOTE

Code Expert uses the description in the machine's NC Configuration settings to identify syncs. See Setting Up Multi-Stream Views to learn more.

To find sync points

Use the Syncs buttons to browse syncs. You can move to the previous or next sync, or jump to the first or last sync in the file.

To view matching syncs

Move the cursor over a sync. Code Expert displays a line connecting the matching syncs in the other streams.

If the matching sync is not visible, right-click and choose Bring matching sync points into view.

Code Expert scrolls the other stream so that the matching sync point is visible.

You can also click the Align Syncs button on the toolbar.

Mastercam inserts white space in each stream so that the syncs are aligned. Mastercam also locks the streams so that they scroll together.

To view syncs in the Navigator pane

  1. Click Navigator at the bottom of the Code Expert window. The Navigator pane appears. (Note that the Navigator pane is available only in multi-stream views).

  2. Place the cursor over a sync point. The Navigator pane highlights streams that contain a sync point with the same ID.

    3. NOTE

    Mastercam only displays streams that are highlighted in the Navigator pane. To toggle a stream's display, click on it in the Navigator pane.

Highlighting syncs and connection lines

You can choose which colors to use for highlighting syncs and connection lines. Choose File, Options, Editor, NC, Fonts and Colors.
