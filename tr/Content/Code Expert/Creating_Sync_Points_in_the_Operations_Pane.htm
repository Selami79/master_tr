Creating Sync Points in the Operations Pane

Follow these steps to create sync points (often referred to as “wait codes”) in the Sync Manager.

  1. Click the source operation.

  2. Drag your mouse to the destination operation.

  3. Release the mouse button. Sync Manager creates the sync points, displaying them as red entries in the operation lists.

    4. NOTE

    When you create a sync point, Sync Manager vertically aligns the operation stream displays for easy comparison.

Tips
  • Operations are split into five major branches: toolchange, approach, motion, retract, and end. Sync point (wait code) output positioning in the resulting NC file is adjustable in the post code and is easily set by the machine definition (.machine) developer.
  • You cannot create syncs on a subprogram node, or drag an existing sync to a subprogram node.
  • Mastercam provides two sync types (primary and alternate). You can set a sync as either a primary or alternate sync type. The difference depends on your selected machine definition, and is set up your selected post. Certain machines/controls may not support alternate sync types, resulting in only the primary type as a sync type as an available option.

  • Click the down arrow for a sync to access some sync options and use Alternate Type. Alternate sync types display in blue. Primary sync types display in red.

    NOTE

    The down arrow does not appear if Alternate Types are not supported by you current machine definition.

  • Right-click a sync line to access a menu providing options for that sync.

    The option to Delete Sync Group is specific to that sync (and its partner). The second section of the menu that includes Delete All Sync Points, Renumber Sync Points, and Use Alternate Type As Default is available anywhere you right click in the operation pane when syncs have been defined. The Use Alternate Type As Default is available when alternate sync points are enabled for the current machine.

  • Click the down arrow next to the Machine Group name to access options, including starting and incrementing values for sync types.

    These options affect the entire job setup or machining job.
