Sync Manager Toolpath Options

As part of the workflow differences between Mastercam Lathe and Mastercam Mill-Turn, a number of toolpath options have been removed from the Mastercam interface and replaced with Sync Manager features. These include the following:

  • Coolant
  • Miscellaneous values (ints and reals)
  • Canned text
  • Selected control definition and Machine Group Properties settings

These options have been replaced because they tend to be machine-specific or are implemented in machine-specific ways. The exact options that are available in Sync Manager are defined in the .machine file by the machine developer and/or the post developer. For more information, click a drop-down below:

Exposing toolpath options

Toolpath options appear in the Sync Manager as small inverted triangles:

M

These options are typically available on individual branches of the operation tree, as well as at the group level. Click on a triangle to see what options are available. The options are different for each triangle; each triangle only shows the options that make sense for that particular branch of the operation. For example, the options for an Approach move might let you select Z-first or X-first motion:

Job setup options

Group-level options are available by clicking on the triangle next to the Machine Group name. These are options that affect the entire job setup or machining job. Typically, these include configuring sequence numbers and sync numbers, header information for the NC file, and other machine-specific settings.

Operation-level options

Settings that are limited in scope to an individual operation are available by clicking on the triangle next to the operation name. This typically includes selecting the coolant strategy for the operation.

Options for toolpath motion

Your .machine file typically has different options available for approach and retract moves, for both milling and turning operations. Each setting is limited in scope to the specific move.

Individual toolpath types have specific options. For example, Lathe Rough operations have options that are not included in milling or turning operations.

Sync options

After you create a sync, click on the triangle to see configuration options for the sync. Typically these will let you edit the number and select whether the sync is a primary or alternate sync.
