Hole panel

The Hole function automates the creation and editing of holes in solids, saving you from creating circles and extruding them. Select the hole position, then set the hole dimensions and type. You can create a cylinder or one of the following hole types:

  • Simple
  • Counterbore
  • Countersink
  • Counterdrill
  • Taper
  • Custom

The Solids Manager displays Hole operations in the Solids History Tree. Right-click an operation, and select Edit parameters to modify it.

How can I reuse common hole dimensions?

You can save commonly used holes in a library that you can use in future sessions of Mastercam. Mastercam converts hole dimensions stored in the library into the part's current unit of measurement (U.S. to metric, or metric to U.S.).

To save a custom hole definition into a library
  1. Use the settings in the panel to define your hole.
  2. Click the Add Presets button.

  3. Name the hole, and choose a category to store it in. Click OK.

To apply a preset hole dimension to a position

In addition to your own custom library, the Hole function includes categories of commonly used hole dimensions. Use the following procedure to apply any of these presets to your solid.

  1. Start the Hole function.
  2. Choose a Plane Orientation.
  3. Select one or more positions, then press [Enter].

  4. From the Template section of the panel, select a Category and a Preset hole.

    Use the Search field to quickly find a preset in that category.

How do I re-order or delete hole presets?

Use the right-click menu options to re-order your list of presets. To delete a hole preset, right click the preset and click Delete.

How do I create a custom hole?

Creating a custom hole is much like creating other types of holes. However, a custom hole is multi-segmented, with each segment having different attributes. The segments allow more complex holes than can be created with the other types. You can start with an existing hole type like a counterbore and then switch to Custom to build a basic tool that can be further customized.

When you choose a hole type of Custom, a table of hole attributes displays, as shown in the following image. Each row in the table is a segment. For each segment, you can define the top diameter, the depth, the bottom diameter, and the amount of taper. To add a segment, click the plus-sign button in the lower right.

Right-click to the left of the grid to expose a pop-up menu with options for moving, adding, and deleting segments, as well as adding chamfers and bottom angles.

How do I move or copy solid hole operations?

The Hole panel allows you to move or copy holes to create new hole operations from existing ones. Suppose you have a bolt circle of eight holes (see the following image) that you want to divide into two operations of four holes each. The following procedure shows how to complete that task.

  1. In the Solids Manager, double-click the hole operation that you want to divide.

  2. In the Hole panel, select the holes to move to a new operation. In the following image on the left, you can see that the user has selected every other hole, starting with the first. Mastercam highlights the selected holes in the graphics window, as shown in the image on the right.

  3. Right-click one of the selected holes, and choose Move to new operation.

    Mastercam moves the four selected holes into a new hole operation. The selected holes vanish from the Position list and in the graphics window, as they are no longer part of this operation.

  4. Click OK in the Hole panel, return to the Solids Manager, and click Regen. You now have two sets of four holes that you can edit independently.

Copying holes to new operations works the same way, except after selecting the holes, you choose Copy to new operation. Mastercam then creates a new operation with the selected holes, while leaving the original operation intact.

Can I break the edge of a hole?

If you need to apply an edge treatment to your hole to satisfy the design or manufacturing intent, you can use the Hole panel's edge treatment options to easily break hole edges in your solid parts. The parameters here create either chamfers or fillets on your hole edge, and control the size of the chamfer by specifying either its diameter or its width.

Additionally, you can smooth the bottom edges of through holes. Select the Match top and bottom option to apply your top edge treatment settings to your bottom edge or enter specific values into the Bottom Edge Treatment section in the function panel to create these dimensions independently.

NOTE

Edge treatments on some hole types are limited.

  • You cannot customize the top edge of a countersink hole.

  • You cannot modify the diameter of the bottom edge of a tapered hole.
