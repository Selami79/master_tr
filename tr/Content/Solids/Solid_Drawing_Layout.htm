Solid Drawing Layout dialog box

The Solid Drawing Layout dialog box lets you create a layout of different part views (for example, top, side, front, and isometric), arranged within a page you define. Use the Solid Drawing Layout dialog box to set the page size and orientation, and to select the view layouts to include.

You can create multiple drawing layouts in the same part file. Each time you create a new layout, make sure you select a new level for it (unless you want to replace an existing layout). If your part file contains multiple solids, after choosing the drawing layout level use General Selection to select the solids to include.

Parameters

Use template file

When selected, this option lets you choose a template file. Template files can contain wireframe geometry that defines the borders of your drawing layout and adds other elements such as labels. Mastercam merges the template entities into the current part file in the same level as the drawing.

Template entities are positioned in the solid drawing layout at the same location and orientation as in the template file. Use the displayed template file (default or last used), or choose Browse to select a different template file. When this option is selected, the paper size list and orientation fields are unavailable because the template file defines them.

Portrait

Sets paper orientation to portrait, where the longer dimension is along the Y axis in a Top Gview.

Landscape

Sets paper orientation to landscape, where the longer dimension is along the X axis in a Top Gview.

Paper size

Sets a paper size for the selected layout. Select a standard size code from the drop-down list, or select Custom. Selecting Custom enables text boxes where you enter dimensions along the X and Y axes. When you select a standard paper size, the X and Y fields display the page dimensions as read-only values. When you set or edit the paper size, the new page has its lower-left corner at X0Y0. The individual views are redistributed evenly in the new page.

X

Enabled when Paper size is set to Custom. Enter the dimension of the custom layout page along its X axis when oriented in a Top Gview.

Y

Enabled when Paper size is set to Custom. Enter the dimension of the custom layout page along its Y axis when oriented in a Top Gview.

Suppress hidden lines

Select to suppress display of hidden lines in the drawing layout. Deselect to view hidden lines in drawing layout.

Radial display angle

Select to add radial display lines in Solids Layout views that represent the surfaces of closed, circular solids (cones, cylinders, spheres, tori). Enter a value from 1.0 to 180.0. Decreasing the angle increases the number of lines along a circular face. Deselect to remove radial display lines in Solids Layout views from the surfaces of closed, circular solids (cones, cylinders, spheres, tori) .

Scale factor

Mastercam automatically scales each view by the scaling factor you enter here.

  • Enter numbers as decimals: for example, use .75, not 75%. To enlarge scale, enter a value greater than one (1.25, 1.5, etc.). To reduce scale, enter a value less than one (0.9, 0.75, etc.). Zero preceding decimal is optional.
  • Use a scale of less than 1 if all the drawings do not fit on the page size you have selected.

Level

Select which level to place the layout on.

Layout method

Choose one of the following layouts:

  • Favorite: Applies a set of saved parameters to the solid layout. If you have not created a favorite, this option will not display.
  • 4 View DIN: Bottom, Front, Left, and Isometric views
  • 4 View ANSI: Top, Front, Right, and Isometric views
  • 3 View DIN: Bottom, Front, and Left views
  • 3 View ANSI: Top, Front, and Right views
  • 1 View Isometric: a single isometric view

Import favorites

Select to import your favorite settings from a previous Solid Layout. This button is only available if you have previously used the Solid Layout function and selected the Save all to favorite checkbox.
