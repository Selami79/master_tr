Editing Solid Models

Mastercam provides a number of tools and functions for editing solid models. Click the drop-downs below for more information.

CAUTION

You can only undo actions applied to solids via Model Prep functions. You cannot delete base operations or tool bodies.

When you edit solid parameters or geometry, Mastercam may ask for additional input, depending on the parameters that you edited. Follow the prompts on your screen.

After you change a solid operation, Mastercam marks the edited operation and its corresponding solid as dirty in the Solids Manager. Regenerate the solid to see your changes reflected in the model. This also applies to any associated toolpaths. Only the toolpath that is directly affected by the change in the solid model is marked dirty. Any other operations are not marked dirty.

Importing solids and solid assemblies

Mastercam allows you to import other CAD file types for your machining jobs. This includes STEP, SOLIDWORKS, 3DXML, or SAT files. When importing solid and solid assemblies, Mastercam's file importers use information from the incoming file to populate both the Solids Manager and the Levels Manager. This includes automatically creating groups and sub-groups for incoming models that contain assemblies, sub-assemblies, and component parts.

For more information, refer to the Levels and Solid Bodies from Imported Assemblies Help topic.

Assign colors to solid faces and features

You can assign different colors to individual solid faces and features. Use these colors to more easily select solid areas for toolpathing and other operations. The Color options are available on the Model Prep tab.

When working with solid colors, be aware of the following:

  • Face and feature colors translate to surfaces when creating surfaces from solids. Attributes options in the Surface From Solids function must be active.
  • Select the Use surface colors option on the Solids property page to map surface face colors to the solid faces when creating solids from surfaces.
Change the order of solid operations

The Solids Manager lists the operations that define each solid in the current file. The operations are listed in the order in which they are performed on the solid. You can change the order by dragging operations to new positions in the operation list. Mastercam rebuilds the solid based on the new operation order.

When dragging an operation, the selected operation is inserted into the operation on which you release it. The cursor changes to a downward arrow if the move is allowed, or to a circle with a slash through it if the move is not allowed.

When you move operations, the following restrictions apply:

  • You cannot move a base operation.
  • You cannot move an operation to a position before a base operation.
  • If an operation is dependent on one or more other operations (for example, a fillet on a boss), you cannot move the dependent operation to a position before the operations on which it depends.
NOTE

Using the Group commands on the right-click menu, you can organize solids into folders by dragging and dropping solids to and from the groups.

Delete solids and solid operations

There are two ways to delete a solid. You can either select the solid in the graphics window and press the [Delete] key, or you can choose the Solids Manager, right-click on the solid or solid operation you want to delete, and then choose Delete.

When you delete operations, Mastercam marks the corresponding solid dirty. Regenerate the solid to see your changes reflected in the model. Also, If you delete a Boolean operation, its tool bodies are restored as distinct solids that are no longer associated with the target body. Mastercam automatically deletes all operations whose input (selected faces, edges, bodies) is dependent on the deleted operation.

Duplicate solids

You can make exact copies of solids, including tool bodies. Each copy is an active solid that you can select and edit. This function is particularly useful if you want to use the same tool body (A solid that is added to, removed from, or overlapped with the target body during a Boolean operation) to affect different target solids.

To duplicate a solid, choose the Solids Manager, right-click on the solid you want to copy, and then choose Duplicate solid.

  • Mastercam automatically renames duplicated solids according to Windows standards. This means that when you duplicate a solid, Mastercam appends "-Copy" plus a number to indicate if there is more than one duplication.
  • If you copy a solid that has already been duplicated, Mastercam continues to add "-Copy" to the name of the solid.

Mastercam adds both types of duplicated solids to the bottom of the Solids History Tree.

In the example above:

  • MySolidBlock - Copy, MySolidBlock - Copy (2), and MySolidBlock - Copy (3) are duplicates of the original, MySolidBlock
  • MySolidBlock - Copy (2) - Copy is a duplicate of MySolidBlock - Copy (2)
  • MySolidBlock - Copy - Copy is a duplicate of the initial duplication, MySolidBlock - Copy

We recommend that you give your solids unique names to minimize confusion!

Edit solid attributes

Use Analyze Entity to view and edit solid attributes. Entity attribute fields for level, color, and lines (style and width) appear the dialog boxes that display when you apply the function to solids.

Regenerate solids

Regeneration is the process of rebuilding a solid so that it matches its associated operations. Regeneration is required when a solid becomes dirty or invalid due to changes that you make to the parameters and/or geometry of one or more of the solid’s operations. When you regenerate a solid, Mastercam incorporates your changes into the solid and restores the solid to a clean state, if possible. Regeneration fails if a solid has one or more invalid operations.

When making many changes to a model, regenerate often. Then, if regeneration fails, you need to make fewer corrections to return the operation to a valid state. In the Solids Manager, you can use the right-click menu to regenerate a selected solid.

To regenerate solids:

  • Single solid: Choose the Solids Manager, right-click on the solid or any component operation, then choose Regen solid.
  • All solids: Choose the Solids Manager, and then choose Regen all at the top or choose Regen all solids from any right-click menu.
Remove solid history

Removing a solid's history strips all operations and features from a selected solid model and converts the solid into a "brick" in the Solids Manager.

  • To remove a solid's history from the Solids Manager, right-click on a solid, and then choose Remove solid history.

  • To choose multiple solids, choose Remove History from the Model Prep tab. Then pick the solids using any selection method in Mastercam's Selection Mode list.
