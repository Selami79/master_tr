Groove Finish Parameters tab

Use this tab to create the finish passes for a groove toolpath. Select the Finish checkbox to activate the options on this tab and create finish passes. Deselect this checkbox if you only want to create a roughing operation.

Use the pictures on the tab as a guide to entering your finishing parameters. Select options on the right side of the tab to enable advanced grooving features, such as lead in/out moves and toolpath filtering.

NOTE

With groove finish passes, if you set compensation in control to automatic, Mastercam changes the compensation in control direction for the clockwise and counterclockwise cut directions.

Parameters

Finish

Select to activate the options to create a finish pass.

Number of finish passes

Enter the number of finish passes. If you choose to make more than one finish pass, then Mastercam determines the offset direction automatically.

Finish stepover

Determines how much stock is removed with each finish pass.

Stock to leave in X, Stock to leave in Y

Determines how much stock remains in the X direction when the operation is finished. Typically, this would be removed by a separate finish operation.

If you are using a Y-axis tool, the value of this field is a Y-axis coordinate, not X. When this happens, the name of this field references the Y axis instead of X.

Stock to leave in Z

Determines how much stock remains in the Z direction when the operation is finished. Typically, this would be removed by a separate finish operation.

Tool back offset number

Defines a second offset number for finishing the groove walls. Mastercam creates one finish pass for each of the walls, and posts this number for the second wall finish pass.

CCW

Starts the finish pass in a counterclockwise direction. The tool then retracts and completes the finish pass in a clockwise direction.

CW

Starts the finish pass in a clockwise direction. The tool then retracts and completes the finish pass in a counterclockwise direction.

Rapid

Retracts the tool at the rapid feed rate. This option is not recommended if you are creating an angled groove.

Feed rate

Converts rapid moves into fast-feed rate moves. The rough and finish passes use the same fast-feed rate, so the interface displays the rate on both the rough parameters and finish parameters pages.

  • in/rev or mm/rev: Calculates the value in inches per revolution or millimeters per revolution.
  • in/min or mm/min: Calculates the value in inches per minute or millimeters per minute.

Corner dwell

This value represents the dwell time at the bottom of the groove. To create a dwell, choose Seconds or Revolutions to specify the dwell units and enter a value in the box. The dwell is engaged at the intersection of the geometry wall/corner to the flat bottom.

NOTE

Corner dwell may not work correctly in odd-shaped grooves.

Middle overlap

Sets the groove finish overlap at the middle of the floor. The first pass cuts to the middle of the floor from one side, and the second pass cuts to the middle of the floor from the other side.

Select overlap

Select to return to the graphics window and select the position where the first finish cut ends and the amount that the first and second finish cuts overlap.

Distance from 1st corner

Determines the position where the first finish cuts ends.

Overlap between passes

Determines the amount that the first and second finish cuts overlap.

Compensation type

Tool compensation adjusts the toolpath to allow for the cutting tool's radius. The options that are available to you are determined by the active control definition.

  • Computer: Mastercam computes the compensated toolpath and does not output control codes for compensation.
  • Control: Mastercam outputs control codes for compensation and does not compute the compensated toolpath.
  • Wear: Mastercam both computes the compensated toolpath and outputs control codes for compensation. Compensation direction in the computer and control are the same.
  • Reverse wear: Mastercam both computes the compensated toolpath and outputs control codes for compensation. Compensation direction in the computer and control are opposite.
  • Off: Mastercam programs the tool tip directly on the chained geometry and does not output compensation codes.

Optimize cutter compensation in control

Eliminates arcs in the toolpath that are less than or equal to the radius of the tool and helps prevent gouging. The default setting for this field is set on the Cutter Compensation, General page in the Control Definition. This field is available only when Cutter compensation is set to Control on the parameters tab, and applies only to that setting.

Roll cutter around corners

Inserts arc moves around the corners in the toolpath. Choose the type of arc move:

  • None: Guarantees all sharp corners.
  • Sharp: Only rolls the tool around sharp corners of 135 degrees or less.
  • All: Rolls the tool around all corners and creates smooth tool movement. The radius of the arc moves equals the radius of the tool.

Wall backoff

Determines how far the tool backs away from the groove wall before retracting. The tool backs off at a 45-degree angle, and will not back off if it is going to hit the part. Choose a % of tool width or a Distance and enter a value.

Ignore undercuts

Select to ignore undercuts on a grooving operation. This is used when cutting a groove with an undercut using a flat tool.

Lead in

Select to have Mastercam add lead-in moves to the toolpath. Click to edit the moves.

Filter

Select to eliminate unnecessary tool moves in the toolpath to create smoother movement. Click to configure toolpath filtering options.
