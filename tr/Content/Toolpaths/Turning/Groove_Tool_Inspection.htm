Groove Tool Inspection dialog box

Use the Groove Tool Inspection dialog box to establish points at which the machine can be programmed to stop for tool inspection during grooving cycles. You can also set these programmed stops to perform other tasks such as clear chips and gauge your part. You can set the conditions on which the tool inspections are performed, such as:

  • Between grooves
  • After the first plunge
  • After each depth cut
  • After specified number of plunges

You can use reference points to safely get the tool in and out of the cut, and then send the tool to either a selected point or to the home position for the tool inspection.

Parameters

Use reference points

Select to open the Reference Points dialog box, where you can select/create the reference point(s) you want to use for your tool inspection. The reference point is a location that the tool goes to on the way to and/or from the tool inspection position. The reference point is also used on the initial approach from home and the final retract going home. If you do not select or create a reference point in the Reference Points dialog box, the Use reference points checkbox remains deselected.

Reference Points

Click to open the Reference Points dialog box, where you can edit your previous settings for tool inspection reference points. This button is only available if you have established reference points for the tool inspection.

Home position

Select to set the machine tool's default Home position as the tool inspection position.

User defined

Select to define a specific position for your tool inspection using the User defined options.

D or X (depending on your construction plane)

Displays the D or X coordinate for the position of the tool inspection. Select the checkbox and enter a value in the field, or click Select to return to the graphics window and select a position to define the coordinates of the tool inspection point.

Z

Displays the Z coordinate for the position of the tool inspection. Select the checkbox and enter a value in the field, or click Select to return to the graphics window and select a position to define the coordinates of the tool inspection point.

Absolute

Positions the tool inspection point relative to the tool plane origin.

Incremental

Positions the tool inspection point relative to the last point in the toolpath before the tool inspection.

Each groove

Sets the tool inspection to occur after each groove.

Each depth cut

Sets the tool inspection to occur after each depth cut.

First plunge

Sets the tool inspection to occur after the first plunge.

Number of plunges

Sets the tool inspection to occur after a defined number of plunges. Enter the number of plunges in the field to the right of this option.

Output comment

Select to output a comment to the NC file. Enter a comment to be included in the NC file where the tool stops. Typical comments include "inspect tool", "clean out chips", or any other relevant information regarding that programmed stop.
