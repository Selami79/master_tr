Creating a Steady Rest Operation: Mill-Turn

Mastercam Mill-Turn gives you several ways to include steady rest operations in your part.

  • Create steady rest operations for a standalone steady rest.
  • Create steady rest point operations for a turret mounted steady rest.
  • Select a part handling strategy that includes steady rest operations.

You can also follow the tool or turret with the steady rest while machining the part. Expand the section below to learn more about each topic.

Before you can create either type of steady rest operation, you need to select the steady rest in Job Setup. See Selecting a Steady Rest: Mill-Turn Job Setup to learn more.

NOTE

Mastercam Lathe and Mill-Turn support steady rests differently. Each uses different steady rest components and operations. See Lathe Steady Rest dialog box to learn about steady rest support in Lathe.

Creating a steady rest operation for a standalone steady rest

Follow these steps to create a steady rest operation. This type of operation is used by standalone steady rests, not turret mounted steady rests.

  1. Select Steady Rest from the Turning contextual tab.

    Mastercam displays the Steady Rest dialog box.

  2. If your .machine file is equipped with more than one steady rest, select the primary steady rest that will be used in this operation. See the section Programming multiple steady rests below to learn more about programming multiple steady rests.

  3. Select the Move type. Choose from the following actions:

    • Linear position. Moves the steady rest to a specified location. You can choose to create a rapid move or a linear move at a specified feed rate.

    • Clamp. Clamps the steady rest. The steady rest must already be in the clamping position (X = 0) before you create this operation.

    • Unclamp. Unclamps the steady rest.

    • Retract. Returns the steady rest to its home position. You can only select this option after an Unclamp operation. You can choose to create a rapid move or a linear move at a specified feed rate.

    • Follow is reserved for use in part handling strategies; these can only be created by machine developers.

  4. Select the Spindle origin. Mastercam populates this list from your Job Setup information. Typically you will choose between the left and right spindle origins.

  5. If you wish you can select a different Data stream. The default stream that Mastercam displays is set in the machine definition by the machine developer.

  6. For Linear position operations, enter the coordinates of the destination point. You can also click the Select button to select a point from the graphics window.

  7. For Linear position or Retract operations, choose whether the move type will be a Rapid move or a Feed move. If it will be a Feed move, enter the feed rate.

  8. For Clamp or Unclamp operations, you can also specify a Dwell time.

  9. When you select Clamp, Mastercam activates the Use boundary in collision detection option. When this option is selected, Mastercam applies a 2D wireframe boundary around the steady rest when it calculates subsequent toolpaths.

    This is separate from the collision checking that Mastercam performs when you simulate your operations. This is a boundary that is applied before you create the toolpaths.

    The boundary is defined when the component is created and is part of the component properties. The following picture shows an example of a steady rest boundary:

    The boundary will be in effect until the steady rest is unclamped.

    NOTE

    The steady rest boundaries will not be visible unless the Update stock option has been turned on.

    If the setting for your operation is Do not update stock, click on the text to toggle the setting. This option is not available in the toolpath dialog box like it is for Lathe operations.

  10. Click OK to create the operation.

Creating a steady rest point operation for a turret mounted steady rest

Follow these steps to create a steady rest point operation. This type of operation is used by turret mounted steady rests, not standalone steady rests.

This operation differs from the standalone steady rest operation because each operation can contain several points or events, whereas the standalone steady rest operation requires a separate operation for each event.

  1. Select Steady Rest Point from the Turning contextual tab.

    Mastercam displays the Steady Rest Point dialog box.

  2. Click the Axis Combination / Spindle Origin button. Mastercam displays the Axis Combination / Spindle Origin dialog box.

  3. Select the desired Spindle Origin. Mastercam populates this list from your Job Setup information. Typically you will choose between the left and right spindle origins.

  4. Click OK to return to the Steady Rest Point dialog box.

  5. Build your list of points and steady rest events in the Motion Points window. After adding a point, click in the Type cell to select the type of motion. You can choose from G0 (interpolated rapid move), G0 - Break (dog-leg rapid move), or G1 (linear move at feed rate).

    See Steady Rest Point (Mill-Turn) dialog box to learn more about using the buttons and other controls in the window.

  6. Enter a Tool number or Offset number. Typically these are equal to the turret index position in which the steady rest is mounted.

  7. When you select Clamp, Mastercam activates the Use boundary in collision detection option. When this option is selected, Mastercam applies a 2D wireframe boundary around the steady rest when it calculates subsequent toolpaths.

    This is separate from the collision checking that Mastercam performs when you simulate your operations. This is a boundary that is applied before you create the toolpaths.

    The boundary is defined when the component is created and is part of the component properties. The following picture shows an example of a steady rest boundary:

    The boundary will be in effect until the steady rest is unclamped.

    NOTE

    The steady rest boundaries will not be visible unless the Update stock option has been turned on.

    If the setting for your operation is Do not update stock, click on the text to toggle the setting. This option is not available in the toolpath dialog box like it is for Lathe operations.

  8. Click OK to create the operation.

Using steady rest operations in part handling strategies

In addition to creating individual steady rest operations, Mastercam also lets you incorporate steady rest operations into your part by selecting part handling strategies. These are created by your machine developer and integrate steady rest operations with spindle and chuck commands. Machines that are equipped with steady rests typically include a mix of part handling strategies both with and without steady rest events. This picture shows an example:

Once you select the desired strategy, enter the destination of each steady rest.

  1. In the Operations list, look for each Position Steady Rest operation.

  2. Enter the coordinates of the destination Position.

Your machine developer can also create part handling strategies that only include steady rest operations if there are sequences of steady rest events that you need to reuse often.

Following the tool with the steady rest

Mastercam lets you create steady rest operations that follow a tool or the other turret while the part is being machined. However, there are several guidelines that you need to follow. These apply to both standalone and turret mounted steady rests. The following outline summarizes the steps and requirements.

  1. Create a steady rest operation that clamps the steady rest. Choose a location on your part where the diameter will not change while the steady rest follows the part. The follow operation will not be successful if the diameter changes.

    To be clear, the turning operation can include sections of the part with different diameters, but the portion of the part over which the steady rest travels must be a constant diameter.

  2. Once the steady rest is clamped, create the turning operation. The only supported operations are rough and finish turning operations.

  3. Create the steady rest operation that will follow the turning operation. Use only Linear or feed rate moves, not rapids.

  4. Ensure that the feed rates for the turning operation and the steady rest are the same:

    NOTE

    When you create the turning toolpath, Mastercam lets you specify the units for the feed rate, but not for the steady rest operation. You do not need to worry about the units, you only need to ensure that the number matches. When the steady rest and turning operations are properly synced (see step 6 below), Mastercam will adjust for the different units in your post and your code will be output correctly.

  5. When you have finished creating operations, click the G1 button to post your operations to the Sync Manager.

  6. In the Sync Manager, sync the turning operation and the steady rest follow operation as shown here. Create one sync between the beginning of the steady rest operation and the beginning of the motion block in the turning operation, and another sync between their ends.

    7. Notice that the first sync must be placed before the Motion block, not the Approach move as shown here:

You can only follow one operation at a time. For each turning operation that you want to follow, you need to create a separate steady rest operation.

Programming multiple steady rests

If your .machine file is equipped with multiple standalone steady rests, Mastercam lets you link one or more steady rests so that they move as one unit. You can do this for linear moves and follow operations; however, each steady rest must be clamped and unclamped individually.

The first step is to select the primary steady rest component for the operation. This is referred to as the driven component. Select it from the Driven Component window. This is the steady rest component that will be moved to the destination coordinates for the operation.

If the operation and its context allow for multiple steady rests, Mastercam activates the Linked Components option. To link additional steady rests to the primary component, select Linked Components, then select the desired steady rests.

The following sequence of operations is typical for a pickoff/cutoff application with multiple steady rests.

  • The yellow and green operations position each steady rest with separate operations, since each is going to a different location on the part.

  • The purple operation moves both steady rests in tandem as linked components, following the pickoff spindle home after the part transfer.

Mastercam supports several different kinematic arrangements of multiple steady rests:

  • Multiple independent steady rests. These are each mounted on separate linear axes so that they can be positioned independently. Program them as described in this section.

  • Tandem steady rests. These consist of multiple steady rests mounted as a block on a single set of linear axes. In this case, you do not need to select a linked component. Select the primary steady rest as the Driven Component, but you do not need to select a linked component. However, you still need to create separate operations to clamp and unclamp each steady rest.

  • Multiple tandem steady rests. In this case, in the Linked Components window select only the single main steady rest from each block.

NOTE

Mastercam only allows you to link steady rests if they are in the same stream. To link steady rests from different streams, create separate operations for each steady rest. Then, sync the operations in the Sync Manager.
