Steady Rest (Mill-Turn) dialog box

Use this dialog box to create an operation for a standalone steady rest in Mastercam Mill-Turn.

Each operation can be one of the following types:

  • Clamp

  • Unclamp

  • Linear move

  • Retract

Create a separate operation for each individual event; for example, moving the steady rest to a clearance position, then to the part, then clamping it. This gives you maximum flexibility for how the steady rest actions interact with your other operations. For example, after clamping the steady rest on your part, you can program the steady rest to follow the tool or turret while the part is being machined. You can also create syncs to individual steady rest events in the Sync Manager.

Begin by selecting the Move type, then complete the other settings. Mastercam enables or disables the other settings based on the selected Move type. See Creating a Steady Rest Operation: Mill-Turn to learn more about creating a steady rest operation for a standalone steady rest.

Machines with multiple steady rests

If your .machine file is equipped with more than one steady rest, they will be listed in the Driven Component window. Select the steady rest to be used in this operation from the list displayed in the window.

For linear and follow moves, Mastercam lets you link one or more additional steady rests that will be moved in tandem with the driven component. To do this, select the Linked Components option and select the desired additional steady rests. The option will only be available in situations when Mastercam allows you to link multiple steady rests, otherwise it will be grayed out.

NOTE

You can only link steady rests that are in the same stream.

NOTE

Mastercam Mill-Turn supports steady rests in two different ways:

  • Steady rest operations are used to program standalone steady rests.

  • Steady rest point operations are used to program turret-mounted steady rests.

Steady rest operations in Mastercam Lathe are different than Mill-Turn steady rest operations. See Lathe Steady Rest dialog box to learn more about Lathe steady rest operations.

Parameters

Driven Component

If your .machine file is equipped with more than one standalone steady rest, select the one that will be used for this operation.

Certain operations let you link multiple steady rests so that they move together.

  • Use this window to select the primary component that will be driven to the specified destination.

  • Use the Linked Components window to select the steady rests that will be linked to this one.

Move type

Select the Move type before entering any other settings. Each steady rest operation contains a single move or action. Choose from the following:

  • Clamp. Clamps the steady rest onto the part. The steady rest must already be in the proper position at X = 0 before you can clamp it; no motion occurs with this action. Use Linear moves to position the steady rest before selecting Clamp.

  • Unclamp. Unclamp or release the part from the steady rest.

  • Linear position. Move the steady rest to a different location. You can choose to output this as a rapid move or at a specified feed rate.

  • Retract. Retracts the steady rest to its home position. You can choose to output this as a rapid move or at a specified feed rate.

  • Follow. This is only available to users with a Developer license. It is used when creating part-handling strategies that incorporate steady rest operations.

The other options on this dialog box are enabled or disabled based on the selected Move type.

Spindle origin

Select the spindle origin that you would like to use for the operation. Mastercam populates this list from the information in your Job Setup; for example, if you are using a twin-spindle machine, typically you would be able to select from the right or left spindle origin.

Z

The Z axis coordinate of the selected Spindle origin.

Work offset

The work offset assigned to the selected Spindle origin.

Data stream

Select the stream in which the steady rest commands will be output. Mastercam selects a stream by default based on information from your machine definition but you can override it here by selecting a different stream.

Position (X, Z)

Enter the destination coordinates when the steady rest is being repositioned. These fields are available when Move type, Linear position is selected. Enter coordinate values directly or click the Select button to choose a destination point from the graphics window.

Rapid

Outputs the steady rest movement as a rapid move instead of a feed rate move.

Feed

Outputs the steady rest movement as a linear move at a specified feed rate. After selecting Feed enter the desired feed rate in the adjacent field. For Follow operations generated by part handling strategies, Mastercam automatically uses the feed rate of the parent operation.

Dwell

You can add a dwell command following the steady rest action by entering a value here.

Use boundary in collision detection

Select this option to have Mastercam collision check against the steady rest boundary when creating subsequent toolpaths. This option is available for Clamp events and is applied as long as the steady rest remains clamped.

  • Mastercam always collision checks against the steady rest component models when you are running Simulation. The option on this page is applied while Mastercam is generating subsequent toolpaths.

  • The boundary used by this feature is a 2D wireframe boundary that is specified as part of the component properties of the steady rest. See Geometry tab: Machine Component Manager (Mill-Turn Steady Rest) to learn more.

  • You can select or deselect this option as the default for your part on the Work Support page in your Job Setup.

Linked Components

Certain operations let you link multiple steady rests so that they move together. When this is allowed, the Linked Components option activates so that you can select the additional steady rest components.

  • Use the Driven Component window to select the primary component that will be positioned at the specified destination.

  • Use this window to select the additional steady rests that will be linked to the primary one.

If Mastercam does not allow multiple steady rests to be linked for the current operation, this option will not be available.

Comment

Enter a Comment to help identify the operation in the Toolpaths Manager. These can be output in the NC file when you post if you select the proper option in your machine group properties.
