Lathe Stock Transfer dialog box

Use the Lathe Stock Transfer dialog box to create an operation which transfers the stock to a chuck on the other spindle. This lets you program operations on the opposite side or back of a part in the same Mastercam file. Stock transfer operations have the following general components:

Handling the geometry

Choose Transfer geometry if you want to create a copy of your part geometry aligned with the repositioned stock, leaving the original geometry unchanged. Click Select to return to the graphics window and select the specific entities or chains that you want to copy. If you wish, you can specify a different level for the new geometry.

Locating the stock

Enter the original and new positions for the stock. Click Select to return to the graphics window and select the locations, or choose From stock back face to automatically use the back face of the stock as currently calculated by Mastercam. The point you choose does not have to be on the face of the stock. You can choose any convenient reference point.

Controlling the chucks

Use the Main spindle position section to enter the location of a reference point on the chuck that is currently holding the stock. Enter the current location in the Original position fields. Enter the location to which the chuck will move after the operation in the Final position fields. The stock transfer will occur while the chuck is at the original position. Enter the values or click Select to return to the graphics window and select the locations.

Then, enter the coordinates of a reference point of the chuck on the sub-spindle. Enter its Original position, and the location where it will pick-off the part. The relationship between the pick-off point and the current face of the stock determines how much of the stock will be clamped. For example, if at the time of the transfer the stock face is at Z=-1, and the pick-off location is Z=-3, then 2 inches of stock will be clamped.

If you choose Get diameter from stock, Mastercam calculates the current stock diameter at the pick-off position and clamps the chuck to that diameter.

The final location of the chuck after it picks off the part is determined by the Transferred position for the stock.

NOTE
  • Before programming a stock transfer operation, you need to define the stock and chuck boundaries in the Stock Setup tab. Make sure you only define stock for a single spindle; Mastercam will not create the operation if stock has been defined for both spindles.
  • The sub-spindle does not have to be defined in Stock Setup, but if it is not, you will not see it simulated on the screen.
Parameters

Transfer geometry

Select to make a copy of your part geometry aligned with the new stock model. Click Select to return to the graphic window and pick the geometry to be copied. Use the Selection Bar to help select the geometry.

Blank original geometry

Select to remove the original geometry on the screen from view. Mastercam does not delete the entities from your part file, so you can Unblank them later to see and work on them.

Keep the same

Places the transferred geometry on the same level as the original geometry.

Copy to level

Lets you specify on which level you want to place the transferred geometry.

Offset by

Mastercam calculates a destination level based on the level number of the original geometry plus the number you enter here.

Stock position: Original position

Enter the coordinates of a reference point on the original stock or click Select to pick a point. Mastercam moves the stock so that this point is positioned at the location entered in the Transferred position Z field. Enter a value or choose Select to pick a point. From stock back face

Select to automatically use the Z coordinate of the back face of the stock as defined in the Machine Group Properties. Mastercam calculates the effects of any machining operations that have affected the stock since it was defined.

Stock position: Transferred position

This is the location to which the stock's original reference point is moved. Enter a value or click Select to pick a point.

Main spindle position: Original position

Enters the coordinates of a reference point on the chuck in the source spindle. This will be the location of the chuck when the stock is transferred. Enter the values or click Select to pick a point.

Main spindle position: Final position

This is the location of the reference point after the operation. The chuck moves to this location after the stock has been transferred. Enter the values or click Select to pick a point. This parameter is available only if the chuck in the source spindle is defined in Machine Group Properties.

Sub-spindle position - Original position

Enter the coordinates of a reference point on the chuck in the destination or sub-spindle or click Select to pick a point.

Sub-spindle position - Pick-off position

Enter the coordinates that the destination chuck will move to receive the stock or choose Select to pick a point.

Get diameter from stock

Select to have Mastercam calculate the diameter of the stock at the pick-off position and automatically clamp the chuck to that diameter. The calculated diameter includes the effects of any machining operations that have affected the stock since it was defined in the Machine Group Properties. If the Z coordinate of the pick-off position does not lie within the stock boundary, Mastercam uses the diameter of the stock face.

Custom Parameters

Select to apply the values for any custom parameters that have been defined for this operation. Click to see or edit the custom parameters. Use the Lathe Part Handling section of the Text page in the Control Definition Manager to define the custom variables.
