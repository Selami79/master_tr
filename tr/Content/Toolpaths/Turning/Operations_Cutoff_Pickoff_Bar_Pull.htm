Operations: Cutoff/Pickoff/Bar Pull tab

Use the Operations, Cutoff/Pickoff/Bar Pull dialog box tab to define the following parameters for each of the operations you create in the Cutoff/Pickoff/Bar Pull Setup tab:

  • Toolpath group name
  • Comment for operation
  • Operation included in post
  • Custom parameters for operation
  • Approach clearance (for applicable operations)
Parameters
NOTE

Parameters and options shown on the right side of the dialog box change depending on the operation you have selected in the Operations list.

Toolpath group name

Defines the toolpath group name in which the operations will be generated. Edit the default group name as needed. Mastercam toolpath groups must have unique names. If you create two separate toolpath groups with the same group name, the Duplicate Toolpath Group Name - Cutoff/Pickoff dialog will appear so you can resolve the conflict.

Operations list

Contains the list of operations that will be programmed based on the operation type selected on the Cutoff/Pickoff/Bar pull tab. Controls and parameters specific to the selected operation display to the right.

Comment

Enter comments for the toolpath. Comments help identify the operation in the Toolpaths Manager. These can be output in the NC file when you post if you select the proper option in the Files tab of the Machine Group Properties.

From operation

Click to use the comment from the currently selected cutoff operation. This button is active if the selected cutoff operation has a comment. This option is unavailable for other types of operations.

Do not post-process this operation

Select if you want this operation to be skipped during post-processing. If selected, this operation will appear ghosted in the Toolpaths Manager.

Custom parameters

Click to open the Custom Parameters dialog box and enter values for any custom parameters that have been defined for a part handling. For each post processor, you can define up to ten variables with floating point values, and ten variables with integer values. This option is unavailable for Pre-position cutoff tool and Cutoff the part operations.

Misc. values

Click to open the Miscellaneous Values dialog box and enter values for custom parameters that have been defined for the current post processor. For each post processor, you can define up to ten variables with floating point values, and ten variables with integer values. This option is only available for Pre-position cutoff tool operations.

Select the cutoff tool pre-position point

Return to the graphics window to select a point that defines the tool position for the pre-position operation, and populates the pre-position X and Z coordinate fields. If the X coordinate of the point is less than the maximum radius of the pickoff spindle chuck jaws, the Pre-position X coordinate field is not updated and a warning appears.

Pre-position X coordinate

Enter the exact X coordinate that the tool will move to before the pickoff spindle advances to support and pick off the part. To avoid tool collisions, this value may not be less than the maximum radius of the pickoff spindle chuck jaws. This option is only available for Pre-position cutoff tool operations.

Pre-position Z coordinate

Enter the exact Z coordinate that the tool will move to before the pickoff spindle advances to support and pick off the part. This option is only available for Pre-position cutoff tool operations.

= Cutoff X

Click to copy the Cutoff X coordinate from the Cutoff/Pickoff/Bar Pull Setup tab to = Cutoff X, so that the pre-position point matches the cutoff point.

= Cutoff Z

Click to compute the pre-position Z coordinate to match the cutoff Z coordinate, and populate the Pre-position Z coordinate field. This ensures that the cutoff tool will be pre-positioned at the proper point to begin the cutoff operation.

Approach clearance

Enter or edit the distance used to compute the chuck Z coordinate for the operation.

Cutoff Operation

Browse to select an operations library that contains cutoff operations. Mastercam adds the selected operation to the Toolpaths Manager.

Use tool from operation

Choose to use the tool stored in the operations library with the cutoff operation. If you have selected a tool to define the width of the cutoff operation on the Cutoff/Pickoff/Bar Pull Setup tab, that tool is automatically selected here by default. You can override this selection. This option is only available for Cutoff the part operations.

Select a tool

Choose to open the Tool Manager and select a tool. If you cancel out of the Tool Manager without defining a tool, Mastercam re-selects the Use tool from operation option. This option is only available for Cutoff the part operations.
