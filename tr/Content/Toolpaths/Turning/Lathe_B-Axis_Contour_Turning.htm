A-Axis, B-Axis Contour Turning panel

Use this function panel to create a rotary contour turning toolpath. This is a finishing toolpath that follows the contour of the chained geometry while simultaneously allowing for A-axis or B-axis rotation. You can:

  • Use automatic mode to compute the optimal tool angles for all points along the toolpath.

  • Define the tool axis vectors manually from tool axis control lines that you select in the graphics window.

The following pictures show some examples of B-axis contour toolpaths.

 

The following picture shows an A-axis contour toolpath using a Y-axis tool. Mastercam automatically creates the necessary planes to orient the tool properly.

 

Each page in the function panel contributes to a top-down structured workflow for programming a rotary contour turning operation.

  • Work through them from top to bottom, but you can return to any page at any time.
  • Click on a topic below to learn more about how to apply the settings on each page.

Each topic applies to both A-axis and B-axis operations; any differences are noted.

Tooling requirements

A-axis contour toolpaths require that you use Y-axis tools. Before you can use a tool to create an A-axis turning operation, the Y-axis option must be selected in the tool assembly/holder settings:

B-axis contour toolpaths can be created with either 2D or 3D tools. However, A-axis contour toolpaths require 3D tools.

For either type of operation, the tool must be set to center compensation.

  • For 3D tools, set the compensation on the Compensation page of Tool Assembly in the 3D Tool Designer function panel.

  • For 2D tools, set the compensation on the Parameters page of the Define Tool - Machine Group dialog box.

    See Working with Y-axis Tools to learn more about creating tools for A-axis contour turning.

Tool page

Select tools for the operation in one of three ways:

  • Mastercam displays all of the available tools in the current toolpath group. Select Show library tools to also display tools from the tool library. Click a tool to use it in the operation.
  • If the tool you want to use is not displayed, click Select library tool. This lets you choose tools from different tool libraries.
  • Create an entirely new tool. The new tool definition will be stored in the current toolpath group unless you choose to save it to a tool library. Depending on the type of toolpath, either right-click in the tool display window or use the buttons below the tool list.

Mastercam uses a distinctive icon to identify Y-axis tools:

When you select a tool, the other fields on the tab update with default values. Override any of them by just typing in the desired values. Default parameters come from the tool definition, machine or control definition, or operation defaults. The machine group properties tell Mastercam where to get the default values.

  • If you select a Y-axis tool, Mastercam automatically filters the list of available planes and spindle origins.
  • If you are creating an A-axis contour toolpath, Mastercam only displays Y-axis tools in the list.
Feeds and speeds

As soon as you select a tool, Mastercam inputs default feed rates and spindle speed. To prevent this behavior for completed toolpaths, select the Lock feedrates option in the Toolpaths page of the System Configuration dialog box. You can calculate the defaults in either of two ways:

  • Read them directly from the tool definition.
  • Dynamically calculate them from the material, operation type, and tool characteristics.

Use the machine group properties to tell Mastercam which type of default you prefer. You can override any of the defaults by entering different values.

Mastercam validates your feeds and speeds against the maximum and minimum values set in the machine definition, and warns you if you try to enter a value outside these limits.

If you later select a different machine for the toolpath group or import the operation into a file with a different machine, Mastercam validates the programmed values against the values in the new machine definition. If the programmed values are out of range, they display in italics until you edit them.

  • The feed rate and spindle speed that you enter here are typically in effect for the entire operation. However, you can use Change at Point to change them for specific moves.
  • Use the Feed page in the control definition to configure the feed rate units, inverse feed rates, or other advanced options.
Machine page

Use these settings to select an axis combination and spindle origin. Together, these define the coordinate system for your turning operation.

  • The axis combination represents the specific spindle/turret combination that you will use for a machining job. This defines which machine axes will be used to represent the part coordinate system. The axis combinations that are available to you come from the machine definition.
  • The spindle origin defines the origin for the part coordinate system and the work offset that is associated with it. The spindle origins that are available to you come from the set of planes that are defined in the current part file. You can define a new origin in this dialog box if you wish. Each spindle origin corresponds to a specific tool plane that you can see in the Planes Manager.

Click on the desired axis combination to select it, then select the desired spindle origin. Mastercam filters the list of spindle origins to eliminate ones that make no sense for the selected axis combination.

For each spindle origin, Mastercam displays the coordinates and associated work offset. A green check mark in front of the Name means that the origin is already used by one or more operations in the part file. See Work Offset Numbers to learn more about work offset numbering.

NOTE

If you are creating a B-axis contour toolpath, use the Tool Orientation on Machine settings to set the A-axis orientation of the tool. These settings are not relevant for A-axis contour toolpaths and so are not displayed.

Operation page

Use these settings to define reference points and several post processing settings.

  • Mill-Turn toolpaths begin and end with a reference position. You select this location in the Sync Manager; it typically corresponds to the G28 home position. Use reference points to define an intermediate location between the first/last point in the toolpath and the reference position. For example, if you are performing ID work, you can use a reference point to define a clear location from which the tool can approach the part.

    Each reference point is optional. Select the checkbox for each approach and retract reference point to enable it and enter its coordinates. Select each axis that will be included in the reference point. For example, if you select the X axis but not Z, Mastercam will output only an X coordinate in your NC file.

  • Select Force tool change to force a tool change to be output in your NC file instead of a null tool change when the same tool is used in consecutive operations.

  • Select To batch to include this operation in a batch post processing job. See Batch Processing Toolpath Operations to learn more.

Enter a Comment to help identify the operation in the Toolpaths Manager. These can be output in the NC file when you post if you select the proper option in your machine group properties.

Geometry page

Use this page to review or edit the chained geometry for this operation.

  • Click the Chain Manager button to edit the chained geometry or select new geometry. See Chaining Overview to learn more.
  • Click the Change at point button to edit the toolpath—such as the feed rate or spindle speed—at a specific point. See Change at Point dialog box to learn more.

The chained geometry can include splines.

Choose the compensation direction and enter the amount of any stock to leave. The compensation direction is relative to the chain direction.

Lead In, Lead Out pages

Use these settings to control how the tool approaches or retracts from the part for each pass in the toolpath. This eliminates the need to create extra geometry for this purpose. You can also combine different types of moves. For example, you can extend a contour and lead in to it with a tangent arc.

Separate Lead In and Lead Out settings let you set different values for entry and exit moves.

  • Select the Entry Vector or Exit Vector checkbox to enable that move. Then, enter the distance and angle of the move. You can also use the angle dial to set the angle.

  • You can create lead in/out moves by extending or shortening the beginning or end of the chained geometry. Select the checkbox to enable the move and specify the adjustment distance.

Basic Motion Control page

Use these settings to control how the tool axis vectors are created throughout the course of the toolpath. You can choose to have Mastercam calculate the tool axis vectors or you can determine them manually.

  • Automatic mode creates safe A- or B-axis angles for all positions along the chained contour based on the shape of your tool insert and holder. The Tool Angle Limits define the range of angles (degrees) that Mastercam is allowed to assign to the tool along the contour.

  • Manual mode allows you to select lines to define the tool axis vectors along the contour. Enter a Tool Offset Angle to offset the tool axis vector from the selected lines. The Tool Offset Angle makes geometry creation easier by allowing you flexibility in how you draw the tool axis control lines.

Click on a topic to learn more about using the other settings on this page.

Editing the toolpath vectors

If you have manually defined the tool axis vectors, Mastercam lets you edit them after they have been created. Do this to fine-tune the resulting toolpath without editing or creating additional geometry.

To do this, select Modified, and then click the Edit vectors button.

Mastercam displays the current tool axis vectors in the graphics window. Click on a vector and use the angle dial to rotate it to the desired orientation. Press the [Shift] key to display an outline of the tool along with the vector.

Double-click anywhere on the contour to add additional tool axis vectors. Sections of the contour where the tool cannot contact the part are highlighted in red; if you double-click in a non-contact area, Mastercam will add the new tool axis vector at the closest possible location.

Press [Enter] to return to the function panel. Mastercam does not edit the original wireframe geometry that was used to create the initial tool axis vector.

Tool angle controls

Tool angle controls let you specify either a leading or trailing angle for the insert. Select the Tool Angle Control option to activate the feature and enter either a leading or trailing angle.

Fanning distance

The fanning distance lets you better control changes in the tool axis as the tool moves around a sharp corner. This can create large changes in the tool axis angle in a very small linear distance. The fanning option lets you specify a distance before and after the corner where the tool axis can transition in a smoother and more controlled manner.

Advanced Motion Control page

The settings on this page give you greater control over your tool and toolpath. These settings include:

  • Cut tolerance: Adjust the maximum value by which the toolpath is allowed to deviate from the chained contour. A smaller Cut Tolerance value leads to a more accurate toolpath, but too small of a value will increase toolpath generation time.

  • Maximum distance between points: The maximum distance allowed between the points from which the toolpath is made and which occur in the posted code. Not all points will be the same distance apart; rather, no points will be farther apart than the distance specified in this field. To display the points in Classic Backplot, enable Show end points.

  • Maximum angle step: The maximum angle by which B-axis rotation may change between two points. If there is a need for a greater angle change between two points, Mastercam will add an intermediate point so that the Maximum angle step value is respected.

  • Holder clearance: The value by which the holder boundary is offset, prior to Mastercam computing the toolpath, to create a gap between the holder and the stock.

    Holder clearance of 0.0 Holder clearance of 1.0
    NOTE

    If the Holder Clearance value is less than double the Cut Tolerance, Mastercam will use double the Cut Tolerance as the Holder Clearance value to keep the holder from contacting the contour.

  • Maximum Insert Depth: This value controls the maximum amount of the insert allowed to contact the contour.

    • Select Depth to enter the value directly. The value is measured in inches or millimeters, depending on the units that you are using. Enter a value of zero to use any exposed portion of the tool insert.

      Applying the Depth value to different types of tools

      The Depth is applied differently depending on the tool type.

      Grooving/parting tools apply the Depth value from a line along the plunge direction, at the distance the user has entered. The resulting portion of the insert is allowed to contact the contour.

      Grooving/parting tools: Maximum insert depth of 0.0 Grooving/parting tools: Maximum insert depth of 2.0

      Boring bars, general turning, and custom tools use the Depth value as follows:

      • For a Depth value that is less than the tool nose radius, Mastercam applies the same method as is applied for grooving/parting tools.

      • For a Depth value that is greater than the tool nose radius, Mastercam computes the allowable contact portion of the insert by creating a circle with a radius that is the Depth value and with a center point at the center of the tool nose radius. Any portion of the insert contained within the circle will be used for cutting. Any portion of the insert outside of the circle will act as part of the holder.

        Boring bars, general turning, and custom tools: Maximum insert depth of 1.5 Boring bars, general turning, and custom tools: Maximum insert depth of 0.25

    • Select Manual to define the usable portion visually in the graphics window.

      Manually defining the usable portion of the insert

      Mastercam also lets you define the allowable portion of the insert manually. Do this when you would rather define it visually instead of entering a dimension, or when you wish to define a usable insert area that is irregular or non-symmetrical. Follow these steps:

      1. Select Manual and click the Edit depth button.

        Mastercam returns you to the graphics window where you can edit the boundaries of the usable portion.

      2. Drag each colored ball to the desired location on the insert boundary. Click and release each ball to drag it; do not drag it while holding down the mouse key.

      3. Click the mouse again to release the ball.

      4. Repeat for the other ball until you are satisfied with the shape.

      5. Press [Enter] to return to the function panel.

    Click Preview to see the updated insert boundaries in the graphics window.

    • Mastercam displays the insert with the proper insert-up or insert-down colors.

    • The picture also incorporates the Holder clearance value.

    • For collision-checking purposes, the unused portion of the insert is treated like it is part of the holder.

Stock page

Use these settings to control how Mastercam updates the stock definition between operations.

Click Adjust stock to edit the stock model at the beginning and end of the toolpath. Do this to control how the tool enters and exits the stock.

When you click the button, Mastercam displays the existing remaining stock model relative to the chained geometry. A green ball indicates the beginning of the chain, and a red ball indicates the end. The gray dashed line indicates the current stock boundary.

Click and release one of the balls to edit the stock model at either end of the chain. Drag the ball to the desired location on the stock boundary, and then press [Enter]. Click-and-release each ball to drag it; do not drag it while holding down the mouse key.

White dots indicate up to three locations to which the cursor will snap:

  • The location perpendicular to the direction of the cut

  • The location parallel to the direction of the cut

  • The location parallel to the direction of the chain

(In these pictures, the white dots are displayed in red for sake of clarity.)

For example, you can drag the ball to the stock margin to extend the stock to the face of the part, or drag it down to create a chamfer.

When you are done, click [Enter] to return to the function panel. Click Reset to undo the changes and restore the original stock.

Settings page

Select the checkbox to adjust the Tool Display settings.

Use these settings to tell Mastercam how the toolpath appears in the graphics window when it is first created or regenerated.

Once a toolpath has been created, use the Toolpaths Manager to toggle toolpath display on and off for individual toolpaths or groups of toolpaths.

NOTE

These options do not affect Mastercam's simulation and backplot features.
