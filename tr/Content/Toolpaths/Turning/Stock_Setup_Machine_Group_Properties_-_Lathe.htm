Stock Setup tab: Machine Group Properties - Lathe

Use the Stock Setup tab to define stock boundaries, chuck jaws, tailstock centers, and steady rest settings for Lathe machine groups. After you create the components, use the Tool clearance fields to define a clearance zone around each boundary. When you are creating a toolpath, Mastercam warns you every time the tool violates these clearance distances.

Setting the stock plane

Select a stock plane to properly orient the stock model and other boundaries with respect to your part. You can align the stock model to any plane saved in your part file. When you choose a stock plane, the edges of the stock model will be parallel with the axes of the selected plane. The chuck, tailstock, and steady rest boundaries will also shift in parallel with the stock boundary.

Select a stock plane when creating toolpaths in a work coordinate system (WCS) other than Top and want to align the stock model to your part. If you have several toolpaths in the toolpath group that use more than one WCS, Mastercam uses the stock plane to keep the stock model constant when the WCS for each operation changes. Mastercam stores the stock plane independently of the WCS.

To select a stock plane, click Stock plane and select the plane from the Plane Selection dialog box.

Note that when you define stock for a VTL, you do not need to select a special plane to create the vertical stock model. When you configure the machine definition as a VTL, Mastercam aligns the planes to the VTL axes.

Parameters

Stock plane

Click to select a plane with which the stock model will be aligned. For example, if you are machining the part in a work coordinate system (WCS) other than Top, you can select the same plane here so that the stock aligns properly with the geometry. Mastercam stores this plane separately from the WCS or tool plane used to create an operation, so even if you change the WCS or tool plane between toolpaths in the group, the stock displays in the plane that you select here. You can select any named plane stored in the part for the stock plane.

Left/Right spindle

Define stock boundaries and chuck jaws for the left spindle, right spindle, or both spindles. Select the spindle in which the component will be mounted, and then choose Properties to create the boundary.

  • Mastercam displays Defined or Not Defined under each spindle indicator to tell you when stock/chuck has been created for it.
  • In order to select a spindle, it must already be defined in the machine definition. If a spindle is disabled and cannot be selected, this means that the spindle component has not been created in the machine definition.

Properties

Define the boundary by creating a component or editing an existing component. You can either create the component parametrically (entering dimensions directly in dialog box fields), by selecting a solid model, or by generating a solid model from wireframe geometry. Mastercam displays the Geometry tab that lets you define the component size, shape, and initial position, as well as other programming parameters.

Delete

Removes the boundary by deleting the selected component. If you choose Reset after operations have been created, the boundary for all operations are deleted from the database. If you define new boundaries, you must regenerate all operations in order to update them. However, the boundary for the stock model or peripheral cannot be reset if you have programmed a miscellaneous operation on that boundary.

NOTE

For stock, chuck jaws, and tailstock centers, clicking this button deletes the component from the component tree. For steady rests, only the boundary is deleted. The component itself remains in the machine definition component tree.

Display options

Selectively display or hide the component boundary in the graphics window. Select the boundaries that you want to see displayed in the graphics window along with the part geometry. Select All or None to quickly select or deselect all the options.

Shade boundaries

Check to shade the stock boundaries with its default color. To change the color, choose File, Configuration, and select the Tools and holders page.

Fit screen to boundaries

Select this option to include the stock, chuck, and tailstock boundaries when fitting the graphics window zoom to the part geometry.
