Spindle Sync dialog box

Use the Spindle Sync dialog box to sync or align the spindles of your machine. Sync locks the spindles together so they turn in unison. Align matches a selected angle on one spindle with a selected angle on the other spindle. Additionally, you have the ability to control to which data stream the operation is output.

Parameters

Sync type

Select the type of sync for the spindles from the drop-down. Your selection determines the subsequent options in the dialog box. You may Align, Sync, or Cancel sync for the sub spindle in relation to the master spindle.

Spindle master

Choose to have the Left or Right spindle designated as the master spindle. The master spindle controls the child spindle when they are synced.

Data stream

Select the data stream in which to place the operation, Upper stream or Lower stream.

Rotary position

Select to activate the rotary position options. Setting Sync type to Align enables this option. Setting Sync type to Sync makes Rotary position optional.

Left spindle angle

Enter the left spindle angle for aligning or syncing. Spindle master can influence how you set this angle depending if the left spindle is the master or sub spindle.

Right spindle angle

Enter the right spindle angle for aligning or syncing. Spindle master can influence how you set this angle depending if the right spindle is the master or sub spindle.

Rotary motion

Choose to have the spindle rotate at the machine's Rapid rate or at the specified Feed rate. Feed rate is specified in degrees per minute.

Spindle start

Select to activate the Spindle RPM field. Available only when Sync type is set to Sync.

Spindle RPM

Enter the RPM to use for synced spindles and choose CW—clockwise, or CCW—counterclockwise spindle rotation.

Comment

Enter a comment for the operation. Comments help identify the operation in the Toolpaths Manager.
