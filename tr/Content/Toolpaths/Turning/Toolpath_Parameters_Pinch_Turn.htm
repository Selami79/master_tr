Toolpath Parameters: Pinch Turn tab

Use the Toolpath Parameters, Pinch Turn dialog box tab to select a tool and set general pinch turn toolpath parameters. This tab is specific to Mastercam's Mill-Turn functionality.

Pinch turn operations use two independent tools, located on two independent turrets in order to remove material more efficiently. With pinch turning, both tools cut the same curve. One tool cuts above the axis of rotation and the other cuts below, mimicking or imitating the part being “pinched.” The feeds and speeds are defined in the Pinch Turn Main, original roughing operation, and automatically applied to the associated pinch turn operation. For more details on the differences and similarities between Pinch Turn Main and Pinch Turn Associated toolpaths, see Pinch turn tab.

Parameters
NOTE

Many of the field and button labels on this tab can be customized and might differ from those listed here, or might be disabled.

Tool display window

Shows the tools available in the current machine group. Click a tool in this list to use it in the current operation. Right-click in this area to add tools to this list from a tool library, create a new tool, or change how this list is displayed.

Show library tools

Adds all the tools in the current tool library to the tool display window. Use Filter to screen the list if there are too many tools to view conveniently. The name of the current library is displayed on the right-click menu.

Select library tool

Opens the Tool Selection dialog box so that you can select a tool from a tool library, or select a different tool library. When you select a tool, Mastercam copies the tool to the current machine group and saves it to the part. The tool is then listed in the tool display window where it can be selected for any operation in the machine group.

Tool filter

Opens the Lathe Tool Filter dialog box to set filter criteria for finding tools.

When you select an axis combination for Mill-Turn toolpaths, the tool filter automatically filters the upper and lower turret tools based on the turret used by the axis combination.

Tool number

Shows the tool number used in the NC program. When Assign tool numbers sequentially is selected in your machine group properties, Mastercam numbers the tools starting with 1, 2, and so on. When this option is deselected, Mastercam takes the tool number from the tool definition. You can also automatically check for duplicate tool numbers. You can override either method by entering a new number.

The post processor can associate the tool number with length and diameter offsets in the control. The control definition determines how Mastercam assigns the offsets for each tool.

Offset number

Information about possible offsets required for each tool in the turret is stored in an offset register. Each offset register is given a one- or two-digit number. Enter a value that corresponds with the number of the offset register you want to use for the specified tool.

Tool angle

Click to open the Tool Angle dialog box where you can rotate the tool to a specific angle for the toolpath. It is only enabled if the active axis combination contains a compatible rotary axis. For a multi-function lathe, this is usually a B axis.

Tool angles

Read-only. Displays the angle and orientation entered in the Tool Angle dialog box.

Turret index position

Sets the turret index position to which the tool is added.

Force tool change

When the same tool is used in successive operations, this option replaces the null tool change in the NCI file with a tool change code, so that an event can be triggered between the two operations. The tool collision avoidance function assumes that the tool will go to the home position immediately after an operation with this option selected, even if the next operation uses the same tool.

Comment

Enter a Comment to help identify the operation in the Toolpaths Manager. These can be output in the NC file when you post if you select the proper option in your machine group properties.

Axis combination/Spindle origin

Opens the Axis Combination/Spindle Origin dialog box to select an axis combination that has been defined for your machine. You need to select an axis combination if your machine tool has multiple definitions for the same axis. Axis combinations typically include a tool turret, spindle, and chuck. You must define axis combinations in the machine definition before they can be selected here. Once you select an axis combination, its name displays on the button.

If your machine tool does not have any axis combinations, the default will be selected and you do not have to do anything.

Tool display

Use these settings to tell Mastercam how the toolpath appears in the graphics window when it is first created or regenerated.

Once a toolpath has been created, use the Toolpaths Manager to toggle toolpath display on and off for individual toolpaths or groups of toolpaths.

NOTE

These options do not affect Mastercam's simulation and backplot features.

Stock update

Access the Stock Update Parameters dialog box to regenerate subsequent operations, define the operation's tool clearance, or use the default tool clearance defined in Job Setup. You can choose to keep either the part boundary, the stock boundary, or both boundaries after the part is cut off.

To batch

Select to automatically tag the toolpath for batch processing and to add the toolpath to the Toolpaths Manager. An NCI file generates after the batch runs.

Reference point

Opens the Reference points dialog box to set reference points. A reference point is a location that the tool moves to between the home position and the start of the toolpath (approach point), and another location that the tool moves to at the end of the toolpath before returning to the home position (retract point).

Mastercam lets you store separate reference points for ID and OD work, and will automatically select the proper ones.

Right-click menu options
  • Create new tool - Use this option to create a new tool instead of selecting one from the list. The tool definition will be stored in the current toolpath group unless you save it to a library.
  • Edit tool - Use this option to edit the tool definition for the selected tool. The changes you make will be stored in the toolpath group and will not affect future parts which use the tool unless you save the changes to a library.
  • Tool Manager - Displays the name of the current tool library, and opens the Tool Manager so that you can work with the library and the tools stored in the current toolpath group.
  • Load tool on machine - Create a Tool component in the machine definition based on the tool definition for the selected tool. Mastercam uses the tool setup information in Tool spindle and Tool locator components to properly position and orient the tool for simulation.
  • View - Display the tools as large icons, list, or as a details list.
  • Arrange tools - Sort the tools by tool number or tool name. Use these options if you have the tools displayed as icons without any column headers.
  • Popup tool - A larger picture of the tool will popup if you let the mouse hover over an icon in the preview window. Use this option to control how long the mouse needs to hover before the larger picture pops up.
  • Save parameters - Use this option to make the current toolpath settings the default for future toolpaths of this type. The current values are saved to the .mcam-defaults file specified in the toolpath group properties.
  • Reload parameters - Overwrite the current values with the values from your defaults file. These will be read from the .mcam-defaults file specified in the toolpath group properties.
  • Re-initialize feeds and speeds - Use this option to recompute the default feed rate and spindle speed.
