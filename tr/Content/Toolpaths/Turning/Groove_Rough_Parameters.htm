Groove Rough Parameters tab

Use this tab to create the roughing cuts for a groove toolpath. Select Rough to activate the options on this tab and create a roughing pass. Deselect this checkbox if you only want to create a finish operation.

Use the pictures on the tab as a guide to entering your roughing parameters. Select options on the right side of the tab to enable advanced grooving features, such as pecking, depth cuts, and toolpath filtering.

There are no tool compensation options for groove roughing. Tool compensation in computer is always on, and Mastercam determines the direction. When cutting a face, Mastercam calculates the optimal plunge point so that each side of the tool removes the same amount of material, balancing tool wear.

Parameters

Rough

Select to create roughing passes.

Finish each groove before roughing next

Select if you want each groove to be roughed and finished before moving on to the next groove. Deselect if you want all grooves to be roughed before any finish cuts are made. This is used when multiple grooves are defined.

Stock clearance

Determines how far above the top of the groove the tool starts the first plunge and retracts after the last cut.

Rough step

Determines the amount of material removed by each cut. This can be determined by choosing an option from the drop-down list and entering a value in the field. The options are:

  • Number of steps
  • Step amount
  • Percent of tool width

Backoff %

Determines how far the tool backs away from the wall of the groove before it retracts. The backoff distance is defined as a percentage of the rough step. The tool backs off at a 45-degree angle, and will not back off if it is going to hit the part.

Cut direction

Determines the direction that the tool roughs the groove:

  • Positive: Machines in the positive direction.
  • Negative: Machines in the negative direction.
  • Bi-Directional, Alternating: Alternates between positive and negative until the required material is removed.
  • Bi-Directional, Positive First: Machines the positive side first, and then machines the negative side.
  • Bi-Directional, Negative First: Machines the negative side first, and then machines the positive side.
  • Chain direction: Machines the direction of the chain. Only with inner/outer boundary grooves.
NOTE

For canned groove toolpaths, only Positive and Negative are available.

Stock amount

Determines the amount of stock above the top of the groove.

Stock to leave in X, Stock to leave in Y

Determines how much stock remains in the X direction when the operation is finished. Typically, this would be removed by a separate finish operation.

If you are using a Y-axis tool, the value of this field is a Y-axis coordinate, not X. When this happens, the name of this field references the Y axis instead of X.

Maximum start diameter / Minimum start diameter

Sets the maximum and minimum diameters as the valid range for the first plunge for a face groove toolpath.

A groove on the face of a part is curved into a circle, so the holder needs to also be curved, and the face grooving insert must be held in a position that supports a curved cut. The curve of the tool limits where the first plunge can happen. The valid range for the first plunge is defined as a maximum and minimum diameter. After the first plunge, the tool can cut beyond that range.

These options are only available when the groove angle is set to Face (0 degrees) or Back (180 degrees) on the Groove Shape parameters tab.

Rapid

Retracts the tool at the rapid feed rate. This option is not recommended if you are creating an angled groove.

Feed rate

Converts rapid moves into fast-feed rate moves. The rough and finish passes use the same fast-feed rate, so the interface displays the rate on both the rough parameters and finish parameters pages.

Plunge

Sets feed rate parameters for the rough groove first plunge or peck. This allows you to set one feed rate for your first peck, and different feed rates for subsequent pecks. Select to activate the field and radio buttons. Enter the feed rate in the field, and choose the appropriate radio button:

  • in/rev or mm/rev: Calculates the value in inches per revolution, or millimeters per revolution for metric.
  • in/min or mm/min: Calculates the value in inches per minute, or millimeters per minute for metric.

Retract

Determines the feed rate the tool moves away from the part for the rough groove first plunge or peck before it moves to the start of the next cut. This allows you to set one retract feed rate for your first peck, and different feed rates for subsequent pecks. Select to activate the field and radio buttons. Enter the feed rate in the field, and choose the appropriate radio button:

  • in/rev or mm/rev: Calculates the value in inches per revolution, or millimeters per revolution for metric.
  • in/min or mm/min: Calculates the value in inches per minute, or millimeters per minute for metric.

Dwell time

This value represents the dwell time at the bottom of the groove. Choose Seconds or Revolutions to specify the dwell units and enter a value in the field.

Steps

Does not create extra moves to smooth the walls.

Smooth

Adds extra moves to the toolpath to remove the steps created when rough grooving a tapered wall. Choose Parameters to configure these moves.

Multiple plunge

Select to cut the groove in multiple plunges or rib cuts. Click to set the plunge dimensions.

Peck groove

Select to cut the groove in pecks. Click to set peck parameters.

Depth cuts

Select to cut the groove in depth cuts. Click to set depth cut parameters.

Filter

Select to eliminate unnecessary tool moves in the toolpath to create smoother movement. Click to configure toolpath filtering options.

Tool inspection

Click to open the Groove Tool Inspection dialog box. You can establish points during grooving operations where the machine can be programmed to stop at a specific position to allow you to perform tool inspections.
