Cutoff/Pickoff/Bar Pull Setup dialog box

Use the Cutoff/Pickoff/Bar Pull Setup dialog box to create a part transfer operation from one spindle to the other and/or a bar pull operation prior to a cutoff toolpath. This dialog box is specifically used for parts cut from bar stock, not parts cut from slugs, castings, or forgings. You can choose to just create the geometry without creating any operation, or you can create the geometry and the following types of operations:

  • Pickoff, Bar pull, Cutoff
  • Pickoff, Cutoff
  • Pickoff
  • Bar pull

Use the controls in the Operations tab to define parameters for each of the operations.

The following conditions must be met to use this function:

  • The red insert arrow must be in a lathe machine group.
  • The machine group must have stock defined in a fixed spindle, or both spindles, and the chuck jaws defined for both spindles. If both spindles can move in Z, the spindle not containing stock is assumed to be the pickoff spindle. If both spindles contain stock in this case, the right spindle is assumed to be the pickoff spindle.
  • A cutoff operation must exist in the currently active operation library.
NOTE

This function uses the left and right stock margins defined in the Component Geometry (Cylinder) dialog box.

Parameters

No operations

Choose to create/transform selected geometry, generate pickoff spindle stock, and/or create a named plane for programming toolpaths in the pickoff spindle.

Pickoff, Bar pull, Cutoff

Choose to create pickoff, bar pull, and cutoff operations as described below:

  1. Pickoff spindle unclamps and ejects the stock it contains, if necessary.
  2. Pickoff spindle approaches to a given distance from the stock.
  3. Pickoff spindle, moves to the grip position on the part.
  4. Pickoff spindle clamps the part.
  5. Main spindle unclamps from the part.
  6. Pickoff spindle pulls the part to the cutoff position.
  7. Main spindle clamps the part. Part is cut off.
  8. Pickoff spindle retracts to its initial position.

Pickoff, Cutoff

Choose this option to create pickoff and cutoff operations as described below:

  1. Pickoff spindle unclamps and ejects the stock it contains, if necessary.
  2. Pickoff spindle approaches to a given distance from the stock.
  3. Pickoff spindle, moves to the grip position on the part.
  4. Pickoff spindle clamps the part.
  5. Part is cut off.
  6. Pickoff spindle retracts to its initial position.

Pickoff

Choose this option to create pickoff operations as described below:

  1. Pickoff spindle unclamps and ejects the stock it contains, if necessary.
  2. Pickoff spindle approaches to a given distance from the stock.
  3. Pickoff spindle, moves to the grip position on the part.
  4. Pickoff spindle clamps the part.
  5. Main spindle unclamps from the part.
  6. Pickoff spindle retracts to its initial position.

Bar pull

Choose this option to create bar pull operations as described below:

  1. Pickoff spindle unclamps and ejects the stock it contains, if necessary. 1
  2. Pickoff spindle approaches to a given distance from the stock. 1
  3. Pickoff spindle, moves to the grip position on the part. 1
  4. Pickoff spindle clamps the part. 1
  5. Main spindle unclamps from the part.
  6. Pickoff spindle pulls the part to a user-defined position.
  7. Main spindle clamps the part.

1 These operations are not performed if the pickoff spindle is already clamped on the part. This allows you to program multiple stock pulls on the same part.

Select geometry

Click to select the geometry to translate to the other spindle or for a bar pull.

Translate geometry

Select to create a copy of the geometry as it would appear after a bar pull operation.

Keep the same

Places the transferred geometry on the same level as the original geometry.

Copy to level

Enter the level number on which level to place the transferred geometry.

Offset by

Mastercam calculates a destination level based on the level number of the original geometry plus the number you enter here.

Create named view

Select to create a new plane for programming pickoff spindle toolpaths. This plane is a copy of the active WCS, but with its origin on the desired face of the part in the pickoff spindle.

View name

Enter the name of the new plane.

Left face/Right face

Choose either Left face or Right face to set the new plane origin on the right or left face of the part. When you select this plane as your default Tplane, the origin of this plane is used for all subsequently programmed turning toolpaths. This eliminates the need to pick a new tool origin for every new toolpath.

Generate pickoff spindle stock

Select to create stock in the pickoff spindle. This stock model will be added to the machine and all operations prior to the part eject operation or first pickoff spindle toolpath, whichever is listed first in the Toolpaths Manager. The stock model is either captured from the programmed cutoff operation, or generated by creating a virtual cutoff operation on the currently active source spindle stock using the parameters defined on this tab.

Retract spindle after bar pull

Select to have the pickoff spindle unclamp the part and retreat to its original position after pulling the stock.

Distance to pull stock

Enter the distance to pull the stock from the stock spindle.

From geo

Click to set the distance to pull the stock from the length of the geometry selected to define the finished part. This is disabled if no geometry has been selected.

Length of finished part

Use this field to define the length of the part being machined. This value is used in the geometry translation computations.

From geo

Click to compute the length of the finished part from the selected geometry. This button is disabled if no geometry has been selected.

Pickoff Z coordinate

Enter the Z coordinate to which the sub-spindle chuck will move to clamp the part for the stock pull or stock transfer operation. Alternately, click Select to return to the graphics window to select the pickoff Z coordinate.

Cutoff width

Use to define the width of the material removed by the cutoff operation. This amount is usually equal to the width of the cutoff tool insert unless clearance cuts are programmed.

Select tool for cutoff width

Click to open the Tool Manager so that you can select a cutoff tool. The width of this tool’s insert displays in the Cutoff width field, and this tool is selected by default for the cutoff operation. You can either select a tool in the database, or copy a tool from the tool library to the database using the Tool Manager. You have full tool editing capabilities while in the Tool Manager. In order to select a tool from a tool library, you must add it to the active machine group.

Cutoff X coordinate

Enter the X, radial coordinate for the cutoff operation.

From stock

Click to calculate the cutoff X coordinate from the diameter of the stock. In Swiss mode, Mastercam adds the tool clearance distance from the machine group properties stock dimension. This ensures that you will not get a collision warning when pre-positioning the cutoff tool.

Note that Swiss mode is determined by a post flag, so you are in Swiss mode when you are using a Swiss post.

Face stock

Defines the amount of stock on the front face of the part.

From stock

Computes the Face stock value using the stock boundary and selected part geometry. This is disabled if no geometry is selected. If cylindrical stock is defined, Mastercam uses the default margin values from the Machine Group Properties Stock page.

Back face stock

Defines the amount of stock on the back face of the part.
