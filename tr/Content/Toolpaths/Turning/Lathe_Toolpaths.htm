Creating a Lathe Toolpath

Mastercam Lathe's toolpaths allow you to boost productivity by turning your part accurately and efficiently, including roughing, grooving, threading, and more. Live tooling, quick setup times, and C- and Y-axis machining all help you meet your machining needs. Combine your Lathe license with a Mill or Mill-Turn license to expand your machining options.

NOTE

Your ability to access specific toolpaths and features depends on your license and the capabilities of the active machine definition.

How do I create a lathe toolpath?

Following is a general overview of how to create a lathe toolpath.

  1. Select a Lathe machine definition. Choose Machine, Lathe, and select a definition from the list.

  2. Double-click Stock setup in Toolpaths Manager to open Machine Group Properties. Choose defaults, define a stock model and material, and set up peripherals like chucks, tailstocks and steady rests.

    NOTE

    Learn about the settings on each page by clicking the Help.

    See Adding Lathe Stock, Chucks, and Peripherals to Your Machine to learn more.

  3. Select a toolpath from the galleries in the Lathe contextual tabs. A description of each toolpath can be found in the What toolpaths are available? section of this page.

  4. The Chaining dialog box opens and Mastercam prompts you to select any necessary geometry. When finished, click OK.

  5. A dialog box specific to your toolpath type opens. For example, the dialog box below is for a Lathe Rough toolpath. Enter your settings.

  6. Click OK to generate your toolpath.

What toolpaths are available?

Mastercam Lathe includes three main categories of toolpaths: General turning toolpaths, part handling operations, and C-axis toolpaths. Depending on your project goals and machine setup, you may wish to use these toolpath categories independently, or in conjunction with each other.

Part handling operations manipulate the stock and program the movements of peripherals like tailstocks, chucks, and steady rests. These operations might output M-code or G-code. You can only program part handling operations for peripherals that are supported by the active machine definition.

C-axis toolpaths let Mastercam Lathe users create simple milling operations that use live tooling. The part can be stationary in the chuck, or the part spindle can be programmed as a rotary axis (C-axis). Use tool planes to specify how the tool is oriented with respect to the part.

What are the differences between C-axis and Y-axis milling operations?

Milling operations in Lathe come in two styles: C-axis and Y-axis.

  • C-axis operations, which are more common, have linear motion in the X and Z axes and rotary motion around the C axis. The C-axis motion is provided by controlled rotation and positioning of the spindle that holds the part.
  • Y-axis operations support the C-axis motion and use an additional linear axis that allows the milling tool to move above and below the spindle's center line.

All Mastercam Lathe C-Axis toolpaths (Lathe Turning, C-Axis) and many Mill toolpaths in Lathe (Lathe Milling), use the Rotary Axis Control property page to select the axis of rotation.

With Y-axis drilling, Mastercam automatically sets the Tplane to the plane of the arc. Because of this, you cannot select points for your drill locations, only arcs. However, with C-axis drilling, because the drill axis is automatically perpendicular to the centerline, you can select either points or arcs.

To learn more about rotary axis motion, see Rotary Axis Control dialog box: Lathe.

What are the differences between Lathe and Mill-Turn toolpaths?

Mastercam Mill-Turn is a separate product from Mastercam Lathe. Users who have a Mastercam Lathe license can also purchase Mastercam Mill-Turn. Mill-Turn users can access all of Mastercam Lathe's turning operations. See Comparing Mill-Turn and Lathe to learn more.

More functionality

Mastercam includes several other operations available in the Utilities group of the contextual tab that can be used to help machine your parts.

  • Stock Model operationsCreates a stock model that is associative to the source toolpath operations. For more information, read the Working with Stock Model Operations Help topic.
  • Toolpath TransformCreates copies of operations in different orientations by transforming them, trimming them, or wrapping them around a cylinder with axis substitution. For more information, read the Transforming Toolpaths Help topic.
  • CViewSet Tplanes and Cplanes when you are creating Mill Contour and Drill toolpaths. For more information read the CView dialog box Help topic.
  • Align to ZAligns a solid body that is used for a turning operation to a new WCS or existing construction plane along the Z axis. For more information read the Align to Z panel Help topic.
What is the Generate toolpath option on the toolpath pages and tabs?

The Generate toolpath option automatically generates the toolpath when you click OK. This option is on the pages and tabs of Mastercam's toolpath dialog boxes, and is on by default for new and edited operations. By selecting this option, you no longer need to select Regenerate dirty operations in the Toolpaths Manager after editing your toolpath.

The Generate toolpath option also works with the Apply button on tree-style milling toolpaths. The Apply button applies your parameter changes to the toolpath you are editing without closing the dialog box.

For more information on how this option functions, refer to the Accepting and Generating Toolpaths Help topic.
