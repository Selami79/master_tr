Holder Designer

Use the Holder Designer to build holders, defining the general attributes, such as name, units, manufacturer, and type. The Holder Designer also supports the import of solid models to define an anti-collision model for simulation. Additionally, the solid is used to derive a second profile used for toolpath calculations such as tool radius compensation and collision avoidance.

Holders that you create are displayed in the Holders tab of the Tool Manager.

Creating a holder

  1. Start the Tool Manager.

  2. In the lower pane of the Tool Manager window, click the Holders tab.

  3. Right-click in the pane, and choose Create holder from the pop-up menu.

  4. In the Holder Assembly pane of the Holder Designer panel, right-click +(add a component), and choose Define from the pop-up menu.

  5. Fill in the information on the Component panel. All of the options have default values, except Model, which you must define before you can move to the next set of options.

  6. Go to the Connections panel.

  7. Fill in the information on the Connections panel. All of the options have default values, except Plane, which you must define before you can move to the next set of options.

  8. When defining adapters and extensions, use the options on the Mating panel to set up workpiece side connections to which to mate child components.

  9. Click OK in the Mating panel and Holder Designer. The new holder appears in Tool Manager's Holders tab.

Where are the parameters from my imported model?

When importing a STEP file or selecting a solid from the Solids Manager, Mastercam does not import values associated with it. This means, for example, that the Thickness parameter under Shank is not updated for holders or the Corner radius parameter under Dimensions is not updated for inserts.

You must manually update these parameters, or you can use Data Entry Shortcuts in parameters to read values from the imported model.
