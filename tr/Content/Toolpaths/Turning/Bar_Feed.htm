Bar Feed dialog box

Use the Bar Feed dialog box to configure a bar feed operation. Create bar feed operations as stand-alone operations; they are not created as part of part-handling strategies.

Mastercam gives you the option to define a new spindle origin at the new location. This creates an entire set of planes whose origin is at the new location. Use these as tool planes for operations that you create at this location. The name that you enter in this dialog box for the destination will be used as the name for the new views. You can also choose to associate a work offset with the new location; this will be invoked automatically whenever you use one of the new planes.

Parameters

Transfer geometry

Select this option to create a copy of your part geometry at the new stock location.

  • Click Select to choose which geometry to copy.
  • Click Show to preview the selected geometry.

Delete previously created geometry

Deletes any geometry that already exists on the level where the new geometry will be placed.

Keep the same

Places the transferred geometry on the same level as the original geometry.

Copy to level

Lets you specify on which level you want to place the transferred geometry.

Offset by

Mastercam calculates a destination level based on the level number of the original geometry plus the number you enter here.

Destination

Tells Mastercam how far to advance the bar.

  • Choose Job Setup - Initial position when the bar is not already in the position described in Job Setup and you need to move it there. This is also useful if you are cutting multiple parts and want to position the stock for a new part.
  • Choose Position to specify any other destination. Use any of the following methods to set the location:
    • Click Select to choose a destination point from the graphics window.
    • Incremental distance specifies the total distance to advance the bar.
    • Distance from chuck face specifies a final location relative to the chuck face.
    • Distance from jaws specifies a final stick-out distance from the chuck jaws.

Rapid

Advances the bar as a rapid move.

Feed

Advances the bar at the feed rate that you enter here.

Data stream

The stream where the NC code for this operation should be output.

Create new spindle origin

Choose this option to have Mastercam create a new set of views whose origin is based on the new stock location. Mastercam creates a copy of each view that was originally created in Job Setup.

Name

Enter a name for the new spindle origin. The name that you enter here will be used in the name of each new plane that Mastercam creates at this location.

Left, Right

Choose whether the bar feed operation takes place on the left or right spindle.

Work Offset

Select Automatic to have Mastercam assign the next available work offset number. Select Manual if you want to use a specific offset number. Enter the offset number in the adjoining field. Mastercam includes this work offset in the views that it creates for the spindle origin so that the offset is automatically called when you use the view in a toolpath.

Comment

Enter a comment for the operation. Comments help identify the operation in the Toolpaths Manager and can be output in your NC program.

NOTE

  • You can only create a bar feed operation if the chuck component for the desired spindle has the Allow through stock option enabled. This option is set in the machine definition.

  • This operation is only available for Mastercam Mill-Turn. Mastercam Lathe users should create a Stock Advance operation for bar feed applications.
