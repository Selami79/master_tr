Center Point parameters

Use these parameters on the Toolpath parameters tab to create a Center Point operation that inserts a lathe center into your machine's turret.

Parameters

Tool display window

Shows the tools available in the current machine group. Click a tool in this list to use it in the current operation. Right-click in this area to add tools to this list from a tool library, create a new tool, or change how this list is displayed.

Show library tools

Adds all the tools in the current tool library to the tool display window. Use Filter to screen the list if there are too many tools to view conveniently. The name of the current library is displayed on the right-click menu.

Select library tool

Opens the Tool Selection dialog box so that you can select a tool from a tool library, or select a different tool library. When you select a tool, Mastercam copies the tool to the current machine group and saves it to the part. The tool is then listed in the tool display window where it can be selected for any operation in the machine group.

Axis combination/Spindle origin

Opens the Axis Combination/Spindle Origin dialog box to select an axis combination that has been defined for your machine. You need to select an axis combination if your machine tool has multiple definitions for the same axis. Axis combinations typically include a tool turret, spindle, and chuck. You must define axis combinations in the machine definition before they can be selected here. Once you select an axis combination, its name displays on the button.

If your machine tool does not have any axis combinations, the default will be selected and you do not have to do anything.

Tool number

Shows the tool number used in the NC program. When Assign tool numbers sequentially is selected in your machine group properties, Mastercam numbers the tools starting with 1, 2, and so on. When this option is deselected, Mastercam takes the tool number from the tool definition. You can also automatically check for duplicate tool numbers. You can override either method by entering a new number.

The post processor can associate the tool number with length and diameter offsets in the control. The control definition determines how Mastercam assigns the offsets for each tool.

Offset number

Information about possible offsets required for each tool in the turret is stored in an offset register. Each offset register is given a one- or two-digit number. Enter a value that corresponds with the number of the offset register you want to use for the specified tool.

Motion points

In this pane, create a list of points to which the tool must travel to get into position. The buttons below the pane let you create and manage points in the list.

Comment

Enter a Comment to help identify the operation in the Toolpaths Manager. These can be output in the NC file when you post if you select the proper option in your machine group properties.

Tool display

Use these settings to tell Mastercam how the toolpath appears in the graphics window when it is first created or regenerated.

Once a toolpath has been created, use the Toolpaths Manager to toggle toolpath display on and off for individual toolpaths or groups of toolpaths.

NOTE

These options do not affect Mastercam's simulation and backplot features.

Generate retract operation

Creates a second operation in the Toolpaths Manager that automatically retracts the center. This operation is separate from the engage operation (the original operation).

For example, the engage operation contains two moves: a rapid to Z96, then the move to engage. The new operation will contain a single retract move to the position immediately before the center engages the part—in this example, Z96. After selecting OK, Mastercam creates two operations in the Toolpaths Manager.

The second operation contains a single move back to Z96. Also, in this case the Generate retract operation option is disabled. Mastercam disables the option whenever the center is already engaged in the part.

Right-click menu options

Create center

Create a new center instead of selecting one from the list. The center definition is stored in the current toolpath group unless you save it to a library.

Edit center

Edit the definition for the selected center. The changes you make are stored in the toolpath group and will not affect future parts that use the center unless you save the changes to a library.

Tool manager

Displays the name of the current tool library, and opens the Tool Manager so that you can work with the library and the tools stored in the current toolpath group.

Load tool on machine

Create a tool component in the machine definition based on the tool definition for the selected tool. Mastercam uses the tool setup information in tool spindle and tool locator components to properly position and orient the tool for simulation.

View

Display the tools as large icons, list, or as a details list.

Arrange tools

Sort the tools by tool number or tool name. Use these options if you have the tools displayed as icons without any column headers.

Popup tool

A larger picture of the tool will popup if you let the mouse hover over an icon in the preview window. Use this option to control how long the mouse needs to hover before the larger picture pops up.

Save parameters to default file

Make the current toolpath settings the default for future toolpaths of this type. The current values are saved to the .mcam-defaults file specified in the toolpath group properties.

Reload parameters from default file

Overwrite the current values with the values from your defaults file. These will be read from the .mcam-defaults file specified in the toolpath group properties.
