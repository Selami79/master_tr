Lead In/Out dialog box: Lathe

Use these settings to control how the tool approaches or retracts from the part for each pass in the toolpath. This eliminates the need to create extra geometry for this purpose. You can also combine different types of moves. For example, you can extend a contour and lead in to it with a tangent arc.

Separate Lead In and Lead Out settings let you set different values for entry and exit moves.

Dynamic Roughing toolpaths let you create separate lead in/out moves for semi-finishing passes. Since most of the stock will have been removed by the roughing passes, this lets you create more compact and efficient moves for the semi-finishing passes. The only difference is that lead in/out moves for semi-finish passes cannot include any modifications to the chained contour; these options will not be present for the semi-finish passes.

NOTE

With Grooving toolpaths, the tabs are called 1st Pass Lead In or 2nd Pass Lead In.

Parameters

Extend / shorten start (end) of contour

You can create lead in/out moves by extending or shortening the beginning or end of the chained geometry. Select the checkbox to enable the move and specify the adjustment distance.

For chained groove toolpaths, set these parameters on the Groove shape parameters tab. Create the lead in and lead out moves separately on each tab.

Amount

Enter the distance by which you want the ends of the chained contour extended or shortened.

For chained groove toolpaths, set these parameters on the Groove shape parameters tab.

Extend

Indicates that you want to extend the start of the chained contour by the value in the Amount text box.

For chained groove toolpaths, set these parameters on the Groove shape parameters tab.

Shorten

Indicates that you want to shorten the start of the chained contour by the value in the Amount text box.

For chained groove toolpaths, set these parameters on the Groove shape parameters tab.

Add Line

Select to create a lead/in out move by adding a line to the chained geometry. Click Add Line to set its dimensions. Create the lead in and lead out moves separately on each tab.

For chained groove toolpaths, set these parameters on the Groove shape parameters tab.

Entry Arc (Lead in tab)

Select to add an entry arc to the start of each pass in the toolpath. Click Entry Arc to set its dimensions.

Exit Arc (Lead out tab)

Select to create exit arcs at the end of each pass in the toolpath. Click Exit Arc to set its dimensions.

Feed rate

Enter the feed rate to be used for entry or exit moves if they will be different from the toolpath. You can set the exit and entry feed rates independently. Deselect Same as toolpath to enable these fields.

Use rapid feed rate for vector moves

Select to use the rapid feed rate for entry or exit motion. If your control breaks rapid moves into separate X and Z components, these moves will be handled the same way.

Same as toolpath

Makes the feed rate and feed rate units for entry and exit moves the same as those for the toolpath.

Use entry vector

Select this option on the Lead in tab to create an entry vector. Use the other parameters to control the length and direction. This move will be added to the toolpath before the entry or exit arc is defined.

Use exit vector

Select this option on the Lead out tab to create an exit vector for the end of the toolpath. Use the other parameters to control the length and direction. This move will be added to the toolpath before the entry or exit arc is defined.

None

Choose to define the vector direction by entering a value in the Angle field or with the Angle dial.

Tangent

Forces the vector to be tangent to either the first move in the toolpath or the entry arc, if one is defined.

Perpendicular

Forces the vector to be perpendicular to either the first move in the toolpath, or the entry arc, if it is defined.

Angle

Enter the angle that the vector will approach (lead in) or retract (lead out) from the part.

Length

Enter the length of the vector.

IntelliSet

Sets the vector angle based on the tool orientation. Choose this option if you know the vector's angle relative to the part. The angle dial points in the appropriate direction. This option is enabled only if you have already selected a tool.

Automatically calculate entry (exit) vector

Select to have Mastercam calculate entry or exit vectors based on the stock, chuck, and tailstock information you have set. You can access this option only if you have defined stock in the machine group properties and Update stock for this operation is selected. It disables all other entry vector parameters.

Minimum vector length

Determines the minimum length for an automatically calculated vector. The calculated vector length may be greater than this value, if required, but never shorter.

Angle dial

Set the angle of the entry or exit vector by dragging the pointer to the desired orientation. Mastercam automatically enters the angle in the Angle field.

Resolution (deg.)

Snaps the angle dial to the closest multiple of this value. For example, if you enter a value of 7 and move the angle dial to 25 degrees, the angle dial will snap to 28 degrees since 28 is the closest multiple of 7.
