Creating Milling Operations in Mastercam Lathe

Mastercam Lathe includes two main techniques for creating milling toolpaths:

  • Use the Lathe C-axis toolpaths that are included with Mastercam Lathe for the most common contour and drill applications. These are available to anyone with a Lathe license and do not require a separate Mill license. When you select one of these toolpaths, Mastercam automatically sets the Tplane and Cplane to the appropriate settings for the selected application. These toolpaths also support Y-axis rotation and axis substitution. Choose the desired toolpath from the C-axis gallery on the Lathe Turning contextual tab. Your machine definition must include the appropriate rotary axis components.
  • If your workstation is licensed for both Mastercam Mill and Mastercam Lathe, you can also create Mill toolpaths. Use Mill toolpaths to machine in planes or orientations not available with the standard Lathe C-axis toolpaths, or to create toolpaths other than contour or drill toolpaths—for example, pocket toolpaths. You can set the Tplane to any view available in your part, or use the CView utility as a guide. Choose Mill from the Machine tab, and select the desired toolpath. The toolpaths that are available to you depend on the level of Mill license and the capabilities of your machine definition. Users with Mill 3D or Multiaxis licenses and multiple rotary axes can access Mastercam's most sophisticated and powerful toolpaths.
NOTE
  • Tool collision, stock, and other boundaries are not used for Mill toolpaths. However, machining jobs often use turning toolpaths to remove most the material before the milling operations are programmed. Since the turning toolpaths can use tool boundaries, you should set them up even if you will be creating milling toolpaths later on.
  • Mastercam' Mill-Turn product is an Add-In which runs on top of Lathe. It provides Mastercam's most complete support for mill-turn applications. See Comparing Mill-Turn and Lathe to learn more.
Contour toolpaths

Click a toolpath to read more about it.

  • C-Axis Contour Toolpaths: Select this toolpath to create a rotary toolpath which wraps around the spindle axis.
  • Cross Contour Toolpaths: Select this toolpath to cut parallel to the spindle axis, for example to cut a slot lengthwise on the part.
  • Face Contour Toolpaths: Select this toolpath to machine a contour on the face of the part. Do not use this for general facing operations; use the regular facing toolpath instead.
  • Contour Toolpaths: 2D: This is the most flexible, general-purpose contour toolpath from Mastercam Mill. Use it to mill a contour in any plane, not just the top plane, or for any application that can not be satisfied with one of the other C-axis contour toolpaths. Unlike the other C-axis drill toolpaths, Mastercam does not automatically set the tool plane (Tplane) and construction plane (Cplane). It is your responsibility to set these so that the tool axis maintains the desired orientation throughout the toolpath. If you wish, you can use the CView utility to help set the Tplane and Cplane properly. You can also include an axis of rotary motion if necessary.
Drill toolpaths

Click a toolpath to read more about it.

  • C-Axis Drill Toolpaths: Select this toolpath to drill from the side towards the centerline of the part.
  • Cross Drill Toolpaths: Select this toolpath to drill from the side either towards the centerline of the part or off-center.
  • Face Drill Toolpaths: Select this toolpath to drill on the face of the part parallel to the centerline, but not necessarily on the centerline.
  • Hole-Making Toolpaths: Mill/Router: This is the most flexible, general-purpose drill toolpath from Mastercam Mill. Use it to drill in any plane, or for geometry which cannot be drilled with the other C-axis drill toolpaths. Unlike the other C-axis drill toolpaths, Mastercam does not automatically set the tool plane (Tplane) and construction plane (Cplane). It is your responsibility to set these so that the tool axis maintains the desired orientation throughout the toolpath. If you wish, you can use the CView utility to help set the Tplane and Cplane properly. You can also include an axis of rotary motion if necessary.
