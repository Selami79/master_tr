Spindle Move dialog box

Use the Spindle Move dialog box to move the active spindle. The options vary based on the selected active spindle, the selected move type, and the machine configuration. Additionally, you have the ability to control to which data stream the operation is output.

Mastercam gives you the option to define a new spindle origin at the new location. This creates an entire set of planes whose origin is at the new location. Use these as tool planes for operations that you create at this location. The name that you enter in this dialog box for the destination will be used as the name for the new views. You can also choose to associate a work offset with the new location; this will be invoked automatically whenever you use one of the new planes.

Parameters

Move type

Select the type of movement for the spindle from the drop-down. Your selection determines the subsequent options in the dialog box. Choose from the following:

  • Linear position: Places the spindle at a selected position.

  • Rotary position: Places the spindle at the entered angle.

  • Retract to home: Retracts the spindle to the home position defined in the machine definition (*.mcam-lmd).

  • Move to stock clearance: Places the reference point of the chuck jaws the entered Distance from the front of the stock

  • Advance to grip length: Places the reference point of the chuck jaws at the front of the stock.

  • Incremental distance: Moves the spindle by the entered distance.

NOTE

The options work in conjunction with the reference point defined in the Machine Component Manager. Refer to the Parameters tab: Machine Component Manager (Chuck Jaws)Help topic for more information.

Active spindle

Select the spindle on which to perform the operation, Left or Right.

Data stream

Select the data stream in which to place the operation, Upper stream or Lower stream.

Distance

Available when Move type is set to Move to stock clearance, Advance to grip length, and Incremental distance. Moves the spindle by the entered amount. A positive value moves towards the home position, a negative value moves away from the home position.

Position

Available when Move type is set to Linear position. Places the spindle at the location. Enter a Z position or use the Select button to return to the graphics window to select a location.

Angle

Available when Move type is set to Rotary position. Places the spindle at the entered angle. The angle is relative to the zero position of the machine.

Feed rate

Choose to have the spindle move at the machine's Rapid rate or at the specified Feed rate. Feed rate is specified in inches per minute, millimeters per minute, or degrees per minute if Rotary position is being used.

Rotary direction

Available when Move type is set to Rotary position. Sets how the spindle should rotate to the entered Angle. Select one of the following:

  • Shortest distance: Travels in the shortest distance, regardless of direction, to get the proper angle.

  • CW: Rotates in a clockwise direction.

  • CCW: Rotates in a counterclockwise direction.

Create new spindle origin

Choose this option to have Mastercam create a new set of views whose origin is based on the new spindle location. Mastercam creates a copy of each view that was originally created in Job Setup.

Name

Enter a name for the new spindle origin. The name that you enter here will be used in the name of each new plane that Mastercam creates at this location.

Work Offset

Select Automatic to have Mastercam assign the next available work offset number. Select Manual if you want to use a specific offset number. Enter the offset number in the adjoining field. Mastercam includes this work offset in the views that it creates for the spindle origin so that the offset is automatically called when you use the view in a toolpath.

Comment

Enter a comment for the operation. Comments help identify the operation in the Toolpaths Manager.
