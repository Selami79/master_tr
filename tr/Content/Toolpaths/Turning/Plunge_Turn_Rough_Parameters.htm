Plunge Turn Rough Parameters tab

Use this tab to create the roughing cuts for a plunge turn toolpath. Select Rough to activate the options on this tab and create a roughing pass using the pictures on the tab as a guide. Select options on the right side of the tab to enable advanced features, such as retraction moves, cut direction, and wide step cleaning.

Deselect this option if you only want to create a finish operation.

NOTE
  • There are no cutter compensation options for plunge turn roughing. Cutter compensation in computer is always on and Mastercam determines the direction.
  • With plunge turn rough passes, there is no option for cutter compensation in control option. Cutter compensation is always on and Mastercam determines the direction.
Parameters

Rough

Select to activate the options to create roughing passes. Deselect if there is no rough passes.

Stock clearance

Determines how far above the top of the groove the tool starts the first plunge and retracts after the last cut.

Stock to leave in X

Determines how much stock remains in the X direction when the operation is finished. Typically, this would be removed by a separate finish operation. For plunge turn toolpaths, if the value is less than 10% of the tool width, you receive a warning message.

Stock to leave in X, Stock to leave in Y

Determines how much stock remains in the X direction when the operation is finished. Typically, this would be removed by a separate finish operation.

If you are using a Y-axis tool, the value of this field is a Y-axis coordinate, not X. When this happens, the name of this field references the Y axis instead of X.

NOTE

For plunge turn toolpaths, if the value is less than 10% of the tool width, Mastercam displays a warning message.

Stock to leave in Z

Determines how much stock remains in the Z direction when the operation is finished. Typically, this would be removed by a separate finish operation.

Depth of cut

Specifies the depth of the cut for each pass. You can define a distance or percentage of the tool width. If the value is greater than 60% of the tool width or less than 10% of the tool width, you receive a warning message.

Minimum cut depth

Use this option to enter the minimum cut depth as a specific distance or as a percentage of the tool width. The cut depth affects the tool's side deflection, which can cause vibration when a wide insert cuts at too small a depth. The minimum cut depth ensures that the cut depth is always appropriate for the insert's width.

Plunge feed rate

Enter the feed rate used for plunge moves and select the appropriate radio button:

  • in/rev or mm/rev: Calculates the value in inches per revolution, or millimeters per revolution for metric.
  • in/min or mm/min: Calculates the value in inches per minute, or millimeters per minute for metric.

Turn feed rate

Enter the feed rate used for turning cuts (side moves) and select the appropriate radio button:

  • in/rev or mm/rev: Calculates the value in inches per revolution, or millimeters per revolution for metric.
  • in/min or mm/min: Calculates the value in inches per minute, or millimeters per minute for metric.

Plunge spindle speed

Enter the spindle speed used for plunge moves and select constant surface speed (CSS) or revolutions per minute (RPM).

Turn spindle speed

Enter the spindle speed used for turn moves and select constant surface speed (CSS) or revolutions per minute (RPM).

Spindle transition dwell

Enter the transition dwell time and select Seconds or tool Revolutions. This is the transition from the plunge spindle speed to the turn spindle speed, and from the turn spindle speed to the plunge spindle speed. Choose None for no transition dwell.

Rapid

Retracts the tool at the rapid feed rate. This option is not recommended if you are creating an angled groove.

Feed rate

Converts rapid moves into fast-feed rate moves. The rough and finish passes use the same fast-feed rate, so the interface displays the rate on both the rough parameters and finish parameters pages. Choose the appropriate option:

  • in/rev or mm/rev: Calculates the value in inches per revolution, or millimeters per revolution for metric.
  • in/min or mm/min: Calculates the value in inches per minute, or millimeters per minute for metric.

Approach clearance

Determines the approach move's distance from the stock on each pass. Approach clearance is similar to stock clearance in groove toolpaths. The tool rapids to the given position, and then feeds into the part.

Cut direction

Select One way positive or One way negative to machine all passes in a single direction. Select Zig zag positive or Zig zag negative to machine each pass in the opposite direction from the previous pass. Zig zag cuts relieve tool pressure when transitioning between plunge cuts and traverse cuts.

Prevent hanging ring

A hanging ring occurs when the tool pushes off, rather than cuts, a small piece of material from the edge of the cut. Select a method, and the toolpath includes plunge moves to remove this extra material properly. Choose from the following: choose from the following:

  • Don't prevent: Hanging ring prevention is off.
  • Bi-Directional: Removes possible burrs from both sides of the cut
  • Positive: Removes a possible burr from the positive side of the cut.
  • Negative: Removes a possible burr from the negative side of the cut.

Cleanup steps

Select to remove steps that are too large to remove on a finish cut. Such large steps can remain when the width of the cut is wider than the width of the insert. The Wide step cleanup function removes steps that are wider than a user definable width, based on the width of the tool’s flat.

Maximum width

Specifies the maximum percent of the tool width used to cleanup steps. Smoothing will take multiple passes if the extra width is more than the specified maximum percent of tool width.

Filter

Select to eliminate unnecessary tool moves in the toolpath to create smoother movement. Click to configure toolpath filtering options.
