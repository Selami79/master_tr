Tool Inspection dialog box

Use the Tool Inspection dialog box to establish points during operations where the machine can be programmed to a specific position and stopped to allow you to perform tool inspections. You can set the conditions on which the tool inspections are performed, such as:

  • After a number of passes
  • After a specified cut length
  • After a specified cut duration

Additional options let you control whether a tool inspection can occur during a pass or near the start or end of a pass. You can also use reference points to get the tool in and out of the cut, and send the tool to a selected point or to the home position for the tool inspection position.

Parameters

Use reference points

Select to set the tool position to a pre-defined reference point. When you select this checkbox, the Reference Points dialog box opens, where you can select/create the reference point(s) you want to use for your tool inspection. The reference point is a point that the tool goes to on the way to and/or from the tool inspection position. The reference point is also used on the initial approach from home and the final retract going home. If you do not select or create a reference point in the Reference Points dialog box, the Use reference points checkbox will remain deselected.

Once you have set a tool inspection reference point, the Reference Points button is activated, allowing you to return to the Reference Points dialog box to edit your settings.

Home position

Select to set the machine tool's default Home position as the tool inspection position. This radio button and the User defined radio button are exclusive options.

User defined

Select this radio button to define a specific position for your tool inspection. This radio button and the Home position radio button are exclusive options. When you select this radio button, the following options are made available, allowing you to define the tool inspection position:

D

Displays the D coordinate for the position of the tool inspection. Select the checkbox and enter a value directly into the field, or click Select to return to the graphics window and pick a position.

X

Displays the X coordinate for the position of the tool inspection. Select the checkbox and enter a value directly into the field, or click Select to return to the graphics window and pick a position.

Select

Returns you to the graphics window, where you can pick a position to define the coordinates of the tool inspection point.

Absolute

Positions the tool inspection point relative to the tool plane origin.

Incremental

Positions the tool inspection point relative to the last point in the toolpath before the tool inspection.

Number of sections

Specify the number of sections to complete before tool inspection occurs. Available only when Section turning is active.

Number of passes

Specify the number of passes before tool inspection occurs. When using this option, the tool inspection always occurs after a pass completes.

Length of cut

Specify the cut length before tool inspection occurs. The length of rapid and entry/exit moves is not considered when calculating the cut length. When using Length of cut as a condition, tool inspection can occur during a pass, unless you also have Only between passes selected.

Time in cut

Specify the time spent in a cut before tool inspection occurs. Time spent in rapid and entry/exit moves is not considered when calculating the cut time. When using Time in cut as a condition, tool inspection can occur during a pass, unless you also have Only between passes selected.

End of operation

Select this option to perform a tool inspection at the end of the operation.

Only between passes

Ensures that tool inspection occurs only after a pass is complete.

Exact length and time

Select to let tool inspection occur during a pass. In this case, use the Lead in/out distance and Start/end of pass allowance values to determine exactly when the tool inspection can occur.

Use lead in/out

Sets the inspection process to use the values from the Lead In/Out dialog.

Lead in/out distance

Specify the distance to retract, at a linear feed rate, when exiting the cut for a tool inspection and when reentering the cut after the tool inspection. Mastercam calculates the lead in/out direction automatically.

Start/end of pass allowance

Specify how close to the start or end of a pass that a tool inspection can occur. If the tool is closer to the start or end of the pass than specified, Mastercam forces the tool inspection to occur before the pass begins or after the pass completes.

Output comment

Outputs a comment to the NC file. Select to activate the field, in which you can enter a comment to be included in the NC file where the tool stops. Enter comments such as "inspect tool", "clean out chips", or any other relevant information regarding that programmed stop.
