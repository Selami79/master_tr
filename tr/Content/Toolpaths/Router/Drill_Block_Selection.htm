Drill Block Selection dialog box

Use the Drill Block Selection dialog box to select a drill block and configure it for a block drill operation in Mastercam Router.

  • The Tool group window on the left lists all of the drill blocks that have been added to the active machine definition. Select a drill block to see its drill layout in the window to the right.
  • Expand the block to see a list of its tool stations. Right-click the selected station to edit the drill mounted in it, or select a different one.
  • For blocks with horizontally mounted drills, enter a safe height. This is a vertical distance measured directly above the table, regardless of which plane is active or used in the toolpath.

When you select a station, Mastercam displays other information about it, such as its location in the block and the work offset assigned to it. The fields are disabled here so you cannot edit them. Use the Machine Definition Manager to edit these fields if necessary.

Parameters

Tool group

Shows the block drills in the current machine definition. Click the plus sign next to a block drill to show the drills assigned to the block. Click on a station to select it; right click on the selected station to edit the drill assigned to it. The first tool added to a block is the lead drill and each subsequent drill is offset from the lead drill.

X / Y / Z offset

  • For an aggregate head, enter the distance between the center of the spindle axis and the center of the tool axis.
  • For a block drill, enter the distance along the X axis between the lead drill and the currently selected drill. The lead drill has a 0 offset, and the remaining drills are offset from 0.
  • For a secondary or piggy-back head, enter the distance that this head is offset from the main head.

Work offset

Enter the code for a work offset that you want associated with the selected station. The work offset code will be automatically output when you select the tool for an operation unless it is overridden in the toolpath parameters. Assigning an offset is optional, and typically depends on your machine or post requirements. Even if you use offset codes, you still need to type in the coordinates of the Head and Tool positions.

Machine view safe height

Creates an additional retract move in absolute coordinates, in the Z-axis of the machine view.

Block number

Enter a number that will be used to identify the block or aggregate head. This often corresponds to a register number in the control. Assign a different number to each block or head that you create. If you load multiple blocks, Mastercam will organize them by block number.

Arc matching tolerance

Mastercam uses this value to determine which drills in the block can drill each selected hole. Mastercam will not drop a drill for a hole if the radius of the hole differs from the drill radius by more than this amount. If there is no drill which fits a selected hole within the tolerance, the hole will not be drilled.

Right-click menu

Edit tool

Change the dimensions or other properties of the tool in the selected station.

Get tool from Tool Manager

Replace the current tool with a different tool selected from a tool library.

Enable tool

Right-click on a tool and select this option to enable the tool for the toolpath.

Disable tool

Right-click on a tool and select this option to disable the tool for the toolpath.
