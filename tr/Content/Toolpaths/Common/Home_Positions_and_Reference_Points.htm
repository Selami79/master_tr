Home Positions and Reference Points

The Home position is where the tool spindle or turret typically travels for tool changes and at the end of the NC program. Reference points lie between the home position and the start or end of the toolpath.

NOTE
This topic does not apply to Mill-Turn. To learn more, see the "Machine Reference Position Manager" link at the end of this topic.
Home positions

Mastercam lets you control the home position from several different sources. You can:

  • Enter a default value for the machine definition, which is available to all operations created for that machine. Use the Machine Axis Combinations dialog box to enter a position for each axis combination. If you don't need to create axis combinations, enter the home position coordinates in the Default axis combination. Use world coordinates to enter the home position in the machine definition.

  • Create an operation-specific default value and store it in the .DEFAULTS file.
  • Store a home position for the tool as part of the tool definition that will be available whenever the tool is selected for an operation. (Lathe only)

Use the Tool page in the control definition to select which home position will be the default for new operations, as shown in the first image following. However, no matter where the default value is coming from, you can override it on the Toolpath parameters tab, as shown in the second image following.

NOTE

The home position for each operation is expressed relative to the tool plane (Tplane) origin used for the operation.

Reference points

Reference points lie between the home or tool change position and the beginning of your toolpath.

  • The approach reference point is the starting position for the entry or lead-in move.
  • The retract reference position is the destination for a exit or lead-out move.
  1. Home position
  2. Reference point - Approach
  3. Reference point - Retract

Create separate approach and retract reference points by selecting the checkbox in front of Approach or Retract and entering the desired coordinates. Select the checkbox for each axis that will be included in the reference point. For example, if you select the X and Z axes, but not Y, Mastercam will output only an X and Z coordinate in your NC file.

Default reference points are stored in the machine definition as part of the axis combination properties. In other words, each axis combination can have its own set of reference points. For lathe machine definitions, you can create separate reference points for ID and OD work. When you are creating an operation, click From machine to use the default reference point from the machine definition; Mastercam will select the proper location based on the current axis combination.

Reference points are not required. If you do not enable them, the tool will rapid directly from the home or tool change position and the first/last programmed position in the toolpath.

Reference points in Lathe and Mill-Turn

For most turning operations, Mastercam only displays two axes—typically, X and Z. Where appropriate, such as operations that use Y-axis tools, all three axes are displayed.

Mill-Turn implements reference points differently than Lathe. In Mill-Turn, each .machine file contains a set of reference positions whose function is the same as reference points in Lathe—the first position in the approach move, or the last position in the retract move. Click the Edit reference positions button to see the reference positions that have been defined for your machine.

In this case, the reference points that you define in the Reference Points dialog box are intermediate locations that lie between the first/last point in the toolpath and the reference position stored in the .machine file.

U.S. and metric values

If you are creating default reference points for a machine definition or axis combination, you should record separate sets of values for U.S. and metric operation. Click the Convert to mm or Convert to inch button to automatically convert one set of values to the other units and write them to the proper fields.

NOTE

You can disable the reference points feature on the NC dialog page in the Control Definition Manager. When this is the case, reference points are not available when creating a toolpath, even if they have been defined in the machine definition.

World coordinates vs. part coordinates

Home and reference points saved in the machine definition are defined relative to the world coordinate system origin. Typically this is the center of the spindle nose for a mill, or the main spindle face on a lathe.

If you are setting reference points for a single operation, the points are defined relative to the tool plane used for that operation.
