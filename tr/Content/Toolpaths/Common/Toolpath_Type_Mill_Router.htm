Toolpath Type page: 2D, C-axis Toolpaths

The Toolpath Type page is the starting point for creating a 2D, 2D high speed, or C-axis toolpath. You can change your toolpath type or select new or additional geometry using the options on this page.

Automatic regions

Use Automatic regions to automatically create Machining, Air, or Avoidance regions based on the selected solid geometry.

You can mix and match Automatic regions with other region selection types; you do not have to use one or the other. You can choose to set manual chains as well. Use these three methods together to create the best chaining results.

To create automatic regions
  1. Click Select automatic regions.

  2. Select the faces to machine.

    At this point, you may wish to click Preview chains.

    Mastercam displays machining regions as red and black stripes, air regions as blue, and containment regions as surrounded by yellow lines.

    If necessary, you can choose to create Machining, Air, or Avoidance regions as well.

  3. When you are satisfied with your chains, click OK in the Solid chaining dialog box. The Chain Options dialog box displays the number of Automatic regions.

To have more control over individual regions, you can convert your Automatic regions into Machining, Air, or Avoidance regions. This allows you to edit each region as necessary.

There are three ways to convert Automatic regions to other regions:

  • Use the Convert automatic regions button in the Chain Options dialog box.
  • Use the Convert automatic regions button in the Toolpath type page.

  • Click Convert to regions in the right-click menu of the Chain Manager.

After you convert to Automatic regions, the number for each type of region updates.

Machining regions

Select an area to be machined. You can select multiple machining regions. Use the Machining region strategy options to determine the cutting strategy. The image below shows the rectangle and the circle as the only selected Machining regions.

NOTE

If the selected Machining regions are open chains, you can extend the open chain to stock with the options under Open chain extension to stock.

Machining region strategy

Select how the toolpath machines the selected geometry.

  • Stay inside: This strategy is enabled by default and keeps the toolpath inside the selected machining geometry. Use this strategy when machining pockets.

  • From outside: Select to use free-flowing motion to machine features, such as standing bosses or cores. A toolpath using this strategy starts from the outside and work its way in.

Open chain extension to stock

Select to extend a selected open chain for the Machining region to the following:

  • None (ignore stock): Select to ignore the stock and to not extend the open chain selected for the Machining regions.

  • Tangent: Select to extend the open chain selected for the Machining regions tangentially to the stock.

  • Shortest distance: Select to extend the open chain selected for the Machining regions by the shortest distance to the stock.

Avoidance regions

Select an area to be avoided during machining. Multiple avoidance regions can be selected. The image below shows the circle as the selected Avoidance regions.

Clearance

Sets the distance the tool will stay away from the selected Avoidance regions. Additionally, the entered amount is added to or subtracted from the entered Stock to leave on walls and Stock to leave on floors values found on the Cut Parameters page.

Air regions

Select an area where there is no material. This allows the tool to travel through it when machining. The image below shows the circle as the selected Air regions.

Use the following guidelines when selecting an air region:

  • An open air region chain can be paired with an open machining region chain. The endpoints must match XY coordinates.
  • An open air region chain can be paired with a closed machining region chain, but the elements of the open air chain (solid edges, lines, arcs, or splines contained in the chain) must lie completely on the machining boundary. However, the endpoints of the open air region chain do not need to match exactly.
  • You can have multiple open and closed air region chains for each closed machining region chain.
  • You can select a segment of a closed machining region chain to allow the tool to approach from the outside of the selected segment.

Expand region

Expands the selected Air regions so that the tool can move while trying to reach the material. This improves the amount of reachable material on certain open pocket shapes.

Containment regions

Select an area outside of which the tool cannot travel. Multiple containment regions can be selected. The image below shows the rectangle as the selected Machining regions, and the polygon and circle as the selected Containment regions.

Entry chain

Select a chain where the tool enters the part. You can select multiple entry chains. The image below shows the black line as the selected Entry chain.

Select

Returns you to the graphics window to select a chain.

Preview chains

Dynamic Mill and Area Mill toolpaths only. Select to preview your currently selected machining regions, air regions, and containment regions. Use Color to customize the colors or change the colors through the Colors page in the System Configuration dialog box.

Color

Dynamic Mill and Area Mill toolpaths only. Select to open the Preview Chains Colors dialog box. Use this dialog box to set the preview chains colors.
