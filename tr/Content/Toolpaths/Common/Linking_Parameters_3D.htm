Linking Parameters page: 3D High Speed Toolpaths

Use the Linking Parameters page to create links between the cutting passes for a 3D high speed toolpath. In general, you can think of linking moves as air moves when the tool is not in contact with the part, compared to cutting moves which are configured on the toolpath's Cut parameters page. In the image below, the cutting motion is colored blue. Other types of motion, some of which are defined on this page, are displayed in other colors.

First/Last Leads page: 3D High Speed Toolpaths

Use this page to control how the tool moves onto and off of the part at the start and end of the operation. The parameters on this page provide you with more precise control over the first and last moves of your toolpath than the Leads and Secondary Leads options on the Linking Parameters page. Finally, select a Fitting option to control how the entry and exit arcs will be fit into each pass.

If you do not enable the Lead In and Lead Out options on this page, the Leads and Secondary Leads options on the Linking Parameters page will control the leads for the first and last pass.

  • Parameters for First/Last Leads page

 

First, select a retract Type. This determines how the tool will move between the end of one pass and the beginning of another. Then, use the Leads fields to control how the tool moves onto and off of the part at the start and end of each cutting pass. These moves are applied to each pass no matter which cutting pass is selected. Finally, select a Fitting option to control how the entry and exit arcs will be fit into each pass.

How do these parameters affect my linking motion?

Mastercam divides the moves which link cutting passes into a number of discrete components so that you can have the maximum degree of control over them.

The picture below shows the relationship between the different parameters when you select the Minimum distance retract type. Mastercam divides the move into two zones: to/from the retract plane, and to/from the part. Each move is a separate arc.

  1. Rapid move to start of next pass.
  2. Part clearance (minimum).
  3. Cutting pass.
  4. Arc fit radius is the radius of the arc as the tool moves away from the retract height.
  5. Radius is the radius of the arc as the tool moves toward the part.
  6. Radius is the radius of the arc as the tool comes off the part.
  7. Linear entry/exit extends the entry and exit vectors.
  8. Arc fit radius is the radius of the arc as the tool moves to the retract height.

Use the Fitting parameters to modify how the entry and exit arcs are actually applied to the cutting pass.

Typically, the height of the retract move is determined by the size of the arcs that you enter plus the linear entry/exit moves. Use Part clearance to define the minimum height that the tool needs to maintain over the part. Mastercam will automatically extend the linear entry/exit moves if necessary to maintain the skim distance.

How does a toolpath know when a move is a retract or a transition?

Mastercam determines when to create a retract move or a transition move based on the Keep tool down within parameter.

  • When the distance between two passes is less than the Keep tool down within amount, a transition move is added.

  • When the distance between two moves exceeds the Keep tool down within amount, a retract move is added.

The type of transition or retract move is set by the Type option in their respective parameter group. You can learn more about these three options in the Parameters drop-down below.

Clearance plane

Defines a safe height above the part that is clear of all part features and fixtures. When Full vertical retract is selected, the tool retracts to this height between cutting passes.

Absolute

Absolute values are always measured from the origin 0,0,0. If you were to translate geometry associated with a toolpath, and the toolpath depth was set to an absolute value, the tool will try to cut to that same absolute depth value no matter where the geometry is located.

Incremental

Incremental values are relative to other parameters or chained geometry.

For 3D high speed toolpaths, the incremental Clearance distance is calculated relative to the adjacent feed-rate move. Note that this option does not result in incremental coordinates being output in the NC program; use the Control Definition Manager to determine that.

Type (Retract)

Select the type of retract move you want to create when the distance between cutting passes exceeds the Keep tool down within amount.

  • Minimum distance: Mastercam calculates a direct route from one pass to the next, incorporating curves on/off the part and to/from the retract height to speed progress.
  • Minimum vertical retract: The tool moves vertically to the minimum Z height needed to clear the surface. It then moves along this plane in a straight line, and drops down vertically to the start of the next pass. The minimum height of the retract is set by the Part clearance.
  • Full vertical retract: The tool moves vertically to the clearance plane. It then moves along this plane in a straight line, and drops down vertically to the start of the next pass. The height of the retract is set by the Clearance plane.

Part clearance

Sets the minimum distance the tool maintains above the part while it is rapiding between passes. The actual retract height is set by the Arc fit radius. However, if the retract height is less than the Part clearance, Mastercam adds an additional linear move between them until the total height equals the Part clearance.

In addition, if there are part features in between the start and end of the linking move, Mastercam increases the height of the move to maintain the Part clearance above those features.

Arc fit radius

Rounds sharp motion found in all retract moves.

Arc fit radius on and set to 0.15 Arc fit radius off

Feed rate (Retract)

Sets the feed rate during retract moves.

Linear entry/exit

Extends the entry and exit vectors.

Keep tool down within

Determines when a toolpath needs to create a retract move or transition move.

  • Distance: If the distance from the end of one pass to the start of the next pass is less than this distance, Mastercam will not create a retract move as defined on the Linking Parameters page. Instead, the tool will stay on the surface and move directly between the passes at the feed rate.
  • Percent of tool diameter: If the distance from the end of one pass to the start of the next pass is less than this distance, defined as a percentage of the tool diameter, Mastercam will not create a retract move as defined on the Linking Parameters page. Instead, the tool will stay on the surface and move directly between the passes at the feed rate.

Micro lift retract

Controls retracts in the toolpath when making a non-cutting move within an area where the tool can be retracted or microlifted. Choose one of the following options:

  • Never: Eliminates retracts from the toolpath for these types of tool moves.
  • Avoiding a boundary: Adds retracts to avoid intersecting boundaries (a gouge boundary, or a boundary of material yet to be milled). The types of boundaries that Mastercam detects and avoids include chained geometry, part features, and stock.
  • Exceeding a distance: Adds retracts when the next cut pass begins at a distance that is greater than the distance you define, either as an absolute Distance or as a Percent of tool diameter. The distance is the allowable length of the down or microlifted move.
NOTE

Choose options that include boolean expressions (OR/AND) to further customize retract moves for your application.

Micro lift distance

Enter the distance the tool lifts off of the part on back moves. Microlifts are slight lifts that help clear chips and minimize excessive tool heating.

Back feed rate

Controls the speed (inches per minute or millimeters per minute) of the tool's backfeed movement. This allows 3D arcs between cuts to have a different feed rate than the rest of the toolpath, which can help reduce cycle time.

Keep tool down between Z depths

Prevents the tool from retracting during moves between Z depths, leading to smoother linking moves.

Feed rate

Controls the feed rate for the moves between Z depths.

Type (Transition)

Select the type of transition move you want to create when the distance between cutting passes is less than the Keep tool down within amount. In the examples below, A represents the front view and B represents the side view of a Waterline toolpath.

  • Tangential Ramp: Creates a true high speed transition between the cutting passes. Arcs are inserted at the beginning and end of the ramp for the smoothest tool motion into and out of the move.

  • Ramp: Moves between passes with a straight line at an angle that you specify.

  • Straight: Moves between passes with a simple perpendicular line.

  • Smooth: Creates high speed arc moves between each pass. Each transition move consists of an arc move into and out of the transition, plus a spline curve which approximately follows the boundary.

For all types of transitions, the tool will follow the surface profile as shown in the side views.

Angle

Enter the angle for the tangential ramp or ramp transition move.

Feed rate (Transition)

Sets the feed rate during transition moves.

Maximum lift

Analyzes transition moves for a total height in Z and compares it to the entered distance. If the transition moves exceed the Maximum lift value, the transition moves are replaced by a simple retract. This helps to avoid long, tall, or steep transition moves.

Maximum lift on Maximum lift off

Apply leads

Adds leads to transition moves. For example, you can add a vertical arc entry or exit move to a pass. You can apply these leads to Open passes or both Open and closed passes.

Apply leads on Apply leads off

Lead In/Out (First/Last Leads page only)

Enables the options for the first and last lead in and out motion.

Method (Leads)

Sets whether or not the lead moves are the same for both lead in and lead out motion.

Type (Leads)

Sets the lead in and lead out motion. Select either None, Vertical, or Horizontal.

Distance

Adds a tangent extension to the end of a lead in/out pass.

Sweep angle

Sets the angle of the lead in/out moves.

Radius

Sets the radius of the lead in/out moves.

Ramp angle

When you enter a value other than zero for Distance (when set to Horizontal), the entry move become a small helix when moving into a new region. Use this field to set the height of this helix.

Secondary leads

Creates a second set of lead in and lead out moves.

Type (Fitting)

Determines how the cutting passes will be modified to fit inside boundaries, walls, and other obstacles. This determines how the lead in and lead out moves are added to the cutting pass.

  • Machine entire pass: The path of the tool matches the Machining geometry, including vertical walls and corners. A lead in/out move will only be inserted at the end of the pass, and then only if it can be done safely without hitting the part.
  • Fully trim pass: The pass is trimmed back so the lead in/out move fits into it, but no nearer than a full machine pass link would be. Select this option when it is important to prevent over-machining.
  • Minimize trimming: The path of the retract will be as close to the Machining geometry as possible, maintaining a minimum distance from the part to fit the lead in/out move.

Maximum trimming distance

Limits the amount of trimming applied to non-horizontal passes. Normally, when a pass is trimmed, the length of pass trimmed off is at most the Radius of the lead in/out move.

For steep finishing passes, the total length of a pass trimmed can be much greater. Enter a value to avoid this. If the amount trimmed would exceed this value, then no lead move is used. Instead, the whole pass is machined and a straight vertical motion is added. Ensure that this value is at least as large as the Radius of the lead in/out move.

Fit transitions

Trims passes to safely fit the transition moves.

Fit transitions on Fit transitions off
