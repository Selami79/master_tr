Holder page

Use the Holder page to select the tool holder to be used in the operation. Select a holder for this operation by selecting it in the list in the left window. You can also create a new holder, load a holder from a library, and edit a holder.

To load a holder from a library, click Open library and select a library of holders. Mastercam replaces the entire list with the contents of the library. To edit a holder, right-click a holder in the list and select Edit Holder. To create a new holder, click New holder in any toolpaths Holder page.

When you create a new holder, it will display in the holders list with an asterisk (*) indicating that the holder has not yet been saved to a library. When you select a holder, Mastercam displays a preview of its geometry.

Holders window

Lists all of the holders available to be selected. The list includes all the holders in the currently open library plus holders that have been defined on this page, whether or not they have been saved. Right-click on a holder to see additional options and functions.

Open library

Click to open a different holder library and display its holders in the window. The current list of holders will be completely replaced by the holders from the newly opened library.

New library

Click to empty the list of holders so that you can create a new list or library.

Save library

Click to save the currently displayed list of holders to a library. You will be prompted to select a name or type a new one, so you can use this command to create a new library from an existing list of holders.

New holder

Use to open the New Holder Wizard where you can create a new holder. The holder definition will be stored in the current toolpath group. It will also be stored in the current library if you are creating it within a library.

Tool projection

Enter the desired exposed length of the tool, projected from the bottom of the holder to the cutting tip of the tool.

Assembly name

Displays the name of the selected assembly. Any changes made to the name of the assembly in the Holders page will reflect in other areas of Mastercam, such as the Tool Manager.

Collision checking

3D high speed and Model Chamfer toolpaths only. When this option is selected, Mastercam will make sure that the holder geometry as described on this page does not come into contact with any geometry in your part model. Mastercam will restrict cutting passes as necessary to ensure this. When selected, Mastercam displays the approximate holder clearance zone in red.

Trim to avoid gouge

Select to trim or modify the toolpath to ensure that the tool holder does not gouge the part. Mastercam identifies areas where the tool holder would collide with the part and may modify the toolpath to avoid these locations.

Tilt to avoid gouge

Select to tilt the tool to avoid gouging the tool holder with the part. This parameter requires you to have a Multiaxis license and a spherical or taper tool.

Additional avoidance

Select additional avoidance geometry to be used in collision checking when Tilt to avoid gouge is selected. Additional geometry is only used in the tilt motion calculation.

Maximum tilt angle

Only available when Tilt to avoid gouge is selected. Enter the maximum tilt angle to apply to avoid a collision.

Shank clearance

Enter a distance that is the closest the tool shaft should get to the part being cut.

Holder clearance

Mastercam uses this value when gouge checking the toolpath against the tool holder. Mastercam ensures that the tool holder will stay at least this distance from the surface model.

If you are leaving stock on the walls of your part, the holder clearance should be at least as large as the amount of leftover stock.

Clearance on holder bottom

Select to apply the Holder clearance value to the bottom of the holder.

Right-click menu options

Depending on where you right-click on the page, you may see different menu options.

Delete Holder

Removes the selected holder from the holder list. This does not remove it from the library until you click Save library.

Rename Holder

Enter a new name for the holder. The new name is not written to the library until you click Save library.

Duplicate Holder

Add a copy of the selected holder to the list with a default name. Typically, you will then edit the new holder. Click Save library to write it to the holder library.

Save Selected Holder

Saves only the selected holder to a holder library. When the Save As dialog box opens, select the library in which you want the holder to be saved, select the library type, and click Save.

Edit composite holder

When you right-click an assembly containing a combination of holders (composite holder), this option replaces the Edit Holder option. Edit composite holder has a fly-out menu, allowing you to choose which holder or extension you want to edit. Choose the option from the fly-out menu to open the Holder Wizard to edit that component.

New Holder

Use to open the Holder Wizard. The holder definition will be stored in the current toolpath group. It will also be stored in the current library if you are creating it within a library.

Edit Holder

Use to open the Holder Wizard, where you can edit an existing holder.

Open Library

Lets you select a library so you can select its holders. The current holder list is cleared when you open the library.

Save Library

Saves the current holder list to a library. You can choose to type in a different name to create a copy of a library.

New Library

Clears the holder list so you can create a new library.
