Process Hole panel

Use this Mill or Router toolpath to apply hole operations stored in an operations file (called the template) to similar holes in the currently loaded part. The template operations, which can be stored in a Mastercam (.mcam) file or an operations file (.mcam-operations), must be solid holes with the same diameter and number of segments as the target holes. The following image shows three segments on a tool, including the tool's tip.

Creating a template for the Process Hole operation

To prepare for the Process Hole operation, you must first create a template file. Your template should contain the following:

  • A base solid. You will add solid holes to this entity.

  • One hole for every type, size, and configuration used in your work.

  • Unique, descriptive names for each hole, as shown in the following example.

  • A toolpath group for each hole type that is named to correspond to the hole type. The names need not match exactly, but should be similar enough that you can see which group goes with which hole type.

  • A toolpath group with toolpaths that apply to a solid hole type. See the Simple 10mm groups at the bottom of the list in the previous image for an example.

  • Valid 2D toolpath parameters and tools that match your typical parts and workflow.

NOTE  

  • You do not need to define tool axis control or a safety zone in your template file. The controls in the Process Hole operation will override these settings.

  • Each toolpath group should contain all the toolpaths necessary to complete the hole to your standards.

  • You need to select just one solid hole when defining each toolpath group in the template file.

  • Set the linking parameters to Incremental as required by the hole geometry, unless you are certain your parts will always have the same top of stock, depth, retract, and so on.

Creating a Process Hole toolpath

Before you attempt to create a Process Hole toolpath, ensure that you have created a template file that contains the operations you want to apply to other parts. The template should contain all of the hole operations you will need to apply to your target files. Keep in mind that the template's hole operations must be solid holes of the same segment diameters and number of segments as the holes to which the template will be applied. The length of the individual hole segments is inconsequential. Store your template file in an easily accessible location, and then follow these steps:

  1. Open into Mastercam the file to which you want to apply the template operations.

  2. On the Toolpaths tab, select Process Hole.

    The Process Hole function panel displays.

  3. Select the holes to which you want to apply the template. A single click selects one hole, whereas [Ctrl+click] selects all of the holes of the selected type. Your selections appear in the Positions list.

  4. Set the options for sorting and avoidance geometry as required for the job.

    NOTE
    To learn what an option does, hover your mouse over the option in the panel to display its tooltip.

  5. Select the Process page.

  6. Click Select library.

  7. Navigate to and select the template file containing the operation to apply.

  8. Select the process to use. The operations appear in the Operation list.

  9. In the Comment field, enter a description of the operation.

  10. Select the Linking page, and set the options as required for your job.

  11. Click the green check to finalize the toolpath.

NOTE

  • The Process Hole toolpath appears as a single entry in the Toolpaths Manager. To see the child operations, right-click the Process Hole operation, and choose Explode Process Hole. You can then view and edit the individual operations. However, you cannot then collapse the child operations back to a Process Hole operation.

  • Only processes that are based on the solid holes selected are displayed in the Process drop-down.

  • Safety zone linking is only available when the selected geometry requires multiaxis motion. Multiaxis motion is required when the hole axis is not perpendicular to the tool plane.

  • Minimum height linking in a Process Hole toolpath is equivalent to automatic linking in Hole Making toolpaths. The height selected is the minimum value that the tool travels above the avoidance geometry.

  • Minimum height and Clearance plane are the only linking options for 2D geometry.

  • If you create a multiaxis Process Hole toolpath, but neglect to define a safety zone, Process Hole automatically creates a rectangular safety zone for you.

Editing Process Hole operations

You can edit Process Hole operations by clicking the Edit operation button on the Process page of the Process Hole panel.

When you click the button, the 2D Toolpaths dialog box for the operation displays. In this dialog box, you can edit the operation as needed. However, some parameters should not be changed; if you do change these parameters, Mastercam reverts them to their original values. These parameters are those on the Tool Axis Control, Safety Zone, and Planes pages, and the Clearance settings on the Linking Parameters page.

When you finish editing an operation, a pencil icon appears on the operation's row, as shown in the following image.

Here are things to remember when you edit Process Hole operations:

  • When you remove geometry or a process, Mastercam disables all but the Cancel button at the top of the panel. The panel remains in this state until you select valid geometry and processes.

  • When you change the type of hole selected, there must be a valid process in your template to regenerate the Process Hole toolpath.

  • When you add solid holes in a different plane, Mastercam changes your toolpath to multiaxis, altering your linking options. Go to the Linking tab to set the safety zone properly. The default safety zone is a rectangle with 0 offset.

  • When you remove holes that do not match the tool plane, you alter your toolpath, changing it to 2D. Edit the options in the Linking tab to reflect this change.

  • If you edit the template used in the Process Hole toolpath, and you want to see the change reflected in your toolpath, you must reload the template and select the process. The template will not update automatically.

  • If you delete geometry selections and then click Cancel, the geometry remains deleted.

  • The Process Hole toolpath regenerates automatically when you edit the toolpath parameters. It does not regenerate automatically when you edit the geometry.
