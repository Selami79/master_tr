Working with Tool Numbers and Offsets

The tool number and tool offsets (length, diameter, and-for lathe-back offsets) that are output to your NC file come from the Toolpath Parameters tab or Tool page for each operation. Whenever you select a tool for an operation, Mastercam generates a default tool number and default tool offsets. Mastercam gives you several different ways to configure how the default tool numbers and offsets are selected.

Tool numbering

You can generate tool numbers in either of two ways:

  • You can choose to number tools sequentially by operation, so that the tool for the first toolpath will be tool number 1, the second operation will use tool number 2, etc.
  • You can read the tool number from the tool definition, as stored in the tool library.

Select the method you want to use in the Machine Group Properties dialog box (Lathe) or Machine Group Setup function panel (Mill or Router) for each machine group as shown in the following picture.

This sets the tool numbering properties for the current machine group only. To make this setting permanent, edit the operation defaults file and save the setting there.

Tool offsets

You can generate tool offsets in either of two ways:

  • You can generate an offset based on the tool number. You can set the offset to be equal to either the tool number, or to the tool number plus another number. For example, you can tell Mastercam to use length offset 1012 when tool number is 12 is used.
  • You can read the tool offset from the tool library. Each tool definition has default length and diameter offsets. For Lathe, these are stored in the Parameters tab of the Lathe Define Tool dialog box. For Mill, these are created through the tool properties in Mastercam's stand-alone Tool Manager. To work in the stand-alone Tool Manager, select the Windows Start menu, and choose Mastercam, Tool Manager.

Select your preferred method in the Tool page in the Control Definition Manager as shown in the following picture.

NOTE
  • If you choose Add to tool, Mastercam only calculates this when you first choose the tool. If you manually edit the tool number, the offset number will not Tool Definitions automatically update.
  • No matter how your tool numbers or offsets are chosen, you can always override them in the toolpath parameters. The values that appear on the Toolpath Parameters tab or Tool page, will always be the values output to your NC program.
  • For Lathe tools, enter default tool length and diameter offsets in the Parameters tab of the Define Tool dialog box. You can also enter a back offset.
