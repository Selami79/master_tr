Material List

Use the Material List to select materials and manage the materials list.

Right-click in the main window to see a menu of options. These let you edit a material definition, create a new material, select different material libraries, and view the contents of a material library in different text formats.

  • Use the options in the Source list to display materials in the current part file or from a material library.
  • The Compress option reads a selected material library file and removes any deleted library entities. This decreases the size of the library file for more efficient storage.
Parameters

Material list window

Displays the list of materials. Right-click in the window to see a menu of material management options.

Display options

Select how the material list window items are displayed:

  • Show all: Displays all materials from the selected material library.
  • Inch: Displays only inch materials from the selected library.
  • Millimeters: Displays only millimeter materials from the selected library.
  • Meters: Displays only meter materials from the selected library.

Source

Select a library from the list to display in the window, or select Mill-current or Lathe-current to display materials that have been selected in the current Mastercam file.

Compress

Reads a material library file and removes any deleted library entities. This decreases the size of the library file for more efficient storage. This button is only available if you selected a library from the drop-down list.

Right-click menu options

Get from library

Opens the material library so that you can load a different material definition. The materials from the library will appear in another instance of the dialog box. Select the new material and choose OK to return.

Save to library

Writes the selected material to the materials library. Choose this option if you have edited a material or created a new one and want to make the change available to other part files. If you do not, the changed definition will be saved with the current part file.

Create new

Opens the Material Definition dialog box so you can create a new material definition. The new material will be saved in the current part file unless you choose Save to library as described above.

Delete

If the list is displaying materials from the current part file, this will remove the selected material from the current file but leave it in the material library. To delete a material from the library, select the library in the Source drop-list, find the material, and then choose Delete.

Edit

Opens the Material Definition dialog box so you can edit the material.

Change library

Use this option to display materials from a different library. The name of the library is displayed in the title bar. This option only appears when you are displaying materials from a library (as selected in the Source drop-list).

Convert a library to text

Outputs the contents of a material library in a text format. You will be prompted to select which library to convert. Mastercam only makes a copy, your source library is unchanged.

Create a library from text

Creates a materials library from a text file. Convert a sample material library to a text file using the Convert... to text option to learn about the proper record structure.

Report

Creates an ActiveReport about the materials in the list. These reports are designed to be easier to read than the Convert a library to text option and can be exported to a variety of formats. ActiveReports are customizable with the ActiveReports Designer.
