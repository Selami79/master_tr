Batch Toolpath Operations dialog box

Use the Batch Toolpath Options dialog box to create lists of operations for batch processing and to run batch jobs. Batch jobs let you post operations from many files at one time without having to open every file individually. You can also repeatedly post the same set of files without having to select them every time by saving the operations list to a batch file.

To run a job saved in a batch file, use Open an existing batch file to open the batch file, and then choose OK to run it. Use the other dialog box controls to build an operations list, create new batch files, or set job options for processing batch files.

Click a drop-down below to learn more about each stage of batch processing.

Building the operation list

The operations that are posted are listed in the Batch file window. Add operations to the list by opening batch files or by opening individual Mastercam files and then selecting operations in the Source file window.

  • To move an operation to the batch list, select it, and then click Add toolpath operations to the batch list.
  • To remove a file from the batch list, click it, and press the [Delete] key.
NOTE

Opening a new batch file overwrites the contents of the Batch file window.

Processing the source files

Once the batch file list is complete, choose how you want to process the source files:

  • You can choose to automatically Regenerate all operations in the source files. Consider using this option if the operations have been automatically tagged for batch processing and have not been regenerated within the source file.
  • Choose whether or not to save each source part file after it has been regenerated.
Posting the selected operations

Select Run post to post the operations and create the NC files. If you do not select this option, Mastercam opens the source files and regenerates toolpaths, but does not create NC programs. Choose Post settings to tell Mastercam whether you want to create NC or NCI files, whether you want to edit them, and whether to transmit them directly to your machine tool control.

  • If you select Run post but do not select Post settings, Mastercam uses the default preferences saved with the control definition.

Mastercam posts each operation using the post and machine definition specified in the operation's machine group and writes to the NC file name specified there. You can include operations from many different machines, using many different posts, in the same batch file. If you have chosen to edit the NC files, each NC file opens in its own text editor window.

Working with batch files

The three buttons above the batch list let you do the following:

  • Open a batch file and add its operations to the list.
  • Create a new batch list.
  • Save the operations in the batch list to a batch file.

When you are satisfied with the list of operations in the batch list, you can choose to either run the job directly by clicking OK, or save the operation list to a batch file, so you can run the same job again in the future.

You can also view or edit any batch file directly in a text editor, by clicking Advanced and choosing Edit now. Opening a batch file in the text editor does not affect the list of operations displayed in the batch list window.

Click OK when the operation list is complete to run the batch job.

Click Advanced to set batch processing time limits on surface operations and batch file editing preferences. You can also open an editing session to view or edit any batch file, without interfering with the one that is currently displayed.

Parameters

Source operations

Click to select a Mastercam file. All of the file's operations are listed in the window so you can choose which operations to add to the batch job.

Move to batch list

Click to move the selected file(s) into the batch operations list.

Batch file

Click to select a batch file. The operations and the name of the source file appear in the list.

New operation list button

Click this button to remove all the operations from the batch list.

Save batch file button

Click to save the current operation list into a batch file. You can use the same name or enter a new one.

Regenerate all operations

When checked, automatically regenerates batched operations before processing.

Save Mastercam file after regeneration

When checked, automatically saves each Mastercam source file after its operations have been regenerated.

Run Post

Select to create NC programs and/or NCI files for the selected operations when you run the batch job.

Post settings

Click to open the Post Setting dialog box, where you can tell Mastercam what types of files to create and edit, and change communication settings.
