Edit Toolpath dialog box

Use the Edit Toolpath dialog box to modify an existing toolpath.

NOTE
  • Cut and Pass fields in the Position section are unavailable for Lathe toolpaths. For other toolpath types, Cut is available only if the toolpath uses depth cuts, and Pass is available only if multiple passes are used.
  • If you are editing a Multiaxis toolpath, you can edit or delete only selected points.
  • To undo all of the changes you made to the toolpath and exit the function, click Cancel. This restores the toolpath to its original state before editing. Any points, passes and cuts that were deleted in the current editing session are also restored.
Parameters

Move point

Moves the point entered in the Position point field to a location you choose by clicking in the graphics window.

Edit point

Opens the Edit Point Parameters dialog box where you can modify the toolpath motion at the selected position point. You can also add other actions to the operation at that point, such as turning coolant on or off, and changing cutter compensation.

Add point

Inserts a new point in the toolpath, before, or after the selected position point. The position of the new point is determined by the Before field setting.

Before

Defines where to position new points in the toolpath when you click Add point. Select to insert new points in the toolpath before the position point. Deselect to insert new points after the position point.

Delete point

Deletes the selected position point from the toolpath.

Delete section

Opens the Section dialog box where you set the endpoint of the section to delete from the toolpath. The section begins with the selected position point.

Delete pass

Deletes the selected position pass from the toolpath.

Delete cut

Deletes the selected position cut from the toolpath.

Point (Position)

Defines the toolpath point position (XYZ coordinate) for editing and highlights it in the graphics window. Use one of the following methods to enter the point:

  • Click a toolpath position in the graphics window.
  • Type a number in the field.
  • Use the up and down arrows to scroll through the points in the toolpath.

Pass (Position)

Available only for Mill and Router toolpaths that contain multi-passes. Defines the toolpath pass (XY motion) for editing and highlights it in the graphics window. Use one of the following methods to enter the pass:

  • Click a pass in the graphics window.
  • Type a number in the field.
  • Use the up and down arrows to scroll through the passes in the toolpath.

Cut (Position)

Available only for Mill and Router toolpaths that contain depth cuts. Defines the toolpath cut (Z motion) for editing and highlights it in the graphics window. Use one of the following methods to enter the cut:

  • Click a cut in the graphics window.
  • Type a number in the field.
  • Use the up and down arrows to scroll through the cuts in the toolpath.

G-code

Shows the G-code associated with the move.

Feed rate

Displays the current feed rate value. To change, use Edit point to access the Feed Rate options in the Edit Point Parameters dialog box.

Canned text

Displays the current canned text setting. To change, use Edit point to access the Canned Text option in the Edit Point Parameters dialog box.

XYZ or DZ

Displays the endpoint coordinates for both circle and linear moves. For G2 and G3 moves, center point coordinates for the circle interpolation also display.

Show pass

Select to show only the specified pass of the toolpath. If deselected, the entire toolpath displays and all passes are highlighted.

Show cut

Select to show only the specified cut of the toolpath. If deselected, the entire toolpath displays and all cuts are highlighted.

Show all

Shows all cuts and passes in the toolpath.
