Probing dialog box

The Probing dialog box is part of the Renishaw Productivity+™ Add-On. Use it to create your Renisaw probe operations. Your selections determine what is probed, how the tool is setup, how to process logic statements, and what happens with the results.

Use the toolbars along the top of the dialog box to make your selections. Enter values in the fields on the right side of the dialog box as needed. Click OK to process your inputs, resulting in an operation in the Toolpaths Manager. Click Import/Attach to import and export probing operations and cycles.

Associativity

Probe toolpaths are associative to part geometry. If a part geometry is moved or changed, probe toolpaths will update accordingly. Editing part geometry may result in dirty toolpaths. Regenerate the dirty operations to update the toolpaths.

Parameters

Cut

Select to remove an item from the probe operation tree.

Copy

Select to copy an item from the probe operation tree.

Paste

Select to paste an item into the probe operation tree.

UnDo

Select to undo the last cut, copy, or paste operation performed.

ReDo

Select to redo the last cut, copy, or paste operation removed by Undo.

Configure

Select to launch the Probe Configuration dialog box. Set the .RenMF, NC, and probe database files to use for your probe operation, as well as a custom macro path.

Import

Displays the Import Probe Operations From File dialog box. Use this dialog box to choose a file from which to import a probing operation.

Import/Attach

Displays a version of the Probing Dialog box for the importing and exporting of probing cycles.

Attach to operation

Click this button to attach the selected probing cycle to the current operation. Choose the cycle to attach from the list in the left hand pane.

Measure point

Select to return to the graphics window for wireframe point selection.

Measure line

Select to return to the graphics window for wireframe line selection.

Measure circle

Select to return to the graphics window for wireframe arc selection. Partial arcs are allowed.

Measure plane

Select to generate a measurement cycle for a plane. Set the measurement type and then enter the required parameters in the Feature Definition area.

Measure 2D corner

Select to generate a measurement cycle for a 2D corner. Use 2D Corner when the corner does not form a right angle. Set the measurement type and then enter the required parameters in the Feature Definition area.

Measure 3D corner

Select to generate a measurement cycle for a 3D corner. Use when the three surfaces / faces involved form a right angle corner. Set the measurement type and then enter the required parameters in the Feature Definition area.

Measure web pocket

Select to generate a measurement cycle for a web pocket. Set the measurement to inside or outside, and then enter the required parameters in the Feature Definition area. Use this cycle for measuring either a pocket or a standing web.

Constructed point

Select to create a measurement cycle based on input data rather than a selected point. Enter the required parameters in the Feature Definition area.

Constructed line

Select to create a measurement cycle based on input data rather than a selected line. Enter the required parameters in the Feature Definition area.

Constructed circle

Select to create a measurement cycle based on input data rather than a selected arc. Enter the required parameters in the Feature Definition area.

Constructed plane

Select to create a measurement cycle based on input data rather than a selected plane. Enter the required parameters in the Feature Definition area.

Inspection cycle

Select to create an inspection cycle in the probe operation tree. At least one child operation must be added that is either for a measured or constructed entity.

Machine update

Select to create an operation to update your machine. Select the update type and then enter the required parameters. This cycle is typically added to a logic statement such as an If... Then.

G-Code block

Select to add a block of G-code to your probe operation. Manually enter the code in the G-code Editor.

Probe calibration

Select to create a calibration cycle for your probe. Child operations are created and populated as you enter parameters.

Tool setting

Select an option from the drop-down list to enter tool setting values. The values are then passed back to your machine for the probe operation.

Reporting

Select to create a reporting cycle based on selected statements in the probe operation tree.

Custom macro

Select to choose a custom macro to run from the configured location. If no macros are present, the list will be unpopulated.

If... Then

Select to generate a logic branch in the probe operation tree. Set the conditions for the If portion using the Condition drop-down options. Add statements to the Then portion. Such statements would typically be a machine update, g-code block, or goto statement.

Else If

Select to generate a logic branch in the probe operation tree. Set the conditions for the If portion using the Condition drop-down options. Add statements to the Then portion. Set another set of conditions for the ElseIf portion.

Else

Select to generate a logic branch in the probe operation tree. Set the conditions for the If portion using the Condition drop-down options. Add statements to the Then portion. Add a statement for the Else branch, resulting in a standard if / then / else statement.

GoTo

Select to add a goto statement to the probe operation tree. An accompanying Label is created at the same time as the GoTo statement.

Label

Select to add a label statement to the probe operation tree. In many cases you should rename the label to make sense for your use of it.
