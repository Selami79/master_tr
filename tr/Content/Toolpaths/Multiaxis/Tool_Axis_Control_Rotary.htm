Tool Axis Control page: Rotary

Use the Tool Axis Control page to establish the tool axis control parameters for your Multiaxis Rotary toolpath. Tool axis control settings determine the tool's orientation in relation to the geometry being cut.

Output format

Output format is locked at 4-axis for a Rotary toolpath. Click Select to return to the graphics window to select a point on the rotary axis.

Backplot rotary axis / Rotary axis

Select the X, Y, or Z-axis to be used in backplot to represent the rotary axis. Match this setting to the rotary axis capabilities of your machine for 4-axis output.

Use center point

Select to generate tool vectors that terminate at the selected point. Deselect to generate tool vectors that are normal to the surface.

Axis dampening length

Enter a value that determines the position of the tool axis based on a specific length away from the surface of the part. A longer axis dampening length creates less angular change between the vectors. A shorter length provides more tool positions and a toolpath that adheres closely to the surface.

Lead/lag angle

Enter the angle to tilt the tool.

  • Lead: Tilts the tool forward in the direction of the toolpath.
  • Lag: Tilts the tool backward from the direction of the toolpath.

Maximum step

Enter a value to control the distance between passes. The distance is measured parallel to the axis of rotation.

Maximum angle increment

Enter the angle the part indexes between passes, in degrees. The space between cuts will vary based on part geometry.

Start angle

Enter an angular value to begin the toolpath. The angle is applied relative to the tool plane used to cut the geometry.

Sweep angle

Enter an angular value to determine the range of the axial cuts. The angle is measured from the start angle value.
