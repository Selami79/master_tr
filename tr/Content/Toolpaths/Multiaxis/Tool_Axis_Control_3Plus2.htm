Tool Axis Control page: 3+2 Automatic Roughing

Use the Tool Axis Control page to control how Mastercam selects and creates planes.

Begin by selecting the plane creation strategy. You can choose to have Mastercam create all of the planes automatically, or you can seed the process with a list of planes that you suggest. Choose a strategy from the Mode list:

  • Automatic. Mastercam analyzes your model and chooses or creates the best cutting planes.
  • Semi-Automatic. You select or specify a list of source planes. Mastercam will use only those planes or create new planes by rotating a source plane within the parameters that you specify.
  • Manual. You select or specify a list of source planes. Mastercam will use only those planes to create the cutting passes.

Mastercam follows this process to calculate the planes:

  • First, Mastercam analyzes your part model. It uses the Max. stock to leave setting on this page to calculate the material to be removed.
    NOTE

    The Max. stock to leave setting on this page must be greater than or equal to the stock-to-leave values on the Model Geometry page.

    If it is not, you will get an error when Mastercam generates the toolpath.

  • If the cutting plane will be created by rotating an existing plane, Mastercam applies the Max. tilt range to determine how much the source plane can be rotated. Mastercam can apply this value in any direction, positive or negative.
  • Finally, Mastercam ensures that the orientation of the calculated plane snaps to the specified Search angle increment.
Creating a list of source planes

When you select the Manual or Semi-Automatic mode, Mastercam displays a grid to store the list of planes. Each plane is represented by the XYZ coordinates of its normalized unit vector, so (0,0,1) is the Top plane. To add a new plane to the grid, right-click in the grid and do either of the following:

  • Choose Add and enter the coordinates of the normal vector.
  • Choose Select tool plane to add an existing plane. You can also select a face of your part or other geometry to define a plane.

Mastercam always processes planes from the grid before creating new planes. It prioritizes the planes based on their order. Use the buttons on the side of the grid to move planes up or down and change their relative priority.
