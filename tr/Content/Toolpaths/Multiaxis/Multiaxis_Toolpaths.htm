Creating a Multiaxis Toolpath

Mastercam's Multiaxis toolpaths allow you to machine for 4-axis and 5-axis. The available settings depend on your current machine setup.

Multiaxis toolpaths are split between two types: Pattern and Application. Pattern toolpaths are general toolpaths that can be used in a variety of ways to manufacture your part. The toolpaths are very flexible and provide a wide range of machining options. Application toolpaths are designed to solve particular machining problems while automating some of the processes. These toolpaths include machining options and strategies specific to the application.

NOTE

Your ability to access specific toolpaths and features depends on your license and the capabilities of the active machine definition.

How do I create a multiaxis toolpath?

  1. Choose a machine that supports multiaxis from the Machine tab.

  2. Select a toolpath type from the Multiaxis gallery, such as Swarf.

  3. Proceed through the branches of the toolpath parameter tree until all necessary toolpath parameters have been set. The tree structure is presented in logical order and will vary based on the toolpath type.

  4. Once you are satisfied with your changes, select OK in the dialog box to create the toolpath.

If you wish to make additional edits, select Parameters under the operation in the Toolpaths Manager to re-open the dialog box. You cannot change the toolpath type or toolpath family after you create the operation.

While creating a toolpath, you can go back and change the toolpath type or family, but be aware that doing so will discard all of the geometry for that operation, and you will receive a warning message. You must reselect all geometry for the new type of toolpath.

What is the Generate toolpath option on the toolpath pages and tabs?

The Generate toolpath option automatically generates the toolpath when you click OK. This option is on the pages and tabs of Mastercam's toolpath dialog boxes, and is on by default for new and edited operations. By selecting this option, you no longer need to select Regenerate dirty operations in the Toolpaths Manager after editing your toolpath.

The Generate toolpath option also works with the Apply button on tree-style milling toolpaths. The Apply button applies your parameter changes to the toolpath you are editing without closing the dialog box.

For more information on how this option functions, refer to the Accepting and Generating Toolpaths Help topic.
