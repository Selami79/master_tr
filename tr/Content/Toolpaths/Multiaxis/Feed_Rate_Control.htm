Feed Rate Control page

Use the Feed Rate Control page to set options related to feed rate, including the feed control zone, feed rate based on the curvature of the surface, feed rates to use for clearance area and rapid distance moves, and more.

Parameters

Feed rate (conventional)

Set an alternate feed rate for the conventional motion of a zigzag cut. If this option is set to zero, Mastercam substitutes it with the standard feed rate. This option is only available when the Cutting method on the Cut Pattern page is set to Zigzag.

Adaptive feed rate

Adapts the feed rate of the tool depending on the engagement volume in the material. Minimum feed rate will be applied in the case of 100% engagement. For example, if the feed rate is set to 1000 mm/min, then with a minimum feed rate of 10%, the smallest possible feed rate is 100 mm/min.

Corner feed rate reduction

Enables the tool to decelerate during the machining of sharp corners, yielding better surface quality.

Final cut feed rate percentage

Specifies the feed on the final passes. Enter the percentage of the cut feed rate.

First cut feed rate percentage

Select the checkbox, and enter the percentage of the machining feed rate to use for the first cut, including intermediate cuts.

Tool contact based feed rate optimization

Attempts to control the feed rate based on the radius of the tool in contact with the surface. For example, if you have a 0.5" ball mill and are cutting only 0.125" up the side (rather than 0.25" up the side, where the diameter is 0.5"), your diameter at that point is 0.25, much smaller than 0.50". In this case, you want a slower feed rate. Tool contact based feed rate optimization takes care of this adjustment.

Minimum feed rate percentage

Prevents the feed rate that is written to the NCI file from falling below the minimum feed rate.

Maximum feed rate percentage

Allows you to limit the already optimized feed rate with a maximum value.

Up feed rate percentage

Optimizes the tool loading and allows you to specify different feed rates for upward cutting directions. In the case of tilted surfaces, the feed rate is proportionally applied by a range of angles. The values are defined as a percentage of the main feed rate.

Down feed rate percentage

Optimizes the tool loading and allows you to specify different feed rates for downward cutting directions. In the case of tilted surfaces, the feed rate is proportionally applied by a range of angles. The values are defined as a percentage of the main feed rate.

Adjust feed rates on edges

Reduces the feed rate in small inner and outer corners. On straight sections, the feed rate is 100%. The smaller the radius, the slower the feed rate. Users can set a minimum feed rate and a maximum feed rate.

Adjust feed rates for fanning

Avoids feed rate acceleration caused by fanning, usually in corners. As the tool tilts around the corner, the tool tip sometimes remains static at a point on the lower curve. Because the feed rate is calculated at the tip (which does not move), the feed rate is now calculated on the upper curve.

Adjust feed rates for cutting depth

Reduces the feed rate according to the cutting depth. The deeper the cutting depth, the higher the cutting forces and the lower the feed rate.

Reference depth

Specifies the value at which you actually start adjusting the feed rate. You can choose Automatic or User defined. The latter lets you enter your own value.

Intermediate slices feed rate percentage

Specifies the percentage of feed rate modification for intermediate slices. The tool load on these slices is low, so the feed rate can be higher, reducing machining time.

Feed control zone

Select the checkbox to enable the options in this group.

  • Geometry: Click Select to return to the graphics window, where you can select the geometry to be used for the control zone.

  • Offset: Enter a distance to offset the selected geometry. Use an offset when you want the tool to slow down before entering the feed control zone.

  • Inside feed rate: Enter the percentage of the feed rate to use when cutting inside the feed control zone.

  • Outside feed rate: Enter the percentage of the feed rate to use when cutting outside the feed control zone.

  • Blending distance: Enter the distance over which Mastercam will modify the feed rate when transitioning to the beginning or from the end of the feed rate control zone.

Radial chip thinning feed control

Dynamically modifies the cutting feed rate, based on the toolpath curvature. Calculates the feed rate difference between tool tip and tool contact points that might occur for tools with large diameters. The option is available only for the adaptive roughing type.

Surface radius based feed optimization

Use this option to adjust the Multiaxis toolpath's feed rate based on the curvature of the surface. Advanced feed rate control parameters allow you to specify up to four feed rate adjustments for curvatures ranging from a sharp corner to a flat surface. The surface curvature is determined at the tool contact point for every position in the toolpath. A linear interpolation is applied for values that fall within ranges. This option is available only for Plane patterns selected on the Cut Pattern page.

  • Infinite radius: Use this adjustment setting when your surface is flat at the tool contact point. Typically this will be the maximum feed rate for your toolpath.

  • Large radius: Enter a value for the larger curvature-defining radius. This value is the upper interval point for your toolpath.

  • Small radius: Enter a value for the smaller curvature-defining radius. This value is the lower interval point for your toolpath.

  • Zero radius: Use this adjustment setting when your surface makes a sharp corner at the tool contact point. Typically this will be the minimum feed rate for your toolpath.

Feed rate percentage

Enter a number to use as the feed rate percentage value. The percentage is multiplied by your toolpath feed rate to establish a new feed rate. This adjusted feed rate is calculated at every tool contact point in the toolpath.

Custom feed rate for clearance blend spline

Controls the feed rate of the clearance blend spline.

Replace rapid with feed

Enables a feed rate entry field. Enter the feed rate to use for clearance area and rapid distance moves.

Area links

Enables a feed rate entry field for Area links. Area links are set up in the Linking page.

Links between slices

Enables a feed rate entry field for Links between slices. Links between slices are set up in the Linking page.

Links between regions

Enables a feed rate entry field for Links between regions. Links between regions are set up in the Linking page.

Adaptive link feed rate

Sets the feed rate for adaptive clearance area link. Available only for the Adaptive roughing pattern.
