Cut Pattern page: Swarf Milling

Use this page to establish the cut pattern parameters for your Multiaxis Swarf Milling toolpath. Cut pattern settings determine the geometry your tool follows and how it travels along that geometry.

Swarf geometries

Click Select to return to the graphics window for solid face, surface, or mesh selection.

Click the Clear all arrow button to deselect all previous selections.

Color control

Use the color swatch to show the areas on the model with which the toolpath interacts. Click the swatch to change the color.

Swarf clearance

Enter an offset for the initial pass.

Floor geometries

Select the checkbox to enable the selection of floor solid faces, surfaces, or meshes. Then click Select to return to the graphics window to select geometry. Floor geometries are compensation geometry at the bottom of the walls. This option adjusts the tool down along the contact line until the tip touches the floor, and then calculates the offset based on the selected geometry's shape and contact with the floor surface. Floor geometries is a useful way to get the toolpath to orient the tool if you get unexpected results.

Click the Clear all arrow button to deselect all previous selections.

Floor clearance

Enter an offset for the initial pass.

Drop tool to floor

Specifies how to manage the tool when the tool tip is above or below the floor.

  • Drop and retract: Drops the tool to the floor when the tip is above the floor geometry, and retracts the tool when the tip is below the floor geometry.

  • Retract only: Retracts the tool when the tip is below the floor geometry.

Tilt lines

Select the checkbox to enable the selection of tilt lines. Click Select to return to the graphics window for tilt line selection. Tilt lines are lines that you can select to set the tilt of the cutting tool/head. The tilt angle is based on the line of contact between the tool and the workpiece.

Click the Clear all arrow button to deselect all previous selections.

Guide curves

Select the Guide curves checkbox to enable the selection of the upper and lower rail select buttons. Click the Select arrow button to close the dialog box, and then make selections in the graphics window. When you have completed selecting from the graphics window, press End Selection to return to the dialog box.

Click the Clear all arrow button to deselect all previous selections.

Swap

Select to swap the upper and lower guide curves.

Orientation

Select to orient the operation to the WCS, Tool Plane, or Construction Plane. This option is helpful if you need to realign the operation's orientation from world Z to one of the listed options.

Swarf Orientation is for ambiguous situations where the toolpath does not or cannot guess the alignment you want.

This option is available only when Output format on the Tool Axis Control page is set to 5 axis.

Side

From the drop-down list, select the side of the workpiece to machine. Use the Autodetect option to automatically set which side of the workpiece to cut. To detect the side of the workpiece, Mastercam uses its built-in logic and the surface's normal direction to evaluate the most logical side to cut.

Direction

From the drop-down list, select the machining direction for the toolpath.

Guide tool at

This option allows you to guide the tool to an upper or lower curve, based on the normal of the surface and the software logic.

Start point

Choose the type of start point you want to use for this toolpath from the drop-down selections. Some of these choices allow you to click the Select arrow button to close the dialog box, and then choose selections from the graphics window. When you have completed your selections, press End Selection to return to the dialog box.

Click the Clear all arrow button to deselect all previous selections.

Cut tolerance

Enter a tolerance value for the surface quality.

Maximum distance

Enter a maximum distance between passes.

Strategy

Select a cutting strategy:

  • Sync. with tilt lines: The tool axis will be aligned to tilt lines along the upper and lower curve.
  • Sync. with (upper/lower) curves: The upper and lower curve will be divided into equidistance steps and the tool axis will be aligned to each step pair.
  • Shortest distance: Aligns the tool with two curves by using the shortest distance between them. This strategy can be used to machine flat shaped surfaces with sharp corners.
  • Automatic: The tool will be placed onto the surface to minimize the undercut to the geometry and deviation from the target geometry.
  • Sync. with main direction: The tool axis will attempt to be as close to the selected axis as possible. The tool will then mainly tilt away to the side and tilting around that axis will be minimized.
  • Normal to guide curve: All edges are automatically extracted from the selected part surfaces.
  • Sync with ISO curves: Follows the ISO curves of a NURBS surface. This option requires that the machining surface be a NURBS surface.
  • Automatic (legacy): The tool will be placed onto the surface in a way that minimizes the area between the contact line from the lower curve to the upper curve and the swarf surface.

Fanning distance

Enter a distance to avoid the tool stopping on short curves.

Type

Select a type of tangential extension.

  • Automatic: Extends the front and rear of the rail curve tangentially by the specified distances. On a closed part, the extensions revolve along the toolpath. You must have non-zero extension lengths to get good results.

  • Align to edges: Extends the start and end of the toolpath by the given distance. The tool axis is aligned to the edges of the surfaces. You must have non-zero extension lengths to get good results.

  • Start/End with angle: Aligns the tool axis to the edges and an additional angle is added to the initial tilting at the start/end.

Extension length at start

Enter the extension length at the start of the toolpath.

Extension length at end

Enter the extension length at the end of the toolpath.

Start angle

Enter the incremental angle that the tool axis will be aligned to at the start of the path, relative to the start edge.

End angle

Enter the incremental angle that the tool axis will be aligned to at the end of the path, relative to the end edge.
