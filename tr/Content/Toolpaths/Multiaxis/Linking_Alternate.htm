Linking page

Use the Linking page to define default linking moves and other overall settings. Subpages for areas, regions, and slices let you select specific strategies for each type of group.

NOTE

This Linking page is used by a number of Mastercam's multiaxis toolpaths. Other multiaxis toolpaths use different versions of this page.

Linking settings determine how the tool moves when it is not in contact with the part (air cutting). Mastercam groups cuts in three ways for the purpose of defining linking moves:

  • Areas. An area is a geographically distinct section of your part that contains a group of cuts, but does not necessarily have a distinct boundary. Areas are only used when creating roughing operations.
  • Regions. A region is a distinct feature of your part, such as a boss or pocket. Regions are defined for both roughing and finishing operations.
  • Slices. Slices are the series of individual cuts as the tool cuts closer to the part.

The picture below illustrates the differences between these types of groups:

  • The green moves are links within an area.
  • The red moves are links between regions.
  • The yellow moves are links between slices.

First entry

Select how the tool approaches the part for the first time. The selected area or distance is where the tool begins the remaining entry motion. Additionally, select whether to use lead-in values set on the Default Lead-in/out page. Direct ignores the clearance area and distances while moving directly to the part.

If the Don't use ramp/Use ramp option is available, you can choose to create the linking move on a ramp into the cut.

Last exit

Select how the tool leaves the part upon completion of the toolpath. The tool motion stops at the selected area or distance. Retract to clearance area through tube center is intended for geometry that is similar to a port. The tool retracts along the tool axis to avoid any collisions. Additionally, select whether to use lead-out values set on the Default Lead-in/out page. Direct ignores the clearance area and distances while ending on the part.

If the Don't use ramp/Use ramp option is available, you can choose to create the linking move on a ramp into the cut.

Start from home position

Select to begin the first entry from the home position.

Return to home position

Select to move to the home position after the last exit.

Home position

Select the home position.

Type (Clearance plane)

Select a clearance area type from the drop-down list. The options are Automatic, Cylinder, Plane, or Sphere.

Direction (Clearance plane)

Sets the clearance plane along an axis.

Height

Sets the height at which the tool moves to and from the part.

Rapid distance

Enter an incremental distance above the part where the tool is moving at a rapid feed rate. This distance is also where the tool axis will be aligned properly for contact with the part. Above this distance the tool axis will be in transition for its orientation in the clearance area.

Feed distance

Sets the height above the part to switch from rapid to feed movements.

Air move safety distance

Enter the minimum distance above the part for the tool to travel in the clearance area.

Smooth corners

Select the check box and enter a value for creating fillets at sharp corners of the toolpath.

Type (Links)

Select the type of links that will be used when large gaps are encountered.

Apply on gaps larger than (in percentage of tool diameter)

Enter a threshold value as a percentage of the tool diameter. Once the gap length becomes greater than the threshold, the link type specified for large gaps will be used.

Radius (Lead in/out)

The desired radius for the lead in/out.

Min radius (Lead in/out)

The minimum acceptable radius for the lead in/out.

Angle step for rapid moves

Enter an angle value to define the maximum rapid move step to take around a cylinder when the tool is in the clearance area.

Select the default strategy to be used for Within groups and Between groups. Then, if the Don't use ramp/Use ramp option is available, you can choose to create the linking move on a ramp into the cut.

Small gaps

Select the type of motion the tool should make when a small gap is encountered. Small gaps are less than the value specified for Small gap size. Additionally, select whether to use lead values set on the Default Lead-in/out page.

Large gaps

Select the type of motion the tool should make when a large gap is encountered. Large gaps are greater than the value specified for Small gap size. Additionally, select whether to use lead values set on the Default Lead-in/out page.

Small gap size

Enter a value for a small gap size.

Type (Clearance area)

Select a custom surface as a clearance area to optimize linking motion. Mastercam aligns all retracts and pending linking motions to the defined surface. Use this only for the roughing machining pattern with 3- and 5-axis output format types.

In percentage of tool diameter

Select and enter a percentage to use for calculating small gaps. Any gap that is greater than the calculated value is considered a large gap.

As value

Enter a value to use for toolpath calculation. Moves or gaps greater than this value are considered as large.

Type

Select a clearance area type from the drop-down list. The options are automatic, cylinder, plane, or sphere. Cylinder is currently the only option for a Port Expert toolpath.

Tool orientation

Use this option to determine the tool orientation with respect to the clearance area.

  • Automatic: Chooses the correct tool orientation based on which side the toolpath is located.

  • Along normal: Sets the tool axis orientation outward relative to the clearance area.

  • Opposed to normal: Sets the tool axis orientation inward relative to the clearance area.

Direction

Sets the clearance plane along an axis or aligns the clearance cylinder to an axis.

Through point / Around point

Enter the coordinates for the cylinder center line to pass through, or the coordinates for the sphere to be centered around. The Select button returns you to the graphics window to select a point to fill in the values.

Radius / Height

Enter a radius value when using Cylinder or Sphere for clearance type. Enter a height value when using Plane. The Select button returns you to the graphics window for height selection. The coordinate of the selected point, relative to the current construction plane and X, Y, or Z radio button selection, is used as the height value.

Incremental height

Enter an incremental height for clearances. The Select button returns you to the graphics window for height selection. The coordinate of the selected point, relative to the current construction plane and X, Y, or Z radio button selection, is used as the height value.

Interpolation tilt angles

Sets the interpolation tilt angle for the approach/exit of the tool motion. Enter values for Angle step for rapid moves and Angle step for feed moves to set the maximum segment length of the cylinder or sphere which is defined by an angle.

Keep initial orientation until distance

Available only when Plane is selected for the Clearance area Type. Select and enter a distance to retract the tool along it's current tool axis orientation. The tool axis will transition to the normal of the selected plane once it reaches the entered distance from the part.

Angle step for rapid moves

Enter an angle value to define the maximum rapid move step to take around a cylinder when the tool is in the clearance area.

Angle step for feed moves

Enter an angle value to define the maximum feed move step to take around a cylinder when the tool is in the clearance area.

Rapid distance

Enter an incremental distance above the part where the tool is moving at a rapid feed rate. This distance is also where the tool axis will be aligned properly for contact with the part. Above this distance the tool axis will be in transition for its orientation in the clearance area.

Entry feed distance

Sets the height above the part at which to switch from rapid to feed movements during the toolpath approach.

Exit feed distance

Sets the height above the part at which to switch from feed to rapid movements during the toolpath retract.

Air move safety distance

Enter the minimum distance above the part for the tool to travel in the clearance area.

Feed distance

Sets the height above the part at which to switch from rapid to feed movements.

Arc fit

Creates tangential arcs for approaching and retracting link segments. You can choose to retract to Clearance area, Rapid distance, or Feed distance.

Arc radius

Set the radius of the tangential arcs.
