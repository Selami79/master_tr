Misc page

Use the Misc page to control miscellaneous settings for your Multiaxis toolpath. This includes tip compensation, blade area roughing, and mirroring your toolpath.

NOTE

Not all options are available for all toolpaths.

Tip compensation

Select whether the compensation point will be the center of the tool or the tip. Compensating to the center of the tool is frequently used with ball cutters where compensation is applied at the control.

Blade Area Roughing

Blade area roughing

Select to enable the Blade area roughing parameters.

Calculation applied

Select one of the following options for when the calculation will be applied:

  • Before tilting: Morphs the toolpath between the impeller blades before performing collision checks on the blade swarfing motion.
  • After collision control: Morphs the toolpath between the impeller blades after performing collision checks on the blade swarfing motion.

Rotary axis around

Use the drop-down menu to select the axis to rotate about.

Rotary axis base point

Returns you to the graphics window to select the base point for the rotary axis.

Maximum stepover

Enter a value for the maximum distance between cutting passes.

Number of cuts per section

Enter the number of cuts to make between a pair of blades.

Cutting method

Selecting a cutting method from the drop-down list:

  • One way (along rotary axis): Starts at upper edge of impeller floor, ends at lower edge, retracts and moves to upper edge for next cut.
  • One way (along revolution rotary axis): Starts at lower edge of impeller floor, ends at upper edge, retracts and moves to lower edge for next cut.
  • Zig zag: Starts at one edge of the impeller floor, cuts to the opposite end, steps over, cuts to the first edge, etc. Cuts are created from left to right between the impeller blades.
  • Zig zag (climb only): Starts at one edge of the impeller floor, cuts to the opposite end, steps over, cuts to the first edge, etc. Cuts are created from the center of the floor surface between the impeller blades and expand outward.

Alternate direction to reduce path length

Select to have the toolpath alternate directions as show below. Available only when Number of cuts per section and Zig zag (climb only) are both selected.

Area

Select an option from the drop-down list:

  • Complete: Entire area between two blades.
  • Left side: Area between the left blade and the splitter blade.
  • Right side: Area between the right blade and the splitter blade.

Trim cuts

Select to enable trim cuts. Choose one of the following options:

  • By percentage of cut length: Select and enter a percentage by which to shorten the cuts. The actual distance is calculated as the percentage of the current cut's length.
  • When curvature exceeds tool diameter: Select to trim the cut in areas such as at the upper radius of the blade.

Extension

Enter a distance to extend the toolpath at the start or at the end of a cut.

Depth cuts

Select to enable depth cuts. Enter the following:

  • Number: Enter the number of depth cuts to create. The number of cuts should be sufficient to remove all material between the impeller floor and the shroud surrounding the impeller.
  • Spacing: Enter the distance along the tool axis to be used between cuts.
  • Start height: Enter a height above the impeller floor along the tool axis to begin the calculation of depth cuts.

Smoothing above splitter

Select to create a morphed toolpath in the area above the splitter blade. This finishing strategy provides a smoother path along the impeller floor when a splitter blade interrupts the normal flow of the toolpath.

Miscellaneous

Maximum angle step for rotations axis

Sets the maximum tool angle change for axis rotations.

Set Y axis machine limits

Enter the lower and upper Y axis limits for your machine. This option is left unselected unless you have a need to limit the Y axis within a specific range.

  • Lower: The lower Y-axis machine limit.
  • Upper: The upper Y-axis machine limit.

Smooth surface normals

Enter a value for the smoothing threshold in the activated field.

Calculation based on tool center

Select to use the tool center for toolpath calculation. The tool pivots around the tool center rather than the contact point with the surface. The result is a toolpath with different tool axis vectors and position coordinates.

Mirroring

Plane normal

Replicates the toolpath motion about an axis or a selected Line. If you choose Line, use the Select button to return to the graphics window to select one or more lines as the plane normal.

Plane base point

Returns you to the graphics window to select a point to define the mirror plane's base point.

Toolpath copy

Keeps both the original and mirrored toolpath motion. When deselected, only the mirrored toolpath motion remains.

NOTE

Selecting this option does not create a new toolpath operation in the Toolpaths Manager. All motion is tied to the original toolpath.
