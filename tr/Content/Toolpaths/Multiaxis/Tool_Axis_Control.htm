Tool Axis Control page

Use the Tool Axis Control page to establish the tool axis control parameters for your Multiaxis toolpath. Tool axis control settings determine the tool's orientation in relation to the geometry being cut.

Parameters
NOTE

The available options depend on what you choose for Output format and for Tool axis control.

Output format

Select 3-axis, 4-axis, or 5-axis from the drop-down list.

  • 3-axis: Limits the output to a single plane.
  • 4-axis: Allows one plane of rotation selected on the 4th Axis page.
  • 5-axis: Allows tool axis rotation in any plane.

WCS Z

Select to use the current Work Coordinate System Z for the tool axis control.

Machining direction

Select to align the tool axis control with the current machining direction (derived from the selected geometry) rather than the current tool plane.

Other direction

Select and enter the XYZ coordinates of a vector to define the tool axis control. Use Tool plane to set the XYZ coordinates to the current tool plane or use Select plane to open the Select View dialog box to choose a plane.

Maximum angle step

Enter the maximum angle the tool can move between adjacent moves.

Tool axis control

Select an option from the drop-down list to control the tilt of the tool axis. The tilt is applied relative to the surface normal.

Side tilt definition

Select the side tilt definition from the drop-down list. Use side tilting to keep the side of the tool in contact with the material being cut.

Set side tilt by

Defines the side tilt, based on one of two parameters.

  • Angle: You choose the side tilt angle, and the algorithm defines the tool contact point.

  • Contact point: You choose the contact point on the tool, and the algorithm defines the angle.

Lead angle to cutting direction

Enter the lead/lag angle to apply to the tool axis in the direction of the cut, relative to the surface normal.

Tilt angle at side of cutting direction

Enter the side tilt angle to apply to the tool axis in the direction of the cut, relative to the surface normal. A positive angle tilts the tool to the left; a negative angle tilts the tool to the right.

Contact point definition

You can define the contact point in two ways:

  • By height: The contact point is defined by the distance from the lowest point of the selected profile section to the contact point in the tool axis direction. The distance is normalized with the length of the selected profile section such that the user can enter values between 0 and 1. A value of 0 means that the contact point is on the lowest point of the selected profile section.

  • By contact height %: The contact point is defined by the distance from the lowest point of the selected profile section to the contact point along the tool profile. The distance is normalized with the length of the selected profile section such that the user can enter values between 0 and 1. A value of 0 means that the contact point is on the lowest point of the selected profile section.

Considered profile section

When defining the contact point using By contact height % and height, you can specify the profile section. There are different options depending on the number of distinct sections of the tool:

  • Full profile: The entire profile from the tip to the end of the flute.

  • Curved profile: The entire profile with a convex curvature (old default).

  • Flatness diameter: The flat side of the tool (only available for Contact height %).

  • Tip radius: The tool tip.

  • Convex Tip: The entire convex area, including the tool tip.

  • Cone: The cone face.

  • Cylinder: The cylindrical face.

  • Barrel section: The barrel face.

  • Full profile: The entire profile.

Use negative side tilt

The tool can be aligned on the defined contact point using a positive and a negative side tilt angle with the same absolute value. By default, the positive angle is used.

Height in percent of flute

The contact point can either be defined by a single value that is used along the whole contour, or you can set an additional value. In the latter case, the contact point moves gradually along the contour from the first to the second value.

Contact height %

Specifies the contact line up the tool. The first value defines the tool tilt. The second value—available if you select the Gradual contact height % change checkbox—allows you to define a tool tilt range over the course of the cut.

Smoothing

Allows the toolpath to be smoothed in its cut direction, as well as the side tilt angle.

Tilting lines maximum snap distance

Enter the maximum snap distance.

Tilt angle

Enter the angle to tilt the tool axis. From the drop-down, select the axis or line to be tilted relative to. The tool will not tilt in areas where the tilt axis is parallel to the surface normal.

Run tool

From the drop-down list, select the contact point for the tool on the drive surfaces. This option is unavailable for Parallel and Morph strategies when using the Exact method of stepover calculation.

Front shift

Enter the distance to shift the tool contact point toward the front of the tool, relative to the cutting direction. Use this to ensure the front part of the tool makes contact with the part.

Side shift

Enter the distance to shift the tool contact point toward the side of the tool, relative to the cutting direction. Use this to ensure the side of the tool makes contact with the part and to avoid cutting with the tip.

Gradual contact height % change

The number you enter here specifies the gradual front shift, which keeps the tool in better contact with the part without placing excess wear on the tool's leading and trailing edges. This value is measured in either inches or millimeters and specifies the amount to shift gradually off the center of the tool as it moves across the part.

Rotary angle

Enter the angle to tilt the tool axis. Select the axis or line the tilt should be relative to. The tool will not tilt in areas where the tilt axis is parallel to the surface normal.

Pole limit

Select to prevent the tool axis from tilting beyond the selected tilt axis. Such tilting can occur when using a slot mill or lollipop cutter on very rounded surfaces.

Rotation angle

Enter the angle to rotate the tool axis. The tool axis is parallel to the surface normal and then the rotation is applied. Select the axis or line from the drop-down to be rotated relative to.

Point tilt type

Set to calculate the tilt angle from the point towards the axis or from the axis towards the point.

Chain

Click the button to return to the graphics window for curve chaining.

Curve tilt type

Select the method for the selected tilt curve to be followed from the drop-down list.

Tilt lines

Click the button to return to the graphics window for line selection.

Use tilt through

Select the method for approximating the tool axis when tilting through lines.

  • All lines weighted by distance: Tilts the tool using a weighted average.
  • Always closest two lines: Tilts the tool from one line to the next, averaging the tilt in between.

Point tilt type

Set to calculate the tilt angle from the point towards the axis or from the axis towards the point.

Impeller rotation axis

Select the impeller rotation axis from the drop-down list. Typically this is the Z-axis. However, you may select the X or Y-axis, as well as a line. Selecting a line activates the Line page where you input the line's endpoints.

Global lead angle

Enter the initial lead angle to apply to the tool when machining an impeller. The tool axis is along the floor surface normal by default.

Global side angle

Enter the initial side tilt angle to apply to the tool when machining an impeller. The tool axis is along the floor surface normal by default.

Approximate

Select an option from the drop-down list.

  • By one vector: Replaces all vectors with a single average vector.
  • By two vectors: Replaces all vectors with two computed vectors.
  • Smooth: Averages the computed vectors across the tool motion.
  • Smooth (local): Averages the computed vectors a short distance around each toolpath point.

Limits

Select to activate the Limits page. Limits keep the tool within specified angles in a selected plane.

Maintain tilt

Maintains the specified Tilt angle, keeping the cutting side of the tool constant.

Run tool

Select the contact point for the tool on the drive surfaces from the drop-down list.

Common direction

Finds and maintains a single tool axis orientation for the toolpath.

  • On all contours: Uses all contours in the toolpath to calculate and maintain a single tool axis orientation.
  • On single contour: Calculates and maintains a tool axis orientation for each individual contour in the toolpath.
