First Entry - Lead In/Last Exit - Lead Out page

Use this page to establish the leads motion for entry / exit, gap motion, links between slices, as well as links between passes. Leads determine how the tool feeds into the material as it approaches the part.

Use default lead-in / Use default lead-out

Select to use the lead motion defined on the Lead-in/out page. Deselect to enter lead motion for the specified type selected on the Linking page.

Type

Select the style of lead motion from the drop-down list.

  • Tangential arc: Created tangential to the part and 90 degrees from the tool axis.
  • Reverse tangential arc: Created tangential to the part and 90 degrees from the tool axis, but reverses direction from the cutting motion.
  • Vertical tang. arc: Created in the same plane as the tool axis.
  • Reverse vertical tang. arc: Created in the same plane as the tool axis, but reverses direction from the cutting motion.
  • Horizontal tang. arc: Created in the same plane as the current part slice independent from the tool axis.
  • Orthogonal arc: Created perpendicular to the part contact point and 90 degrees from the tool axis.
  • Tangential line: Created in the same plane as the current part slice independent from the tool axis.
  • Reverse tang. line: Created in the same plane as the current part slice independent from the tool axis, but reverses direction from the cutting motion.
  • Orthogonal line: Created perpendicular to the part contact point and 90 degrees from the tool axis.
  • Reverse vertical profile ramp: Created as a ramp following the part profile for the specified length and height.
  • Position line: Created at a certain position and orientation.

Flip

Select to have the lead created from the opposite direction.

Tool axis orientation

Select the tool axis orientation to apply to the lead motion.

  • Fixed: The orientation does not change during the lead move.
  • Tangential: Changes the orientation throughout the lead move to avoid a sudden rotation as the tool makes contact with the part. (Recommended)
  • Tilted: Maintains the tilting strategy selected on the Tool Axis Control page.

Maximum angle change

Enter the maximum change in angle allowed when using a tangential lead. The value is the total change in tool orientation, not the change from point to point.

Fillet

Enter a value for the fillet of the orthogonal line for the lead move.

Width

Enter a value that defines the width of the bounding rectangle for the lead move.

Length

Enter a value that defines the length of the bounding rectangle for the lead move.

Arc sweep

Enter the included angle for the arc sweep of the lead move.

Arc diameter / tool diameter percentage

Enter the percentage of the tool diameter to use as the lead arc diameter. For example, if the tool diameter is 10 mm and you enter a value of 200 in the Arc diameter / tool diameter % field, the arc diameter will be 20 mm.

Height

Enter an incremental value to move the lead move from the cutting motion. Creates a ramp when using a line lead move and a helix when using an arc lead move.

Feed rate percentage

Enter a number to use as the feed rate percentage value. The percentage is multiplied by your toolpath feed rate to establish the feed rate for lead moves.
