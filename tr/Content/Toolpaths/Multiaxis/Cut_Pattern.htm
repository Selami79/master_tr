Cut Pattern

Use the Cut Pattern page to establish the cut pattern parameters for your Multiaxis toolpath. Cut pattern settings determine the geometry your tool follows and how it travels along that geometry.

Parameters
NOTE

Not all options are available for all toolpaths. For Triangular Mesh toolpaths, view the Triangular Mesh Parameters topic.

General

Pattern from / Pattern to

Returns you to the graphics window to select the first curve or surface.

Parallel to

Returns you to the graphics window to select the curve, surface, or angle your tool will follow.

Machining angle in X, Y

Enter the angle of the cuts in the XY plane, relative to the plane selected with the Select machining angles button.

Machining angle in Z

Enter the angle of the cuts in the Z plane, relative to the plane selected with the Select machining angles button.

Select machining angle

Returns you to the graphics window to select a plane. The plane establishes the axes for creating the toolpath slices.

Constant Z

Sets the Machining angle in Z equal to 0. Deactivates Machining angle in X,Y.

Parallel

Sets the Machining angle in Z to 90 and activate Machining angle in X,Y. Set the fields to values needed for the best parallel pattern on your part.

Lead

Returns you to the graphics window to select the lead curve. Passes are created perpendicular to the lead curve so care should be taken in creation and selection of the curve. Click the Clear all arrow button to deselect all previous selections.

Projection

Returns you to the graphics window to select the projection curves. Projection curves should lie on or above the surface, within the Maximum projection distance. The curves are projected onto the surfaces along the surface normals, possibly changing the shape and length of the curves.

Drive surface

Returns you to the graphics window for drive surface selection. Click the Clear all arrow button to deselect all previous selections.

Drive surfaces offset

Enter a value to offset the drive surfaces. The value is applied in all three directions. A positive value acts as a stock to leave amount and prevents the tool from coming closer than that distance to the part.

Maximum projection distance

Enter the maximum distance a projection curve can be from the part. Curves that lie farther from the surface than this distance will not be projected.

Cutting side

Changes the cutting side of the toolpath to be the Center, Left, or Right.

Projection direction

Select the projection method from the drop-down list. The options are along the X, Y, or Z-axis, along a line, or using a surface normal.

Type

Select the type of projection to use from the drop-down list. Not all of these options are available for all pattern types.

  • Offset: Creates offset patterns of the selected curve based on the stepover value.
  • Parallel: Creates parallel cuts on multiple depths based on the machining angle.
  • Adaptive: Ensures that the cutting conditions remain constant. This type avoids full-width cuts by constantly measuring the engagement of the tool with material and gradually removes material off the remaining stock.
  • Radial: Creates a radial pattern based on the center point, inner and outer radii, and included angle for the pattern.
  • User-defined: Projects the user selected curves to create the pattern.
  • Spiral: Creates a spiral pattern based on the start and end angles, using the stepover value for the pitch.
  • U-Shape: Creates a U-Shape cut projected onto machining surface, based on the start point and length of the edges of the rectangle.
  • Morph: Creates morph patterns between a set number of curves while the toolpath is contained.
  • Line: Projects a line for the machining pattern based on the machining angle for X, Y and Z.

Cutting side offset

Enter a value to offset the Cutting side.

Area

Type

Select the method for cutting the toolpath area from the drop-down list. The type selected determines where the tool starts, ends, and area cut.

Number of cuts

Enter the number of cuts to create in the operation. Available only when Determined by number of cuts is selected.

Round corners

Activates the Round Corners page of the tree structure. Use to add fillets to the operation for internal corners of your part.

Extend / trim

Activates the Extend / Trim page of the tree structure. Use to add or remove cutting motion when the tool gets to the end of a surface.

Containment

Activates Containment and the Containment page of the tree structure. Click Select to select a chain to control the cutting area of the part.

Angle range

Activates the Angle Range page of the tree structure. Use to set the cutting motion by entering two angles.

Sorting

Flip stepover

Reverses the cutting direction for the operation.

Cutting method

Select the cutting method from the drop-down list. Not all of these options are available for all toolpath types.

  • One way: The tool cuts each pass in the same direction.
  • Zigzag: The tool cuts each pass in the opposite direction from the previous one.
  • Spiral: The tool cuts in a spiral motion on the part.
  • Clockwise: The tool cuts in a clockwise direction.
  • Counterclockwise : The tool cuts in a counterclockwise direction.

Cut order

Select the cut order from the drop-down list. Not all of these options are available for all toolpath types.

  • Standard: The default option which usually cuts from one side of the part to the other.
  • From center away: Cuts from the approximate center of the part to the outer edges.
  • From outside to center: Cuts from the outer edges of the part to the approximate center.
  • From top to bottom: Cuts from the top of the part to the bottom of the part.
  • From bottom to top: Cuts from the bottom of the part to the top of the part.

Direction for one way machining

Select a direction from the drop-down list. There are different options available depending on what Cutting method is selected.

Enforce cutting direction (assume closed contours)

Available only when One way is selected for the Cutting method. Cuts open contours as if they were closed, enabling the cutting direction to be maintained.

Start point

Activates the Start Point page of the tree structure. Use to set the tool's initial contact point with the part.

Machine by

Select an option from the drop-down list. Note that not all options are available for all toolpaths.

  • Depth: Cuts the part by its depth.
  • Levels: Cuts the part by alternating between pockets for each level.
  • Lanes: Cuts the part with the selected pattern.
  • Regions: Splits the toolpath into regions when the tool encounters a gap. Use when the part geometry creates excessive air motion around gaps.
Surface quality

Cut tolerance

Determines the accuracy of the multiaxis toolpath. A smaller cut tolerance value creates a more accurate toolpath, but may take longer to generate the toolpath and create a longer NC program.

Maximum distance

Select and enter a value for the maximum distance between toolpath positions in the cutting direction. Use in operations where you need to have more points defined, such as on flat surfaces.

Damp

Allows a more flexible application of the axial shift.

Stepover

Maximum stepover

Enter a value for the maximum distance between cutting passes.

Scallop height

Specifies the height of the remaining material between passes when using a rounded tool. Mastercam calculates the stepover based on the value that you enter here and the selected tool. The Maximum stepover and Scallop height fields are linked so that when you type a value in one field, Mastercam automatically updates the other.
