Trochoidal Motion page

Mastercam's High Speed toolpaths have been specially designed for high speed machining and hard milling applications, specifically Area Roughing and Horizontal Area toolpaths. Because of this, it is important to detect and avoid circumstances where the tool is fully buried or is engaging too much material for the high speed strategies to be safely employed.

For example, in the part shown here, you can see the loops that Mastercam inserts as the tool approaches the area between the two bosses. As the tool is forced to engage more material because of the part geometry, Mastercam calculates smaller loops.

Minimize burial (on) Minimize burial (off)

Inserting the loops typically results in increased machining time. If you are confident that a fully buried tool does not pose a problem for your application, you can de-select Minimize burial. For most applications, however, this should be turned on by default, since Mastercam only inserts the loops if it detects problem areas in the toolpath.

Area Roughing example

Consider the part shown below. It is being roughed with an Area Roughing toolpath with the machining strategy set to From Outside. Tool burial typically isn't a problem for many area roughing applications because the tool approaches the bosses in multiple passes starting from the outside, but in this part, the tool will be buried when it passes between two bosses (red area). Even when it makes a sharp change of direction (orange areas), it will be more fully engaged than expected.

When Minimize tool burial is selected, Mastercam looks for locations where the tool burial will be too great, and inserts trochoidal loops in the toolpath as the tool approaches those areas, so the tool is never fully engaged. These help you maintain the high speed cutting motion without excess tool wear or the risk of broken tools.

Parameters

Off

Disables the trochoidal loops feature. No loops will be inserted in your toolpath. The other controls on this page will be disabled.

Minimize burial

Activates the trochoidal loops feature. Mastercam will insert loops in your toolpath in sections where the amount of tool burial is slightly greater than what would normally be created by the maximum stepover distance.

Stepover adjustment

Controls the threshold for creating loops. Typically, the threshold for creating loops is whenever the tool burial slightly exceeds what would be created by the maximum stepover. You can set this threshold higher by entering a value in this field. Mastercam applies this percentage to the Maximum stepover on the Cut Parameters page to calculate a higher tool burial threshold. The higher the value that you enter here, the fewer loops will be created. The default is 0, meaning that the amount of tool engagement that is allowed will be no greater than what would be allowed by the Maximum stepover.

Initial loop radius

This determines how large the loops are. In the left field, specify the radius as a percentage of the tool diameter. The field to the right shows the actual value.

Retry loop radius

If Mastercam does not have enough room to create loops at the initial size, it will create successively smaller loops. The Retry loop radius specifies how large each new loop will be as a percentage of the previous loop. For example, if your Initial loop radius is 16, and the Retry loop radius is 50, Mastercam will create an 8 loop if the 16 loop is too large. If the 8 loop is too large, it will then create a 4 loop.

Minimum loop radius

This is the smallest loop that Mastercam will create. In the left field, specify the radius as a percentage of the tool diameter. The field to the right shows the actual value.

Feed rate adjustment

Specify a Feed rate adjustment to slow down the tool, if the Minimum loop radius is too large to fit in a given region, Mastercam will continue the toolpath with no loops. Use the slower feed rate to compensate for the greater tool burial in those portions of the toolpath. The Feed rate adjustment is specified as a percentage of the Feed rate on the Tool page.

Keep loop inside machining region

Unavailable for 2D high speed toolpaths or for Area Roughing toolpaths with the From Outside strategy enabled. Select this option to ensure that loops created in the outer regions of your surfaces stay completely inside the machining region. If this option is not selected, loops may overlap the boundaries of your surface.
