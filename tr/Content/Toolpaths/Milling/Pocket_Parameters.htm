Pocket Parameters tab

Use the Pocket Parameters tab to set up roughing and finishing passes for a rough pocket toolpath. Rough pocket is used for machining cavities or bosses on surfaces and solid mode.

Parameters

Rough

Select this option to create a roughing operation and to enable the roughing parameters.

Cutting method

Click on a cutting method to select it. Not all parameters listed in the Roughing page are available for every cutting method.

Stepover percentage

Sets the distance between cutting passes in the X and Y axes as a percentage of the tool diameter. Changing this value automatically adjusts the stepover distance.

Diameter / Flat

Sets the distance between roughing passes in the X and Y axes. Choose to have the stepover measured as a percentage of the tool diameter or of the flat of the tool. Of flat is useful with bullnose tools.

Stepover distance

Measures the distance between cutting passes in the X and Y axes. This option calculates as a distance in the X and Y axes and equals the stepover percentage multiplied by the tool diameter.

Roughing angle

Sets the angle that the tool moves at during the roughing passes.

Minimize tool burial

Clears out areas around pocket islands to avoid tool damage due to burying the tool in too much stock. Applies to Zigzag, Spiral, Spiral Clean Corners and Constant Overlap pocket toolpaths.

Spiral inside to outside

Applies to all spiral pocket toolpaths. The toolpath spirals from the center (inside) to the wall (outer) of the pocket. If you do not select this option, the toolpath spirals from the outer wall of the pocket to the center.

Use quick zigzag

Enabled when the Cutting method is set to Zigzag. Roughs the pocket in a linear motion according to the roughing angle.

High speed

Enabled when the Cutting method is set to High Speed. Allows you to enter the corner smoothing radius as well as the loop radius and loop spacing for trochoidal (high speed) cuts.

Finish

Use the options listed to create finish passes. The Stock to leave on walls value you define in the Cut Parameters page specifies the amount of material for the finish pass to remove.

Passes

Enter the number of finish passes to create on the selected geometry. Entering 0 creates no finish passes.

Spacing

Enter the distance between finish passes.

Spring passes

Enter the number of spring passes for the toolpath. Spring passes create additional finish passes along the same path as the last finish pass. This can be used for parts with thin material that may have flexed away from the tool during previous passes.

Finish containment boundary

Creates a finish pass around the containment boundary.

Compensation type

Choose how you want to handle cutter compensation:

  • Computer: Computes the compensated toolpath and does not output control codes for compensation.
  • Wear: Computes the compensated toolpath and outputs control codes for compensation. Compensation direction in the computer and control are the same.
  • Reverse wear: Computes the compensated toolpath and outputs control codes for compensation. Compensation direction in the computer and control are opposite.

The active control definition determines the available options.

Override feed speed

Allows you to override the speed and feed values for finish passes, providing better control for floor finishing. Select one or both of the following options:

  • Feed rate: Select to enter a different feed rate for the finish passes than the feed rate used for roughing. You may want to increase the feed rate because the tool usually cuts less material during finish passes.
  • Spindle speed: Select to enter a different spindle speed for the finish passes than the spindle speed used for roughing. You may want to increase the spindle speed because the tool usually cuts less material during finish passes.

Lead in/out

Select the checkbox to use lead in/out moves for the finish passes. Click the button to enter the dimensions and other parameters.

Thin wall

Select the checkbox to divide each finish pass into multiple depth cuts for parts with thin walls. Then enter the dimensions and other parameters.
