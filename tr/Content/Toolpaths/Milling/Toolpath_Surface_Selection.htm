Toolpath/Surface Selection dialog box

Use the Toolpath/Surface Selection dialog box to select all the geometry needed for a surface toolpath. Each surface toolpath requires drive geometry selected from the graphics window or from a CAD file. Optionally you can select check geometry and a tool containment boundary. Geometry selection for surface toolpaths always defaults to standard selection. You will be able to select a solid model in standard selection. However, if you need to select parts of the solid model, you will need to use solid selection.

More on check and drive surfaces

Check surfaces offer little or no usefulness with rough pockets. For that reason, the Check section in the Toolpath/surface selection dialog box is disabled for these toolpaths.

For other toolpaths, the Mastercam product level with which you are working affects the available check- and drive-surface options, as follows:

  • With the Mill 3D product level, you can have any number of check surfaces and drive surfaces.
  • With the Mill product level, generally you can have any number of check surfaces, but only a single drive surface. Certain toolpaths, such as Parallel and some pockets, do allow multiple drive surfaces.

If you select check surfaces, and then select a toolpath strategy that does not support them, Mastercam automatically treats the check surfaces as additional drive surfaces. Mastercam will display a message to alert you when it does this.

In your .config file there is an option to automatically use all non-drive surfaces as check surfaces. Because the new high speed toolpaths do not support check surfaces, you should ensure that this option be set to None or Prompt.

You can find this setting in the Toolpaths page of the System Configuration dialog box.

The options available in this dialog box change depending on the type of toolpath you are creating. Additional geometry selection options include a start point, curves, a radial point, or flowline data.

You can prevent this dialog from displaying by deselecting the Show toolpath/surface selection dialog option on the Toolpaths page of the System Configuration dialog box.

To change the geometry selection after the toolpath has been created, choose the geometry icon for the toolpath in the Toolpaths Manager.

Parameters
NOTE

Not all options are available for all toolpaths.

Drive

The surfaces, solid faces, solid bodies, or CAD files that will be cut.

Check

The surfaces, solid faces, or solid bodies that you want the tool to avoid.

Containment

One or more closed chains of curves that limit tool motion.

Select

Returns to the graphics window to add or modify the selection.

Deselect all

Removes all the selected geometry element from the toolpath.

CAD file

Imports an STL file for the drive geometry. To select an STL file that you have imported as a mesh entity, use Select instead of the CAD File button. Mesh entities offer most of the benefits of a full entities, including shading, selecting as drive and check surfaces, entity analyzing, and more. However, a mesh cannot be used as a drive surface for a Flowline toolpath.

Show

Returns to the graphics window with the selected geometry highlighted.

Approximate start point

Selects the point near where the toolpath will start, and activates the Use approximate start point parameter on the toolpath parameters or cut parameters tab. This feature functions differently depending on the toolpath type:

  • Standard (Non-high speed) toolpaths: In a rectangular shaped part with a finish parallel toolpath, the approximate start point indicates the corner where the tool will start and sets the starting side and stepover direction. In a part with multiple bosses or cavities with finish contour toolpath, the approximate starting point indicates the region where the tool will start. If you do not select a starting point, the toolpath will start in a corner or region toward the lower left of the machining view; set using the Planes button on the Toolpath parameters tab.
  • High speed toolpaths: If the toolpath has several cutting regions, the approximate start point indicates which region will be machined first.

Radial point

Available only for Surface Rough and Finish Radial toolpaths. This example shows a surface radial toolpath radiating from a point on the upper-left side of the surface.

Entry point

Available only for the Surface Rough Pocket toolpath. You can optionally select the entry point for the toolpath. Selecting an entry point enables the Use entry point parameter on the Rough parameters tab.

Grid

Available only for Surface Rough Plunge toolpaths. Selects the lower left and upper right points on the drive surfaces. Mastercam automatically creates a grid of plunge points based on the surface and the selected points.

Flowline

For Surface Rough Flowline and Finish Flowline toolpaths, choose Flowline to set the cut direction, start point, offset, and whether edges are shared or free.

Blend

Available only for Surface Finish Blend toolpaths. Select two chains of curves. Mastercam projects the curves onto the drive surface and creates tool motion in a blend between the chains.

Curves

Available only for Surface Rough and Finish Project toolpaths. Selects curves to be projected onto one or more drive surfaces.

4th axis

Sets the axis about which the 4th (rotary) axis rotates. This option is available only when 4-axis is selected as the output format in the geometry selection dialog box, and when a rotary axis is defined on the current mill machine definition.
