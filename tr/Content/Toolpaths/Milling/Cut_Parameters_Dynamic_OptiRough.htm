Cut Parameters page: Dynamic OptiRough

Use the Cut Parameters page to enter values for cutting parameters and compensation options for Dynamic OptiRough toolpaths, a high speed roughing toolpath capable of machining very large depths of cut.

Cutting method

Orients the cutting direction. Select one of the following options:

  • Climb: Cuts in one direction with the tool rotating in the opposite direction of the tool motion.
  • Conventional: Cuts in one direction with the tool rotating in the same direction as the tool motion.
  • Zigzag: Cuts each pass in the opposite direction to the previous pass. A short linking motion connects the two ends.

Conventional feed rate

Sets an alternate feed rate for the conventional motion of a zigzag cut. Enter a percentage of the programmed feed rate or a value. Only available when the Cutting method is set to Zigzag.

Tip compensation

Select whether the compensation point will be the center of the tool or the tip. Compensating to the center of the tool is frequently used with ball cutters where compensation is applied at the control.

Optimize stepups

Defines the cut order Mastercam applies to different cutting passes in the toolpath.

  • By depth: Mastercam machines all cut passes Z-level by Z-level.

  • Next closest: Mastercam moves to the closest cut from where it finished the previous cut.

  • By pocket: Mastercam first machines all the stepdowns, moving from pocket to pocket. After all stepdowns on a Z-level are machined to completion, Mastercam machines the stepups by next closest, in the safest cut order.

Optimize stepdowns

Controls the cut order Mastercam applies to different cutting passes in the toolpath. When the tool finishes a machining pass, it must select a start point to resume. The start points can be set as the following. The images below show a toolpath with Classic Backplot to help show the differences between the options. The numbers indicate the order of machining to also help visualize the options.

  • Material: Set by default. Starts at the material that is closest to the entire tool. Mastercam attempts to do smaller pieces within a general area when pieces of material become isolated before moving on.

  • None: No optimization is applied to the cutting passes.

  • Air: Starts nearest to the tool. Mastercam attempts to create the shortest possible air moves between machining areas.

Stepover

Sets the distance between cutting passes in the X and Y axes. Enter a percentage of the tool diameter, an absolute distance, or an angle. The value or percentage is limited to 95% of the tool's flat diameter in the Dynamic cutting method. A larger value will automatically be reduced to the 95% limit after a warning message. Be aware when using a bullnose tool that the percentage is a calculation based on the tool's flat diameter.

NOTE

The field becomes a drop-down selection when the following conditions exist:

  • An ISCAR CHATTERFREE tool is selected
  • The Iscar HEM checkbox in the Tool page is selected

Your options then become 5%, 7%, and 10%.

Maximize engagement

When enabled, Mastercam produces more nominal chips and minimizes the number of small chips. Additionally, Mastercam moves towards the entered Stepover as quickly as possible.

Stepdown (Passes)

Determines the Z spacing between adjacent cutting passes.

Uniform stepdowns

When this option is selected Mastercam creates equal steps when the Stepdown value is not a perfect division.

Stepup

When this option is selected, Mastercam calculates and adds stepup cutting passes (+Z) to the toolpath. Enter the stepup value to use between up cutting passes as either a percentage of tool diameter or an absolute distance.

Mastercam also inserts extra cut passes when stepping up to clear all detected flat areas. Stepup cutting passes help remove material left behind between the stepdowns, for example, islands or large scallops on angled walls, using a dynamic rest motion. Deselect to disable stepup cutting for the toolpath. The toolpath then uses dynamic milling motion to remove material only on stepdowns, avoiding any islands.

NOTE

When you set the Stepup value equal to the Stepdown value, Mastercam calculates up cutting passes for only flat areas.

Mill vertical walls

Select to always calculate a rest area for stepup cuts. This results in a waterline cut motion on vertical walls. Deselect to machine only the actual rest area Mastercam calculates from the stepup cuts.

Minimum toolpath radius

Enter the minimum toolpath radius to create in the operation. Enter the value as a percentage of the tool diameter or an absolute value. Mastercam uses the specified minimum toolpath radius in combination with the Microlift distance and Back feed rate parameters to calculate 3D arc moves between cut passes. The Minimum toolpath radius value is beneficial when the following conditions are present:

  • Sharp corners in your machining chain: If you machine a rectangular-shaped pocket, each of the four corners will have a radius. The Minimum toolpath radius is the radius of those corners.
  • Arc on and off: The arc motion on and off during toolpathing tries to target 45% of the tool's diameter for its size. However, if the percentage prevents the tool from going into a tight space, Mastercam makes this value less than 45%, but never less than the Minimum toolpath radius value.

Micro lift distance

Enter the distance the tool lifts off of the part on back moves. Microlifts are slight lifts that help clear chips and minimize excessive tool heating.

Back feed rate

Controls the speed (inches per minute or millimeters per minute) of the tool's backfeed movement. This allows 3D arcs between cuts to have a different feed rate than the rest of the toolpath, which can help reduce cycle time.
