Setup page: FBM Drill/Wizard Holes

Use the Setup page to set the following basic parameters for the FBM Drill or Wizard Holes operation.

  • Define how to group the FBM-generated drill cycles in the Toolpaths Manager.
  • Optionally sort all drill points in each FBM-generated drill cycle and, when activated, choose the 2D sort method.
  • Create operations that are independent of the solid model. This selection automatically creates a point at the top of the hole, associative to the independent operation.
  • Create additional geometry (points and vectors) on a level you choose. The new geometry is non-associative (independent of any toolpath or solid model feature).
NOTE
  • If you choose to Create operation independent of solid (create points) and also select Points at top of hole, place the additional geometry on a different level than the toolpath geometry. Otherwise, you will create duplicate points.
  • If you plan to use the Drill toolpath to machine hole features in your solid model, consider using FBM Drill first to efficiently create the necessary vectors and bottom point geometry.

Automatic initial hole / feature detection

When selected, Mastercam automatically detects features when you choose the FBM operation. If you save the setting to your toolpath .mcam-defaults file and use that file in the active machine group, Mastercam automatically detects features every time you choose the FBM operation. Otherwise, this setting affects only the current Mastercam session when the current toolpath defaults file is used.

Detection analyzes the solid based on current FBM parameters and determines which features to include in the Features page (FBM Mill and FBM Drill). When you accept the FBM operation, Mastercam creates toolpaths for all features in the Features page list that can be machined.

  • Options at the top of the dialog box allow you to Save parameters to defaults file and Reload parameters from defaults file.
  • Select Detect in the FBM dialog box at any time to analyze the solid based on your current parameters, and to update the Features page.

Grouping (FBM Drill/Wizard Holes)

Controls how all FBM-created drill cycles are organized in the Toolpaths Manager. Regardless of the primary grouping method you choose, Mastercam orders operations within groups into subgroups by operation type. The first subgroup includes all spot and pre-drilling operations. The next subgroup includes all finish tool and chamfering operations.

  • None: Orders operations only by operation type (all spot and pre-drill operations followed by finish tool and chamfer operations).
  • Plane: Groups all operations based on the plane of the hole.
  • Tool: Groups operations by the tool used for the operation.

Sorting

Select to activate sorting for the FBM Drill or Wizard Holes operation. The order that will be used to sort each set of points in a flat plane displays as text and as a picture on the Sort order button. The red point indicates the starting point. Click the button to open the 2D Sort dialog box and choose a different sort order. Deselect if you do not want to sort drill points in the FBM-generated drill cycles.

Subprogram

Sets values in the subprogram lines in the NCI file to indicate a section of the operation that is repeated at a different location (for example, a drill cycle or depth cut). This information is then passed to the post processor. The control definition must be set to support subprograms for this function to work.

In some instances, you might be able to select Absolute or Incremental subprograms. An absolute subprogram contains absolute coordinate values; in other words, the same points are machined every time the subprogram executes. With an incremental subprogram, the points which are machined are relative to the tool's location when the subprogram is called. Typically an absolute subprogram has a G90 code while an incremental one has a G91.

Comment

Enter comments for the toolpath. Comments help identify the operation in the Toolpaths Manager. These can be output in the NC file when you post if you select the proper option in the Files tab of the Machine Group Properties. Comments entered for FBM Mill and FBM Drill operations are copied to their generated toolpaths. For example, the same comment you enter for an FBM Drill operation appears in all of its generated drill cycles. This can help you maintain a relationship between FBM "parent" and "child" operations.

Create operations independent of solid (create points)

Select to create drilling operations that are independent of the solid model. This selection automatically creates a point at the top of each detected hole feature. The point is associative to the independent drilling toolpaths that FBM generates to machine the feature.

NOTE

If you also select Points at top of hole under Create additional geometry, place the additional points on a different level than the toolpath geometry. Otherwise, you will create duplicate points.

Include chamfers (Wizard holes only)

Select to include chamfers in the hole features that FBM Drill detects. If unselected, FBM Drill excludes chamfers in detected holes; no operations are created to machine them.

Create additional geometry

Select one or more options to create geometry (points and vectors) for detected hole features without generating toolpaths. The geometry is saved to the level you choose in this section and is non-associative (independent of any toolpath or solid model feature).

Mastercam uses the hole feature data to create the point at the top of the hole. Drill cycle toolpath data is used to create the vectors and bottom points.

The vector length is defined by the theoretical drill path while it is engaged in the cut. The vector starts at the top of the hole and extends to where the drill tip of the finish tool would stop. The end point of the vector includes any Tip Compensation and Additional Break Through defined for the toolpath.

NOTE

If you choose to Create operation independent of solid (create points) and also select Points at top of hole, place the additional geometry on a different level than the toolpath geometry. Otherwise, you will create duplicate points.

Level

Displays the level where Mastercam creates the selected geometry types (points and vectors). To choose a level, type the number in the field, or click the button to update the field with the first unused level Mastercam finds in the part file that is greater than the specified level.

4th axis output

Select to output the appropriate tool planes for FBM-generated drill cycles that support drilling in a rotary axis environment. This checkbox is available only when a rotary axis is defined in the active machine group. Deselect to output tool planes that do not take into consideration a 4th rotary axis.

Axis of rotation

Select Machine Definition (default) to generate tool planes for the machine definition’s rotary axis configuration that matches your machine capabilities. To override the machine definition's rotary axis assignment, select another axis of rotation (X, Y, or Z).
