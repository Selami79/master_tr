Revolved Parameters tab

Use the Revolved Parameters tab to enter the toolpath parameters for a Revolved Wireframe toolpath. This toolpath revolves a cross-section or profile through either the X- or Y-axis to simulate a surface.

  • First, select the surface shape to create, either concave or convex.
  • Select the axis of rotation, either X or Y. Use the Axis depth to position the toolpath above or below the chained geometry.
  • Enter other toolpath dimensions and compensation options.
  • Finally, select the trimming options. You can trim either the Height or the Width. Click Select to select width trim lines from the graphics window. If the toolpath would extend past the trim plane, it is clipped; if the trim plane is outside the toolpath, the toolpath is extended to it.
Parameters

Clearance

Sets the height at which the tool moves to and from the part. Select the checkbox to activate the clearance height. If it is not selected, the Retract value is used as the clearance plane. Click the button to set or change the height. You can select a point on the geometry or enter a value.

Absolute

Absolute values are always measured from the origin 0,0,0. If you were to translate geometry associated with a toolpath, and the toolpath depth was set to an absolute value, the tool will try to cut to that same absolute depth value no matter where the geometry is located.

Incremental

Incremental values are relative to other parameters or chained geometry.

NOTE

This option does not result in incremental coordinates being output in the NC program; use the Control Definition Manager to determine that.

Only at the start and end of operation

Rapids to the clearance plane only at the start and end of the toolpath.

Retract

Sets the height that the tool moves up to before the next tool pass. Select the checkbox to activate the retract plane, and then click the button and select a point on the geometry or enter a value. This option is off by default. The retract height should be set above the feed plane. If you do not enter a clearance height, the tool moves to the retract height between operations. In multiaxis toolpaths, retract is an incremental distance from the current tool position.

Retract

Sets the height that the tool moves up to before the next tool pass. Select the checkbox to activate the retract plane, and then click the button and select a point on the geometry or enter a value. The retract height should be set above the feed plane. If you do not enter a Clearance height, the tool moves to the retract height between operations.

Feed plane

Sets the height that the tool rapids to before changing to the plunge rate to enter the part. The tool also moves to this height between operations if you do not enter both the Clearance height and the Retract height. Select the checkbox to activate the feed plane, and then click the button and select a point on the geometry or enter a value.

Step size

Sets the distance between cuts in the current construction plane.

Retract between passes

Rapids up to the feed plane, then makes a rapid move between passes in the toolpath. If off, a feed move is used between passes.

Compensation in computer

Generates a toolpath with compensation for the tool radius, but does not allow for tool wear adjustment at the control. Left and Right offset the tool by its radius; Off causes the tool to cut the center line of the geometry.

Stock to leave

Enter the amount of material to leave on the part model when the operation is finished.

Tip compensation

Select whether the compensation point will be the center of the tool or the tip. Compensating to the center of the tool is frequently used with ball cutters where compensation is applied at the control.

Concave

Sets the center of the tool to travel above the axis of rotation.

Convex

Sets the center of the tool to travel below the axis of rotation.

Axis: X

Sets the X axis of the current construction plane as the axis of rotation.

Axis: Y

Sets the Y axis of the current construction plane as the axis of rotation.

Axis depth (abs)

Sets the absolute coordinate of the axis of rotation on the Z axis of the current construction plane. The system sets this value automatically to the depth of the cross-section.

Nothing

Disables the toolpath trimming function. Click here to learn more about trimming revolved toolpaths.

Height

Trims the toolpath using a plane parallel to the current construction plane. The coordinate entered determines the height of the trimming plane.

Width

Trims the toolpath to a fixed width between two parallel lines. The lines must be in the same orientation as the current construction plane. To select the two parallel lines, you can either enter From and To coordinates, or choose Select lines to select the lines from the graphics window.

From, To

Represents one of the parallel lines that determines the trimming width. You can also choose Select lines to enter this value.

Select lines

Returns you to the graphics window to select two parallel lines for trimming the toolpath to a fixed width.
