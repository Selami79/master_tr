Toolpath Control page: Blend

Use the Toolpath Control page to create a containment boundary and set other containment parameters for your 3D High Speed Blend toolpath. Tool containment boundaries are used to control the tool's position around the boundary that you define. The boundary is a closed set of wireframe curves which enclose the area to be machined. Mastercam will not create tool motions that violate the boundary, regardless of the selected cut surfaces. They can be any wireframe curves, and do not have to be associated with the surfaces that are machined. For example, you can create custom guide geometry to precisely limit the tool movements. The curves do not have to lie on the part; they can be at any Z-height.

Boundary chains

Select one or more closed chains that limit tool motion.

Automatic boundary

Mastercam creates a boundary around the selected geometry and uses it as a containment boundary in addition to any selected Boundary chains. Select from the following:

  • None: No boundary is created. The Include stock option is automatically applied.

  • Silhouette: Creates a boundary that is a silhouette of the selected Machining geometry.

  • Bounding rectangle: Creates a rectangle that bounds the selected Machining and Avoidance geometry.

  • Bounding circle: Creates a circle that bounds the selected Machining and Avoidance geometry.

NOTE

The boundaries (except for None) accommodate the Contain options and the Compensate to options.

Include stock

Includes the stock settings from the Stock page for the selected Automatic boundary. If Automatic boundary is set to None, this option is automatically applied.

Contain

Select one of the following options:

  • Tool tip: Confines the tool center inside the containment boundary.
  • Tool contact point: Confines the tool contact points inside the containment boundary instead of the tool center. The tool center may run outside the selected boundary, but the contact point of the tool will not.

Compensate to

Select how the toolpath compensates to the selected Boundary chains:

  • Inside: Results in a toolpath where the entire tool remains inside the Boundary chains. Selecting this option allows you to adjust the boundary's offset distances.
  • Center: Results in a toolpath where the center of the tool travels to the containment boundary. You cannot make adjustments to the offset distances.
  • Outside: Results in a toolpath where the outer edge of the tool is bound to the containment boundary. Selecting this option allows you to adjust the boundary's offset distances.

Offset distance

Adjusts the inside or outside tool containment. For example, to ensure that the tool is completely clear of the containment boundary, you might enter a value of 0.050 inches. Or, to ensure that the tool overlaps the boundary, you can enter of a value of +/- 0.050 inches. For both inside and outside containment, enter a positive value to move the tool away from the boundary, and a negative value to pull it towards the boundary.

Include tool radius

Adds the tool radius (not the corner radius) to the Additional offset field to calculate the total offset applied to the containment boundary. Use this option to ensure that the tool is always clear of (or overlaps) the tool containment boundary by Offset distance. When you deselect Include tool radius, Offset distance represents the total distance between the tool center and the containment boundary, regardless of the tool size.

Total offset distance

Read-only. Displays the net distance between the Boundary chains and the center of the tool. It takes into account the following dimensions:

  • The size of the selected tool.
  • The amount of the Offset distance.
  • Whether Include tool radius has been selected.

Approximate start point

Select the point near the start of the toolpath. If the toolpath has several machining regions, the Approximate start point indicates which region is first.

Curve

Returns you to the graphics window to select curves. When curves are selected, the toolpath is created outward using the curves as a starting point.

NOTE

If the selected Curves lie entirely outside of the set Minimum depth or Maximum depth on the Steep/Shallow page, then you receive an error when generating the toolpath. The curves must be inside the set limits.

Project

Sets where the created curves will lie. Select one of the following:

  • 2D: Keeps the passes equidistant by Z depth (when Contain (Curves) is set to Tool tip) or as a straight down projection (when Contain (Curves) is set to Tool contact point).

    Tool tip Tool contact point

  • 3D: Keeps the passes equidistant by a 3D distance (when Contain (Curves) is set to Tool tip) or by the surface normal (when Contain (Curves) is set to Tool contact point).

    Tool tip Tool contact point

Contain (Curves)

Select how to contain the tool center in the selected Curves by one of the following options:

  • Tool tip: Confines the tool center inside the selected curves.
  • Tool contact point: Confines the tool contact points inside the selected curves instead of the tool center. The tool center may run outside the selected curves, but the contact point of the tool will not.
NOTE

The Contain options are only available when Project on the Cut Parameters page is set to 3D. Additionally, the Tool contact point option is only available when a ball tool is selected for the toolpath.

Points

Returns you to the graphics window to select points that will be projected onto the selected surfaces or solids.
