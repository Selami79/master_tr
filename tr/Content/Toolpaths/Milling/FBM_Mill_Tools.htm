Restmill Tools, Roughing Tools page: FBM MIll

Use the Restmill tools and Roughing tools pages to manage tool selection for the finish operations FBM Mill creates. In the large window, use right-click menu options to select up to 10 preferred tools for the operation type.

Mastercam uses the selected tools only when Preferred tooling is enabled in the Setup page, as shown in the following picture.

The Automatic tool selection parameters in the tools pages apply to some or all of the following types of tools.

  • Tools in the part file
  • Tools in the selected library
  • Tools created by Mastercam for the operation
  • Tools used to fillet internal feature edges in the X and Y axes.
NOTE

You can save the default list of preferred tools and other FBM Mill parameters in the operations .mcam-defaults file. Click Save parameters to defaults file located at the top of this dialog box.

Preferred tool list

Use right-click menu options to add up to 10 tools to the list of preferred tools for the operation type. Tool types supported for preferred tooling are Flat, Bull, Ball, Face. When Preferred tooling is activated in the Setup page, Mastercam searches your preferred tools list from the largest to smallest tool to find an appropriate tool for each FBM-generated toolpath. The search ends with the first tool small enough to machine the feature (even if a more appropriate tool is available in another selected source). No operation is created if Preferred tooling is active but the preferred tool list is empty.

Automatic tool selection

Applies only when tool selection options Use tool library or Create as needed are selected in the Setup page. Use these fields to control how Mastercam selects tools from the specified library, or creates them.

Maximum diameter

Enter the maximum tool diameter for automatic tool library selection or tool creation. FBM will not select or create tools larger than this diameter.

Tool size increments based on maximum diameter

Defines the size gap between tools that Mastercam uses to find or create the next available tool size. The Value and % reduction fields are associative and are based on the Maximum diameter defined for automatic tool selection. Entering a value in one automatically updates the other. A value of 25% means that the next tool selected or created will be 25% smaller than the previous one, starting with the maximum diameter tool. If you specify a small tool increment, Mastercam selects or creates more tools between the automatic tool selection's maximum and minimum diameters. Larger tool increments result in fewer tools selected or created.

Minimum diameter

Enter the minimum tool diameter for automatic tool library selection or tool creation. FBM will not select or create tools smaller than this diameter.

Internal XY fillets

Choose a strategy for selecting or creating tools to machine internal fillets in the XY axes. This strategy applies only when the minimum tool diameter is less than or equal to the fillet. Choose one of the following options:

  • Match tool with arc: When selecting or creating the tool, Mastercam uses the fillet size plus the stock to leave value you defined in the Floor finish and Wall finish cut parameters pages.
  • Use next smaller tool: Choose a tool that is the next size smaller based on your automatic tool selection settings (preferred tools, part file, library, or create as needed). If Mastercam must create the tool, it creates the next smaller standard size for the tool type.
  • % of arc size: Selects or creates a tool no larger than the percentage of the arc you specify.

Match wall draft with tapered tool

Selects a tapered tool that matches the draft on the part wall. Disables the Tapered wall stepdown field on the Wall finish page. The depth cuts values on the Wall finish page are used for the toolpath.

Allow use of

Select the tool type Mastercam should choose or create for the operation type. The tool type you choose applies when searching for tools in the Mastercam file or the selected tool library, and when creating tools, provided that these options are activated in the Setup page. Note that in the Facing tools page, Ball tool is unavailable; it is replaced with Face mills. Face mills are available only in the Facing tools page.

Right-click menu options

Create new tool

Opens the Tool Wizard where you create the new tool.

Edit tool

Opens the selected tool definition for editing in the Tool Wizard.

Delete tool

Removes the tool from the FBM tool list for the selected toolpath type.

Select tool from part

Opens the Tool Selection dialog box which lists only the tools that are currently saved in the Mastercam part file.

Select tool from library

Opens the Tool Selection dialog box where you can select a tool from the current tool library, or from any tool library you choose.

Duplicate tool

Quickly creates a new tool using one that is already in the list. Select a tool in the list then use this option to open the Tool Wizard. All values from the selected tool display and can be edited before saving the new tool.
