Depth Cuts page: Pocket

Select Depth cuts to activate the options in this page, and to add the following features to a 2D Pocket toolpath:

  • Divide the total pocket depth in increments. These will be used for both the roughing cuts and the finish passes on the walls.
  • Create finishing passes for the floor of the pocket.
  • Create tapered walls and islands.

In addition, you have the option of outputting subprograms for each depth. This option works only if the proper subprogram functionality has been enabled in your control definition.

To set the number of depth cuts, enter a maximum rough step. Mastercam divides the total depth into equal steps. Or, enter the exact number of finish steps and the size of each finish step. Mastercam never creates unequal rough depth cuts.

Maximum rough step

Sets the maximum amount of material removed in the Z axis with each rough cut. Mastercam calculates equal rough cuts no larger than the maximum rough step until it reaches the final Z depth. The value is calculated as a percentage of the tool diameter if you select Use tool's step, Peck, or Coolant option in your machine group properties.

Number of cuts (Finish)

Sets the number of finish cuts for the toolpath. This number multiplied by the finish step value equals the total amount of stock cut by the finish passes. Setting the number of finish cuts to 0 creates no finish cuts.

Step (Finish)

Sets the amount of material removed in the Z axis with each finish cut. This number multiplied by the number of finish passes equals the total amount of stock cut by the finish passes. The value is calculated as a percentage of the tool diameter if you select the Use tool's step, Peck, Coolant option in your machine group properties..

Override feed speed

Allows you to override the speed and feed values for finish passes, providing better control for floor finishing. Select one or both of the following options:

  • Feed rate: Select to enter a different feed rate for the finish passes than the feed rate used for roughing. You may want to increase the feed rate because the tool usually cuts less material during finish passes.
  • Spindle speed: Select to enter a different spindle speed for the finish passes than the spindle speed used for roughing. You may want to increase the spindle speed because the tool usually cuts less material during finish passes.

Keep tool down

Determines whether or not to retract the tool between passes.

NOTE

Even if Keep tool down is selected, other parameters, such as Entry method, may require the tool to retract.

Use island depths

When this option is selected, Mastercam passes the tool over the island until it reaches the depth of the island. It then goes around the island until it reaches the final depth. When this option is deselected (default), the toolpath goes around the island regardless of the island depth.

Subprogram

Sets values in the subprogram lines in the NCI file to indicate a section of the operation that is repeated at a different location (for example, a drill cycle or depth cut). This information is then passed to the post processor. The control definition must be set to support subprograms for this function to work.

In some instances, you might be able to select absolute or incremental subprograms. An absolute subprogram contains absolute coordinate values; in other words, the same points are machined every time the subprogram executes. With an incremental subprogram, the points which are machined are relative to the tool's location when the subprogram is called. Typically an absolute subprogram will have a G90 code while an incremental one will have a G91.

Depth cut direction

Sets the depth cut direction:

  • Stepdown: Cuts from top to bottom.
  • Stepup: Cuts from bottom to top.

Undercut (undercut tool only)

Select when using an undercut tool to support tool compensation.

Depth cut order

Determines the order of the depth cuts based on the selected option.

  • By pocket: Performs all depth cuts in a region, contour, or pocket before moving to the next contour or region.
  • By depth: Creates depth cuts at the same level in every contour or region and then descends to the next depth cut level. This option can be used for thin-walled parts using soft materials such as aluminum or graphite.

Tapered walls

Angles the depth cuts from the top of stock to the final depth. Not available if you use subprograms. Note that if you have chained the bottom of the pocket instead of the top, the upper cuts will be outside the chained geometry.

Taper angle

Sets a taper angle for the contour or pocket walls.

Island taper angle

Sets the taper angle for islands.
