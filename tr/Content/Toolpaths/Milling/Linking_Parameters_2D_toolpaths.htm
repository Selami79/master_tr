Linking Parameters page: 2D Toolpaths

Use this page to define how Mastercam calculates clearance height for 2D toolpaths. Other settings control tool movement for retract, feed plane, top of stock, and depth heights. Set these heights as either absolute or incremental values.

How do I create linking parameters relative to a stock model?

When Use stock is selected on the Stock page, toolpaths are created relative to the stock model. This creates more accurate and efficient toolpaths when using an uneven stock model.

When Use stock definition is set to Feed position, Depth, or Both on the Stock page, the corresponding values are locked to Incremental.

Calculate incremental values from holes/lines

Calculates the Retract, Feed plane, and Top of stock values from the top of the hole or line. When not selected, Mastercam uses the Retract, Feed plane, and Top of stock values specified in the fields.

Automatic linking

Modifies your Clearance, Retract, Top of stock, and Depth parameters to avoid collisions with the solid model or the selected avoidance geometry. When you select Automatic linking, Mastercam warns you that the settings have been changed.

Arc fit maximum radius

Attempts to fit 90-degree arc moves into rapid and clearance moves. If the entered radius does not fit, then the output is created as an 180-degree arc.

Output feed move

Outputs all rapid moves between passes as a feed rate move instead of a rapid move. This is helpful when the tool needs to make many irregular moves per pass to jump between different areas of the part. This option is also useful for older machines which create dogleg moves for rapids.

Arc fit maximum radius and Output feed move off Arc fit maximum radius and Output feed move on

Clearance

Sets the height at which the tool moves to and from the part. Select the checkbox to activate the clearance height. If it is not selected, the Retract value is used as the clearance plane. Click the button to set or change the height. You can select a point on the geometry or enter a value.

For Model Chamfer toolpaths, Clearance is an absolute value from start and end. However, if there is a feature on the solid model that is higher than the entered Clearance value (shown as 1 below), Model Chamfer will instead use the entered Retract value (shown as 2 below) to clear it.

Only at the start and end of operation

Rapids to the clearance plane only at the start and end of the toolpath.

Retract

Sets the height that the tool moves up to before the next tool pass. Select the checkbox to activate the retract plane, and then click the button and select a point on the geometry or enter a value. The retract height should be set above the feed plane. If you do not enter a Clearance height, the tool moves to the retract height between operations.

For Model Chamfer toolpaths, the incremental value moves away from the current position and it is gouge checked against the selected solid model and the avoidance geometry. If the distance will gouge, Model Chamfer raises the value higher until it clears.

For 2D Contour toolpaths, when moving between chains, if the incremental retract height does not clear the top of stock by a safe distance, a minimum retract of .100 inches or 2 millimeters will be used.

Feed plane

Sets the height that the tool rapids to before changing to the plunge rate to enter the part. The tool also moves to this height between operations if you do not enter both the Clearance height and the Retract height. Select the checkbox to activate the feed plane, and then click the button and select a point on the geometry or enter a value.

For Model Chamfer toolpaths, Feed plane is an incremental distance above the next cut depth.

Top of stock

Sets the height of the material in the Z axis. Many other toolpath parameters are measured from this position in Incremental mode, such as Clearance, Retract, and Feed planes. Select the checkbox to activate the retract plane, and then click the button and select a point on the geometry or enter a value. Incremental values are measured from the chained geometry.

Depth

Determines the final machining depth and the lowest depth that the tool descends into the stock. The default Depth is the depth of the first chain selected. Select the checkbox to activate the retract plane, and then click the button and select a point on the geometry or enter a value. Incremental values are relative to the chained geometry, not the Top of stock.

Absolute

Absolute values are always measured from the origin 0,0,0. If you were to translate geometry associated with a toolpath, and the toolpath depth was set to an absolute value, the tool will try to cut to that same absolute depth value no matter where the geometry is located.

Incremental

Incremental values are relative to other parameters or chained geometry.

For most Mill and Router toolpaths, Depth and Top of stock parameters are relative to the location of the chained geometry. Clearance, Retract, and Feed plane are relative to the Top of stock.

Associative

Associative values are measured relative to a selected point on the feature. Use Select point to choose the point from which the associative value will be measured. Use the Delete point option to remove the associative point.

Break through

Use the Break through option to drill through the bottom of the stock by an amount that you specify. Break through causes the tool to cut into the spoil board to ensure a clean cut through the material. Always enter the break through amount as a positive number. Mastercam adds the break through amount to the final depth of the toolpath to ensure through-cutting.
