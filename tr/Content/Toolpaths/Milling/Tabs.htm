Tabs page

Use the Tabs page to configure tabs for a 2D Contour toolpath by:

  • Setting the tab position by choosing a method that Mastercam uses to place tabs along the contour.
  • Defining the tab motion and configuring the size of the tabs.

If your layout includes many parts of different sizes, you can tell Mastercam the size of the largest part for which to automatically create tabs. Typically, tabs are only used for smaller parts. To do this, choose Create tabs on shapes less than, and then enter the dimensions of the largest part that requires tabs.

NOTE

When selecting tab positions, you cannot place a tab on a solid. Tab positions can only be placed on wireframe geometry.

After completing options in this page, choose the Tab Cutoff page page to select the type of cutoff operation.

Automatic

Select to have Mastercam calculate and evenly position tabs along the contour toolpath's chained geometry. Then choose the calculation method to use (number of tabs or maximum distance). Deselect to define a manual method of positioning the tabs.

Number of tabs

Enter the number of tabs to be placed on each part (closed chain) in the operation. Mastercam calculates and creates the specified number of tabs, evenly spaced along the contour.

Maximum distance between tabs

Enter the maximum distance you want between tabs. Mastercam calculates and creates the fewest number of tabs that can be evenly spaced along the contour without exceeding this parameter.

Create tabs on shapes less than

Creates tabs only on parts that are smaller than the specified dimensions. Enter a width and length.

Width

Enter the maximum width of the parts to be tabbed.

Length

Enter the maximum length of the parts to be tabbed.

Tab all

For automatic tabs, places tabs on all parts in the contour toolpath, regardless of size.

Manual

Select to manually choose tab locations. Then choose Position to return to the graphics window and select each point for tab placement along the contour toolpath's chained geometry. Deselect to have Mastercam automatically create tab positions based on the number of tabs or maximum distance between tabs.

Position

Returns you to the graphics window so that you can select the tab locations. Use the AutoCursor options to automatically “snap” to endpoints, midpoints, or other key locations.

Use square point for tab position

Automatically locates a tab at square points. These are point entities that have been formatted with the square point attribute. Select the square point style from the Attributes group on the Home tab and create points at each desired tab location. The points need to be created before you choose this option.

Start

The point that identifies the tab location at the start of the tab.

Midpoint

The point that identifies the tab location at the midpoint of the tab.

End

The point that identifies the tab location at the end of the tab.

Full

Sets the height of the tab to the full thickness of the stock.

Vertical moves

The tool moves straight up to the tab height at the beginning and end of each tab.

Ramp moves

The tool moves to the tab height at an angle at the beginning and end of each tab.

Ramp angle

Enter the angle at which the tool moves to the tab height.

Width

Sets the linear dimension of the tab along the toolpath.

Tab thickness

Enter the amount of stock to leave for the tab.

Partial

Sets the height of the tab to less than the full thickness of the stock. Enter a Tab thickness to tell Mastercam how much stock to leave in the tab.

Use feed plane for full height tabs

When retracting from the stock while moving from the tab to the next position in the toolpath, the tool retracts to the feed plane instead of the clearance or retract height. When this option is not selected, Mastercam uses either the retract or clearance planes (whichever is set). If both are set, Mastercam uses the higher of the two.

Overwrite edited tabs

Select this option if tabs have already been created and manually edited, and you wish to overwrite those changes. If this option is not selected, Mastercam preserves any changes or new tabs that were added through the tab editing in Chain Manager. (Tabs can be edited by clicking the geometry icon, selecting the chain(s), right-clicking and choosing Edit tabs from the menu.)
