Pocket Toolpaths: 2D

Use 2D Pocket toolpaths to clean out material from an enclosed boundary. Mastercam offers many options for integrating rough and finish operations and tailoring the toolpath to your geometry. Use the following outline as a guide for creating pocket operations and organizing the different pocketing options. Click drop-down below to see more information about the step.

Chain geometry and select a tool
  1. Choose Pocket from the 2D gallery on the Toolpaths contextual tab.
  2. Then, chain the geometry for the pockets. All geometry used to define a pocket and any islands must be in the same construction plane. You cannot select a 3D chain for a pocket toolpath. Mastercam automatically interprets closed boundaries within the chained geometry as islands and adjusts the toolpath accordingly.

  3. Use the Tool page to select a tool and enter general toolpath parameters.

    NOTE

    Consider using a surface pocket toolpath if you need to create a pocket with complicated geometry.

Select a pocket type

Use the Cut Parameters page to select a pocket type. Mastercam lets you select from several pocket types, which are based on your pocket geometry:

  • Standard: Mastercam intelligently analyzes all selected chains and considers the largest of the closed chains to be the stock to machine. The tool is constrained by the machined area (the largest closed boundary selected). All other smaller closed chains within the largest chain are considered non-machinable areas and are avoided.
  • Facing or island facing: Mastercam automatically adjusts cut depths for islands.
  • Open pockets: Mastercam automatically enters and exits the pocket through the opening.
  • Remachining: Mastercam calculates the stock remaining from a previous operation (or from a roughing tool dimension) and just machines the leftover stock.

When you select a pocket type other than Standard, Mastercam displays additional parameters below the field. Use these parameters to fine-tune the cutting action.

Use other options on this page to set other general cutting parameters.

Select a cutting method
  • Use the Roughing page to select a cutting method and set other options for the roughing passes. Mastercam offers different cutting patterns so you can choose the one best suited to your part geometry and application requirements.
  • Use additional options in the Entry Motion page nested below the Roughing page to define tool plunge motion. You can define tool entry motion as helical or ramp, or you can turn it off for the toolpath.
Create a finishing pass
  • Use the Finishing page to create a finish pass. You can override the feed rate and spindle speed and set special options for thin wall pockets. Mastercam outputs the finish pass as a separate operation so that you can further edit and refine it separately from the roughing operation.
  • In addition to finishing pockets in Z depth increments, you can use spring passes to finish thin wall pockets. Spring passes are additional finish passes along the same path as the last finish pass. They can be used for thin wall parts where the tool may have flexed away from the walls during previous finish passes and missed some material. To create the spring passes, enter the number of passes in the Finishing page for standard pocket toolpaths, or the Pocket parameters tab for surface rough pocket toolpaths.
  • Use additional options in the Lead In/Out page nested below the Finishing page to create separate lead in/out moves for the finish pass. These moves are distinct from the entry moves defined for the roughing passes.
Create a separate finishing operation

To add a separate finish operation to the Toolpaths Manager after the pocket operation:

  • Select Create a separate finishing operation on the Cut Parameters page
  • Then select Finish on the Finishing page to enable finishing passes for the toolpath.

The new finish operation uses the same parameters and geometry as the original pocket toolpath, but consists only of finishing passes. This allows you to select a different tool for the pocket finishing passes.

Specifying an entry location

To specify an entry location for a pocket toolpath, follow this procedure:

  1. Create a point where you want the toolpath's entry location in the graphics window. Optionally, you can skip this step and create / select the point in Step 3.
    • The entry point can be outside the pocket.
    • Mastercam uses only the XY position of the selected point; the Z depth is not used.
  2. Choose Pocket from the 2D gallery on the Toolpaths contextual tab.
  3. In the Chaining dialog box, chain the point you created in Step 1, or choose the point option to create and chain the point.
  4. Then chain the pocket.
  5. Click OK when finished chaining.
  6. Select your desired settings in the pages displayed in the 2D Toolpaths - Pocket dialog box.
  7. If the entry point you selected is outside of the pocket, click OK to exit the 2D Toolpaths - Pocket dialog box and create the toolpath.
  8. If the entry point you selected is inside of the pocket, in the Entry Motion page of the 2D Toolpaths - Pocket dialog box , choose Helix.
  9. Then select Center on entry point.
  10. Click OK to exit the 2D Toolpaths - Pocket dialog box and create the toolpath.
NOTE

After defining the entry point, you can use Mastercam's transform functions to move the point anywhere you like. After moving the point, you must regenerate the toolpath.
