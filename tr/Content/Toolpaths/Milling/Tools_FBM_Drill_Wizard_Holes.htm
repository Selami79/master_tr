Tools page: FBM Drill/Wizard Holes

Use the Tools page to control the tools FBM selects for the drill cycles it creates.

Mastercam attempts to match the hole diameter with the closest sized tool using the tool sources you select here and other tool creation settings. However, this does not guarantee that the hole can be successfully machined and that all material will be removed.

You can limit tool selection to any combination of the following sources:

  • Tools in a block drill (if available in the current Router machine group)
  • Tools in the current file
  • Tools in a specified tool library
  • Standard or custom-sized tools that Mastercam creates as needed

If you select Use block drill for the FBM Drill operation, Mastercam assigns matching block drill tools first. Then it assigns tools based on the following hierarchal order:

Red lines connecting the tool source checkboxes show the hierarchical order.

Example 1 Example 2 Example 3

Block drill tools are given the highest priority for tool assignments to minimize tool changes in the FBM Drill-generated toolpaths. If you do not select a specific block drill and more than one is configured, Mastercam assigns the first one in the machine definition's component list. You can assign a specific tool to spot drill cycles in the Spot Drilling page. This assignment overrides any Tools page settings.

Diameter matching tolerance

Mastercam uses this value to determine how closely the diameter of the tool and the hole must match before selecting an appropriate tool.

Use both inch and metric tools

Select to consider U.S. and metric tools when matching the hole diameter with the closest sized tool. Deselect this option to consider using only tools that have the same unit of measure as defined in your current Mastercam configuration file (U.S. or metric).

Tool selection

Select one or more of the following options to tell Mastercam where to locate tools for the FBM-generated toolpaths. The options display in the same hierarchical order Mastercam uses when searching for an appropriate tool. If an appropriate tool is not available in the first selected source in the hierarchy, Mastercam ends the search if no other tool selection options are activated. Otherwise, the search continues with the next selected source in the hierarchy. The tools must match within the specified Diameter matching tolerance.

  • Use tools in part file: Looks in the current Mastercam file for an appropriate tool. The tools do not have to be used in previous operations to be available to the FBM operation.
  • Use tool library: Searches the selected tool library for the necessary tools.
  • Create as needed: Creates the necessary tools using the Tool creation parameters you define.

Hole bottom geometry

Blind holes created by the SOLIDWORKS Hole Wizard may not always have a flat bottom. Choose one of the following options to define the relationship of the bottom hole geometry to the tool tip geometry.

  • = geometry angle: The tool tip angle must match the hole bottom geometry within the specified Tool tip match tolerance.
  • > geometry angle: The tool tip angle must be larger than the floor angle geometry. The resulting hole may not be drilled as deep as it is defined in the SOLIDWORKS model.
  • < geometry angle: The tool tip angle must be smaller than the floor angle geometry. The resulting hole may be drilled deeper than it is defined in the SOLIDWORKS model.

Tool tip match tolerance (degrees)

Determine how closely the tool tip geometry must match the hole bottom geometry for proper tool selection. Specify the tolerance in degrees.

Use Block Drill

Available only when the current Router machine group includes one or more block drills. Deselect to exclude block drill tools from assignment in the FBM Drill-generated toolpaths. Select to allow FBM Drill to assign available block drill tools to detected holes, and to generate block drill toolpaths for these assignments. Then use Select/edit block drill to open the Drill Block Selection dialog box. Block drill tools are given the highest priority when tool assignments are made, minimizing tool changes in the FBM-generated drill operations. Note that, if you do not select a specific block drill and more than one is configured, Mastercam assigns the first one in the machine definition's component list.

Select/edit block drill

Click to open the Drill Block Selection dialog box where you can view and select a drill block for use in the FBM Drill operation. You can also edit the block drill tool stations and configure other settings.

Tool creation

Available only when Tool selection includes Create as needed. Mastercam uses the parameters you choose in this section to create new tools for FBM-generated toolpaths. Note that, if this is the only tool selection option activated for an FBM Drill or Wizard Holes operation, Mastercam adds 10% of the tool's flute length to calculate the tool's shoulder length and overall length.

Create only standard sizes

Select to prevent Mastercam from creating non-standard tools. Only industry standard sized tools are created for FBM-generated operations. When deselected, Mastercam creates non-standard sized tools that most closely match the hole diameter, within the specified Diameter matching tolerance. To determine standard tools sizes, Mastercam references entries in the following files installed in Mastercam's \Common\FBM directory:

  • FbmToolTable.xls: Used if Microsoft Excel is installed or accessible from your Mastercam workstation
  • FbmToolTable.csv: Used if your Mastercam workstation does not allow access to Microsoft Excel. You can customize these files using your preferred XML or text editor. However, do not change the file names or their locations.

Consider flute length

Select to use the tool's flute length when creating new tools. Then choose one of the following options to define what to do when the hole depth exceeds the tool's flute length.

  • Create one tool for all depths: Creates a single tool capable of meeting the longest depth requirement, based on detected features and FBM operation parameters.
  • Create one tool for each depth: Creates a separate tool for each unique depth.
  • Create tools in length increments of: Creates tools in length increments using a percentage of the current tool. For example, if you enter 25%, Mastercam create the next new tool 25% longer than the original tool.
