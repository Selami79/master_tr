Hybrid Toolpaths

The 3D High Speed Hybrid toolpath utilizes both scallop and constant Z approaches in a single toolpath, addressing both the steep and shallow areas of your part. Hybrid attempts to apply these methods of generating machining passes onto your model by analyzing the topology of the part and the distances between passes. The Hybrid toolpath includes options to influence where it generates the scallop offset filler passes and options on handling flat features.

The Hybrid toolpath typically results in uniform spacing of passes across the entire model.

 

Toolpath Guidelines

Hybrid toolpaths are finishing toolpaths. In most cases, it is best practice to use a Hybrid toolpath after the roughing operations are complete.

What makes the Hybrid toolpath different than other 3D high speed toolpaths?

The Hybrid toolpath is part of the 3D high speed toolpath family, but it does include unique options not found in other toolpaths. These are listed below:

Toolpath Control page

The Projected boundary smoothing option in the Toolpath Control page attempts to improve the projected boundary in cases where it is jagged or noisy when you are using Avoidance geometry and no containment boundary. The best practice to create your toolpath, measure the amount of noise, and adjust this option as necessary.

Cut Parameters page

The Step group of settings in the Cut Parameters page are unique to the Hybrid toolpath. Use these controls to control how Mastercam defines steep and shallow areas.

The Z stepdown value defines the constant Z distance between adjacent stepdowns. Mastercam uses these stepdowns in combination with the Limiting angle and 3D stepover values to calculate cut passes for the Hybrid toolpath.

Mastercam first slices the entire model into different sections defined by the Z stepdown distance. Then, analyzes the machining geometry's slope transition between each stepdown along the specified Limiting angle.

1. Steep stepdown

2. Limiting angle (applied)

3. Steep

4. Shallow

  • If the machining geometry's slope transition distance within the stepdown distance is less than the applied Limiting angle, the Hybrid toolpath considers it steep and a single 2D waterline cutting pass is generated.

  • Otherwise, the stepdown is defined as shallow. Mastercam uses the 3D stepover value to create 3D scallop cut passes along the shallow slope for the shallow stepdown.

Advanced page

The Remove tight passes option on the Advanced page allows you to filter out any pass or pass sections less than the entered distance. You enter the closest distance allowed between a shallow and a steep pass.

What are similarities Hybrid toolpath share with other 3D high speed toolpaths?

The Hybrid toolpath shares many similarities with the other available 3D high speed toolpaths.

Geometry Selection

Geometry selection for the Hybrid toolpath is the same as the other 3D high speed toolpaths. After selecting the toolpath from the ribbon, the toolpath parameters dialog box opens. Use the Model Geometry and Toolpath Control pages in the toolpath dialog box to set machining regions, avoidance regions, and more.

Parameters

  • Hybrid operations include cut styles for Open contour directions and Closed contour directions. These options orient the cutting direction for open and closed contours. This is shared with Area Roughing, Waterline, and Equal Scallop 3D high speed toolpaths. You can learn more about these methods in the Cut Parameters page.

  • Hybrid, along with other 3D high speed toolpaths, include advanced linking parameter options. These options give you more control over leads, retracts, and transitions. You can access these options on the Linking Parameters page.

  • The Optimize cut order option on the Cut Parameters page optimizes your machining to help reduce travel time and minimize tool retracts.

  • Hybrid, along with other 3D high speed toolpaths, allow you to enter a maximum and minimum Z depth value on the Steep/Shallow page to establish the heights of the highest and lowest cutting passes. You can learn more in the Using Z Limits to Create a Machining Zone Help topic.
