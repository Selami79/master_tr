Feature Detection page: FBM Mill

Use the Feature Detection page to control the types of features FBM Mill detects. For example, you can include pockets that extend through the part and set a minimum diameter to exclude small drill holes from detection.

You can choose a pocketing or hole milling strategy to machine the round features FBM Detects. If you choose Hole milling, use the parameters on the Hole Milling and the Circle Mill or Helix Bore pages to define the toolpath settings.

During feature detection, Mastercam creates edge curve geometry for the FBM Mill-generated toolpaths. Use options on this page to assign a level to this geometry.

Only features that match your detection criteria appear in the Features page for review. Mastercam automatically creates toolpaths for detected features when you save the FBM Mill operation.

Choose Detect (shown below) in this dialog box to detect features. You can also configure Mastercam to detect features when the FBM Mill dialog box opens. To do this, select Automatic initial feature detection in the Setup page, and then save the setting to the operations .mcam-defaults file.

Through features

Select to allow FBM Mill to detect and include pockets that extend through the whole model. Choose one of the following options to control how Mastercam cuts the through pockets. Then enter the corresponding value in the field.

  • Stock to leave: Sets the amount of material to leave at the bottom of the stock. Use this option to protect what is underneath the part during the machining process.
  • Additional break through: Sets the depth of the cut beyond the bottom of the part.

Edge curves

During feature detection, Mastercam creates the edge curve geometry that is required by the FBM-generated toolpaths to machine detected features. Choose one of the following options to tell Mastercam where to place this geometry. Note that the edge curve geometry is blanked; Use Unblank on the Home tab to view it.

  • Place edge curves on level: Defines the level where the FBM-generated geometry is created.
  • Place edge curves on first unused level greater than: Places the FBM-generated geometry on the first unused level it finds that is greater than the specified level.

Recognize holes greater than this diameter as features

Eliminates from detection any through holes that are not through pockets, for example, drilled holes. FBM Mill detects only through holes with a diameter that is greater than the value you enter in this field. All holes that are equal to or smaller than the entered diameter are excluded from detection.

Mill round features with this cycle

Choose one of the following strategies for machining the round hole features that FBM Mill detects in your solid model:

  • Pocket: (default) Generates pocketing toolpaths for round hole features, allowing you to output separate rough, rest, and finish toolpaths.
  • Hole milling: Generates a single Circle mill or Helix bore toolpath to machine round hole features. After choosing Hole milling, use the parameters on the Hole Milling, and the Circle Mill, or Helix Bore pages to define the toolpath settings.
