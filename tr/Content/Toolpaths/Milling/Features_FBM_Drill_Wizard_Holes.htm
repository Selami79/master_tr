Features page: FBM Drill/Wizard Holes

Use the Features page to manage the list of hole features that the FBM drilling operation detects in the solid model. The holes and machining information in the list are based on the parameters you set in the operation's other pages.

For each hole listed, you can use right-click menu options to change the hole type, tool plane, and finish tool. You can also delete holes from the list (select and press [Delete]), or use the right-click Suppress option to exclude them from the operation.

When you accept the operation, Mastercam creates drill cycles for all holes in the Features list that can be machined.

Make these modifications only at the final stage of creating the FBM drill operation. Detect rebuilds the Features page list, overwriting any hole type, plane, and finish tool modifications.

NOTE
  • The Features page settings are saved within the part file after choosing OK. The saved settings—including assigned hole types, finish tools, suppressed states, and planes—are applied to existing features even if Detect is selected again.
  • Imported SOLIDWORKS® data is the exception to this rule. On redetect, Mastercam replaces all imported SOLIDWORKS Hole Wizard® data with feature data detected in the Mastercam part file.
  • To delete all saved data and regenerate new data for the FBM Drill operation, activate the Delete existing features on detect checkbox in the Hole Detection page, and then detect features.
  • Different options display when you right-click in the Hole type, Plane, and Finish tool columns.
  • The Z1 column displays the depth where the hole feature starts. For co-axial features, the Z1 value is also the bottom (end) of the feature that is positioned immediately above the next co-axial feature. The Depth column shows the distance between Z1 and the designed bottom of the hole.

  • Hole information is not listed until you click Detect, located at the top of the dialog box. Choose Detect whenever you make changes to FBM Drill parameters. Then view results in the Features page carefully before saving the operation.

  • Use [Shift+click] and [Ctrl+click] methods to select more than one hole in the list.

  • Features with this status indicate that Mastercam cannot completely machine them using the current tool selection and FBM drilling parameters.

  • If you are working with an imported SOLIDWORKS part, and you have SOLIDWORKS installed on your local PC, click Select SOLIDWORKS file to open the model's .sldprt file and update the features list with imported SOLIDWORKS Hole Wizard data.

Display all normals

Select to display normal direction arrows in the graphics window for all detected holes. Deselect to remove the arrows from the display.

To set the normals of all detected holes to the same normal, right-click anywhere in the features list, and choose Select normal point from the right-click menu. Then choose a point in the graphics window.

You can set the normals of all detected features in a direction towards the current tool plane, if this is possible. To do this, select Set normals toward tool plane on detect in the Hole detection page. Then choose Detect to make the change. (FBM Drill only)

Icon Name and Description

Feature suppressed: Your current parameters do not allow machining this feature. For FBM Drill/Wizard Holes, this state indicates that the feature was manually suppressed. Hole type column

Use to change the hole type currently assigned to features. First select one or more features to change. Then right-click in the column, choose Hole type, and select a new hole type from the list.

Plane column

Use to change the tool plane currently assigned to features. First select one or more features to change. Then right-click in the column, choose Plane, and select a new plane from the list.

The list displays only valid machining planes for the selected features (perpendicular planes are omitted from the list). If you choose holes from non-parallel planes, such as Top and Left planes, only planes that are parallel to the first hole in the list can be changed. If you select a Top plane hole feature along with holes in Left and Bottom planes, Mastercam changes only the Top and Bottom planes. Selecting a [reverse plane] option will reverse all selected holes regardless of initial plane.

Z1 column

The Z1 column displays the depth where the hole feature starts. For coaxial features, the Z1 value is also the bottom (end) of the feature that is positioned immediately above the next coaxial feature.

Depth column

The Depth column shows the distance between the Z1 depth and the designed bottom of the hole.

Finish tool column

Use to change the hole feature's finish tool. First select one or more features to change. Then right-click in the column, choose Finish tool, and select a new tool from the list. The three-character prefix in the finish tool list indicates that the tool is from one of the following tool sources:

  • MCAM: Mastercam part file
  • LIB: Selected tool library
  • XLS: Created for the operation using FBM's standard tool definitions file. FBM uses the tool definitions in this file to create new tools if your automatic tool selection settings on the Tools page include Create as needed. It is used exclusively if Create only standard sizes is also selected on that page.

Select (FBM)

Click to return to the graphics window and select one or more features in the solid model. When you end selection, these features are also selected in the Features page list.

Select / Unselect all (FBM)

Click to select or unselect all features in the list.

Select common features

Select to activate the option to select all holes in the Feature list with common attributes. Then click Select common features to access the Feature Selection dialog box where you can choose the attributes to match in the list.

Select co-axial features

When activated, automatically selects all co-axial hole features in the list (co-axial holes are multiple holes that share a common axis).

Select SOLIDWORKS file

Available only if the current workstation includes a valid SOLIDWORKS installation. Allows FBM Drill to utilize the SOLIDWORKS Hole Wizard data when generating operations for detected features. Click this button to select the SOLIDWORKS part file that corresponds to the current solid model and import its Hole Wizard data into the features list. During the import, Mastercam matches the SOLIDWORKS Hole Wizard data to features in the list by comparing hole diameter, coordinates, and plane information. For each match, if the information in the features list is different, Mastercam redefines it using the SOLIDWORKS Hole Wizard definition.

Right-click menu options
  • Hole type: Available only in the Hole type column. Use to change the hole type currently assigned to one or more selected features.
  • Suppress: Removes toolpaths for suppressed features from those generated by the FBM Drill-operation. However, when FBM Drill generates toolpaths for unsuppressed features, it considers suppressed feature data when calculating incremental clearance and retract moves.
    • Selected: Suppress the holes selected
    • Previously machined: Suppress previously machined features
    • Features with warnings: Suppress all features with warnings
  • Plane: Available only in the Plane column. Use to change the plane currently assigned to one or more selected features. The list displays only valid machining planes for the selected features (perpendicular planes are omitted from the list). If you choose holes from non-parallel planes, such as Top and Left planes, only planes that are parallel to the first hole in the list can be changed. If you select a Top plane hole feature along with holes in Left and Bottom planes, Mastercam changes only the Top and Bottom planes. Selecting a [reverse plane] option will reverse all selected holes regardless of initial plane.
  • Finish tool: Available only in the Finish tool column. Use to change the tool currently assigned to one or more selected features. The three-character prefix in the finish tool list indicates that the tool is from one of the following tool sources:
    • MCAM: Mastercam part file
    • LIB: Selected tool library
    • XLS: Created for the operation using FBM's standard tool definitions file. FBM uses the tool definitions in this file to create new tools if your automatic tool selection settings on the Tools page include Create as needed. It is used exclusively if Create only standard sizes is also selected on that page.
  • Set normal direction:
    • Towards a point: Sets the normals of all selected holes towards a point you choose in the graphics window.

    • Away from a point: Sets the normals of all selected holes away from a point you choose in the graphics window.

      NOTE

      Select Display all normals at the top of the Features page to display normal direction arrows for all holes in the graphics window.

  • Delete: Deletes the selected holes from the list. To restore deleted holes, run Detect again.
  • Summary: Opens the Feature Summary dialog box where you view a summarized list of all detected features grouped by common hole types, diameters, and finish tools.
