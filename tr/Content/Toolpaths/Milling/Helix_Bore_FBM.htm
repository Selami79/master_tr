Helix Bore page: FBM

Use the Helix Bore page to create roughing and finishing passes, and to define cutting parameters and compensation options for the Helix Bore toolpaths that FBM generates.

Choose Finish to enable the finish pass options. Typically, if you use this option, you would set the Stock to leave values to 0.

You can set feed rate overrides separately for each type of cut (rough and finish).

Disable pre-drilling

Available only for FBM Drill and Wizard Holes operations. Select to ignore the settings on the Pre-Drilling page, and exclude pilot hole drill cycles for features included in FBM-generated Circle mill or Helix bore toolpaths. Deselect to allow FBM Drill or Wizard Holes to consider pre-drilling parameters. If pre-drilling is active for the FBM operation, pre-drilling cycles are generated in addition to Circle mill or Helix bore toolpaths for hole milling features.

Rough pitch

Sets the distance that the tool advances with each rotation.

Number of rough passes

Tell Mastercam how many rough passes to create. Set this value to more than one if you want to provide a rough pass stepover value.

Rough pass stepover

If you specify more than one pass, enter the distance the tool moves between the passes.

Long tool rough pass stepover

If you specify more than one pass, enter the distance (as a percentage of the tool's diameter) that you want the tool to move between passes made using long tools. Setting a separate stepover distance for passes made with long tools can help reduce the amount of tool deflection. Note that, in FBM Mill, you define long tools for finishing passes by entering a length to diameter ratio in the Cut parameters, Floor finish, and Wall finish pages. In FBM Drill or Wizard Holes, long tools are defined in the Deep drilling page.

Feed rate

Enter a different feed rate for the finish passes than the feed rate used for roughing. This lets you adjust the chip load for the finishing operation.

Spindle speed (RPM)

Enter a different spindle speed for the finish passes than the spindle speed used for roughing. This lets you adjust the chip load for the finishing operation.

Stock to leave on walls

Sets the amount of stock on selected vertical machining geometry. Enter a positive value if you do not want to remove all stock or enter a negative value to remove more stock. If the machining geometry (or a portion of it) is not vertical, Mastercam interpolates a value between the Stock to leave on floors and Stock to leave on walls amounts.

Stock to leave on floors

Sets the amount of stock on selected horizontal machining geometry. Enter a positive value if you do not want to remove all stock or enter a negative value to remove more stock. If the machining geometry (or a portion of it) is not horizontal, Mastercam interpolates a value between the Stock to leave on floors and Stock to leave on walls amounts.

Start angle

Sets the angle where the toolpath begins.

Entry/exit arc sweep

Sets the included angle of each entry and exit arc. If the entry/exit arc sweep is less than 180 degrees, then Mastercam applies an entry/exit line.

Start at center

Begins the toolpath at the center of each selected entity before moving to the next entity. The image below is a Helix Bore toolpath with Start at center selected and End at center deselected.

Perpendicular entry

Enters the toolpath perpendicular to the first tool move. This option is particularly helpful when cutting multiple holes to avoid unexpected motion.

Overlap

Sets how far the tool goes past the end of the toolpath before exiting for a cleaner finish.

Output arc moves for helixes

Writes the entry helix to the NCI file as arcs. Using this option can create shorter NC files. If you turn off this option, the helix breaks into linear segments in the NCI file.

Tolerance

Controls the accuracy of the entry helix when broken into linear segments. A lower tolerance value produces a more exact entry helix, but may create larger NC files and take longer to process. Not available if you select the Output arc moves option.

Compensation type

Choose how you want to handle cutter compensation:

  • Computer: Computes the compensated toolpath and does not output control codes for compensation.
  • Control: Outputs control codes for compensation and does not compute the compensated toolpath. If the tool diameter is greater than or equal to the hole radius, Mastercam applies an additional 10% to the tool radius value and uses that value as the arc radius for entry and exit moves.
  • Wear: Computes the compensated toolpath and outputs control codes for compensation. Compensation direction in the computer and control are the same.
  • Reverse wear: Computes the compensated toolpath and outputs control codes for compensation. Compensation direction in the computer and control are opposite.
  • Off: Programs the tool tip directly on the chained geometry and does not output compensation codes.

The active control definition determines the available options. For Helix Bore toolpaths, when the Compensation type is set to Control and the tool diameter is greater than or equal to the hole radius, Mastercam applies an additional 10 percent to the tool radius value and uses that value as the arc radius for the entry and exit moves.

Use long tool values when the length to diameter ratio is greater than (L/D)

Defines long tools using the length to diameter ratio you enter. When long tools are used for FBM-generated toolpaths, Mastercam applies the % Long tool values you set in the XY stepover and Depth cuts fields for roughing, restmill, floor finish, and wall finish cut parameters.
