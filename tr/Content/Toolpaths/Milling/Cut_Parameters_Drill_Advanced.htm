Cut Parameters page: Advanced Drill

Use the Cut Parameters page to divide your hole into individual segments and customize the machining action for each segment. This is useful for spot drilling, deep hole drilling, back spot facing, and other advanced applications. Follow this general outline.

  1. The large grid displays all the segments plus the Retract move from the bottom of the hole. Click the + button to define new segments, and enter the Depth for each. The Depth for each segment is the distance from the top of the hole to the bottom of the segmentâ€”it is not the depth of the segment itself.

  2. Enter the other basic cutting parameters for the segment in the grid.

You can also right-click in the grid and use the context menu to add or remove segments, as well as access the standard cut, copy, paste commands.

If a solid hole operation is selected as the machining geometry, you can use the right-click menu to import and replace segments from the solid hole operation. This option imports and replaces any segments from the solid hole operation except for the retract.

Use the other controls on the page to apply additional features to each segment. The values and settings that are displayed apply only to the highlighted segment.

  • Select Tip Compensation to drill past the bottom of the segment and ensure that the entire length of the segment is machined with the full width of the tool. Enter the break-through distance in the Extend depth field or leave it at 0.0 to have Mastercam calculate it for you.
  • You can choose to drill the segment in pecks. Select Full Segment for conventional pecking, Chip Break to retract after each peck, or None for no pecking.
  • Use the Segment manual entry fields to output a comment or an additional block of NC code with the drill commands for that segment.

Modifications to the segment that are not output with the tool motion are listed in the Segment modification order grid. Click on an entry and use the Up or Down buttons to change the order in which they will be output.

You cannot associate any modifications with the Retract move.

NOTE

Unlike other drilling operations, this toolpath creates long-hand output only. Canned cycle definitions and custom drill parameters that have been defined in your post are not available. Mastercam outputs the tool movements as individual rapids and linear feed rate moves.

Add/Delete segment

Icon Description
Adds a new segment to the grid above the currently selected row.
Removes the selected segment from the grid.

Tip compensation

Select to drill past the bottom of the segment and ensure that the entire length of the segment is machined by the full width of the tool.

Extend depth

Determines how far the tool goes past the bottom of the segment. A positive value removes that amount from the depth, whereas a negative value adds that amount to the depth. A value of zero uses the segment depth specified, taking Tip compensation, if it is active, into account.

Type

Select the type of pecking action to apply to the segment.

  • None No pecking. Mastercam drills to the bottom of the segment in a single move.
  • Full Segment The segment is drilled in pecks.
  • Chip Break The segment is drilled in pecks and the drill retracts after each peck.
  • Top of Stock The drill retracts to the top of stock position after each peck. This is helpful when you need to move to the top of the stock rather than the top of the segment to clear the chips.

First peck

Sets the depth for the first peck move, which plunges in and out of the material to clear and break chips.

Nominal peck

Sets the desired depth for all peck moves after the first peck.

Minimum peck

Sets the minimum depth of a peck move. For example, if the First peck and Nominal peck are 0.2, and the depth of the segment is 0.5, the final peck will be 0.1. If the Minimum peck is greater than this, Mastercam will combine this with the previous peck and create one longer peck.

Retract amount

Sets the amount that the drill retracts every time it makes a peck move. Enter a positive value that is measured from the bottom of the peck. This option is available when Chip Break is selected as the peck Type.

Comment

The text of an optional comment that will be output in your NC file when the segment is drilled. Your post will apply the proper comment delimiters in your NC file.

Code

The text of an optional block that will be output in your NC file when the segment is drilled. No comment delimiters will be applied to this text, it will be output exactly as you type it in here as a stand-alone block.

Segment modification order

Modifications for this segment that are output as separate NC blocks are listed here. Use the buttons below the grid to change the order in which they are output.

Icon Description
Moves the selected modification up.
Moves the selected modification down.
