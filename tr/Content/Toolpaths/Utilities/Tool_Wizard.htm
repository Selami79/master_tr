Tool Wizard

The Tool Wizard is used to create and edit milling tools, including custom tools. Click any of the drop-downs below to learn more about creating and editing a tool.

NOTE

To create tools for turning, refer to either Tool Designer for Lathe or Defining and Editing Lathe Tools Help topics.

Creating and editing tools

Creating a tool from the classic Tool Manager

You must choose a Mill or Router machine definition and have geometry on which you will apply a toolpath before starting this procedure.

  1. Right-click in the tools list window in either the Tool Manager or any toolpath Tools page, and choose Create new tool.
  2. Select a pre-defined tool type which shape best fits the tool you are creating and click Next. This is the preferable method because it gives you a starting point for the tool profile that you can customize according to your needs.
  3. On the Define Tool Geometry page, enter the geometric values to define the tool shape.
  4. On the Define Tool Geometry page, you can use the buttons located above the graphics window to further refine your custom geometric profile.
  5. Use Back to select a different tool type, or Next to finalize any property values.
  6. Click Finish when complete. This action saves the new custom tool to your current database.

Once a custom tool is created from geometry by importing it from a different part file, that tool is linked to the referenced file with the geometry until you break the link. To break the link to the referenced part file, use Break link.

After breaking the link, the tool profile is stored in the current part file rather than linked to a referenced file. Breaking links to referenced files cannot be undone, and you receive a message to that effect.

Creating a tool from the stand-alone Tool Manager
  1. Open the stand-alone Tool Manager by choosing Tool Manager from your Start menu's Mastercam folder.
  2. Open an existing TOOLDB or create a new one.

  3. Select the Home tab, which displays both the New Milling and New Hole Making galleries.

  4. Double-click a pre-defined tool type which shape best fits the tool you are creating.
  5. On the Define Tool Geometry page, enter the geometric values to define the tool shape.
  6. On the Define Tool Geometry page, you can use Import and link to custom geometry from a file located above the graphics window to import a DXF file from which you can create the tool.
  7. Use Back to select a different tool type, or Next to finalize any property values.
  8. Click Finish when complete. This action saves the new custom tool to your current database.
Defining Tool Geometry page toolbar buttons
Icon Name and Function
Import custom geometry from a level: Choose to create the tool from a profile stored on a level.
Save profile geometry to a level: Choose to save the tool profile to a level.
Import custom geometry from a file: Choose to create the tool profile from a file. MCAM, DXF, and STEP are valid file types.
Reset profile geometry: Choose to discard any profile that the tool may have, and reset the tool to a Parametric format. This button becomes available only after either of the import buttons are chosen.

Editing existing tools

Editing a tool

Tools can be edited in the Tool Wizard by right-clicking on a tool in any of the toolpath dialog boxes and selecting Edit tool or by selecting the tool directly from the Toolpaths Manager.

From within the stand-alone Tool Manager, select a tool in the Tools document and edit the properties as needed in the Properties Grid.

Adding a tool to an assembly
  1. Click the tool in the Tools document. Tool Manager highlights the tool.

  2. Drag-and-drop the tool to the assembly item in the Assembly Tree.

Adding and editing tool manufacturers

You can add manufacturers to a library during tool creation. From the Tool Wizard's Finalize Properties page, click the plus button to the right of the Manufacturer name field to add a new entry to the list.

You can remove or edit an existing manufacturer. When you expand the list you will notice two buttons to the right of each name. The left button can be used to edit the name and the right button can be used to remove that item from the list.

Guidelines when creating tools

  • When creating an end mill tool, Mastercam does not compensate for the Edge break option under Tip/Corner treatment. Due to this, Mastercam interprets the tool to have a 90-degree sharp corner at the tip.

    • If you need to compensate to the tip shape on a tool, you may want to define the tool as a Bull mill, Chamfer mill, Face mill, or a different tool type.

  • When creating an Oval form tool, the Cutting length and Shoulder length must be the same value.

  • When creating a Center drill, the Cutting length is a read-only field. Changing the Shank diameter, Drill diameter, Drill length, or Shoulder angle dimensions will automatically update the Cutting length.

  • When creating a Custom tool, please note the following:

    • Depending on the tool type you are creating, the Define Tool Geometry page offers different editable settings.
    • You may have to adjust the Shoulder length to make it shorter than the tool projection length (from the front face of the part to the face of the spindle). This length is set as the Tool projection in either the Properties Grid of a selected tool assembly in the stand-alone Tool Manager or on the Holder page for any tree-style dialog box.
    • Shank diameter changes do not affect the toolpath. They are only used for collision avoidance purposes.
    • DXF file types must be in ISO 13399 format.
