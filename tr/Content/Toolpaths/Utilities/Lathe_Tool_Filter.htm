Lathe Tool Filter dialog box

Use the Lathe Tool Filter dialog box to specify which tools will display in the tool list area when you need to select a tool. Filtering is useful when you edit a tool library or search for a specific tool within a tool library. For example, if there were 200 tools in a specified library and you wanted to use a right-handed boring bar mounted in the top turret, you could set the appropriate filter options and reduce the number of tools in the tool list.

Filtering is automatic when you are in a toolpath dialog box. For example, if you are creating a drilling toolpath, only drilling tools display in the tool parameters tool list.

Parameters

Tool type

Click a button to select a type of tool:

  • General turning
  • Threading
  • Grooving / Parting
  • Boring Bar
  • Drill / Tap / Reamer
  • Custom

Operation masking

Use to display tools based on whether or not they have been used in an operation, or have no masking.

Unit masking

Use to only display U.S. or metric tools, or no unit masking.

All

Selects all tool types represented by the six Tool type buttons. Click individual Tool type buttons to deselect them.

None

Clears all of the tool type selections represented by the six Tool type buttons. Click individual Tool Type buttons to select them.

Right hand tools

Select to only display tools which cut from the right side.

Left hand tools

Select to only display tools which cut from the left side.

OD tools

Use to only display tools that can cut the outer diameter (OD) of the part.

OD tools

Use to only display tools that can cut the inner diameter (ID) of the part.

Tool material

Select specific tool materials to only display tools made from those materials.

  • All: Selects all tool materials.
  • None: Deselects all tool materials.
  • Copy job material: Each material definition includes a list of tool materials. The tool materials in this list are allowed to cut the material defined in the material definition. Click to filter the tool list based on the allowed tool materials from the stock material definition. To use this feature you need to have selected a stock material in the Tool Settings tab of the Machine Group Properties dialog box.

Top turret/Bottom turret

Select to only display tools that are loaded into the top turret or bottom turret (for VTLs, choose left or right turrets). Select both options to display tools from either turret.

IC diameter

Select this option and enter a value to only display those tools which meet the inscribed circle (IC) diameter threshold. The IC diameter is determined by placing a circle in the insert shape and measuring the circle's diameter.

Corner radius

Select this option and enter a value to only display those tools which meet this corner radius threshold.

Center tool

Specifies whether Mastercam displays Lathe centers in the tool list.
