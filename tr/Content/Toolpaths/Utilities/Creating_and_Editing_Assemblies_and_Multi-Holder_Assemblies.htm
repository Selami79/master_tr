Creating and Editing Assemblies and Multi-Holder Assemblies

Use the Assembly Tree to open, create, and edit assemblies in the open TOOLDB database. Tool Assemblies may contain one tool and one or more holder components. Use the Assembly Tree to navigate the individual assembly items. Any changes you make are automatically and immediately saved to the database.

Creating tooling assemblies must be done in the stand-alone Tool Manager.

Creating and editing assemblies
  1. Open an existing TOOLDB or create a new one.

  2. Click Assembly on the Home tab. Tool Manager opens the Assemblies document, and the Assembly Tree.

  3. Open the Tools and Holders documents by double-clicking those database items in the Database Explorer.
  4. To add assembly components:
    1. Drag and drop assembly components from the Tools and Holders documents into the Assembly Tree as needed.
    2. With more than one instance of Tool Manager open, drag and drop assembly components from another database to add the component to the assembly.
    3. Drag and drop assembly items from the Search Window to add assembly components.
    4. Use the buttons on the Home tab to open the Tool Wizard and Holder Wizard to create new tools and holders and add them to the current database for use in assemblies.
  5. To edit an assembly:
    1. Rearrange assembly holder components in the Assembly Tree by dragging and dropping a holder onto the desired parent item.
    2. Edit assembly properties in the Properties Grid, or click any of its components to edit their properties in the Properties Grid.
    3. Select an item and press [F2] to rename it.
    4. Dynamically edit the tool holder and extension positions along the length of the tool by holding down the [Ctrl] key while clicking and dragging the components in the graphics window.
  6. To delete an assembly or one of its components, right-click the tool in the Assembly Tree, and choose Delete. Alternatively, you can select the assembly/component, and press the [Delete] key.
Creating and editing composite holders (Multi-holder assemblies)

Creating Composite Holders

  1. Open an existing TOOLDB or create a new one.

  2. Click Assembly on the Home tab. Tool Manager opens the Assemblies document, and the Assembly Tree.

  3. Drag and drop assembly components from the Tools and Holders documents into the Assembly Tree as needed.

  4. To create a composite holder, place the holders in the Assembly Tree in the order that they will be assembled. Below is an example:

  5. Rearrange assembly holder components in the Assembly Tree by dragging and dropping a holder onto the desired parent item.

Editing Composite Holders

  1. Create a toolpath, or for an existing toolpath, click Parameters in the Toolpaths Manager.
  2. Select the Holder page.

  3. In the Holder page, right-click the composite holder you want to edit and choose Edit composite holder, then choose the holder/extension of the assembly that you want to edit.

  4. This opens the Holder Wizard where you can edit your selected holder/extension for the assembly.
Creating a new tool and adding it to an assembly
  1. Open Tool Manager from your Start menu's Mastercam folder.
  2. Choose File, Open.
  3. Choose the TOOLDB you want to open.
  4. The default location for tool database files is ...\<Shared Data Folder>\mill\Tools. However, you may have to navigate to a different location if you have customized your default location for tool database files.

  5. Click Assembly on the Home tab, then navigate and open the assembly in which you want to add the tool.

  6. Double-click Cutting Tools in the Database Explorer to open the Tools document.

  7. Select the type of tool you want to create in the Home tab. This opens the Tool wizard.
  8. Use the Tool Wizard to set the tool's parameters and save it to the database. You can always edit the tool properties after you create it through the Properties Grid.
  9. Select the new tool in the Tool Document, and drag and drop it into the appropriate position in the Assembly Tree.
NOTE
  • Putting multiple tools in an assembly is not allowed and, dragging another tool onto an assembly will replace the existing tool.
  • You cannot place a holder component after a tool component.
