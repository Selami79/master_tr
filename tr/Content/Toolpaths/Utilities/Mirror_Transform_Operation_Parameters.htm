Mirror tab: Transform Operation Parameters

Use the options in the Mirror tab to create a mirror image of a toolpath by reflecting it symmetrically with respect to a defined axis or point, including:

  • Horizontal axis of the specified Mirror plane or the source operation's tool plane
  • Vertical axis of the specified Mirror plane or the source operation's tool plane
  • Specified angle
  • Selected line
  • Theoretical line defined between two points
NOTE

The Mirror transform tab is unavailable for Wirepath transform operations.

Define a mirror axis by selecting a point, or entering a value.

You can choose to mirror the toolpath in a different plane.

Cutting direction settings control whether Mastercam applies the same cut direction, cutter compensation, start point, and selection order defined in the source operation to the mirror transformed operation.

You can also swap lead in/out parameters (including Adjust start/end of contour) to make the Lead in parameters for the original toolpath become the Lead out parameters for the resulting toolpath, and the Lead out parameters for the original toolpath become the Lead in parameters for the resulting toolpath.

Parameters

Y axis select point

Choose to select a reference point (X value) in the graphics window that defines the vertical axis used to mirror the selected operations.

X axis select point

Choose to select a reference point (Y value) in the graphics window that defines the horizontal axis used to mirror the selected operations.

Polar select point

Choose to select a reference point in the graphics window that, with the specified angle, defines the angular axis used to mirror the selected operations.

Select line

Choose to select a line in the graphics window. The line's end points define the axis used to mirror the selected operations.

Select two points

Choose to select two points in the graphics window that define the axis used to mirror the selected operations.

Mirror plane

Select to mirror the transformed toolpaths in a different plane than the source operation's tool plane.

Select plane button

Opens the Plane Selection dialog box, where you can select a plane.

X-Y-Z (From point)

Sets the coordinates of the reference point for the original toolpath. Click Select to choose a point from the graphics window.

X-Y-Z (To point)

Sets the coordinates of the reference point for the destination toolpath. Click Select to choose a point from the graphics window.

Transfer buttons

Transfers values from the one group of fields to the other.

Reset points

Resets all XYZ values in the From point and To point fields to 0.0.

Select buttons

Returns you to the graphics window to select the point or other geometry. Mastercam copies the coordinate positions into the proper dialog box fields where you can edit or override them, if necessary.

Reverse order

When this option is selected, Mastercam maintains the same start point for the mirror transform toolpath as the source toolpath. However, it reverses the order of the mirror transform toolpath's geometric entities, including chain direction, point order, and, for surface toolpaths, selection order. This also reverses cut direction and cutter compensation settings in the mirror transformed operation. If you select Reverse order, Mastercam chooses the Maintain start point option.

Deselect this option to apply the same cut direction and cutter compensation settings in the source toolpath to the mirror transformed operation. You can choose whether to maintain the same start point, or maintain the same start entity for the mirror transform toolpath as the source toolpath.

Maintain start point

Choose this option when you want the mirrored transform operation to use the same start point as the source operation. The cutting direction may differ, however, based on whether you also choose to reverse order. If you select Reverse order, Mastercam chooses the Maintain start point option.

Maintain start entity

Available only when Reverse order is deselected. Choose this option when you want the mirrored transform operation to use the same entity in the chained geometry as the source operation and the same cutting direction.
