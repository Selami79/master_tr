Parameters tab: Define Tool (Turning/Boring Bar)

Use this tab to set default values for different cutting parameters and to properly calculate cutter compensation and tool collision boundaries. When you select this tool while creating a toolpath, many of the values that you enter here automatically transfer to the parameters for the toolpath you are creating. Use the Machine Group Properties to enable or disable this feature. Any time you create a toolpath, you can override the values that you enter in this tab.

Parameters

Tool number

Lathe turning system tools are stored in a turret. Each position in the turret is described as a station. When a tool is placed in a turret station, it is given a one- or two-digit identifying number. The tool number identifies in which turret station the tool is placed.

Tool offset number

This number corresponds to a value that the programmer uses to define the offset value from the center of the turret to the programmed point on the tool.

Tool station number

Auxiliary number used to define the turret face into which the tool is loaded. If you select Station Number Equals Tool Number in Machine Group Properties, this number defaults to the default tool number. If you deselect this option, the number defaults to -1.

Tool back offset number

Defines a second offset number for finishing the groove walls. Mastercam creates one finish pass for each of the walls, and posts this number for the second wall finish pass.

Feed rate

You can choose to use this value as the default feed rate whenever the tool is selected for an operation as set in the Tool Settings tab of the Machine Group Properties. Enter the default feed rate value and choose the unit:

  • in/rev or mm/rev: Calculates the value in inches per revolution or millimeters per revolution.
  • in/min or mm/min: Calculates the value in inches per minute or millimeters per minute.

Plunge rate

You can choose to use this value as the default plunge rate whenever the tool is selected for an operation, as set in the Tool Settings tab of the Machine Group Properties. Enter a default plunge rate for the tool and choose the unit:

  • in/rev or mm/rev: Calculates the value in inches per revolution or millimeters per revolution.
  • in/min or mm/min: Calculates the value in inches per minute or millimeters per minute.

% of Material feed/rev

Determines what percentage of the base material feed/rev value will be used. Mastercam uses this value when you tell it to automatically calculate a feed rate from the selected stock material.

Spindle speed

Enter a default spindle speed for the tool. You can choose to use this value as the default whenever the tool is selected for an operation, as set in the Tool Settings tab of the Machine Group Properties. Enter a spindle speed value in constant surface speed (CSS) or revolutions per minute (RPM).

% of Material CS

Determines the percentage of the current material cutting surface speed to use for spindle speed and feed rate calculations. Mastercam uses this value when you tell it to automatically calculate a feed rate from the selected stock material.

Coolant

Click to select default coolant options for the tool. You can choose to use these settings as the default coolant options whenever the tool is selected for an operation, as set in the Tool Settings tab of the Machine Group Properties. Be careful when selecting coolant options that are machine dependent or that are not available in all of your machine definitions. When you select this tool for an operation, Mastercam will match the default coolant selected here to the coolant options available on the machine solely by its number. For example, if you select the 4th coolant option here as the default, when you select this tool for an operation, Mastercam will try to activate the 4th coolant option in the active machine definition, if it has been defined.

Compute from material

Sets the feeds and speeds to those set in the material definition. If no material has been selected in the Machine Group Properties for the active machine group, you will be prompted to select a material.

Amount of cut (rough)

Enter a default value to determine how much stock the tool will remove during each roughing pass. Mastercam uses this value as the default whenever the tool is selected for an operation if you select Use tools's step, peck coolant in the Tool Settings tab of the Machine Group Properties.

Amount of cut (finish)

Enter a default value to determine how much stock the tool will remove during each finishing pass. Mastercam uses this value as the default whenever the tool is selected for an operation if you select Use tools's step, peck coolant in the Tool Settings tab of the Machine Group Properties.

Overlap amount (rough)

Determines the distance the tool overlaps the previous cut before repositioning and making the next cut. When face roughing, this distance is perpendicular to the X axis. Mastercam uses this value as the default whenever the tool is selected for an operation if you select Use tool's step, peck coolant in the Tool Settings tab of the Machine Group Properties.

Retraction amount (face)

Determines the distance the tool moves away from the face of the part before it moves to the start of the next cut. Mastercam uses this value as the default whenever the tool is selected for an operation if you select Use tool's step, peck coolant in the Tool Settings tab of the Machine Group Properties.

X overcut amount (face)

A rectangle is used to define the material removed by the facing toolpath. The overcut amount determines how far past the rectangle the tool will cut, which is typically a small distance that the tool cuts past the part centerline. Mastercam uses this value as the default whenever the tool is selected for an operation if you select Use tool's step, peck coolant in the Tool Settings tab of the Machine Group Properties.

Compensation: Corner

Programs the toolpath from the corner of the tool corner radius.

Compensation: Center

Programs the toolpath from the center of the tool corner radius.

Compensation: Tip

Programs the toolpath from the tip of the tool corner radius.

Compensation: Edge

Programs the tool movements with respect to the front edge and bottom of the tool insert.

Tool Clearance

Select to set required clearance angles for the side and end of the tool.

Metric values

Select to interpret values you enter as metric values.

Tool name

Enter a name for the tool, using up to 75 characters. Lathe tools must have a name. If you do not enter one, a warning prompts you to enter a tool name.

Manufacturer's tool code

Enter a comment of up to 75 characters that is displayed in the setup sheet to be used by the operator. This comment is not reflected in the toolpath.

Requires tool locator

This option is only used by Mill-Turn, for tools that will be mounted in a tool spindle or B-axis head. Select this option for tools (such as stick tools) that require a tool locator to be properly mounted in a tool spindle. Leave this unselected for tools that do not require a tool locator, such as HSK or Capto tools.

Save to library

Saves the current tool and its settings to a tool library. You can select a different library if you wish. If you do not click this button, the tool definition or any changes you have made will only be saved to the copy of the tool stored in the machine group, and will be unavailable to other Mastercam files.

Draw tool

Displays the tool geometry in the graphics window.

NOTE

To save the tool geometry to a Mastercam file or to a level in the part file, use the Tool Manager. Right-click on the tool and choose the desired save option.

Setup tool

Opens the Lathe Tool Setup dialog box where you can choose the default mounting position, tool angle, spindle rotation direction, turret, active spindle and home position for programming the tool.
