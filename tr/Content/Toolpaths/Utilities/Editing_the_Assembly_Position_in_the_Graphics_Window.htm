Editing the Assembly Position in the Graphics Window

In addition to editing assemblies by changing settings in the Properties Grid, you can reposition the assembly (or its extensions) along the center line of the tool in the graphics window.

Changing the projection distance of the tool using the ruler
  1. Select the assembly you want to work with.

  2. Hold down the [Ctrl] key and click the holder or extension in the graphics window. The assembly is highlighted (glowing) and a dynamic ruler displays.

  3. The ruler always displays the dimensions in the absolute mode, which means that the tool projection value is always shown as the value measured from the tool tip to the face of the holder
  4. Once the ruler displays, move your mouse back and forth along the center line of the tool to change the length that the tool extends from the holder. You will see the dynamic ruler floating and the text box displaying the tool projection value changes with your mouse movement.
Changing the projection distance of the tool using the edit box
  1. Select the assembly you want to work with.
  2. Hold down the [Ctrl] key and click the holder or extension in the graphics window. The assembly is highlighted (glowing) and a dynamic ruler displays.

  3. To change the distance that the tool is projected from the end of the holder, type in the new value. The new value is displayed in the graphics window text box, the dynamic ruler repositions to reflect the projection value change, and the graphic representation of the extension relocates to its new position.

Changing the distance between holders and extensions
  1. Select the assembly you want to work with.
  2. Hold down the [Ctrl] key and click the holder or extension in the graphics window. The assembly component is highlighted (glowing) and a dynamic ruler displays.

  3. Once the ruler displays, move your mouse back and forth along the center line of the tool, or type in a new value to reposition the assembly component.

      

  4. If you are repositioning the assembly component that clamps the tool, it changes the tool projection distance. If you are repositioning a "parent" component of the assembly (a component that connects to another part of the assembly, not the tool), it changes the distance between that parent component of the assembly and its "child" assembly component, as shown above.
NOTE
  • When you are changing the Tool Projection value in the Graphics Window, notice that the Tool projection field in the Properties Grid reflects those changes. You can also use the Tool projection field in the Properties Grid to change those values.
  • When the dynamic ruler is active, you can change the ruler increments by zooming in or out by rolling your mouse wheel. This will reduce or increase the number of graduations on the ruler. This is helpful if you want to be more or less precise in your Tool Projection value. The number of graduations are appropriate to the selected units specified in the Options dialog box.
