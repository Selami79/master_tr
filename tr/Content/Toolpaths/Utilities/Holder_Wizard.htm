Holder Wizard

The Holder Wizard is used to create and edit holders. Click any of the drop-downs below for more information.

Creating a holder

The Holder Wizard has two pages. Follow these steps to create or edit a holder. As you make edits to the holder, your changes are automatically saved to the database.

  1. On the Define Holder Geometry page:
    • Set or edit the upper and lower connection types, and add segments as needed.
    • After the holder and/or segment parameters are set, you can edit the top, height, and bottom values in the properties grid.
    • Select the holder or segment in the graphics window to change focus to the associated datagrid line for that geometry, and edit the properties.

    • The upper connection type and size use the holder catalog database. If you specify an upper connection type and size that is not supported in the current database, you will receive an error message as shown below.

  2. Click Add Segment to add and set properties for an additional segment to the holder.
  3. Click Next to advance to the Finalize Properties page.
  4. On the Finalize Properties page, set the miscellaneous property values for the holder and click Finish when done.
Editing a holder

Holders can be edited in the Holder Wizard by right-clicking on a holder in any of the toolpath dialog boxes and selecting Edit tool.

For holders accessed from the Holders document of the stand-alone Tool Manager, click a section of the holder. The holder section becomes red, and that section's properties appear in the Property Grid, where you can edit the properties of the section holder .

Adding and editing holder manufacturers

You can add manufacturers to a library during holder creation. From the Holder Wizard's Finalize Properties page, click the plus button to the right of Manufacturer name to add a new entry to the list.

You can remove or edit an existing Manufacturer. When you expand the list you will notice two buttons to the right of each name. The left button can be used to edit the name and the right button can be used to remove that item from the list.

Defining Holder Geometry page graphics window buttons
Icon Name and Function
Import custom geometry from a level: Choose to create the holder from a profile stored on a level.
Save profile geometry to a level: Choose to save the holder profile to a level.
Import custom geometry from a file: Choose to create the holder profile from a file. Mcam, DXF, and STEP are valid file types.
Reset profile geometry: Choose to discard any profile that the holder may have, and reset the holder to a Parametric format. This button becomes available only after either of the import buttons are chosen.
Defining Holder Geometry page right-click menus

The right-click menu on the Define Holder Geometry page allows you to customize the datagrid by moving selected holder segments, inserting new ones, or removing them.

You can also right-click in the graphics window and change the view of the current holder.
