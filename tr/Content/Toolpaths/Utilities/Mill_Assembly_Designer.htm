Mill Assembly Designer panel

Use the Mill Assembly Designer function to create and edit mill tool assemblies within Mastercam. You can import tools and holders from a library and/or create components from scratch as you build your tool assembly. You can also add multiple holder components from different libraries into an assembly and set the projection for each holder component.

The Details pane displays information about the selected holders and tools.

How do I create or edit an assembly?

To begin creating an assembly, follow the procedure below.

  1. In the Tool Manager dialog box, right-click in the Part window or the Library window (in the Assembly tab), and select Create tool assembly.

  2. In the Mill Assembly Designer function panel, enter a Name for the new assembly.

  3. Right-click Add a holder and select one of the following:

    1. Define a holder: Opens the Define Holder wizard to create a new holder.

    2. Import from library: Opens the Browse for Item dialog box for you to select an existing holder. Use the Browse button to select a new .TOOLDB if necessary.

  4. Repeat step 3 for adding a tool.

  5. You can then rename components or delete components to select new ones.

    You can also add more holder components. Follow the steps in the How do I add another holder component to my assembly? section to learn more.

  6. Once you are satisfied with your tool assembly, click OK in the function panel to finalize the creation.

To edit an assembly, right-click on the assembly you wish to modify and select Edit tool assembly from the menu. The Mill Assembly Designer function panel opens and you can make the necessary edits.

How do I add another holder component to my assembly?

You can continue to add holder components to your assembly after adding the initial components. To do so, right-click on the location below where you want to insert the component and select Add component above. Select either Define a holder or Import from a library.

Define a holder opens the Define Holder wizard to create a new holder. Import from library opens the Browse for Item dialog box for you to select an existing holder. Use the Browse button to select a new .TOOLDB if necessary.

How do I edit the holder projection while in this function?

You have the ability to edit the projection of components at any time while creating an assembly. To modify the projection, do the following.

Select the component you want to modify in the graphics window. A ruler then displays next to the component.

Then manually modify the projection using your mouse to place the component at a new location or type in a value and press [Enter] twice.

When modifying the projection for an assembly with multiple holder components, modifying the projection of the lowest component in the designer modifies the projection of all components. For example, if you edit the projection of Holder 3 then Holder 2 and Holder 1 are also moved. If you edit the projection of Holder 2, then Holder 1 is moved but Holder 3 is not.

NOTE

You can save your created assemblies to other tool libraries. In the Tool Manager, right-click on the tool assembly and select Save assembly to library. For more information refer to the Save to Library panel Help topic.
