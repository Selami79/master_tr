Translate tab: Transform Operation Parameters

Use the Translate tab to tell Mastercam how to run an operation at one or more new locations when transforming a toolpath. The options that are available depend on the translate method you choose.

NOTE
  • The Pattern origin shift parameters (or Coordinates are world coordinates parameters if you are using the between-points method) offset the first copy relative to the source operation. You can select an origin and destination point.
  • Use the Sort point to point option to set the linking between each transform instance.
  • To translate the operation to a different plane, select To plane. Then click Select plane to pick a plane from the Plane Selection dialog box.
Parameters

Rectangular

Creates one or more copies of the toolpath at regular X and Y offsets. Use the Rectangular X and Y fields to enter the spacing between copies in each axis. In the Instances fields, enter the number of copies in each direction and define how to distribute the copies in the specified spacing. Select the Sort point to point checkbox to set the order for the transformed operations. You can position the resulting transform operations by entering world coordinates in the Pattern origin shift fields.

Between points

Copies the selected toolpath from one reference point to another. Enter the coordinates of the source and destination points in the From point and To point fields. Create multiple copies of the toolpath by entering a value in the Instances field.

Delta

Similar to the rectangular method. Delta translates the selected toolpath incrementally in a radial direction using delta XYZ coordinates instead of X and Y distances. Enter the coordinates directly in the XYZ fields, or enter a polar angle and distance to define them. Or define the coordinates by choosing a line, or a start and an end point in the graphics window. You can position the resulting transform operations by entering world coordinates in the Pattern origin shift fields.

Between planes

Translates the selected operations to a new plane without making more than one copy. Use the From plane and To plane sections to specify the source and destination plane. If you do not select a From plane, Mastercam uses the source operation's tool plane. Mastercam transforms all the operations according to the relationship between the selected plane (source and destination). For example, if the destination plane is rotated 90 degrees clockwise about the Y-axis from the source plane, Mastercam rotates the translated operations 90 degrees about the Y-axis, regardless of the actual plane the operations were created in or are translated to. You can position the resulting transform operations by entering world coordinates in the Pattern origin shift fields.

Sort point to point

Available only for Rectangular. Controls the order of the transformed operations. When selected, the resultant transform instances are output in a zigzag pattern. The XY spacing of the transformed instances determine the order of the zigzag pattern. Mastercam creates rows in the axis in which transformed instances have the shortest distance between them and connects the rows in the axis where the distance between instances is greatest.

When deselected, the resultant transform instances are output by rows and then columns so that the tool goes from the last instance in the current row to the first instance of the next row (in a manner similar to a manual typewriter).

Lathe users should note that the coordinates of the From and To points are in world coordinates. To translate a typical turning operation along the spindle, enter X coordinates instead of Z.

X

Sets the distance between repetitions of the operation in the X axis. The Rectangular options are unavailable for Lathe toolpaths because Lathe often requires a Z-axis translation. For Lathe toolpaths, use the Between points options instead, and select a From point and To point to position the translated toolpath.

Y

Sets the distance between repetitions of the operation in the Y axis.

3D

Determines whether the delta transformation is a 2D or 3D move.

  • Select to define the delta coordinates in all axes (XYZ).
  • Deselect to define the delta coordinates in the X and Y axes, or only in the Z axis. When deselected, entering a value in the X or Y field sets the Z field to 0.0. Entering a value in the Z field sets the X and Y fields to 0.0.

XYZ (Delta)

Defines a transformation vector using Cartesian (XYZ) coordinates that you enter for the transformed entities. The 3D checkbox must be selected to enter coordinates in XYZ. Otherwise, define the delta coordinates as a 2D move in the X and Y axes, or only in the Z axis. Specifying coordinates automatically calculates and displays values in the polar angle and distance fields.

Select from/to points

Defines a transformation vector using the distance and angle between two points that you select in the graphics window. Selecting points automatically calculates and displays values in the Delta XYZ and polar angle and distance fields, based on the start and end points. If you have not yet defined a transformation vector, choosing either button lets you select both points. If you have already defined a transformation vector using either the select points or select line method then the two buttons function as follows:

  • Select from point: lets you reposition the start or FROM point in the graphics window.
  • Select to point: lets you reposition the end or TO point in the graphics window.

Select Line

Allows you to choose a line in the graphics window and define a transformation vector using the distance and angle between the line's end points. Selecting a line automatically calculates and displays values in the Delta XYZ and polar angle and distance fields, based on the line end points.

Angle

Sets the angle for a delta translation. Used with the polar distance field to define the Delta XYZ coordinates. This field updates when the XYZ fields are changed directly, or when the Select line and Select from/to point options in this section are used to set the delta coordinates.

Distance

Sets the incremental translation distance for a delta translation. Used with the polar angle field to define the Delta XYZ coordinates. This field updates when the XYZ fields are changed directly, or when the Select line and Select from/to point options in this section are used to set the delta coordinates.

X (Instances)

Determines the number of times to repeat the operation along the X axis.

Y (Instances)

Determines the number of times to repeat the operation along the Y axis.

# (number of instances)

Type the number of copies to create from the original operations. This parameter is unavailable if you have chosen the Between planes transform method in the Translate tab (Transform Operation Parameters dialog box).

Distance between/Total distance

Available only when you choose a rectangular translation method. This field tells Mastercam how to use the distance defined in the Rectangular X and Y fields to evenly distribute the number of translated operations in the X and Y axes. Select one of the following options.

  • Distance between: For each axis (X, Y), sets the distance between each translated operation along the axis equal to the distance defined in the Rectangular field.
  • Total distance: For each axis (X, Y), evenly distributes the number of Instances of translated operations along the axis within the distance defined in the Rectangular field.

From plane

If you are transforming operations between planes, select this option to choose the source plane. Mastercam transforms all operations consistently based on the relationship between the From plane and the To plane. For example, if the To plane is rotated 90 degrees around the Y axis from the From plane, all the operations are transformed 90 degrees around the Y-axis. This way, if several source operations are in different planes, they retain their relationship with one another after they have been transformed.

Select the checkbox to activate the option, then click Select plane to choose the source plane from the Plane Selection dialog box.

To plane

Use this option if you want the locations of the transformed toolpaths to be calculated in a different plane. Select the checkbox to activate the option, and then click Select plane to choose the destination plane from the Plane Selection dialog box.

Select plane button

Opens the Plane Selection dialog box so you can select a plane.

From point

Sets the coordinates of the reference point for the original toolpath. Click Select to select a point from the graphics window.

To point

Sets the coordinates of the reference point for the destination toolpath. Click Select to select a point from the graphics window.

Copy buttons

Transfers values from the one group of fields to the other.

Reset points

Resets all XYZ values in the From point and To point fields to 0.0.

Select button

Returns you to the graphics window to select the point or other geometry. Mastercam copies the coordinate positions into the proper dialog box fields where you can edit or override them, if necessary.
