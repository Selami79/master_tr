Creating and Using Custom Lathe Tools

Use the Geometry tab of the Define Tool dialog box to define custom lathe tools. These are tools that cannot be defined using any other tool types. To do so, you must be able to select geometry which represents the profile of the custom tool. This geometry can be created and saved in one of the following file types:

  • A separate file
  • A separate level in the current Mastercam file

Once you have the insert and holder geometry saved to a file or to a specified level, you can begin to create the custom tool using the Define Tools dialog box.

Steps to create custom tool geometry

Before you begin the step-by-step procedure, please note the following:

Important Standards:

  • Make sure that the compensation point of the insert, which is the virtual sharp corner, is located at the origin: 0,0,0.
  • If the tool profile geometry is located on a level in the current Mastercam session, the insert and holder geometry must be the only geometry on that level.
  • The geometry must consist of two closed boundaries: one for the insert and one for the holder. There must only be those two contours.
  • Insert and holder lines should be different colors. Recommended colors are #14 for the insert up color, #138 for the insert down color, and #116 for the holder.
  • Custom lathe tool geometry cannot be created with splines. Only arcs and lines are supported.
  • Custom 3D lathe tools are not supported.

Procedure Steps:

  1. Open Mastercam. You do not need to load a machine definition because you are just working in Design, creating geometry.
  2. In the Top CPlane, loaded by default when in Design mode, create the geometry for the insert.
  3. Move the insert geometry so the tool nose radius is tangent to the X and Y axes.

  4. Create the geometry for the holder from lines and arcs. Your custom tool geometry should look similar to this:

  5. Create the tool geometry in a unique level in the part file, or save it as its own Mastercam part file. Create the tool on a unique level, so it does not interfere with other geometry. You can also save it as other CAD file types such as IGES and DXF.
  6. After creating the custom tool profile geometry, you can use the following procedures to use and further define the parameters of that tool in a toolpath.
Setting tool parameters for the custom tool in a toolpath

For this example, we create a Rough toolpath on a line.

  1. Load a Lathe machine type.
  2. Create a line on which you will create the lathe Rough toolpath.
  3. Choose Rough toolpath, chain the geometry, and click OK. This opens the Toolpath Parameters tab of the Lathe Rough Properties dialog box.

  4. Right-click in the tools list and choose Create new tool to open the Define Tool dialog box.

  5. In the Type tab, click Custom. This opens the Geometry tab, where you can choose and define the parameters for your custom tool.

  6. In the Tool geometry section, choose either File or Level, depending on how/where you saved the custom tool geometry that you created.

  7. In the Corner radius field, enter the radius of the insert of your custom tool, or click Select to pick the radius from your geometry.

  8. Use the radio buttons in the group boxes to set the tool orientation in the Top Plane and the Turret.
  9. Click Setup tool to open the Lathe Tool Setup dialog box. Use the settings to establish the physical orientation and positioning of a tool, which are part of the tool definition.
  10. Set final parameters for your custom tool in the Tool Type and Parameters tabs of the Define Tool dialog box. Unless you save the tool definition to a tool library by selecting Save to library when you exit this dialog, they will apply to the current part file only.
  11. Click OK to accept your tool definition and continue to set up your Rough toolpath.
NOTE
  • If you are defining a custom tool that is mounted at an angle in the turret, the tool geometry should represent the tool mounted at 0 degrees. You can define the tool angle in the Lathe Tool Setup dialog box by choosing Setup tool.
  • When defining a custom boring bar, you should use orientation 4, not 8.
