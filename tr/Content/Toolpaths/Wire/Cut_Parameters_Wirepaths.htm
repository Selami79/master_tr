Cut Parameters page: Wirepaths

Use the Cut Parameters page to:

  • Define the type and number of cuts you will make in the wirepath.
  • Optionally divide the cuts into separate operations.
  • Suppress thread or cut flags.
  • Control stop codes.

Mastercam supports three main categories of cuts: rough and skim cuts before the tab, tab cuts, and skim cuts after the tab. The Cut Parameters page lists the programmed cuts and indicates their grouping as operations.

Contour Parameters

Perform rough cut

Select to create a rough cut for the contour. The rough cut uses the Start Pass # from the Wire power settings library. Clear this checkbox in operations where you want only a tab or skim cut.

Additional skim cuts (before tab)

Enter the number of passes before cutting the tab. These passes use the Contour lead in/out settings.

Tab width

Enter the width of the tab.

Number of tab cuts

Enter the number of passes the wire makes to cut the tab. To use the power settings for the rough cut, Pass 1, select Reset pass number on tab cuts.

Make tab cutoff move with skim cut

Select to combine the tab cut with the last skim cut. If you have no skim cut before the tab, the tab cut combines with the rough cut. Clear to cut tabs separately.

Automatic

Mastercam Wire places the number of tabs you specify on each part in the operation. The tabs are spaced equally apart.

Manual

Lets you choose the locations for the tabs. Choose Manual then Position to return to the graphics window to place tabs on the part.

Use square points

Requires that you place a square-style point on the part where you want each tab to be positioned. Mastercam automatically places a tab at each location marked by a square point. You need to create the points before choosing this option. Select the location of the point in relation to the tab.

Start

The square point marks the start of each tab.

Midpoint

The square point marks the midpoint of each tab.

End

The square point marks the end of each tab.

Skim cuts after tab

Enter the number of passes the wire makes after cutting the tab, also known as finish passes. Skim cuts after a tab cut use the Finish lead in/out settings.

Together

Enter the number of skim cuts on a contour before moving to the next contour in the operation.

Separate

Enter number of skim cuts on all contours, one pass at a time per contour. Activated only when the operation contains more than one contour.

One way

Programs multiple passes in the same direction. To cut more than one pass using the one way method, the system must cut and re-thread the wire after each pass, unless the contour is closed and you are not using tabs.

Reverse

Programs multiple passes to be bi-directional. Each pass reverses the direction of the previous pass at the end of the contour or at the tab.

Reset pass number of tab cuts

Select to make the tab cut use the same power setting as the first rough cut. Mastercam uses the starting pass power setting for the rough cut and then increments the power pass number for each additional skim cut before tab. Deselect the checkbox to use the next power pass number for the tab cut. This field is disabled if there are no tab cuts or if there is only 1 tab cut and Make tab cutoff move with skim cut is checked.

Expand operation

Select to separate passes into individual operations. Passes are separated into operations by type: rough cuts for a contour become one operation, tab cuts for the same contour become another operation, and the cuts after tab (skim cuts) become one operation. Allows parameters such as tab width to be modified for each contour in a multi-contour operation. Use with All cuts together to keep operations (rough cut, tab cut, skim cut) for a single contour one after another in the Toolpaths Manager. Available only when first creating an operation, but not when editing an operation.

Suppress all wire threads

If selected, no thread flags will be written to the wirepath's NCI file.

Suppress all wire cuts

If selected, no cut flags will be written to the wirepath's NCI file.

Linearization tolerance

Used when converting 3D arcs and 2D or 3D splines in the chained geometry from curves to lines. Smaller linearization tolerance values make more accurate wirepaths, but may take longer to generate and create a longer NC program.

Total cuts for each chain

Display only; shows the sum of rough, tab, and skim cuts.

4 Axis Parameters

Perform rough cut

Select to create a rough cut for the contour. The rough cut uses the Start Pass # from the Wire power settings library. Clear this checkbox in operations where you want only a tab or skim cut.

Additional skim cuts (before tab)

Enter the number of passes before cutting the tab. These passes use the Contour lead in/out settings.

Equal

Sets the UV and XY tab widths equally. When this option is chosen, the XY tab width field is active, but the UV tab width field is not.

Proportional

Calculates the UV Tab Width value by multiplying the length of the final entity on the UV plane by the ratio of the XY tab width value to the length of the final entity on the XY plane. When this option is chosen, the XY Tab Width field is active, but the UV Tab Width field is not.

Independent

Allows you to set the tab width on both the XY and UV planes. Choosing this option activates the XY Tab Width and UV Tab Width fields, where you can enter the distance that the tab will extend along the appropriate edge of the contour.

XY tab width

Enter the distance that the tab will extend along the XY edge of the contour (the width of the tab). If you choose either the Equal or Proportional options, Mastercam calculates the UV tab width in relation to the value you enter here.

UV tab width

Enter the distance that the tab will extend along the UV edge of the contour (the width of the tab). The UV Tab Width field is only available for editing when you select Independent.

Number of tab cuts

Enter the number of passes the wire makes to cut the tab. To use the power settings for the rough cut, Pass 1, select Reset pass number on tab cuts.

Make tab cutoff move with skim cut

Select to combine the tab cut with the last skim cut. If you have no skim cut before the tab, the tab cut combines with the rough cut. Clear to cut tabs separately.

Skim cuts after tab

Enter the number of passes the wire makes after cutting the tab, also known as finish passes. Skim cuts after a tab cut use the Finish lead in/out settings.

One Way

Programs multiple passes in the same direction. To cut more than one pass using the one way method, the system must cut and re-thread the wire after each pass, unless the contour is closed and you are not using tabs.

Reverse

Programs multiple passes to be bi-directional. Each pass reverses the direction of the previous pass at the end of the contour or at the tab.

Rest pass number of tab cuts

Select to make the tab cut use the same power setting as the first rough cut. Mastercam Wire uses the starting pass power setting for the rough cut and then increments the power pass number for each additional skim cut before tab. Deselect the checkbox to use the next power pass number for the tab cut.

Suppress all wire threads

If selected, no thread flags will be written to the wirepath's NCI file.

Suppress all wire cuts

If selected, no cut flags will be written to the wirepath's NCI file.

Total cuts for each chain

Read-only. shows the sum of rough, tab, and skim cuts.
