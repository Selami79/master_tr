Edit Wirepath dialog box

Use the Edit Wirepath dialog box to modify an existing wirepath. You can also view information associated with the selected wirepath point position, including the G-code, canned text settings, feed rate, and endpoint coordinates.

Listed below are procedures on deleting or adding to a wirepath using the Edit Wirepath dialog box, accessed by using the right-click menu in the Toolpaths Manager. Select the drop-downs below to view a procedure.

Be sure to make all other changes to the wirepath, such as wire or parameter changes, before performing this procedure. Because these edits are written directly to the NCI file, not to the geometry, Mastercam locks the wirepath and prevents regeneration.

Deleting a pass from a wirepath
  1. Using Pass, select the pass that you want to delete. The selected pass is highlighted in the graphics window.
  2. Click Delete pass.
  3. Choose OK to return to the graphics window.
Deleting a point from a wirepath
  1. Using Point, select the point that you want to delete. The selected point is highlighted in red.
  2. Click Delete point.
  3. Choose OK to return to the graphics window.
Deleting a section from a wirepath
  1. Use Pointto select the start point of the section you want to delete.
  2. Click Delete Section. The Section dialog box opens.

  3. Scroll through the endpoints to select the section that you want to delete. The selected section is highlighted in the graphics window.

    NOTE

    The section begins with the point position value you defined in the Edit Wirepath dialog box, and ends with the point position you enter here.

  4. Click OK. The Select dialog box closes, the Edit Wirepath dialog box opens, and the section you chose is deleted.
  5. Choose OK to close the Edit Wirepath dialog box and return to the graphics window.
Adding a point to a wirepath
  1. Using Point, select a point in the wirepath close to the position where you want to add a point. The selected point is highlighted in red.

  2. Select or deselect Before to place the new point before or after the position you designated.

    NOTE

    The before/after setting is relative to the direction of the chain.

  1. Click Add point. The Edit Wirepath dialog box minimizes.
  2. Click in the graphics window or use the AutoCursor to position the new point. The wirepath redraws showing the new point and the Edit Wirepath dialog box reappears.
  3. Choose OK to return to the graphics window.
Moving a point in a wirepath
  1. Using Point, select a point in the wirepath close to the position where you want to add a point. The selected point is highlighted in red.
  2. Click Move point. The Edit Wirepath dialog box minimizes.
  3. Click in the graphics window or use the AutoCursor to reposition the point. The wirepath redraws showing the repositioned point and the Edit Wirepath dialog box reappears.
  4. Choose OK to return to the graphics window.
Editing a point in a wirepath
  1. Select a point in the wirepath that you want to change using Point. The selected point is highlighted in red.
  2. Choose Edit Point. The Edit Point dialog box opens and the wire move prior to the selected point is highlighted in red.
  3. Make changes to the selected point in the dialog box.
  4. Choose OK to return to the Edit Wirepath dialog box.
  5. Choose OK to return to the graphics window.
Change control flags in a wirepath
  1. Select a point in the wirepath that you want to change using the Pass and Point fields. The selected point is highlighted in red.
  2. Choose Edit Point. The Edit Point dialog box opens and the wire move prior to the selected point is highlighted in red.
  3. Click Control flags and then the Control flags button.
  4. In the Control Flags dialog box, check the box next to the setting and set the related values.
  5. Click OK to accept changes and return to the Edit Point dialog box.
  6. Choose OK to return to the Edit Wirepath dialog box.
  7. Choose OK to return to the Toolpaths Manager.
Change control settings in a wirepath
  1. Select a point in the wirepath that you want to change using the Pass and Point fields. The selected point is highlighted in red.
  2. Choose Edit point. The Edit Point dialog box opens and the wire move prior to the selected point is highlighted in red.
  3. Click Control settings and then the Control settings button.
  4. In the Control Settings dialog box, check the box next to the setting and enter a value.
  5. Click OK to accept changes and return to the Edit Point dialog box.
  6. Choose OK to return to the Edit Wirepath dialog box.
  7. Choose OK to return to the graphics window.
Parameters

Move point

Moves the point set in the Point parameter to a location you choose by selecting in the graphics window.

Edit point

Opens the Edit Point dialog box from which you can change the attributes for the selected point.

Add point

Adds a new point in the wirepath, before or after the selected point. The position of the new point is determined by the Before parameter.

Delete point

Deletes the selected point from the wirepath.

Delete section

Opens the Section dialog box where you set the endpoint of the section to delete from the wirepath.

Delete pass

Deletes the selected position pass from the wirepath.

Before

Defines where to position new points in the wirepath when you click Add Point.

Point

Defines the wirepath point position (XYZ coordinate) for editing and highlights it in the graphics window.

Pass

Available only for wirepaths that contain multi-passes. Defines the wirepath pass for editing and highlights it in the graphics window.

G-Code

Read-only. Displays the G-code associated with the move.

Canned text

Read-only. Displays the current Canned Text setting.

Feed rate

Read-only. Displays the current feed rate setting.

X / Y

Read-only. Displays the endpoint coordinates for both circle and linear moves. For G2 and G3 moves, center point coordinates for the circle interpolation also display.

Show UV

Displays the UV contour (upper contour) in addition to the XY contour (lower contour).

Pass

Select to show only the specified pass of the wirepath. If deselected, the entire wirepath displays and the selected pass is highlighted.

All

Shows all passes in the wirepath.
