Edit Point dialog box: Wire

Use the Edit Point dialog box to change the point parameters of a wirepath. This gives you control over the motion in a wirepath on a point-by-point basis. For example, you can use this dialog box to add taper to a single point or a series of points, assign control flags for machine events such as flush or power, or change corner type.

Rapid move is unavailable for arcs and Feed rate is disabled for rapid moves.

Parameters

Taper

Check to enable the field where you can enter a value to change the angle of the taper.

Gradual

Applies the taper angle over the length of the wirepath segment.

Immediate

Applies the taper angle immediately.

Left

Tilts the taper to the left.

Right

Tilts the taper to the right.

Corner type

Check to enable the drop-down list where you can choose the shape to create when encountering a sharp corner in the plane that contains the geometry. Enter a radius for the corner types Other, Fixed, or Fishtail.

Apply to end of pass

Check to apply Corner Type or Arc Type settings to entire pass.

Arc type

Check to enable the drop-down list where you can choose the shape to create when encountering an arc in the plane that contains the geometry. Enter a radius for the corner types Other or Fixed. The Arc Type section is disabled unless the point position in the Edit Wirepath dialog box is set to a point on an arc.

Radius

Enter the radius for the Arc type or Corner type.

Wire compensation

Check to enable the drop-down list where you can choose a new setting for the wire compensation in control. Left or Right offsets the wire to the left or right of the chained entities. Cancel turns off compensation. This setting stays in effect until you make another change in wire compensation.

Rapid move

Check to create a rapid move at the top speed of the machine. Rapid move is unavailable for arcs.

Dwell

Check to enable the adjacent dwell time field where you can specify a number of seconds to stop the wire.

Feed rate

Check to enable the adjacent feed rate field where you can specify a speed at which the wire cuts along the wirepath.

Manual entry

Select a checkbox to activate the field to enter text to be inserted either before, with, or after the selected point.

Canned Text

Check to enable Canned text, which you click to open the Canned Text dialog box. In that dialog box, you can assign up to 20 variables ranging from 1 to 99 to the selected point. The post processor uses these variables to control machine-specific functions.

Control Settings

Check to enable Control settings, which you click to open the Control Settings dialog box. In that dialog box, you can generate a G1010 code (wire condition change) that overrides auxiliary register settings entered in the Aux. Registers page. Control settings stay in effect for the remainder of the wirepath until a new control setting is encountered.

Control Flags

Check to enable Control flags, which you click to open the Control Flags dialog box. You can use that dialog box to output a control flag at the selected point to set machine events such as stops, glue stops, threading the wire, cutting the wire, or flush.
