Change at Point dialog box: Wire

Use the Change at Point dialog box to make associative changes to a contour wirepath. You can assign taper, corner types, and wire parameters to entity endpoints on the geometry. Change at Point is available only on contour wirepaths.

NOTE
  • Changes made with Change at Point remain in effect if you change the geometry. However, Mastercam Wire erases the point changes if you re-chain the geometry.
  • When selecting entity endpoints, the endpoints of two adjacent entities may lie at the same location and appear to be the same point in the graphics window. However, when using wire compensation to the outside of the geometry, these two endpoints actually correspond to different locations on the contour because the wirepath has corner arcs.

Use the following procedures to make changes using Change At Point:

Assign canned text to a point
  1. In the Toolpaths Manager, select Geometryfor the contour wirepath.
  2. In the Chain Manager, highlight the chain to edit, right-click, and choose Change at point.
  3. When prompted, click the point to change on the contour wirepath.
  4. In the Change At Point dialog box, select Canned text, and then select the Canned text button. The current canned text settings display in the Canned text dialog box.
  5. To assign canned text to a point, select a canned text variable from the list, choose to add it before, with, or after the selected point, and then choose Add.
  6. Click OK to close the Canned text dialog box.
  7. Make additional changes to the selected point, as necessary, and then choose OK to close the Change At Point dialog box.
  8. To change other points, return to step 3. Otherwise Press [Esc], and then click OK to close the Chain Manager.
  9. Choose Regenerate all selected operations in the Toolpaths Manager to regenerate the contour operation and apply point changes.
Change control flags
  1. In the Toolpaths Manager, select Geometry for the contour wirepath.
  2. In the Chain Manager, highlight the chain to edit, right-click, and choose Change at point.
  3. When prompted, click the point to change on the contour wirepath.
  4. In the Change At Point dialog box, select Control flags, and then choose the Control flags button. The current control flags are shown in the Control Flags dialog box.
  5. To change a control flag, check the box next to the setting and set the related values.
  6. Choose OK to close the Control Flags dialog box.
  7. Choose OK to close the Change At Point dialog box.
  8. Choose Regenerate all selected operations in the Toolpaths Manager to regenerate the contour operation and apply point changes.
Change control settings
  1. In the Toolpaths Manager, click Geometry for the contour wirepath.
  2. In the Chain Manager, highlight the chain to edit, right-click, and choose Change at point.
  3. When prompted, click the point to change on the contour wirepath.
  4. In the Change At Point dialog box, select Control settings , and then choose the Control settings button. The current control settings are shown in the Control Settings dialog box.
  5. To change a control setting, check the box next to the setting and enter a value. Select All passes to apply the control setting to the selected point for the entire wirepath.
  6. Choose OK to close the Control Settings dialog box.
  7. Choose OK to close the Change At Point dialog box.
  8. Choose Regenerate all selected operations in the Toolpaths Manager to regenerate the contour operation and apply point changes.
Make associative parameter changes to a point in a wirepath
  1. In the Toolpaths Manager, click Geometry for the contour wirepath.
  2. In the Chain Manager, highlight the chain you want to edit and right-click.
  3. Choose Change at point from the right-click menu.
  4. When prompted, click the point to change on the contour wirepath.
  5. Make parameter changes as needed, and click OK. The Change at Point dialog box minimizes and you are prompted to select another point.
  6. Press [Esc] when done, and choose OK to close the Change At Point dialog box.
  7. Choose OK to close the Chain Manager.
  8. Choose Regenerate all selected operations in the Toolpaths Manager to regenerate the contour operation and apply point changes.
Parameters

Taper

Changes the taper angle by the degrees specified. Choose to have the taper take place immediately or gradually over the length of the segment, and whether the taper will tilt left or right.

Gradual

Applies the taper angle over the wirepath segment. Enter the angle of taper in degrees in the fields to the right.

Immediate

Applies the taper angle immediately.

Restore

Returns taper to original programmed settings.

Left

Tilts the taper to the left.

Right

Tilts the taper to the right.

Cancel after pass

Returns the taper angle to its original programmed value after the pass number specified in the field to the right.

Corner type

The shape to create in the UV contour when encountering a sharp corner in the XY contour. Choose the corner shape for the UV contour in the field to the right. Other allows you to specify a radius for a custom corner defined in the post processor. Mastercam draws the corner as a straight line since it can not anticipate what type of corner the post processor will specify.

Arc type

Choose the shape to create in the contour when encountering an arc corner in the plane that contains the geometry. In the field to the right, choose the shape to create in the contour when encountering an arc corner in the plane that contains the geometry. Other allows you to specify a radius for a custom corner defined in the post processor. Mastercam draws the corner as a straight line since it can not anticipate what type of corner the post processor will specify.

Radius

Enter the radius.

Pass

Enter the pass number to apply the parameter changes to the selected point.

Wire compensation

Choose a new setting for the wire compensation in control. Left and Right offset the wire to the left or right of the chained entities. Cancel turns off compensation.

All passes

Select to apply the parameter change to the selected point for all passes in the wirepath.

Rapid move

Creates a non-cutting move at the top speed of the machine. Rapid move is unavailable for arcs.

Dwell

Wire stops for the number of seconds specified. In the field to the right, specify the number of seconds to dwell.

Feed rate

Change the speed at which the wire cuts along the wirepath. In the field to the right, change the speed at which the wire cuts along the wirepath. Select Restore to return the feed rate to the original programmed speed.

Manual entry

Enter text to be inserted before, with, or after the selected point. In the field to the right, enter text to be inserted. The radio buttons associated with the Manual Entry option places the manual entry text before, with, or after the wire move to the selected point.

Canned text

Assigns a variable ranging from 1 - 99 to the selected point. These variables are used by the post processor to control machine-specific functions. Select the checkbox, then click the button to open the Canned Text dialog box where you can insert canned text commands in your NC program.

Control settings

Activates the Control settings button, which opens the Control Settings dialog box where you can change the wirepath control settings on a point-by-point basis. These changes override auxiliary register settings entered in the Aux. Registers page. Control settings persist for the remainder of the wirepath unless a new control setting is encountered.

Control flags

Activates the Control flags button, which opens the Control Flags dialog box where you can assign control flags to a point in a wirepath that triggers machine events such as stops, glue stops, threading the wire, cutting the wire, or setting the water flush.
