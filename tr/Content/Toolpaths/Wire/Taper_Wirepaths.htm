Taper page: Wirepaths

Use this page to taper the contour wirepath inward or outward and specify the location of the land, the point at which the taper begins. You can also select how to apply the taper to passes and specify the tilt direction of the taper to the left or right of wirepath.

You can also use this page to set the following taper heights:

  • Z height of the upper wire guide for rapid moves
  • Location of the upper wire guide (UV Trim Plane) and the lower wire guide (XY Trim Plane)
  • Height of the part (the upper contour of the wirepath)
  • Plane at which the wire pivots to the taper angle (available only for land up or land down taper styles)

  • Location of the XY plane (the lower contour of the wirepath)

Parameters

Contour taper style

Choose a contour taper style by selecting the radio button below that represents the type of part you are creating. Contour wirepaths can taper inward or outward, and you can specify the location of the land, the point at which the taper begins.

Initial taper

Sets the taper angle in degrees. Taper angle remains in effect for the entire wirepath unless changed at specific points using Change at Point for Contour wirepaths only, canceled after a specific pass, or applied after a specific pass.

Pass settings

Taper all passes applies the taper angle to the entire wirepath. Cancel taper after pass returns the wire to upright after cutting a taper. Enter the pass number after which to return the wire to upright. Apply taper after pass leaves the wire upright until the specified pass then tilts the wire to cut the taper for the remaining passes in the operation.

Pass number

Enter the pass number after which to cancel or start applying the taper angle.

Reset pass number

Select to use the power settings stored as Pass 1 in the wire power settings library for the first tab cut. Mastercam Wire uses the Pass 1 settings for the rough cut. Additional tab cuts use Pass 2 settings, Pass 3 settings, and so on. An operation does not necessarily have to start at pass 1. Clear the checkbox to use the next pass number in the library.

Taper direction

Tilts taper to the left or right of the wirepath. Compensation direction determines whether the taper tilts toward or away from the center of the contour. Make certain to check the compensation direction as you set the taper tilt.

Chain height

Select one of the options to indicate where the geometry was chained, which is the XY or UV height or, if there is a land, at the land height. Select from the following choices:

  • Top: Geometry was chained at the UV height.
  • Middle: Geometry was chained at the land height. Only available when Land style is selected.
  • Bottom: Geometry was chained at the XY height.

Rapid height

Z depth of a rapid move. Can be higher than the UV height, UV trim, or UV extension planes in order to clear any clamps.

  • Absolute: Locates the plane to a Z depth relative to the system origin.
  • Incremental: Locates the plane relative to the Z depth of the chained geometry.

UV trim plane

Location of the upper guide on the wire machine that may be required by the control to locate the guide in relation to the part. Trim plane values are written to G-code 1015 in the NCI file. When not using the UV trim plane, enter the same value as the UV height.

  • Absolute: Locates the plane to a Z depth relative to the system origin.
  • Incremental: Locates the plane relative to the Z depth of the chained geometry.

UV height

Location of the XY plane that contains the geometry for the upper contour of a 4-axis part. When using incremental values, the UV height is relative to the Z depth of the XY plane of the chained geometry. When absolute, UV height is relative to the system origin.

Land height

When using taper, the plane at which the wire pivots to the taper angle. Available only when Land up or Land down is selected for Contour taper style.

XY height

Location of the XY plane which usually contains the geometry for the lower contour of a 4-axis part. When using incremental values, equals the Z depth of the XY plane of the chained geometry. When absolute, XY height is relative to the system origin.

XY trim plane

Location of the lower guide on the wire machine that may be required by the control to locate the guide in relation to the part. Trim plane values are written to G-code 1015 in the NC file. When not using the XY trim plane, enter the same value as the XY height.

  • Absolute: Locates the plane to a Z depth relative to the system origin.
  • Incremental: Locates the plane relative to the Z depth of the chained geometry.
