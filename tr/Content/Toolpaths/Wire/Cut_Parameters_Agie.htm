Cut Parameters page: Agie

Use this property page to establish the punch or die cut parameters, including the cut separation and clearance, the minimum radius for the last cut, and the option for a reverse cut. For 4-axis wirepaths, you can also set the wirepath 4-axis output format, sync option, linearization tolerance, and the step size.

If you have chosen to synchronize the contours, you can also set the linearization tolerance, used when converting 3D arcs and 2D or 3D splines in the chained geometry from curves to lines. If you have chosen not to synchronize the upper and lower contours, Mastercam Wire places points on the wirepath so the wire can follow the chain. If sync is set to By branch and there are no branch points in the 4-axis geometry, Mastercam Wire will not create the 4-axis wirepath. Use the Step size field to specify how far apart the points are placed.

For detailed, machine control-specific information, please refer to your Agievision manufacturer's documentation.

Parameters

4 Axis taper

4 Axis wirepath only. Outputs only UV/XY linear moves. All circular moves are broken into linear moves based on the linearization tolerance value. The format may be modified by customizing your post processor.

Direct 4 Axis

4 Axis wirepath only. This format outputs both XY/UV linear and arc moves. The format may be modified by customizing your post processor.

Sync option

4 Axis wirepath only. Determines how Mastercam divides the chained contours and places synch points between the lower (XY) and upper (UV) contours. For many cases, the By branch option is an effective first choice. Branch, entity, node, and point use the linearization tolerance to calculate the wirepath between sync points. Entity, node, and point require the chains to have an equal number of items so there is a one-to-one correspondence between the chains. The following are synchronization choices:

  • None: Synchronizes the chains by dividing them into an even number of segments using the Step size.
  • By branch: Requires branch lines to be added to the geometry to create sync points as shown in the example above.
  • By entity : Matches the endpoints of each entity and requires both chains to have an equal number of entities.
  • By node: Applies only to parametric splines and synchronizes the two chains by nodes on the splines.
  • By point: Synchronizes the chains by point entities found along the chain.

Step size

4 Axis wirepath only. When creating a 4-axis wirepath with the sync option set to None, Mastercam Wire places points on the wirepath so the wire can follow the chain. The value you enter for Step size determines how far apart these points are placed. We recommend that you keep this value small to retain accuracy in the corners.

Linearization tolerance

Used when converting 3D arcs and 2D or 3D splines in the chained geometry from curves to lines. Smaller linearization tolerance values make more accurate toolpaths, but may take longer to generate and create a longer NC program. In 4-axis wirepaths, linearization tolerance provides a factor that Mastercam Wire uses to place sync points along chains when using the by branch, by entity, by node, and by point sync option.

Punch / Die

Choose Punch or Die to apply the cut parameters to closed chains. When you choose Punch, the offsets are applied to the outside of the chain. When you choose Die, the offsets are applied to the inside of the chain.

Left / Right

Choose Left or Right to apply the cut parameters to either the left or the right side of open chains.

Minimum radius last work

Check to apply the minimum Agievision radius to the last cut.

Optimize

Eliminates arcs in the wirepath that are less than or equal to the radius of the wire.

Infinite look ahead

Checks for wirepath self-intersections along the entire contour before creating the wirepath. If Mastercam finds a wirepath intersection, it modifies the wirepath so that it does not cut the portion of the part that comes after the intersection. This checkbox is not available if Compensation Type is set to Off, or if it is set to Control and Optimize is not checked.

Reverse cut

Check to apply a reverse cut strategy.

Expand operation

Select to separate passes into individual operations. Passes are separated into operations by type: rough cuts for a contour become one operation, tab cuts for the same contour become another operation, and the cuts after tab (skim cuts) become one operation. Allows parameters such as tab width to be modified for each contour in a multi-contour operation. This checkbox will only appear in AgieVision’s contour and collar operations and only be enabled if there are multiple chains in a single operation.

Separation cut

Enter the separation cut length. This field is unavailable for Agie No Core rough cuts.

Separation cut clearance

Enter a value to define the clearance increment of the separation cut. This field is unavailable for Agie No Core rough cuts.

Clearance

Enter the clearance value between the punch and die chains. The clearance value is added to the offset. This field is unavailable for Agie No Core rough cuts.

Working st. wo. Inversion

Enabled when Reverse Cut is selected.
