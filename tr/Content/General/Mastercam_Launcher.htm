Mastercam Launcher dialog box

Use the Mastercam Launcher dialog box to control the start-up options for a particular seat (license) of Mastercam. You can set the following start-up options:

  • Assign product (platform) levels and availability, such as Mill Entry or Router.
  • Edit the availability of Mastercam Add-On products and the status of their use.
  • Start Mastercam with either a HASP, software licensing, or NetHASP.
  • Use Reseller, Industrial, Educational, Learning Edition, or Subscription functionality (editing capabilities for Mastercam Resellers only).
  • Run Mastercam with or without Maintenance (editing capabilities for Mastercam Resellers only).
  • Create a customized Mastercam shortcut on the desktop based on your current settings.
  • Save the current settings to the registry and/or open Mastercam when you click OK.

Mastercam Launcher is primarily for NetHASP users and Mastercam Resellers. It is used to customize the start-up settings for a particular Mastercam license. Some options on this dialog box (such as running Mastercam with or without Maintenance) are available only if you have a Reseller's HASP or software licensing. Regular HASP, software licensing, and NetHASP clients will not see these options. Mastercam Launcher recognizes which type of license you have on your machine, and this dialog box displays only those options that you can change.

Parameters

License information

Displays the customer name, HASP, NetHASP, or Software license serial number, the user type, its expiration date, and the maintenance expiration date.

NOTE
  • If using a Permanent License, the HASP expiration displays the word "Permanent" rather than a date.
  • Maintenance expiration date shows the date maintenance on the license expires. For temporary codes, the license expiration date and the Maintenance expiration date are the same.
  • A Reseller License expiration date, and maintenance date appear the same.

Platform

Displays a grid with the name of each of the Mastercam product lines (platforms), the available level of that platform on your license, and the status of the platform. If a platform is not enabled on a license, the entire row is inactive.

The Level column has a drop-down arrow that allows you to select a specific level of a product. The Status column also has a drop-down arrow that allows you to choose from the following status settings:

  • Inactive: The product is not available to be used (acts as if you are not licensed for the product).
  • Startup: Mastercam starts up in that product.
  • Active: The product is available to be used.

Add-On products

Displays a grid of all of the available Add-On products, how many licenses are currently available, and their status.

  • A Not licensed status means that the Add-On is not available on your License, and therefore you do not have access to that Add-On.
  • Not all Add-Ons are checked out at start-up. Those items are marked Automatic checkout and are checked out automatically as needed.
  • Add-On products that are not available on the license are inaccessible. If an Add-On is on the license, but all of the instances are in use, the available number in the Licenses available column will be 0, and you will not be able to use it.

License type

Displays your current license type. You can switch your license between HASP, NetHASP, and Software depending on your access to each of those devices at the time you start Mastercam Launcher.

If you select Software, you can further refine the search by setting it to Default search (local first, then network), Local search only, or Network search only.

Window positions

Select Reset to reset the dialog positions to their default locations. This is useful when going from a dual screen setup to a single screen and some dialogs seem to be missing.

Create shortcut

Creates a shortcut on the desktop based on the current settings in Mastercam Launcher and prompts you for a name. The shortcuts contain all of the switches needed to open Mastercam according to the settings you chose in Mastercam Launcher. When you create a shortcut and do not enter a name for it, Mastercam names the first shortcut as Mastercam (1), and increments the number for each customized shortcut you create without specifying a name.

Save on exit

Saves the settings in the Mastercam Launcher to the registry when you click OK to exit the dialog box. If you click Cancel to exit the dialog box, the settings are not saved to registry even if you have selected this option.

Start Mastercam

Launches Mastercam when you click OK. If you click Cancel to exit the dialog box, Mastercam does not start even if you have selected this option.

Emulation

Resellers only. Allows Mastercam Resellers to set up Mastercam to run with Reseller, Industrial, Educational, Learning Edition, or Subscription functionality. Mastercam can also be set up to run with or without maintenance.

Expiration date

Displays the expiration date of the license. For temporary codes, the license expiration date and maintenance expiration date are the same.

Maintenance expiration date

Displays the date when maintenance on the license expires. For temporary codes, the license expiration date and maintenance expiration date are the same.

Command lines and shortcuts

You can use command-line switches for specific tasks when launching Mastercam. These include:

  • Overriding the initial license type.
  • Loading licenses for selected product levels and Add-Ons.
  • Requesting specific network licenses on start up.
  • Opening Mastercam in a particular product or product level, or to suppress products with single-user licenses.
NOTE

Shortcuts created by the Mastercam Launcher include the appropriate command-line switches.

Command-line switches

Listed below are all command-line switches for Mastercam products. However, not all products may be included with your Mastercam installation.

NOTE

If you use a command-line switch to suppress products or check out a lower level product than what is supported by your license, a message displays the products enabled by the license and modified by the command-line switch.

Use these command-line switches to override the default registry settings.

  • /H = Sets the initial license type to HASP.

  • /N = Sets the initial license type to NetHASP.

  • /Software = Sets the initial license type to Software (local, then network).

  • /SoftwareLocal = Sets the initial license type to Software (local only).

  • /SoftwareLan = Sets the initial license type to Software (network only).

Using Shortcuts and Command-lines

Use the following procedure to set command-line switches for a Mastercam shortcut:

  1. From your desktop, right-click on the Mastercam shortcut and select Properties.

    The Mastercam 2025 Properties dialog box displays.

  2. Select the Shortcut tab.

  3. Add one or more command-line switches to the Target parameter.

    Precede each command by a "/" and separate each by a space. There must also be a space between "...\Mastercam.exe" and the first /Command. For example, to request a single Mill Entry license at start up, enter:

    "C:\Program Files\Mastercam 2025\Mastercam\Mastercam.exe" /ME

    To request a Mill 3D license and a Router license:

    "C:\Program Files\Mastercam 2025\Mastercam\Mastercam.exe" /M3 /R2

    NOTE

    The Target must have quotation marks around the directory path and the command-line options must be outside of the quotation marks.

  4. To accept the command-line changes to the Mastercam shortcut, click OK to exit the Properties dialog box.
