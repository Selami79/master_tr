Exploring Fundamental Tools

Mastercam offers built-in fundamental tools for ease of use and cross-project functionality. Use these tools to add accuracy to your project while improving your workflow. These tools include:

  • Data entry shortcuts (also known as SelCue)
  • 2D/3D construction mode
  • Built-in calculator
  • AutoCursor
  • Fast Point mode
  • Machine groups

Data Entry Shortcuts (SelCue)

Mastercam offers convenient shortcuts for entering data into almost any kind of field. Click the following drop-downs for details.

Automatic U.S./metric conversion

Mastercam automatically performs U.S./metric conversions as you enter data. When in a field, type in the value followed by either "mm" for metric or "in" for U.S. Regardless of the current configuration file, Mastercam converts the number to the equivalent value in the other system. For example, in the metric configuration, if you type "2.0 in" and move your cursor to another field, Mastercam converts that inch value to the equivalent metric value of 50.8 mm.

Input Value and Units Converts to Default Units
Specific data range limitations

Certain data fields in Mastercam only allow specific ranges of data to be entered. If unacceptable data is entered into one of these fields, a "clue window" appears, providing acceptable numeric ranges. The possible messages for this situation are:

  • greater than 0.0
  • minval or greater (e.g., 0.0 or greater)
  • less than 0.0
  • maxval or less (e.g., 100.0 or less)
  • invalid expression
  • minval to maxval (e.g., 0.0 to 360.0)
Reading values from geometry

Mastercam can also read dimensions, coordinate positions, and other values directly from geometry in the graphic window. For example, type [X] in a field, and press either [Tab] or [Enter]. Mastercam returns you to the graphics window to select a point or entity location and reads the X coordinate directly into the field. Right-click in the field to see the complete list of shortcuts. You can either select the shortcut from the menu or type the shortcut key.

When choosing Angle menu, T=Two lines, the order that the lines are picked determines the angle between the lines.

Circle and arc diameters

Specify diameters quickly and without consulting printed drill charts. Type [#] and a drill number from 1-97, or type [#] and a drill letter from A-Z in a calculable field to automatically enter the correct diameter.

You can also right-click on the field and select from the menu items to choose diameters from a drill or tap table. Choose Solid hole diameter from the right-click menu to have Mastercam return the diameter of a selected solid hole.

2D/3D Construction Modes

When creating geometry, use the 2D/3D toggle button on the Home tab, Status Bar, or graphics window right-click menu to set the drawing mode. This button switches from 2D construction mode to 3D construction mode. The default setting for this button is 3D construction mode.

In both modes, use AutoCursor or the selected function or dialog box fields to enter coordinate values. There are some exceptions to this that apply to specific entity types.

The following functions always provide 3D solutions, regardless of the construction mode setting:

  • Fillets and Chamfers
  • Convert to NURBS
  • Control Points NURBS
  • Transform: Translate, Translate 3D, Mirror, Rotate, Rectangular Array, Scale

In 2D mode, all the geometry is created parallel to the current construction plane and at the current Z depth setting. However, you can override the Z depth setting by typing coordinate values that include a Z depth value different from the current Z depth setting.

In 3D mode, the X, Y, and Z coordinates are all read from AutoCursor when sketching dynamically. The Z depth setting in the Home tab is not used. When working in a 3D environment, you can create entities whose points lie in different planes, or create entities (such as arcs or 2D splines) in planes other than the predefined planes.

NOTE
  • This setting has no effect on Analyze and Toolpath functions.
  • In 3D mode, if you choose to bisect two lines that are not in the same plane or are not parallel to the current Cplane, a message informs you that a 3D solution does not exist and asks you to specify the Z depth to use for the 2D solution.
  • When editing entities using the Trim or Break functions, all solutions are calculated based on the current Cplane setting. If the entities cannot intersect in the current Cplane through a 2D projection, they cannot be modified using this function.

Built-in Calculator

Fields that accept number values have a built-in calculator. Use the calculator function to enter mathematical formulas directly into an input field. Along with calculating the solution to formulas, the calculator can also eliminate the need to reference drill charts and can convert entire expressions to a specific unit type. Learn more about Mastercam's Built-in Calculator.

AutoCursor

Use AutoCursor to easily select specific points on your part. AutoCursor detects and highlights endpoints and midpoints of curves, lines, arc center points, and point entities. For example, in the following image, AutoCursor is set to Arc Center and shows the arc center visual cue whenever the user mouses over an arc.

AutoCursor can also snap to angle, nearest, tangent, perpendicular, horizontal, and vertical conditions.

You can learn more about AutoCursor in AutoCursor Overview. To learn how to adjust the settings of AutoCursor, see Selection page: System Configuration.

Fast Point Mode

Use Fast Point mode when AutoCursor is active to input X, Y, Z coordinates to set an endpoint. Fast Point mode is enabled by default and can be activated by pressing the space bar. Input your coordinates and hit [Enter]. To learn more about input values and modality, see Fast Point Mode.

Machine Groups

Use machine groups to organize your toolpaths by machine. Rename, delete, edit, and organize your machine groups and their toolpaths. Machine groups are created automatically when you select a machine from the Machine tab. Mastercam displays machine groups in the Toolpaths Manager. To learn more about machine groups, see Working with Machine Groups
