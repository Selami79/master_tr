Using the Selection Bar

The Selection Bar, an on-screen selection control that is located in the graphics window, is used to select entities for most Mastercam functions. Selection in Mastercam is consistent with Windows selection methods.

The Selection Bar has a Standard Selection mode and a Solids Selection mode. Many functions on the Selection Bar can also be added to your graphics window right-click menu (context menu) through the Context Menu page of the Options dialog box.

NOTE

When you are selecting machining and avoidance geometry, you do not need to activate solid selection; it is activated automatically. Also for machining and avoidance geometry, Mastercam disables the Standard Selection, Solid Selection, Edge Selection, Face Selection, and Body Selection buttons.

Parameters

Toggle AutoCursor Lock

Use this button to ignore settings defined in the AutoCursor Settings dialog box, and to detect and snap only to the point type you choose from the AutoCursor list on the Selection Bar.

AutoCursor Override

Select a point type from the drop-down to detect and snap to as you move your cursor over geometry on the screen. AutoCursor becomes active whenever Mastercam prompts you to select a position on the screen.

AutorCursor Configuration

Opens the AutoCursor Settings dialog box, where you can customize the AutoCursor.

Standard Selection

From Solid Selection mode, click this button to return to Standard Selection mode. This button cancels any locked selection method, and returns you to the default Standard Selection mode.

Activate solid selection

Activates solid selection functions appropriate for the current solid and operation. In cases where you initiate a Mastercam function that might apply to solids in addition to other types of entities, you can use this button to switch from the default Standard Selection mode to the Solid Selection mode.

Select edge

Toggles edge selection on or off. When on, you can select solid edges on a model. When off, you exclude solid edge selection.

Select face

Toggles face selection on or off. When on, you can select solid faces on a model. When off, you exclude solid face selection.

Select body

Toggles whole solid body selection on or off. When on, you can select whole solids on a model. When off, you exclude whole solid body selection.

From back

Toggles "from back" selection on or off. When on, you can select solid edges or faces only on the back of a model. When off, you can select solid edges or faces only on the front of a model.

Selection Methods

Sets the method by which you select entities in the graphics window. Selecting a method option limits selection to that method. Left-click a method to choose it for a single selection event. You can return to the standard selection method from any of the following selection methods by clicking Standard Selection. Note that it is not necessary to select a method in advance of selecting entities. Window and Single are the standard selection methods, requiring you to only begin selecting entities in the graphics window without choosing a method in advance.

To finalize multiple-entity selections, press [Enter], or click End selection. You can also end a series of selections by double-clicking the last entity of the selection. For example, to select three lines using the single selection method, click the first two lines. Then double-click the third line to end the selection process. (The double-click method does not work with polygon and vector selections.)

  • Chain: Selects/chains entities that are connected to other entities. For example, clicking one side of a rectangle selects all four sides and clicking one line that is connected to another selects both lines.
  • Window: Lets you select entities by drawing a window around them. This selection method is governed by your choice in the Entity selection settings drop-down list.
  • Polygon: Lets you select entities by drawing a polygon around the entities. Double-click to complete polygon. This selection method is governed by your choice in the Entity selection settings drop-down list.
  • Single: Lets you select a single entity with your mouse.
  • Area: Selects multiple nested shapes with a single mouse click.
  • Vector: Lets you select multiple entities by drawing a vector line through them. Any entities physically intersected by the vector line will be selected.

Entity selection settings

Determines how the polygon and window selection methods select entities.

  • In: Selects only entities that lie completely inside the window.
  • Out: Selects only entities that lie completely outside the window.
  • In +: Selects entities that lie completely inside and entities that intersect the window.
  • Out +: Selects entities that lie completely outside and entities that intersect the window.
  • Intersect: Selects only entities that intersect the window.

Temporary center point

Creates a temporary center target point for base positions after you select geometry. This button is only available when you are in a function that prompts you to pick an origin position.

NOTE

The temporary center point of the selected entities respects the 2D switch and Cplane settings.

Toggle verify selection

Opens a small window where you can cycle through entities in the graphics window until you reach the one that you intend to select.

NOTE

This feature is only available when Mastercam detects multiple solutions. The entities detected by Mastercam are based on your cursor position.

Invert selection

Reverses entity selection (deselects selected entities and selects unselected entities).

Select last

Reselects the entity (or entities) selected for the previous operation. Useful when performing sequential functions on selected geometry, or when you need to recover from a failed operation. This function ignores entities that are not supported by the current operation or active selection mask.

NOTE

To reselect previously chained entities, use Last in the Chaining dialog box.
