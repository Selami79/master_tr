Selection page: System Configuration

The Selection page configures the behavior of the AutoCursor option in the Selection Bar. Use the checkbox options to define the points to which AutoCursor snaps. For example, if you select the Horizontal/Vertical checkbox, the next time you draw a line, your cursor snaps to a nearby horizontal or vertical position.

Use AutoCursor when rotating geometry with Dynamic to snap geometry to horizontal, vertical, or tangent positions.

Parameters

Snap position checkboxes

When checked, sets the AutoCursor to snap to that type of position.

  • After setting your AutoCursor positions/conditions, your cursor displays a visual cue for each type of position as you move your cursor around the graphics window to choose a position.
  • When you select Angular, the field next to it becomes active. This function works in increments. For example, if you enter 10 , angles will snap to 0 , 10 , 20 , 30 , etc.
  • Tangent is only active when creating a line or arc. Angle, Perpendicular, and Horizontal/Vertical are only active when creating a line.
  • When Mesh Vertices and Along Edges Only are selected, the AutoCursor positions are only available on vertices along the edges of mesh entities. When Mesh Vertices is selected, but Along Edges Only is disabled, AutoCursor positions are available along the vertices of all mesh entities.

Restore defaults

Resets all snap position checkboxes to their default setting.

Disable all

Clears all snap position checkboxes in this dialog box.

Default to Fast Point mode

Initiates Fast Point mode when AutoCursor is active and you begin typing a number or numeric formula. Fast Point opens an empty field in the upper-left corner of the graphics window.

Enable power keys

Select to use power keys to override AutoCursor settings and force AutoCursor to snap to only a specific point type. Power keys are one-character shortcut keys associated with some of the point types that you set in this dialog box. The list of AutoCursor shortcut keys can be found in AutoCursor Overview.

When AutoCursor is active and power keys are enabled, you can press a power key to temporarily override the current AutoCursor settings and snap only to the power key point type. For example, to force AutoCursor to detect only arc center points, press the Arc Center power key [C].

Temporary midpoints delay

When using AutoCursor, controls how many seconds before midpoints are displayed when hovering over an entity. The default is 1 second.

Tangency tolerance

The maximum distance that two entities can be out of exact tangency to be considered tangent.

Allow pre-selection

Lets you pre-select entities in the graphics window before choosing an operation to perform on these entities.

Auto-highlight

Select to highlight an entity under your cursor before selection.

Solids by faces

Select to highlight solid faces rather than entire solids.

Delay

Enabled by selecting Auto-highlight. Lets you control how fast the feature gets accentuated. The default is No Delay.

Use glow highlighting

Highlights the entity under your cursor with a glow effect. Glow highlighting is automatically disabled during Dynamic Rotation.

Use stipple on solids/surfaces

Displays stipple overlay effect on highlighted or selected surfaces, meshes, and solids.

Use dashed wireframe

Displays the dashed effect on auto-highlighted or selected wireframe.
