Solids Mode: Chaining

In Solids mode, the Chaining dialog box provides chain selection methods including Edges, Loop, Face, Partial loop, Outer open edges, Outer shared edges, Cavities and Bosses.

Some selection modes can be enabled simultaneously, while others exclude some options. Modes that require edge selection, and are mutually exclusive, are Edges, Loop, and Partial loop. Modes that require face selection are Face, Outer open edges, Outer shared edges, Cavities, and Bosses. You can also deselect and reselect solid chains, reverse chaining direction, and move the start of a chain.

Guided chaining

Guided chaining is available in 3D and Cplane modes with Edges selected. You can guide and select the next entity in a chain when a branch point is reached.

Click an edge to begin chaining. As shown in the following example, Mastercam displays a green arrow at the start point of the chain. When a branch point is reached, a red arrow points to the default direction or entity. One or more blue arrows point in the direction of any alternate branches.

Use the Previous, Next, and Adjust controls to guide the direction of the chain.

Alternately, you can click on the red or blue arrow directly to advance the chain in that direction.

Button Description
Click Previous to remove the last edge from the chain.

Click Next to extend the chain to include the edge indicated by the red branch arrow. [Shift+Click] extends the chain to include any tangent edges.
Click Adjust to select a different direction/entity as indicated by a blue arrow.

Use the following methods to extend, edit, or create a new chain.

  • Select an edge that touches the end of the chain, as indicated by the blue and red arrows.
  • [Shift+Click] to extend the chain to include any tangent edges found.
  • Click a chained edge in the middle of an existing chain to remove the chained edge, splitting the single chain into two chains.
  • Click the last chained edge to remove it from the chain, returning to the last chained point.
  • Select an entity that is not contiguous to the current chain to cause the current chain to end and a new chain to start.
How does Mastercam create a lathe solid chain?

When chaining a solid for a lathe toolpath, Mastercam creates a spun profile of the part. This profile is created by virtually spinning the geometry about the selected axis and generating a close approximation of the actual profile.

When chaining, both the upper and lower profiles are displayed initially. Once you chain either the upper or the lower profile, the unchained profile no longer displays in the graphics window.

NOTE

Every edge located on the solid is used in the calculation of the spun profile. If you modify an edge in the solid, you may need to regenerate your toolpath to update the spun profile.

Parameters

Wireframe

Selects Wireframe chaining mode. If only Wireframe is available in the part, then that mode is the default. If both Wireframe and Solids are present in the drawing, then Wireframe is the default selection. After your initial selection, the last used mode becomes the default.

Solids

Selects Solids chaining mode. If only Solids are available in the part, then that mode is the default . If both Wireframe and Solids are present in the drawing, then Wireframe is the default selection. After your initial selection, the last used mode becomes the default.

Options

Opens the Chaining Options dialog box so that you can refine the way Mastercam chains entities, sets masks, modifies the sorting order, and sets the chaining tolerance.

Cplane

Chains only edges, loops, and faces that are parallel to the current construction plane (Cplane).

This mode uses the Plane tolerance parameter in the Chaining Options dialog box to define the maximum distance an entity can be from the plane and still be chained. Plane tolerance also determines which entities are co-planar when using the Edges selection method.

3D

Chains edges, loops, or faces regardless of their orientation in the construction plane. If an edge is shared, move your mouse to highlight the other available loops before selecting.

Edges

Creates a chain from the selected edge. [Shift+Click] to select tangent solid edges.

When in Cplane mode, only edges that are parallel to the Cplane are chained, and the chain will propagate to include adjacent edges at the same depth relative to the Cplane.

Loop

Creates a chain from the selected loop. Hovering over an edge will highlight the loop associated with it. If an edge is shared, move your mouse to highlight the other available loops before selecting.

When in Cplane mode, only loops that are parallel to the current construction plane are chained.

Face

Creates a chain from the selected face.

When in Cplane mode, double-clicking or triple-clicking will only select faces that are parallel to the Cplane and at the same depth as the face that was selected. Window select will only select faces that are in the window, parallel to the current Cplane, and at the application's current Z depth.

Partial loop

Creates an open chain from a loop with two mouse clicks. In the graphics window, click the entity where you want to start the chain, and then click where you want to end the chain. Mastercam creates an open chain, consisting of all adjacent entities between these two points.

Outer open edges

Chains, in a clockwise direction, the outer open edges of a selected face.

Outer shared edges

Chains, in a counterclockwise direction, the outer shared edges of a selected face.

Cavities (Selection method)

Selects all cavities on a face. This selection method creates counterclockwise chains by default. These chains can be individually edited in Chain Manager as individual loops.

Cavities (drop-down)

Available only if the Cavities selection method is selected. Use the drop-down menu to filter your selection.

NOTE

Nothing will be selected if cavities and bosses are not available in the part.

Bosses

Selects all bosses on a face. This selection method creates clockwise chains by default. These chains can be individually edited in Chain Manager as individual loops.

Last

Re-selects entities selected during the previous chaining session. This is useful when correcting chains that did not generate properly. You can edit the re-selected chains to correct chaining direction and order. Using Last avoids the need to completely rechain the entities.

Unselect

Deselects the active chain. Repeat Unselect to remove single chains, one at a time.

End chain

Ends the current chain, allowing you to start a new one.

Display all selection arrows

Available when multiple chains are selected. Displays all chain selection arrows. Full loops display only one chain direction arrow; partial loops display both the start and end arrows. This option lets you edit chains without leaving the Chaining dialog box. You can also select individual chains.

Once enabled, you can do the following:

  • Use Reverse to switch the direction of all active chains. If you select only one chain, you can reverse its direction instead.
  • Use Back, Forward, or Dynamic to reposition the start or end of an individual chain.
  • Use Unselect to clear the selection of all active chains. You can also select individual chains and clear the selection of only that chain. This is helpful when you use Window selection (or another selection method) to create multiple chains and need to remove one or more of them from the selection.

Previous

Allows movement from the end of the chain back to the end of the previous edge in the chain. The button is enabled when the active chain is an edge chain, and contains more than one edge. [Shift+Previous] removes any edges tangent to the last edge of the active chain. This is not supported for lathe chains. Previous is only available when Edges is selected.

Adjust

Cycles the chaining direction (shown as a red arrow) through alternate available chaining directions as indicated by blue arrows. Adjust is only available when Edges is selected.

Next

Continues the chain in the direction of the red arrow. Next is only available when Edges is selected.

Start point Forward/Start point Back

Use Start point Forward and Start point Back to move the start of a chain from one entity endpoint to another.

Dynamic

Moves the start or end of a selected chain to any position along an edge, without restricting the start/end to an edge endpoint. Choose Dynamic, then click the start or end point of a chain to drag it to a new position.

Reverse

Swaps the chaining direction. Reverse applies to both open and closed chains.

Parameters - Lathe Solid Chaining

The parameters listed below only display when chaining for a Lathe toolpath.

Wireframe

Selects Wireframe chaining mode. If only Wireframe is available in the part, then that mode is the default. If both Wireframe and Solids are present in the drawing, then Wireframe is the default selection. After your initial selection, the last used mode becomes the default.

Solids

Selects Solids chaining mode. If only Solids are available in the part, then that mode is the default . If both Wireframe and Solids are present in the drawing, then Wireframe is the default selection. After your initial selection, the last used mode becomes the default.

Last

Reselects entities chained during the previous chaining operation. This is useful when correcting chains for a solid that did not generate properly. Mastercam rechains the entities so that you can edit the chains and correct chaining direction or order. Using Last avoids the need to completely rechain the entities.

Unselect

Deselects the active chain. Repeat Unselect to remove single chains, one at a time.

End chain

Ends the chain so you can extend the current chain or start a new one.

Display all selection arrows

Available when multiple chains are selected. Displays all chain selection arrows. Full loops display only one chain direction arrow; partial loops display both the start and end arrows. This option lets you edit chains without leaving the Chaining dialog box. You can also select individual chains.

Move start backward/Move start forward

Use Move start backward and Move start forward to move the start of a chain from one position on the chain to another.

Dynamic

Moves the start or end of a selected chain to any position along an edge, without restricting the start/end to an edge endpoint. Choose Dynamic, then click the start or end point of a chain to drag it to a new position.

Reverse

Swaps the chaining direction. Reverse applies to both open and closed chains.

Move end backward/Move end forward

Use Move end backward and Move end forward to move the end of a chain from one position on the chain to another.

Show solid

Select to display the solid associated with the current chain. Useful for when you want to see how the chain interacts with the solid part. Deselect to easily pick a specific piece of the chain.

Tolerance

Use the Tolerance option when converting the solid into a turning profile for chaining. The tighter the tolerance the more accurate the solid's spun profile will be.
