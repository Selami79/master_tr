Wireframe Mode: Chaining

In Wireframe mode, the Chaining dialog box provides chain selection methods including Chain, Partial, Window, Polygon, Point, Area, Single and Vector. Other controls let you manage the selection process (Last, End Chain, Unselect, Unselect All) as well as control a chain's onset, direction, and termination. Chain Similar allows you to automatically select chains based on the Chain Similar Options.

Guided chaining

Guided chaining is available when using the Chain or Partial chain selection methods. With it, you can determine the next entity in a chain when your chain reaches a branch point. It is also possible to create multiple disconnected chains in the same wireframe session.

Select wireframe geometry to begin chaining. As shown in the following example, Mastercam displays a green arrow at the start point of the chain. When the chain reaches a branch point, a red arrow points in the default direction. One or more blue arrows point in the direction of any alternate branches.

Use the Previous, Next, and Adjust controls to guide the direction of the chain.

When an entity that is not contiguous to the current chain is selected, the current chain is ended and a new chain is automatically started.'

Button Description
Click Previous to remove the last edge from the chain.

Click Next to extend the chain to include the edge indicated by the red branch arrow. [Shift+Click] extends the chain to include any tangent edges.
Click Adjust to select a different direction/entity as indicated by a blue arrow.

Alternately, you can click on the red or blue arrow directly to advance the chain in that direction.

Parameters

Wireframe

Selects Wireframe chaining mode. If only Wireframe is available in the part, then that mode is the default mode. If both Wireframeand Solidsare present in the part, then Wireframe is the default mode. After your initial selection, the last used mode becomes the default selection.

Solids

Selects Solids chaining mode. If only Solids are available in the part, then that mode is the default mode. If both Wireframeand Solids are present in the drawing, then Wireframe is the default mode. After your initial selection, the last used mode becomes the default selection.

Options

Opens the Chaining Options dialog box so that you can refine the way Mastercam chains entities, set masks, modifies the sorting order, and sets the chaining tolerance.

Cplane

Chains only entities that are parallel to the current construction plane and at the same Z depth as the first entity you chain. Mastercam uses the plane tolerance setting in the Chaining Options dialog box to determine whether or not an entity lies in the construction plane.

Cplane chaining recognizes co-planar branches but does not recognize or pause at non-planar branches. This plane mask is useful when you need to chain geometry that has many non-planar branches, as Mastercam can create chains without stopping at each non-planar branch point.

3D

Chains entities defined in the X, Y, and Z axes simultaneously. Recognizes co-planar as well as non-planar branches. At non-planar branch points, Mastercam prompts you to select the next entity in the chain.

Chain

Lets you create a chain by clicking on an entity. This default chaining method is active the first time the Chaining dialog box opens in Wireframe mode. The default chaining mode can also be set in the Chaining page of the System Configuration dialog. Use [Shift+Click] to select tangent entities.

Chaining behavior depends on which mode is selected, Cplane or 3D. When Cplane is selected, Mastercam chains entities that have adjacent endpoints and lie in the current Cplane, without pausing at non-planar branches. When 3D is selected, Mastercam pauses chaining at all (co-planar and non-planar) branches and prompts you to select the next entity.

Partial

Creates an open chain with two mouse clicks. In the graphics window, click on the entity where you want to start the chain, then click where you want to end the chain. Mastercam creates an open chain consisting of all adjacent entities between these two points. Guided chaining is also available for partial chains

Partial chaining behavior depends on which mode is selected, Cplane or 3D.

Window

Selects multiple chains by drawing a rectangle around them. Mastercam orders the chains based on the position of the start point. You can change this behavior by deselecting the Prompt for window start point on the Chaining page of the System Configuration dialog box. By default, Window chains all entities inside the chaining window. To modify this behavior choose another option on the Entity selection settings dropdown list.

Polygon

Selects chains within a multi-sided, irregular window that you draw around the entities. To modify this behavior choose another option on the Entity selection settings dropdown list.

Point

Collects points as chained entities. Use this method to chain a point to serve as a plunge point or as a move-to point in a toolpath.

Area

Selects all chains within a closed boundary with a single mouse click. When using this chaining method, you can turn Infinite nesting in area chains (in the Chaining Options dialog box) on or off, giving you two methods for chaining closed boundaries.

Single

Creates a chain out of a single line, arc, or spline. Useful for creating ruled surfaces that require two opposing entities for the surface boundaries.

Vector

Creates chains from entities that intersect a vector line, extending chains in both directions until the end point of the entity is encountered.

Entity selection settings

This drop-down menu provides options that modify Window and Polygon chaining. Select any of the following:

  • In: Chains all entities within the window.
  • Out: Chains entities that lie outside the window.
  • In +: Chains entities inside and that intersect the window.
  • Out +: Chains entities that lie outside and that intersect the window.
  • Intersect: Chains entities that intersect the window.

Wait

Works only with Partial chaining. Allows the same entity endpoint to be selected so that you can chain a single or partial chain in both directions.

Last

Reselects entities chained during the previous chaining operation. This is useful when correcting chains for a surface that did not generate properly. Mastercam rechains the entities so that you can edit the chains and correct chaining direction or order. Using Last avoids the need to completely rechain the entities.

Chain similar

Instructs Mastercam to add chains to the Chain Manager based on criteria set in the Chain similar options dialog box.

Unselect

Unselects the last selected chain. Repeat to remove single chains one at a time. Unselects all chaining selected if Window or Polygon chaining was used.

End chain

Ends the chain so you can extend the current chain or start a new one. Often used in conjunction with the Wait option.

Chain similar options

Opens the Chain similar options dialog box so you can set the way Mastercam will recognize related chains. Refine the chain selection through geometric and attribute matching.

Display all selection arrows

Available when multiple chains are selected. Displays all chain selection arrows. Full loops display only one chain direction arrow; partial loops display both the start and end arrows. This option lets you edit chains without leaving the Chaining dialog box. You can also select individual chains.

Once enabled, you can do the following:

  • Use Reverse to switch the direction of all active chains. If you select only one chain, you can reverse its direction instead.
  • Use Back, Forward, or Dynamic to reposition the start or end of an individual chain.
  • Use Unselect to clear the selection of all active chains. You can also select individual chains and clear the selection of only that chain. This is helpful when you use Window selection (or another selection method) to create multiple chains and need to remove one or more of them from the selection.

Previous

Allows movement from the end of the chain back to the end of the previous entity in the chain. The button is enabled when the active wireframe chain contains more than one entity. [Shift+Previous] removes any edges tangent to the last edge of the active chain. This is not supported for lathe chains.

Adjust

Cycles the chaining direction (shown as a red arrow) through all available chaining directions as indicated by blue arrows. Adjust is only available when a branch point is reached with the Chain or Partial methods.

Next

Continues the chain in the direction of the red arrow. Next is only available when a branch point is reached with the Chain or Partial methods.

Start point Forward/Start point Back

Use Start point Forward and Start point Back to move the start of a chain from one entity endpoint to another. Not available if Break closest entity to thread point in the Chaining Options dialog box is selected.

Dynamic

Moves the start or end of a selected chain to any position along an entity, without restricting the start/end to an edge endpoint. If Break entities in Dynamic is selected (Chaining Options dialog box or Chaining property page), using this function automatically breaks the entity at the start/end position. Choose Dynamic, then click the start or end point of a chain to drag it to a new position. This button is disabled for Wire chaining.

Reverse

Swaps the chaining direction. Reverse applies to both open and closed chains.

End point Forward/End point Back

Use End point Forward and End point Back to move the end of a selected chain from one entity endpoint to another. Available only on open chains and partial chains. Not available if Break closest entity to thread point in the Chaining Options dialog box is selected.
