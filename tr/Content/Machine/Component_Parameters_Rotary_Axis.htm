Parameters tab: Machine Component Manager (Rotary Axis)

Use the Parameters tab to configure a rotary axis component.

Identifying the axis component

First, select the Machine coordinate that will identify the axis associated with this component. This is typically the axis referred to when programming parts and operations. Select from A, B, or C.

Then, map the Machine coordinate axis to its physical orientation on your machine tool using the diagram in the tab. Select the desired axis by clicking on it. For example, on a typical 2-axis lathe, Machine coordinate=C will map to Axis of Rotation = X+. For a typical 3-axis vertical mill, on the other hand, X+ would be used for Machine coordinate=A.

Finally, select the initial position and rotation direction by clicking on the proper options.

  • Click Tilt Angle to configure a rotary axis that is tilted about an axis other than the axis of rotation.
  • Enter the initial position in the Defined angle field. If your machine definition will be used for simulation, this typically represents the position of the solid model for this component within your machine model.

The Axis label identifies the coordinate label that will be output in the NC code. This is typically only used with Mastercam products that support machine simulation and event-driven programming. Other Mastercam users should select the same coordinate as the Machine coordinate.

Rotation parameters

Each rotary axis in the machine tool can have its own feed rate limits. Enter the limits directly or click From Machine to read them from the Axis traverse rate limits tab of the General Machining Parameters dialog box. The general machining parameters can also be set to force each axis component to automatically use the general machine limits; if that is the case, this tab displays the message, Set from machine general parameters and these options are disabled.

Other parameters let you fine-tune your component's rotary motion capabilities.

Center of Rotation and Travel Limits

These fields are typically only used with Mastercam products that support machine simulation and event-driven programming. For the center of rotation, enter the coordinates of any point along the axis of rotation, or click Select to choose one from the graphics window.

Parameters

Name

Enter the name that will be used for the component in the machine definition component tree.

Machine coordinate

Select the coordinate label that is associated with this axis. Identifies the axis associated with this component. This is typically the axis referred to when programming parts and operations. Select from X, Y, Z, or A, B, C. The Macro option is typically only used with Mastercam products that support machine simulation and event-driven programming.

Define macro-driven axis feed rate in axis motion events

Select to set the axis feed rate to be explicitly defined in axis motion events for this axis. If the box is not checked, the current tool path feed rate is used. This checkbox is used for event-based programming, and only enabled if Machine coordinate is set to Macro.

Axis label: absolute, incremental coordinates

Identifies the coordinate label that will be output in the NC code. This is typically only used with Mastercam products that support machine simulation and event-driven programming. Other Mastercam users should select the same coordinate as the Machine coordinate. Separate fields let you select different axis labels for absolute and incremental output.

Axis of rotation

Click in the graphic to select the axis to rotate about.

0 degree position

Click in the graphic to select the axis that lies along the 0 degree position of the rotary axis.

Direction

Select CW or CCW for the primary direction of rotation.

Tilt angle

Click to open a dialog box where you describe a rotary component that rotates about one axis and is tilted about another axis.

Defined angle (from 0 deg.)

Enter the initial position of the rotary axis relative to the zero-degree position selected elsewhere on this tab. This information is typically only used by Mastercam products that support simulation or event-driven programming. If your Mastercam product does not support these features, you do not need to enter anything in this field.

Center of rotation

Enter the location of the center of rotation. The coordinates should be in world coordinates, relative to the rest of your machine, not part coordinates. Click Select to choose a point from the graphics window. This information is typically only used by Mastercam products that support simulation or event-driven programming. If your Mastercam product does not support these features, you do not need to enter anything in these fields.

Minimum / Maximum travel limit

These travel limits are relative to the zero degree axis position and positive direction, in terms of an axis that winds up continuously. The travel limits represent the physical limits of motion for the rotary axis.

Minimum / Maximum reposition angle

The reposition angles are used to force a retract and reposition in cases where the cut approaches a travel limit. Once the reposition angle is reached, the axes are reset to continue the cut.

Break rotary motion

These parameters enable checking for rotary motion that exceeds travel angles, and apply only to the rotary axis component. The rotary motion can be broken by using the chordal deviation setting on the Tolerances page in the control definition, or by setting a Maximum rotary move angle, or by using both settings. See the Motion control tab of the General Machining Parameters dialog box for settings that control combined rotary axis motion.

From machine

Copies the value from the Axis feed rate limits tab of the General Machine Parameters dialog box.

Maximum feed rate

Enter the feed rate limit for the rotary axis as recommended by the machine tool manufacturer. This value can be used by the post to check that feed rates on individual axes are not exceeded. If you selected Apply to all rotary axes in the Feed rate limits tab of the General Machine Parameters dialog box, the feed rate limits will be preset from the machine parameters and unavailable for editing in the component properties. If you add the rotary axis component from a library and want to retain feed rate limits of the component, deselect Apply to all rotary axes.

Fixed/continuous positioning

Choose between fixed positions and continuous positioning. When fixed positioning is selected, enter the index increment. The angle must divide equally into 360 degrees. For continuous positioning select a type:

  • Signed continuous: rotates in one direction only, continuously (does not need to unwind)
  • Signed direction, absolute angle 0 - 360 degrees: rotates in one direction up to 360 degrees, then needs to unwind
  • Shortest direction, absolute angle 0 - 360 degrees: rotates in either direction within limits to position in the shortest distance possible
