Files page: Control Definition Manager

Use the Files page to tell Mastercam where to find the files needed by this control definition, and to establish default settings for writing different types of files.

  • The File usage window lists the types of files for which defaults can be set. Click on an item to see the current default file for that type.
  • Use the Data paths window to set the default paths for NC and NCI files.

Use the other option groups to configure the posting process, such as how to log errors and other messages, set Post Processing dialog box defaults, and how to handle NC parameters.

NOTE
  • When creating a new control definition, the default names and paths of the toolpath defaults files are read from the Files section of the System Configuration dialog box.
  • Use the Operation defaults page to edit and create new toolpath defaults files.
  • Use the Files tab of the Machine Group Properties dialog box or Machine page of the Machine Group Setup function panel to change the toolpath defaults file, tool library, or operations library for a particular machine group.

This page is used by all machine types.

Parameters

File usage

Choose a default file for each item in this list. Many of these items will appear as defaults in the machine group properties. This can be overwritten on an individual machine basis.

Selected item's file name

Displays the default file for the item which is highlighted in the File usage window. Click the file cabinet button to choose a new path.

Enable chooks from misc values button

Select to replace the misc values button/page with a button to launch the assigned Add-In. You will no longer have access to the misc values button/page until this option is deselected.

Enable chooks from aux registers button

Select to add a button to the Aux. Registers page that will launch the assigned Add-In. You will still have access to the aux registers. Only available after loading a machine definition in Wire.

Data paths

Choose a default path for each item in this list. Many of these items will appear as defaults in the machine group properties. This can be overwritten on an individual machine basis.

Selected item's data path

Displays the default path for the item which is highlighted in the Data paths window. Click the file folders button to choose a new path.

Keep error file

Choose the option which describes which post processing events you want recorded to the log file. The selection you make here sets the err_file$ post variable.

Message options

Choose the option which describes how you want to log post processing errors. The selection you make here sets the error_msg$ post variable.

Suppress variable reinitialize messages

When this option is on, variable initialization messages are not written to the error log.

Transform/Nesting operation options

Allows you to control how operation parameters are processed when processing a transform operation. Select which type of operation parameters you want Mastercam to write to the .ops file.

Your choices are:

  • Transform/Nesting operation parameters only: Processes only transform operation parameters.
  • Source operation parameters only: Processes only source operation parameters.
  • Transform/Nesting and source operation parameters: Processes both source and transform operation parameters.

NC file extension

Enter the file extension that your machine tool expects for part programs. Typically, this is NC.

Post Processing dialog

Click to set default options for creating and editing NC and NCI files when operations are posted. These values will appear in the Post Processing dialog box whenever this control definition is used to post operations. Defaults for some of the other settings in the Post Processing dialog box (for example, communication settings and the name of the post processor) are set elsewhere in the control definition.

Right-click menu options

Default

Reset either the current page or the entire control definition to the default values.

Import

Import values for either the current page or all pages from a different control definition. However, importing values will not change the following:

  • selected post processor, control type, and name
  • toolpath/operation defaults
  • post text
  • miscellaneous integer and real values

You can import post text and miscellaneous values from the Text page.
