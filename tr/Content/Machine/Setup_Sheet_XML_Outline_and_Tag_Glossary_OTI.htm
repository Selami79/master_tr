Setup Sheet XML Outline and Tag Glossary: Optional Tool Information

Raw tool data is supplied for more advanced setup sheet customization. The XML tags listed here are all nested within every operation parent node and Mill and Lathe tool nodes.

"OTI-ACTIVESPINDLE" Active spindle for turning
"OTI-ACTIVETURRET" Active turret for turning
"OTI-COMMENT" Tool description
"OTI-COOLANT" Coolant 8=off, 16=flood, 32=mist, 64=thru-tool
"OTI-CRAD" Tool corner radius
"OTI-CRAD-TYPE" None / Corner / Full (1 of 3 strings )
"OTI-DIA" Tool diameter
"OTI-DOFF" Diameter offset number
"OTI-FEEDRATE-ABS" Absolute value of the feed rate
"OTI-FEEDRATE-TYPE" Type of feed rate. (M) per minute, (R) per revolution, (S) surface finish in microns
"OTI-FEEDRATE-VALUE" Feed rate (no prefix or suffix, just the numeric value).
"OTI-FEEDRATE" Feed rate
"OTI-FPT" Feed per tooth
"OTI-GROUP" Component group to which tool belongs (abstract number)
"OTI-ISSCALABLE" Is scalable
"OTI-LEVELNUMBER" MC tool reference level number
"OTI-LOFF" Length offset number
"OTI-LTL-ANGLE" Lathe tool insert back angle
"OTI-LTL-BACKUSED" The back edge of the insert is defined
"OTI-LTL-BACKV" Vector representing the back edge of the insert: angle and width
"OTI-LTL-FRONTUSED" The front edge of the insert is defined
"OTI-LTL-FRONTV" Vector representing the front edge of the insert: rake angle and height
"OTI-LTL-HEIGHT" Lathe tool insert height
"OTI-LTL-ORIENT" Lathe tool orientation 1-8
"OTI-LTL-ORIENTANGLE" Spindle orientation angle in degrees
"OTI-LTL-QUADRANT" Lathe tool quadrant, 0 (invalid) or 1-4 for final b-axis tool orientation
"OTI-LTL-RADIUS" Lathe tool insert corner radius
"OTI-LTL-RAKEANGLE" Lathe tool insert front angle
"OTI-LTL-TCENTER" Lathe tool insert center point
"OTI-LTL-TOOLANGLE" Angle that lathe tool is mounted in degrees
"OTI-LTL-WIDTH" Lathe tool insert width
"OTI-MACHINEGRPIDN" Machine group
"OTI-MATL" 1 = HSS, 2 = Carbide, 3 = Ti Coated, 4 = Ceramic
"OTI-MAXSS" Maximum spindle speed (lathe)
"OTI-MC" MC tool reference geometry file
"OTI-MERGEDNCI" Merged from ASCII NCI file
"OTI-MM" Values in metric
"OTI-NFLUTES" Number of flutes
"OTI-PLSURFFINFEED" Plunge feed rate is a surface finish
"OTI-PLUNGE-ABS" Absolute value of the plunge
"OTI-PLUNGE-TYPE" Type of plunge rate. (M) per minute, (R) per revolution, (S) surface finish in microns
"OTI-PLUNGE-VALUE" Plunge value for this tool
"OTI-PLUNGE" Plunge rate
"OTI-RADTYPE" None, corner or full
"OTI-RETRACT" Retract rate
"OTI-RPM" Spindle speed
"OTI-SFM" Surface feet per minute.
"OTI-SLOT" Internal tool id number
"OTI-STATION" Station number for block
"OTI-SURFFINFEED" Feed rate is a surface finish
"OTI-THDS" Threads per inch/ thread pitch (mm)
"OTI-TIPANGLE" Tool tip
"OTI-TIPLENGTH" Tool's tip length - calculated by Mastercam
"OTI-TLNO" Tool number
"OTI-TOOLDISPTYPE" MC custom tool display comes from auto (0), file (1), or level (2)
"OTI-TYPE"

Tool type ID number

"OTI-USECSS" Spindle speed is CSS
