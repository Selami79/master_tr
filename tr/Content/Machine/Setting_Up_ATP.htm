Setting Up ATP

Before you can use Mastercam’s Automatic Toolpathing (ATP) to process files, you need to set up groups of levels and associate (map) each of the levels to toolpaths to create strategies. Once the level group is set up, you will need to chain the geometry in those levels, and save the operations to a Mastercam operations library file. When the level groups and operations library are set up, you can create strategies to complete the ATP setup.

To set up ATP, you must follow these procedures, which are listed here in the order in which they must be performed:

  • Creating a Strategy
  • Creating a Level Group: Level Scan
  • Creating Toolpaths and Operations Libraries

Once these procedures have been completed, you can use the Mastercam ATP dialog box to process the part files in ATP.
