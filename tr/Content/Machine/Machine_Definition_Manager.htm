Machine Definition Manager

Machine definitions are the first step in programming your part. Use the Machine Definition Manager to assemble machine components, set machine and component properties, select a control definition, and select a post processor. Once a machine definition has been created and selected, you can begin creating toolpath operations for the machine.

Machine Configuration

The Machine Configuration window displays a schematic model of your machine tool. Individual components and groups of components are organized in a tree structure.

  • Right-click in the tree to add new components or to set properties for individual components.
  • Add components and groups by dragging them from the other windows.
  • Use the checkbox in front of each component to select it for simulation.

Control Definitions and Post Processors

The machine definition is tightly integrated with the control definition and post processor.

  • The control definition tells the post processor about your control's capabilities and requirements. It initializes many NC settings and post variables.
  • The control definition tells Mastercam which post processors have been configured for this machine. Most machine and control definitions are only configured for a single post processor.
  • First, select the control definition. Then, if more than one post processor has been configured for it, select the desired one.
Machine Definition Manager toolbar

Toolbar buttons let you:

  • Perform basic functions, such as opening, saving, and password-protecting your machine definitions.
  • Set general machine properties.
  • Create axis combinations and reference positions.
NOTE
  • Many Machine Definition Manager tools and features are only used by Mastercam Mill-Turn. If you are not building a machine for Mastercam Mill-Turn, many machine definition features do not apply to you and can be ignored.
  • The Machine Definition Manager can only open the machine types associated with the Mastercam products you have purchased and that are enabled on your SIM. For purchasing information, contact your local Mastercam Reseller.
Parameters

Unused component groups

Shows the available component groups for the machine. Use drag and drop to add or remove component groups from the machine configuration tree.

Component file

Shows the components available in the current component library. Use drag and drop to add components to the machine configuration tree. Choose the folder button to open a different component library or machine definition file.

Description

Enter a name for the machine. This name displays in the Toolpaths Manager next to the machine group properties icon. If you do not enter a description, Mastercam uses the machine definition file name for the description.

Machine was defined using Mastercam Extensions

This option will be selected if you are running a Mill-Turn machine definition. It is only available to Mill-Turn machine developers.

Control definition

Shows the name of the control definition file to be used with this machine definition. The file that is selected here determines which post processors will appear in the Post processor list. Use the folder button to select a different control definition file. This field is not available for Mill-Turn machine definitions.

Post processor

Shows the current post processor for the selected control definition. A control definition can have a list of post processors associated with it. This field is not available for Mill-Turn machine definitions.

Machine configuration

Shows the components in the machine. Drag and drop components onto the tree. Right-click on a component to edit its properties. Choose the + next to a component group to expand the group.

Right-click menu options—Machine configuration tree

The available options are slightly different when you right-click on a group name, component name, or the machine base. Not all of the options listed here are relevant for all nodes in the tree.

Add component—Add a new component at the current location in the tree. Select the desired component type from the sub-menu.

Properties—View or edit properties for the selected component.

Rename—Edit the component name.

Remove—Remove the selected component group from the tree. It is added back to the Unused component group pane.

Delete—Remove the selected component from the tree.

Delete all children—Remove all components inside the current node in the component tree, but not the component itself.

Delete component and all children—Remove all components inside the current node plus the selected component itself.

View—Display the component's solid model in the graphics window, if one has been selected for the component. This is typically only useful for Mill-Turn machine definitions.

View component group—Display the solid models for all of the components in the component group. This is typically only useful for Mill-Turn machine definitions.

Expand all, Expand group—Expand all of the nodes in the component tree or the selected group.

Collapse all, Collapse group—Collapse all of the nodes in the component tree or the selected group.

Export to XML file—Generate an XML file with information about the components in the tree. This is only used by Mill-Turn machine developers.

Convert units to metric—This option is available to Mill-Turn machine developers who are working with tool locator libraries. It is only visible when you right-click on the Machine base component. Use it to convert an U.S. tool library to metric, or vice versa. This will convert both the solid models and all of the settings for each tool locator.

Other right-click menus

Other panes of the Machine Definition Manager also have right-click menus:

Unused component groups

Shows the component groups that have not been added to the machine configuration tree. Right-click to display components by icons or in a list.

Component library

This pane shows the components in the current library. Right-click to change the way the components are displayed and how the list is sorted.
