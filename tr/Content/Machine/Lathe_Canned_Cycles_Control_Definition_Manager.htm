Lathe Canned Cycles page: Control Definition Manager

Use the Lathe Canned Cycles page to tell Mastercam how your control supports canned cycles for turning and threading toolpaths. For each toolpath type, select the checkbox to enable it if your control supports it.

  • For canned turning cycles, you will get an error message if you try to create a non-supported toolpath.
  • For threading, if a cycle or output format is not enabled, you can still select it, but it will generate long-hand output.

Groove and thread toolpaths also have a number of secondary options which indicate if your control supports specific toolpath features. These correspond directly to options and parameters you see when creating the toolpaths. Options not enabled here will not be activated or visible. These options occur on the following lathe toolpath tabs:

  • Canned groove shape parameters
  • Canned groove rough parameters
  • Thread cut parameters
Parameters

Enable canned rough turning

Tells Mastercam that the control supports this cycle. If this option is not selected and you try to create the toolpath, you will get an error message.

Enable canned rough pattern repeat

Tells Mastercam that the control supports this cycle. If this option is not selected and you try to create the toolpath, you will get an error message.

Enable canned roughing undercuts

Activates Plunge Parameters on the Canned rough parameters tab so that you can specify how the tool plunges into recesses on the part.

Enable canned finish

Tells Mastercam that the control supports this cycle. If this option is not selected and you try to create the toolpath, you will get an error message.

Enable anticipated pulloff for long hand thread

This option affects the Anticipated pulloff field on the Thread cut parameters tab if you select Longhand as the NC code format.

Enable thread equal depth cuts

This option affects the Determine cut depths from options on the Thread cut parameters tab.

Enable thread equal area

This option affects the Determine cut depths from options on the Thread cut parameters tab.

Enable thread anticipated pulloff

This option affects the Anticipated pulloff parameter on the Thread cut parameters tab.

Enable thread multiple starts

This option affects the Multi Start button on the Thread cut parameters tab.

Multiple first start closest to part

This option affects the First start closest to part option on the Multi Start dialog box accessed from the Thread cut parameters tab.

Multiple first start far from part

This option affects the First start furthest from part option on the Multi Start dialog box accessed from the Thread cut parameters tab.

Enable canned groove wall taper

This option affects the Taper angle field on the Canned groove shape parameters tab.

Enable canned groove radius on corners

This option affects the Radius fields for corners on the Canned groove shape parameters tab.

Enable canned groove radius on chamfers

This option affects the Bottom radius and Top radius fields on the Groove Chamfer dialog box accessed from the Canned groove shape parameters tab.

Enable canned groove rough pecking

This option affects Peck Groove on the Canned groove rough parameters tab.

Enable canned groove rough depth cuts

This option affects Depth Cuts on the Canned groove rough parameters tab.

Enable canned groove chamfer on corners

This option affects the Chamfer fields for corners on the Canned groove shape parameters tab.

Enable canned groove dwell

This option affects the Dwell Time options on the Canned groove rough parameters tab.

Right-click menu options

Default—Reset either the current page or the entire control definition to the default values.

Import—Import values for either the current page or all pages from a different control definition.

However, importing values will not change the following:

  • selected post processor, control type, and name
  • toolpath/operation defaults
  • post text
  • miscellaneous integer and real values

You can import post text and miscellaneous values from the Text page.
