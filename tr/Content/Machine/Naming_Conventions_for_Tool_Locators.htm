Naming Conventions for Tool Locators

It is important to use a consistent, clear naming convention for your tool locators. Not only does it help you keep your own work organized, but it will be a big help to your users when they need to use the Tool Setup Manager—especially for large libraries that have many tool locators.

The standard used internally by Mastercam for tool locators is to include the following items of information in the tool locator name, separated by hyphens:

  • Manufacturer or OEM part number (if any). This should be the first piece of information in the name.
  • Short phrase indicating the size and type of the tool locator—for example, 40mm Boring Bar.
  • The number of stations in the tool locator, if there are more than one.

In addition, the tool locator name should include a description of its tool stations. Use the codes in the following table.

Code Meaning
UT Upper turret.
LT

Lower turret.

The UT/LT code should immediately follow the OEM part number (if there is one), otherwise it should be the first element in the tool locator name.
LIU Left-side, insert up.
LID Left-side, insert down.
RIU Right-side, insert up.
RID Right-side, insert down.
HIPM Station for half-index turret position—offset from primary position in minus or negative direction.
HIPP Station for half-index turret position—offset from primary position in positive direction.
RPID Station located at primary index position, for right-side tools, insert-down.
RPIU Station located at primary index position, for right-side tools, insert-up.
LPID Station located at primary index position, for left-side tools, insert-down.
LPIU Station located at primary index position, for left-side tools, insert-up.

Since lists of tool locators are usually sorted alphabetically, a naming convention like this ensures that when your users see a long list of tool locators, the list is organized logically. Items that do not apply or are not relevant to a specific tool locator are simply omitted.

The following table shows some examples of tool locator names.

Type Tool Locator Name
Boring bar 7.074.067 - UT - 40mm Boring Bar - Left
Cross turning (two stations) 7.074.001 - UT - 25mm Cross Turning - 2 Station - LIU - RID
Face turning 7.077.312 - UT – 1in Face Turning - LID
Live cross 7.073.360 - UT - ER16 Live Cross
Live face 7.073.280 - UT - ER16 Live Face – Left
Cross turning (one primary station, one half-index station) UT - 20mm Cross Turning - 2 Station - Left - HIPM - LPIU
