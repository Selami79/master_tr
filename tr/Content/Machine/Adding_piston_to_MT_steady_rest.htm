Adding a Piston to a Mill-Turn Steady Rest

When a steady rest component for Mastercam Mill-Turn contains arms or pistons, the arms and pistons are defined as individual subcomponents. This is because they move independently of the base.

Follow these steps to add a piston to your steady rest. These steps apply to both turret mounted and standalone steady rests.

See Adding a Turret Mounted Steady Rest to a Component Library (Mill-Turn) for an overview of the entire process of creating a steady rest component.

  1. If necessary, display the properties for the steady test component. Double-click the steady rest component in the Mill-Turn Component Library dialog box to display its properties.

    The Mill-Turn Turret Mounted Steady Rest dialog box opens with the Geometry tab displayed.

  2. Right-click on the steady rest in the Subcomponents window and select Add subcomponent, Add piston

  3. If desired, edit the component name.

    • Right-click on the piston and select Rename.

    • You can also choose to read the component name from the solid model, as described in the next step.

  4. Follow these steps to select the solid model for the component.

    1. You can choose to read the name or color of the component from the model. To do this, select either of these options before selecting the model:

      • Set name from solid
      • Set color from solid

    2. Select the model by clicking the button next to the Model name field. Select only a single solid body; do not try to select any other geometry, such as construction geometry or other subcomponents.

    NOTE

    Before you can select the model, it must already have been imported into the component library. See Importing Solid Models into a Component Library to learn more.

  5. Enter the maximum distance that the piston can move. Enter a positive value only.

    Enter an inch value, then click the Convert to mm. button, or enter a millimeter value then click the Convert to in. button.

  6. Enter the feed rate at which the piston moves. Enter a positive value only.

    Enter an inch value, then click the Convert to mm. button, or enter a millimeter value then click the Convert to in. button.

  7. Enter the angle at which the vector along which the piston moves is offset from the world (machine coordinate system) Y-axis. For most pistons, this vector is parallel to the axis.

    • For a lower turret steady rest, this value is typically 0.

    • For an upper turret stead rest, this value is typically 180.

    NOTE

    If the turret is mounted on a slant bed, do not attempt to adjust for that angle here. Mastercam reads that value from elsewhere in your machine definition and applies it properly.

Continue setting the steady rest properties or with Adding an Arm to a Mill-Turn Steady Rest.
