Geometry tab: Machine Component Manager (Collet (Parametric))

Use this tab to define a collet parametrically. This means that Mastercam will model the collet from the settings that you enter on this tab, instead of selecting geometry for the model.

  1. Begin by specifying the Inside diameter. Mastercam calculates the outside diameter for you based on the dimensions of the steps.
  2. Enter the number of slots and the spacing between them.
  3. Create steps to define the profile of the collet along its length. Use the buttons above the Steps grid to create and manage steps. Click the Initialize collet definition button to create a new set of steps.

    • Use the other buttons to add, remove, or reorder the steps.
    • Refine the profile by typing new values for any dimension directly in the grid.

    As you add, remove, and edit steps, the Preview window updates in real time to display the current profile.

Parameters

Name

Enter the name that will be used for this component.

Geometry

Choose Chain or Parametric to tell Mastercam how you want to define the profile of the jaws.

  • Selecting Parametric displays the options described in this topic.
  • See Geometry tab: Machine Component Manager (Chuck Jaws (Chain)) to learn about defining the chuck jaw profile with wireframe geometry.

Color

Select the color for the component. This color will appear during simulation and when the component is previewed on the screen.

Chord tolerance

Use this setting to control how finely Mastercam will draw the model that represents this component. A smaller value will result in a finer, more detailed simulation; however, this could slow down simulation. If your workstation's performance during simulation is sluggish, consider increasing this value.

Translucency

Use this option to make the component appear more or less translucent when it is displayed on the screen. This will be applied both when the component is previewed and when it is used in simulation.

Inside diameter (ID)

The diameter of inside opening in the collet. Mastercam uses this value in Job Setup to limit the maximum stock diameter.

Number of slots

The number of slots in the face of the collet. The Preview window shows only a single "slice" of the model between two slots.

Slot width

The width of the slots in the face of the collet.

Steps

Use the following buttons to edit the steps in the grid or add new steps.

Icon Description
Initialize the grid with a new set of steps. Any steps that are already in the grid will be removed.
Set all steps to the same width as the selected step.
Set all steps to the same height as the selected step.
Reverse the order of the steps. Go to the Parameters tab to change the clamping from OD to ID.
Add a new step.
Remove the selected step.

Preview window

Displays a live interactive 3D model of the component. Right-click in the window to change the view, dynamically rotate the component, or zoom in on component details. Depending on the type of component, other buttons or controls are available for additional preview options.

The Preview window also uses the Translucency setting found on the Geometry tab. Adjust the Translucency slider to examine hidden details of the component or model without leaving the dialog box.

This picture shows a chuck, but the Preview window is used by a number of different components so you might see a different component on your screen.
