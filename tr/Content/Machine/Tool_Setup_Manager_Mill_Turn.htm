Tool Setup Manager: Mill-Turn

Use the Tool Setup Manager to manage how the tools that have been selected for your operations are loaded onto the turret or tool spindle. You can do any of the following:

  • Select or re-assign tool locators.
  • Move tools to different turret positions.
  • Adjust the projection distance for tools.
  • Edit tool locator settings.

Mastercam automatically loads the tools for each operation when you enter the Sync Manager. If Mastercam is unable to load one or more of your tools, use the Tool Setup Manager to load the tools manually.

NOTE

Everything that you do in the Tool Setup Manager is "live." This means that any changes or edits are immediately effective in your part, even if you have not clicked OK. There is no undo and there is no way to cancel any changes.

Working with machine configuration files

Machine configuration (.mtConfig) files store sets of tools and tool locators that have already been loaded onto a turret. Configuration files can contain any number of tools, from a single tool or locator component to a fully loaded turret.

  • Click the Load machine configuration button to launch the Load machine configuration function panel. From here, you can select a machine configuration file, inspect the components included in it, and select which components to import into the current file.
  • Click the Save machine configuration button to save some or all of the tooling from the current job setup in a new machine configuration file.

Once a configuration file has been loaded, its components will appear in the tree display just like other components. You can continue setting up additional tools or edit the imported components.

NOTE

Machine configuration files can contain other types of components than tools and locators. These other components are available to you when you load a configuration file from Job Setup. When you load a configuration file from Tool Setup Manager, Mastercam only loads the tools and locators.

Loading tools into angled milling heads

The workflow for tools mounted in angled milling heads is different than for other tools. Follow these steps:

  1. While working in the toolpath settings, select the desired tool.

  2. Right-click on the tool and select Get angled head.

    Mastercam opens the Tool Setup Manager.

  3. From the Tool Locators window, select the angled head and drag it to the desired index position on the turret. If necessary, open a different component library that contains the angled head.

  4. If the selected index position does not match the tool number in the toolpath settings, you will need to edit the tool number.

    1. In the Tools in the machine group window, right-click on the tool for the operation and select Edit Tool.

    2. Edit the tool number to match the index position where the head is mounted.

  5. Load the tool by dragging it to the component tree and releasing it on the angled head.

  6. Click OK to return to your toolpath settings.

  7. To use the tool again in additional operations, simply select it like any other tool. You do not need to repeat the steps in Tool Setup Manager.

Another common approach is to use the Tool Setup Manager to load all of the tools before you begin programming your toolpaths. Once a tool has been mounted into an angled head, you can select it for an operation like any other tool.

Parameters

Filter

Use the options in this section to filter the tools and tool locators that are displayed in the Tools in the machine group and Tool Locators lists. This can make it much easier to identify proper locators for each tool or turret. Select one or more of the following filters.

  • Location on machine

    Filter the displayed items by upper or lower turret.

  • Display

    Choose to display all tools loaded in the machine group, only those tools that are used in operations, or only tools that are used in the operations that are currently selected in the Toolpaths Manager.

  • Tool type

    Choose to display milling tools, turning tools, or both.

  • Tool orientation

    Choose to display only tools and locators that are oriented for face milling, cross milling, or angled heads.

  • Only tools not loaded on the machine

    Select to display only those tools that have not yet been loaded.

Tools in the machine group

Displays the tools that are available in the machine group. Right-click to add, delete, edit, or load a selected tool onto the machine. You can also drag-and-drop tools between the tool list and the Machine tooling setup pane. When you load a tool onto the machine, it automatically loads onto the appropriate turret or spindle (depending on the type of tool), and displays in the Machine tooling setup tree. You can also use the right-click menu to open the Tool Manager, where you can add tools to the machine group from a tool library. These tools will display in the list.

Tool locators

Displays the list of tool locators in the current component library (.mcam-gmd file). Tool locators define the position and orientation of tools on a turret or tool spindle.

  • Use the folder button to browse to and open a different component library to access different tool locators.
  • To add a tool locator, drag and drop it onto a turret in the Machine tooling setup tree.
  • Use the right-click menu to change the way the list is sorted and displayed.

Display tool locators for selected tools only

Filters the list of tool locators so that Mastercam only displays tool locators that are compatible with the selected tool from the Tools in the machine group window.

Machine tooling setup

Displays a schematic model of your machine tooling components. Individual components and groups of components are organized in a tree structure. Right-click in the tree to edit properties for tool locators or individual tool stations and access other options.

Load tools from filtered list

Click to load all of the tools that are displayed in the Tools in the machine group list. Mastercam searches for and selects compatible tool locators from the current component library.

Load machine configuration

Click to browse for and load a machine configuration file. Mastercam displays the Load Machine Configuration function panel where you can select which tools and tool locators you want to import.

Save machine configuration

Click to create a machine configuration file and save it. Mastercam displays the Save Machine Configuration function panel where you can select which tools and tool locators you want to save.

Right-click menu options

Properties

View or edit properties for a tool locator. (This is not available for turret or tool components.)

Set projection length

Return to the graphics window to adjust how far the tool projects from the holder.

Remove

Unload the selected tool.

Remove all children

Unload all tools or components inside the current node in the component tree.

Remove component and all children

Unload all tools or components inside the current node in the component tree plus the selected component.

Remove tools

Unload all tools in the component tree.

Remove empty locators

Remove from the turret any tool locators that do not have a tool loaded in them.

View component

Display the component's solid model in the graphics window.

View component and all children

Display the component and all the components attached to it in the graphics window.

View all

Display all the components from the component tree in the graphics window.

Expand all

Expand all of the nodes in the component tree.

Collapse all

Collapse all of the nodes in the component tree.

Export to XML file

Generate an XML file with information about the components in the tree. This is only used by Mill-Turn machine developers.
