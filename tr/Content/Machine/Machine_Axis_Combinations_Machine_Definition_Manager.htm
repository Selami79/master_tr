Machine Axis Combinations dialog box: Machine Definition Manager

Use the Machine Axis Combinations dialog box to define valid axis combinations for machines that have redundant axes. Axis combinations tell Mastercam which Z-axis is being used. In addition, you can define a home position and reference points for each axis combination. To use an axis combination, click Axis Combo's on the Toolpath Parameters tab when creating a toolpath, or select the Axis Combination page.

Mastercam automatically creates a default axis combination for any machine definition that you create. In many cases, the default axis combination will work as it is. However, depending on the complexity of the machine tool, you may need to set up additional axis combinations. Axis combinations must meet the following requirements:

  • An axis combination requires two component groups.
  • Each component group must end in either a part-holding component or a tool-holding component.

Click on a drop-down below to see product-specific notes.

Lathe

Lathes with multiple spindles and turrets typically require axis combinations to pair tooling with the correct spindle. For example, an axis combination might pair the left spindle with the upper turret.

Router

For router machine definitions that have multiple heads, the head positioning offsets are passed to the NCI through the Multiple Head Selection dialog box in the operation's Toolpath parameters tab. You do not need to set up all possible axis combinations for machines with multiple heads.

Mill

Most mills do not require axis combinations. However, one example is when your machine includes a programmable quill. In this case, you need to define axis combinations to distinguish the quill from the regular Z axis.

Wire

A typical wire machine definition consists of three component groups: the machine table group, the upper wire guide group and the lower wire guide group. Both wire guide groups can support two sets of linear axes: U and V for the upper guide, and X and Y for the lower guide. For these four linear axes to work together, they need to be part of the same axis combination.

Start position

The start position represents the default start point for wire operations. Mastercam reads this start position from the machine definition into each wire operation. You can override this start position in the operation through the Start Position page in any wirepath dialog box.

Parameters

Name

Enter the name that will be used for the component in the machine definition component tree.

Components used

This pane displays all the components in the machine definition. Select a component to add to the axis combination by clicking the checkbox. Mastercam requires that an axis combination contain components from two component groups, a part-holding group and a tool-holding group.

Data stream

For Mill-Turn machines, which support multiple stream outputs, this defines the data stream for NC output for the selected axis combination.

Reference points

A reference point is a location that the tool moves to between the home position and the start or end of the toolpath. Reference points that you assign to an axis combination are read into the Reference Points dialog box for an operation.

Home position

The home position is the position the axes retract to for a tool change. You can assign a default home position for the axis combination.

Right-click menu options

Axis combination

When an axis combination is selected the following options are available. With no selection, only New is available.

  • New adds an axis combination to the list.
  • Delete removes the selected axis combination from the list.
  • Rename lets you change the name of an axis combination.

Components used

  • Expand all/Collapse all opens and closes the component groups in the tree.
