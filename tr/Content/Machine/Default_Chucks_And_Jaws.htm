Default Chucks, Jaws, and Collets

Mastercam Mill-Turn users can create and store multiple sets of chucks, jaws, and collets. These are saved in the same component libraries (.mcam-gmd files) as the tool locators. Users can also designate which chucks and jaws will be used as the default for each spindle.

Mastercam applies the default selections when it loads the .machine file and creates a new machine group. This means that the component library with the default chuck or jaws needs to be saved as part of the .machine file, instead of a stand-alone library.

Mastercam applies the default chuck or jaws selection when it builds the Job Setup and related planes for your Mill-Turn part. If you wish, you can select a different chuck or jaws in the Machine Configuration page in Job Setup.

To designate which chuck and jaws will be the default for each spindle, follow these steps:

  1. While the .machine file is open in Code Expert, double-click the component library to open it in Mastercam.

  2. Drag the desired chuck and jaws components from the Components window to the Default Components window and drop them on the desired node in the tree.

    • If a component already exists on that node, Mastercam will replace it.
    • Removing or replacing one of the default components does not remove it from the library.
    • When it is OK to drop the component, Mastercam displays the following icon:

    NOTE

    In many component libraries, the chucks and jaws are named "Left" or "Right." However, for most applications, you can use any chuck or jaws on either spindle (or select it as the default) and Mastercam will orient the component properly.

  3. Save the component library.
  4. When you return to Code Expert, save the .machine file.

You can also choose to use a collet as the default workholding component. However, you need to ensure that a collet chuck is selected as the default chuck, not a chuck for jaws.

  1. Right-click on the chuck component and select Delete component and children.

  2. Drag a collet chuck onto the spindle node.
  3. Drag the desired collet onto the collet chuck.

Most .machine files include both U.S. and metric component libraries.

  • If you are a developer building the .machine file for others, the best practice is to repeat this in both libraries so that the default selections are the same.
  • If you are an end-user who always works in the same units, you are not required to make the change in both files.
NOTE

The ability to select default chucks, jaws, and collets is limited to Mastercam Mill-Turn. You cannot do this in Mastercam Lathe.
