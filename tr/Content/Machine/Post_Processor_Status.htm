Post Processor Status

Before you can create an NC program using a particular post (.pst file), Mastercam needs to know that it has been properly configured. A properly configured post meets three criteria:

  • A control definition has been created which associates the post with a control type and saved to a control file.
  • Mastercam knows where the .pst file is stored on your system and has verified that it is actually there.
  • A header exists in the .pst file with the control type and the name of the control file. This is followed by the post text for that control.

When you select a post to work on in the control definition, Mastercam displays an icon next to it which tells you its status.

Refer to the following table for an explanation of each possible status icon:

Image Description
The .pst file exists and the post has been properly configured in the control definition. Status is OK for posting. This is the only status that is approved for posting.
Mastercam can't find the .pst file and the post has not been configured in the control definition.
The .pst file exists and the post has been added from the edit list. You see this icon when you first select a post from the list. Save the control definition to a control file to change its status to OK.
The post has been configured in a control definition, but Mastercam can't find the .pst file. This can appear if you move the Mastercam file to another workstation which does not have the necessary .pst file loaded on it.
The .pst file exists, but has not been configured for use with a control definition or Mastercam is not sure that it is the right post for that control. Select the post and create a control definition for it. You can use it after it has been saved to a control file.
Unknown; Mastercam is unable to determine its status.
NOTE

All of the comments in this topic apply equally to binary post files (.psb files).

The post status icons do not indicate that a post has been properly customized for a particular machine or application. Creating a control definition and configuring the post as described here is not a substitute for writing, editing, and properly testing the .pst file to make sure that its output is compatible with a particular machine. That must be done in addition to the steps described here, and is outside the scope of this help file. For more information, contact your Mastercam Reseller.
