Component Position/Orientation On Machine tab: Machine Component Manager (Mill-Turn)

In Mastercam Mill-Turn, use this tab to adjust the position of the solid model for this component relative to the rest of the machine model.

This tab is rarely used. It is typically only used if the solid model for a specific component has not been drawn in its proper position. For example, this might happen if you get a CAD model for a specific component directly from the manufacturer instead of drawing it yourself.

If the component is already drawn in the proper position, the Component is drawn in position on the machine option should be selected.

One scenario when this tab is required is when you are defining a right-spindle chuck for a Mill-Turn machine. In this case, you need to tell Mastercam where it is located relative to the main spindle origin.

  1. Deselect Component is drawn in position on the machine.
  2. Enter the distance from the main spindle origin to the back face of the chuck. For a horizontal lathe, this is usually a positive X value, measured in mm.
  3. Set the View to Bottom.
Positioning the component
  1. Deselect the Component is drawn in position on the machine option. This activates the other options on the tab.
  2. Select the Position on Component location. This will be a reference point that you will use to position the component. For example, for a cylindrical component such as a spindle, this position might lie along the centerline on one of the faces. Click Select to choose a point on your model from the graphics window.
  3. If necessary, select a plane in the Orientation on Machine section. Use this to flip the model on its side, for example, or position it on a slant relative to the rest of the machine.
  4. Select the Position on Machine. This locates the Position on Component point relative to the rest of your machine geometry.
  5. Preview your stock model and make sure it is properly positioned relative to your part geometry. Use View component on the toolbar to see the component relative to either the component group, the entire machine, or by itself.
Parameters

Name

Enter the name that will be used for this component.

Component is drawn in position on the machine

Select this option if you do not need to move the component from where it has been created or defined on the Geometry tab. Selecting this option disables the other controls and options on this tab. If you are using imported solid models, and they have already been drawn in the proper position and orientation, you should select this option.

Position on component

Click Select and choose a point on your component that you will use as a reference point for positioning it on the machine. The location that you select here will be drawn at the Position on Machine coordinates. Choose a convenient location that lets you easily locate it relative to your other machine components. This point is not otherwise used by programming operations or events, it is only used in this tab. Components that require a reference location for programming purposes typically specify that on the Parameters tab. If you are working with a lathe, remember that this position is specified in 3D world or machine coordinates, not 2D lathe coordinates, so that a typical spindle center line will lie along the X axis, not Z.

Orientation on machine

Click Select to choose a plane. The solid model defined on the Geometry tab will align with the plane that is selected here. If the solid model is already drawn in the proper orientation, you do not need to select a plane here. Select a plane only if you need to change the orientation of the solid model. The origin of the selected plane does not matter. The location of the solid is determined by the base point from the Geometry tab, as adjusted by the Position fields on this tab.

Position on machine

These coordinates represent the location of the reference point on the stock relative to the origin of the machine coordinate system. For example, if you are defining a lathe chuck, the Position on Stock might be (0,0,0), representing the center line of the back face of the chuck. If the chuck will be mounted on a left spindle, 20 inches to the left of your machine zero, the Position on Machine would be (-20, 0, 0). The location of the zero point for your machine definition depends on how you have defined your machine base and other coordinates, and will vary from machine definition to machine definition. If you are working with a lathe, please remember that this position is specified in 3D world or machine coordinates, not 2D lathe coordinates, so that a typical spindle center line will lie along the X axis, not Z.

View machine

Select this option to display the machine geometry when selecting the component's position in the graphics window. This is useful when the point that you want to select is located on another component. De-select this option to hide the geometry while you select the location. This is useful when the location that you want to select is hidden by another component.

Select

Click to go to the graphics window and select the position on the machine.

= Position on stock

Click to copy the coordinates from Position on Stock to the Position on Machine fields.

NOTE
  • If your component has been automatically positioned by another component—for example, chuck jaws—all the options on this tab will be disabled.
  • All coordinate positions on this tab are relative to the machine zero point. If you select a new plane in the Orientation on Machine section, Mastercam uses its planar orientation only. The origin of the selected plane does not affect the coordinate positions entered on this tab.
