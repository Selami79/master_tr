Parameters tab: Machine Component Manager (Aggregate)

Use the Parameters tab to add aggregate tooling to a Mill or Router machine definition. The aggregate component requires a spindle as a parent component.

Follow these general steps to define an aggregate:

  • Select the type of aggregate.
  • Add the stations to the aggregate by right-clicking in the Stations window. Add a tool to each station by selecting the station, right-clicking, and then selecting a tool from the library.
  • Enter the angular position (machine plane angle), the tool axis length, offset from the spindle Z axis, and a work offset, if any, for each station.
  • Enter the coordinates of the anchor point for the block on the Position/Orientation On Machine tab .
About aggregate tooling for Mill and Router

Mastercam Mill and Router organize aggregate tooling information in a hierarchy of heads, blocks, and tool stations.

  • Heads are complete assemblies that respond to axis commands and move as a unit. The router spindle component is a head.
  • Blocks are mounted on heads and hold the individual tool stations. An aggregate component is a block.
  • Tool stations are mounted on the block and hold individual tools.
NOTE

Mastercam Mill-Turn also supports tooling blocks and milling heads but differently than Mill and Router. This topic applies to Mastercam Mill and Router only. See Angled Milling Heads and Milling Tool Locators (Mill-Turn) to learn more.

Parameters

Name

Enter the name that will be used for the component in the machine definition component tree.

Type

Select the type of aggregate block that matches your machine tool. The vertical selection provides a method for defining a vertical block with an offset.

Stations

Displays the positions associated with the aggregate head. Use the right-click menu to manipulate this list.

Block number

Enter a number that will be used to identify the block or aggregate head. This often corresponds to a register number in the control. Assign a different number to each block or head that you create. If you load multiple blocks, Mastercam will organize them by block number.

Orientation

Select Fixed for an aggregate block that does not perform any rotary motion. Select Rotary for an aggregate that has an adjustable rotary mechanism that is non-programmable.

Tool change type

Select the type of tool change mechanism used by this block.

Machine view angle

Enter the angular position of the tool station. Many aggregate blocks have tools at two or four positions: 0, 90, 180, 270 degrees. The angle is measured relative to the machine or table, not the active view or tool plane inside Mastercam.

Tilt angle

Enter the angle from the machine view Z-axis to the tool axis. This is enabled only for compound-angle aggregate blocks.

Tool axis length

Enter the length from the pivot point to the collet face (for compound-angle blocks) or the center point of the block to the collet face (for right-angle blocks).

X-Y-Z offset

For an aggregate head, enter the distance between the center of the spindle axis and the center of the tool axis. For a block drill, enter the distance along the X axis between the lead drill and the currently selected drill. The lead drill has a 0 offset, and the remaining drills are offset from 0. For a secondary or piggy-back head, enter the distance that this head is offset from the main head.

Work offset

Enter the code for a work offset that you want associated with the selected station. The work offset code will be automatically output when you select the tool for an operation unless it is overridden in the toolpath parameters. Assigning an offset is optional, and typically depends on your machine or post requirements.

Right-click menu options

Create station

Adds a station to the aggregate.

Create tool

Adds a tool to the station. Opens the Create Tool wizard.

Edit

Available only when a tool is assigned to a station. Opens the Edit Tool wizard.

Get tool from Tool Manager

Opens the Tool Manager so you can select an existing tool for the aggregate.

Cut/Copy/Paste

Use to move a station in the tree, or to create a new station based on parameters of an existing station.

Delete

Removes a station or a tool.

Rename

Adds a descriptive name to the station.
