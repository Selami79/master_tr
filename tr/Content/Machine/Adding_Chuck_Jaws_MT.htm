Adding Chuck Jaws and Collets (Mill-Turn)

Creating a collet or a new set of chuck jaws and mounting them on your machine typically requires two separate tasks:

  • Define the new jaws or collet and add them to a component library.
  • Select the new jaws or collet to mount on the machine.

Defining the New Component

Follow these steps to define chuck jaws or a collet in Mill-Turn. These steps assume that you are adding them to one of the component libraries that are part of your .machine file.

  1. In Code Expert, double-click the desired component library.

    Mastercam opens the component library in a new session of Mastercam.

  2. Right-click on a Chuck Jaws Group or Collet Group and select Add chuck jaws or Collet.

    I don't see a Collet or Collet Chuck group in my library

    If your component library does not include a Collet or Collet Chuck group, follow these steps to add one. For example, if your .machine file has been migrated from an earlier version of Mastercam, these groups might not be present.

    1. Right-click on Component library at the base of the component tree and select Groups.

      The Groups dialog box displays.

    2. Select each group that you want to include in the library. Remember that if you want to mount a collet on your machine, you will also need a collet chuck component, not a jawed chuck.
    3. Click OK.
  3. Click the Geometry tab. Use it to decide how you want to represent the shape of the component.
  4. Select Parametric or Solid entity from the Geometry drop-down. For chuck jaws, you can also choose a Chain option. Expand the following drop-downs to see the next steps. As you work, the Preview window updates in real-time to show the overall profile of the jaws.

    Defining your jaws with a wireframe profile

    You can model your jaws by selecting a chained profile of wireframe geometry. This technique is often used when your jaws have been machined to fit the shape of your part.

    Select Chain from the Geometry list, then click the Select Geometry button to select the geometry from the graphics window. The geometry should be a 2D profile drawn in the upper quadrant. No part of the profile should cross the spindle center line.

    You can create either rectangular or pie-shaped jaws with this method. See Geometry tab: Machine Component Manager (Chuck Jaws (Chain)) to learn more.

    Defining your component with a solid model

    You can model your collet or jaws by selecting a solid model. The model can be either a Mastercam solid on a level in your part file, or an external file in .STEP format.

    Select Solid entity from the Geometry list, then select the model.

    • In the picture below, click the left button to select a model from an external file.
    • Click the right button to select a model from the Solids Manager or the graphics window.

    Use the Align function panel to orient the model and align it with the chuck and spindle, then enter the other settings on this tab. See the following topics to learn more:

    • Align Chuck/Jaw/Collet panel
    • Geometry tab: Machine Component Manager (Collet (Solid))
    • Geometry tab: Machine Component Manager (Chuck Jaws (Solid))

    Defining your component parametrically

    Select Parametric from the Geometry list to define the collet or jaws by entering their dimensions directly in this dialog box. This lets you model them without creating or selecting any geometry.

    1. For jaws, choose either the Rectangular or Pie-style option and enter the related dimensions.
    2. Click the Steps: Initialize button to begin defining the jaw steps.

    3. Refine the dimensions for each step by entering the values directly in the grid. Use the other buttons to add or remove steps or invert the order.

    See Parameters tab: Machine Component Manager (Chuck Jaws) or Geometry tab: Machine Component Manager (Collet (Parametric)) to learn more.

  5. Click the Parameters tab to define the location and clamp position.

    1. Choose whether to clamp the part on the Inside diameter or Outside diameter.
    2. Click the Select Reference Point button and select a point on the profile. This will be the programmed point for chuck or part-handling operations. It aligns with the clamping position on the part.
  6. Use the live Preview window to see the changes as you enter new values. Right-click in the window to change the view and see other options.
  7. Click OK when you are done and close the Component Library dialog box.
  8. Close the Mastercam session and return to Code Expert.

Selecting Collets or Jaws and Mounting Them on Your Machine

Once the collets and jaws have been created and saved to a component library, use the Machine Configuration page in Job Setup to mount them on your machine. You can also select and use components that have been saved in a library different from the one that is included with your .machine file.

  1. Click Job Setup in the Toolpaths Manager.

  2. Select the Machine Configuration page.
  3. Right-click on the Chuck Jaws node for the desired spindle, and choose Select new chuck jaws or Edit chuck jaws. The same steps apply to collets.

    • Select new chuck jaws lets you select a set of jaws that have been saved in the current component library. You can also select and open a different component library.
    • Edit chuck jaws lets you redefine the chuck jaws' shape, profile, and other settings. Note that any changes made here apply to this part only; they will not override the settings of the jaws that are saved in the component library.

    There are additional steps to complete if you are replacing jaws with a collet or vice versa. See Job Setup: Work Holding page to learn more.
