Machine Reference Position Manager

NOTE

This feature is only used for Mastercam Mill-Turn machines. If you are using any other Mastercam product, you do not need to use this feature.

Licensed Mill-Turn Developers use the Machine Reference Position Manager to define named reference positions that can be called by name when working with the .machine file. These include:

  • the home position for each turret
  • tool change positions
  • the starting and ending positions for an operation
  • locations for pre-positioning the tool for cutoff operations
  • turret park locations

These reference positions are stored in the machine definition that is associated with the .machine file. They can only be created and edited by licensed Mill-Turn Developers.

End users who are not licensed Developers can create their own custom reference positions or turret park locations. Do this to meet requirements specific to an individual part, such as accommodating fixtures or unusual part geometry. Reference positions that are created by end users are stored in the Mastercam part file, not the .machine file. This means that they are not available in other parts.

To create a new reference position, right-click in the list of reference positions, and select either New reference position or Duplicate selected reference position.

Parameters

Reference Positions

Displays the currently defined machine reference positions. Selecting a position initializes the controls below, allowing you to edit/define values for the selected position. Right-click in the control to bring up a menu that allows you to add, delete, or rename items in the list. The right-click menu for a reference return also lets you set the default for the specified axis combination.

Name

Displays the name of the selected reference position.

Comment

Enter a comment for the reference position.

NC code output

Specifies the optional NC code to be output. This is only available to licensed Developers.

Type

Choose a type to make changes to machine positions.

Axis combination

Displays all of the axis combinations and machining modes that have been defined in the current machine definition. Select the desired axis combination from the drop-down.

Linear axes / Rotary axes

These controls allow you to define the axis positions for the individual machine axes. These values are set in units (either U.S. or metric) for the linear axes and degrees for the rotary axes. You do not have to specify motion for all axes. You can choose to:

  • Enter an exact coordinate for the selected axis
  • Drive the axis to either of its travel limits
  • Position the axis relative to either of its travel limits
NOTE

If you select the travel limits options, the coordinate will be read from the appropriate machine axis component (minimum/maximun travel limit) when the reference position is used in an operation. Units are the units in which the machine is defined. Units are converted when the Machine Definition is read into a part file with different units.
