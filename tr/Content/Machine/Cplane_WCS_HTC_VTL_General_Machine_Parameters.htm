Cplane, WCS, HTC/VTL tab : General Machine Parameters

Use the Cplane, WCS, HTC/VTL tab to establish the default coordinate system for your lathe. The planes that you select here will be activated automatically whenever this machine is selected for an operation. This tab is used for Lathe machine definitions.

Default Cplane

Click on the desired plane or coordinate system.

  • To use diameter coordinates, select the DZ option corresponding to the desired quadrant—for example, +D+Z for the upper-right quadrant.
  • To use radius coordinates, select the XZ option corresponding to the desired quadrant—for example, –X+Z for the lower-right quadrant.
  • Select one of the standard planes (for example, Top, Front, or Back) to establish a 3D coordinate system in the desired orientation.
  • Select None to use whatever tool and construction planes are active when the machine is selected.
Parameters

Default Cplane

Select a construction plane here to be automatically activated when this machine is selected. This also sets a matching tool plane. Only system-defined, not user-defined, planes are listed.

Orientation

Select the lathe type. Selecting Vertical turret lathe will display the X axis vertically and the Y axis to the left, which automatically creates graphics views so that you can view the part as it would appear on the VTL. Selecting VTL will not affect the Cplane, WCS, or tool planes.

Update WCS using

Use this setting to automatically set the work coordinate system (WCS) when this machine definition is loaded.

  • None leaves the coordinate system as it was set in the part file before you selected this machine definition.
  • Top sets the WCS to Top. Use Top for programming parts for a lathe that are already drawn with their axis of rotation about the world X axis. For most 2D turning applications, the lathe Z axis equals the world X axis.
  • Lathe Z = World Z assigns a WCS created specifically for programming parts for a lathe that are drawn with the axis of rotation about the world Z axis. When using this as a WCS, the part axis of rotation is the WCS X axis, as required for Lathe toolpaths.
