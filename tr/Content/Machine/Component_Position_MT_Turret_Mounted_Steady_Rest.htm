Position tab: Machine Component Manager (Mill-Turn Turret Mounted Steady Rest)

Before you can use a steady rest, it must be properly mounted on the turret. Use the settings on this tab to select the desired index position and provide other connection information.

  • Select the proper Attachment Position. Attachment positions define where the point that is the origin of the model mates to the turret at each index position. These are typically defined by the machine developer who configures the turret component in the component library. If every index position on the turret is the same, the turret might have only a single attachment position. More complex turrets might have additional positions defined.

    Attachment positions are used by both tool locator and steady rest components; see Working with Attachment Positions to learn more.

  • Select the index position on the turret where the steady rest will be mounted from the Turret Index Mounting Position list.

  • Some steady rests are large enough to span several index positions. If this is the case, use the Disabled Index Mounting Positions list to select any index positions that cannot be used when the steady rest is mounted. This ensures that Mastercam will not try to load a tool in those positions. The position selected as the Turret Index Mounting Position is disabled automatically and does not appear in the list.

Troubleshooting connection problems

Complex turrets might have several sets of attachment positions that let Mastercam mount different types of components in different ways. If several attachment positions are available and you do not know which one to choose, you can inspect the properties of each by examining the turret component.

  1. Double-click the turret component to display its properties.

  2. Select the Tool Locator Positions tab. The positions are displayed in the Tool locator attachment positions window.

  3. Right-click on a position and select Modify to review its properties. For example, the attachment position shown here is intended for odd-numbered positions only and should not be selected if you intend to mount the steady rest in an even-numbered position.

It is also common for attachment positions to be disabled for specific positions that the developer wants to leave open for turret park moves. You should be aware of this before selecting the index position for the steady rest.

It is also possible that the proper attachment position does not exist, and you or the machine developer will need to create one. If this is the case, you must create the attachment position in two places:

  • the turret component stored in the machine definition (.mcam-lmd file)
  • the turret component stored in the component library (.mcam-gmd file)

Note that only users with a Developer license can edit a Mill-Turn .mcam-lmd file.

Finally, whenever you import a steady rest component from another library, the recommended best practice is to re-select the attachment position. This is because the turret component in the source library might have a different set of attachment positions than the destination library.

NOTE

This topic only applies to Mill-Turn steady rests. Steady rests for Mastercam Lathe are supported differently. See Adding a Steady Rest (Lathe) to learn more.

NOTE

This topic only applies to turret mounted steady rests. Standalone steady rests have different Position settings. See Position tab: Machine Component Manager (Mill-Turn Standalone Steady Rest) to learn more.

Parameters

Name

Enter the name that will be used for this component.

Attachment Position

Attachment positions identify the location on the turret index position that mates with the origin of the model. Many turrets only have a single attachment position, but if your turret has more than one, select the desired position.

Turret Index Mounting Position

Select the index position where you would like to mount the steady rest. Only primary index positions are displayed in this list, you cannot select a half-index position.

Disabled Index Mounting Positions

If mounting the turret causes any other index positions to be inaccessible or otherwise not available for mounting tools, select them in this list.
