Local Machine Definition dialog box

Use the Local Machine Definition dialog box to work with the machine definition from the current Mill-Turn machine group. You can do any of the following:

  • Preview the solid models of a component, a component group, or the entire machine.
  • Edit common machine settings, such as feed rate limits.
  • Create new reference or turret park positions.
  • Inspect properties of any machine component.

To open this dialog box, click the Edit button on the Files tab in the Machine Group Properties dialog box.

In Mill-Turn, the machine definition (.mcam-lmd file) is serialized inside your .machine file:

Mastercam stores a copy of this .mcam-lmd file in your part file as part of the machine group. The Local Machine Definition dialog box lets you work with this copy—not the master copy stored in your .machine file. Because of this, any changes that you make in this dialog box apply only to this part file. There is no way to push changes to your .machine file.

Right-click in the large window and select Expand all to see the components in the machine definition.

NOTE

This dialog box is only used in Mill-Turn.

Displaying the machine or components

This dialog box gives you several ways to display your machine or its components.

  • Right-click on any component and select View to see it displayed in the graphics window. You can also click the View Component button on the toolbar:

  • Right-click on any component and select View component group to see the entire group displayed in the graphics window. You can also click the View Component Group button on the toolbar:

  • Right-click on Machine Base and select View to see the complete machine displayed in the graphics window. You can also click the View Machine button on the toolbar:

  • You can hide any component or group by clicking on the yellow box in the component tree before viewing it:

Right-click menu options

Not all of the options listed here are available for all nodes in the tree.

Properties—Inspect the properties for the selected component.

View—Display the component's solid model in the graphics window.

View component group—Display the solid models for all of the components in the component group. In the component tree, the component groups are bold; for example, this machine definition includes four component groups:

View component and parent—Display the solid models of the selected component and the component immediately above it in the tree.

Expand group—Display all of the components in the selected component group. Right-click on the group name to see this option.

Collapse group—Collapse all of the nodes in the selected component group. Right-click on the group name to see this option.

Expand all—Display all of the components in the tree.

Collapse all—Collapse all of the nodes in the component tree.

Toolbar buttons
Icon

Description
View entire machine: Displays all the components in the machine definition.
View component group: Displays all the components in the current component group.
View component: Displays the solid model of the selected component.
General Machine Parameters: Opens the General Machine Parameters dialog box for editing. Use this to view or edit settings such as feed rate limits.
Machine Reference Positions: Opens the Machine Reference Position Manager so that you can create custom reference positions or turret park locations for this part.
Axis combinations: Inspect the axis combinations that have been defined for this machine. You can see which components have been assigned to each axis combination, but you cannot make any changes.
