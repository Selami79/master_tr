Parameters tab: Machine Component Manager (Spindle Tool Locator)

Use the settings in this tab to configure a tool locator for turning tools that will be mounted in a tool spindle or B-axis head.

Without a tool locator, Mastercam assumes that turning tools mounted in a B-axis head or tool spindle have the insert tip aligned with the spindle center line. This is typically not true for turning tools that have a "stick" holder or shank. Spindle tool locators let you more accurately simulate the tool by showing the actual position of the insert, in addition to the holder block itself.

Each tool locator component includes at least one tool station. Each tool station identifies a group of settings that together tell Mastercam how to load a tool for a specific orientation or application. For example, a tool locator might be able to support both insert-up and insert-down tools. For this application, you should define two stations, one for each orientation. As a general guideline, most tool locator components have between one and four tool stations.

Solid models for spindle tool locators

Each tool locator component includes a solid model. This represents the block or fixture which sits between the face of the spindle and the tool holder or shank. The model is displayed in Simulation and is used for collision-checking.

  • The zero point for the tool locator model should match the center point of the face of the tool spindle. This location typically aligns with the Tool Reference Position from the Tool Spindle properties.

  • Orient the model so that the tool axis is parallel to the Y axis and points to –Y.
  • Orient the model so that the base of the locator (which is the same as the face of the tool spindle) is parallel to the XZ plane.

This picture shows a typical spindle tool locator, properly drawn and oriented. It has three stations so it can accommodate up to three different tools.

Editing tool definitions to support spindle tool locators

Lathe tool definitions include a special setting to support spindle tool locators.

When the Requires tool locator option is selected, Mastercam’s automatic tool loading routines will look for a spindle tool locator before loading the tool.

If this option is not selected, the tool will still be loaded, but it will be loaded directly in the spindle, without a tool locator.

Because spindle tool locators were not used before Mastercam 2018, if your tool libraries have been migrated from earlier versions of Mastercam, this option might not be set properly. This option is not selected by default.

NOTE

If this option is not selected and the tool is loaded directly in the spindle, you can still select a tool locator manually in the Tool Setup Manager.

NOTE
  • Spindle tool locators are only used for turning (Lathe) tools. Milling operations in Mastercam Mill-Turn use the same holders and holder libraries as Mastercam Mill.
  • Tool locators for tool spindles or B-axis heads are a different component type than tool locators for turrets. See Geometry tab: Machine Component Manager (Turret Tool Locator) .
  • Tool locators are only used in Mastercam Mill-Turn.
Parameters

Name

Specifies the tool locator's name. See Naming Conventions for Tool Locators for guidelines on creating proper tool locator names.

Comment

Enter a comment that describes the tool locator. This will be visible in the Tool Setup Manager and should help the user decide if this tool locator is appropriate for their needs.

Stations

Lists the stations on the locator. Right-click to add or remove stations.

Name (Station details)

Enter a name for the currently selected station.

Comment (Station details)

Enter a comment for the currently selected station.

Rotation angle

The angle around the tool spindle axis at which the station is located. This is usually the A-axis rotation for the station. The zero position is relative to how the model is drawn.

Allowable tool types

Select the types of tools to be allowed in the station. For most stations, you will only select one option.

Tool orientation

Select the type of tool holder: Axial, Radial, or Angular.

Tilt angle

For Radial or Angular orientations, enter the Tilt angle. This angle is the face of the tool locator, measured relative to the tool axis.

Should I enter a positive or negative angle?

Visualize the tool locator model in top view. Choose a positive or negative angle based on the quadrant of the tool angle face.

  1. +45 degrees
  2. -45 degrees

Tool holder position to match

A reference point on the tool holder or shank that will be mapped to the tool locator. The Tool position on locator is the corresponding point on the tool locator model. These two points will line up when the tool is mounted. Position 5—the center of the shank—is generally used for round tool holders.

Tool position on locator

A reference position on the tool locator model where the tool will be mounted. The Tool holder position to match is the corresponding point on the tool holder or shank. These two points will line up when the tool is mounted.

  • The Y value is typically the length of the tool holder, entered as a negative value.
  • The X and Z values represent the offset of the tool’s programmed point from the spindle center line.

Projection adjustment

Adjusts the stickout length of the tool when mounted in this tool locator. Select the checkbox to activate the field, and then enter the adjustment value. Use positive values to extend the tool from the locator and negative values to retract the tool.
