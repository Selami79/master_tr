Coolant Commands tab: General Machine Parameters

Use the Coolant commands tab in the General Machine Parameters dialog box to define the coolant options for your machine tool and how they will appear to the user in Mastercam’s toolpath interface.

You can define up to 10 different coolant options. For each option, you can set:

  • The name that will be displayed to the user.
  • The name of the “coolant on” and “coolant off” events. The default labels are On and Off but you can enter any label that makes sense for the specific option, such as an Mcode.

Enter two double-quotes ("") to disable any coolant options that are not necessary. Do this to prevent users from selecting coolant options that are not supported or defined.

Multiple coolant settings can be set for a single operation. For example, a long drill may require both thru-tool and flood coolant. You can also use the coolant settings to operate other devices such as an air compressor. Your post processor will need to be modified for you to take advantage of the coolant options.

Customizing a post to support new coolant options is a sophisticated task that should only be undertaken by experienced post writers.

Enabling coolant options

Each coolant option has three strings associated with it:

  • Enter a name for the option in the Event field. The table below summarizes the rules for this string.

  • Enter a label that is used when the coolant option is turned on in the Event Enable field. The default is On but this can be any string you wish.

  • Enter a label that is used when the coolant option is turned off in the Disable field. The default is Off but this can be any string you wish.

Follow these rules for enabling or disabling the option:

Event string Actions
"" two double quotes Disables the coolant option. The Enable/Disable options are disabled if the event is disabled.
Event label is left blank Enables the coolant option and labels it with default text. Enable/Disable options can be set separately.
Event label contains any other text Enables the coolant option and labels it with the text.
Parameters

Support coolant using coolant value in post processor

The ability to define custom coolant options was introduced in Mastercam X. Because the new coolant techniques require a completely different implementation in your post, this option was added to maintain backwards compatibility with older posts. Select this option if your post still uses the coolant logic from Mastercam Version 9 or earlier.

It is important to understand that even if your post has been updated to the latest version of Mastercam, it might still use the legacy coolant logic. This is most likely to happen if the post was originally written for Mastercam Version 9 or earlier and has simply been updated with UpdatePost since then, instead of being explicitly rewritten to support the new coolant logic.

Tree control coolant text

Use this field to customize the text that appears on the Coolant button in the toolpath settings. For example, you might change this to Flushing for a Wire machine. This text is only in toolpaths that have a tab-style interface with a Coolant button, not toolpaths that organize their settings in a tree-style control or function panel.

Event

Enter a name for the option. This enables the option to appear in the toolpath settings interface.

Enable/Disable

Enter text for the coolant on and off settings. The off event will be disabled if there is no text in the corresponding event on field.
