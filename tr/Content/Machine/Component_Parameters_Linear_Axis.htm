Parameters tab: Machine Component Manager (Linear Axis)

Use the Parameters tab to configure a linear axis component—typically, X, Y, or Z. Depending on your post processor and level of Mastercam, you might not need to enter all of the information.

  • The Machine coordinate and Positive axis direction are required for all machine definitions. All users should enter this information.
  • Travel limits and rapid traverse rate limits are only necessary if your post explicitly requires them.
Axis direction and orientation
  • The Positive axis direction is expressed in world coordinates. It maps the Machine coordinate to the physical layout of your machine tool. For a typical vertical Mill or VMC, the axis orientation will match the Machine coordinate. For a horizontal Mill or most Lathes, they will be different.
  • Most horizontal Mills will map Machine coordinate=Z to Y+.

  • A typical Lathe will map Machine coordinate=Z to X+ for a left spindle, or X- for a right spindle.

  1. Coordinate system for horizontal turret Lathe.
  2. World coordinate system in machine definition.
Tilt angle

If the linear axis is positioned at an angle relative to another axis in the machine, choose Tilt Angle to define the angle.

NOTE

You may need to create an axis combination to make use of this axis and pair it with the correct tooling.

Physical motion vs. programmed motion

For some axis components, the physical motion of the component differs from the programmed motion. This typically occurs when the axis component is mounted at a non-orthogonal angle to another axis component. In this case, both axis components need to move and the programmed move is interpolated from both components.

For these components, follow these steps:

  1. Deselect the Programmed axis motion = Physical axis motion option.
  2. Select Physical axis.
  3. Enter values for the following settings:
    • Positive axis direction
    • Tilt angle
    • Defined axis position
    • Travel limits
  4. Select Programmed axis.
  5. Enter another set of values for the following settings:
    • Positive axis direction
    • Tilt angle
    • Defined axis position
    • Travel limits
Travel Limits/Defined Axis Position

These fields are usually only used with Mastercam Mill-Turn.

Use these fields to identify the travel limits of the axis relative to the solid model of the component. Use the Defined position to identify the location of a reference point on the component. Then enter the Minimum and Maximum limits of this point along the axis of travel.

Because the values that you enter in these fields depend on where your model is drawn, you should follow this sequence:

  1. Draw the model and enter its properties on the Geometry tab.
  2. If necessary, make any position adjustments on the Position/Orientation On Machine tab.
  3. Enter the Defined position and travel limits on this tab.

You need to enter both an U.S. and metric set of values. Use Convert to enter one set of values and automatically convert them to the other.

Rapid Traverse Rate

Use these fields to set the motion used for rapid moves. You can set this value two ways:

  • Individually for each linear axis by entering the values here.
  • Globally for all linear axes. In this case, use the Axis traverse rate limits tab of the General machine parameters dialog box.

You should enter both an U.S. and metric value. Use Convert to enter one value and automatically convert it to the other.

Parameters

Name

Enter the name that will be used for the component in the machine definition component tree.

Machine coordinate

Select the coordinate label that is associated with this axis. Identifies the axis associated with this component. This is typically the axis referred to when programming parts and operations. Select from X, Y, Z, or A, B, C. The Macro option is not currently supported.

Output lathe X or Y coordinate as a diameter

Allows you to output diameter instead of radius for the X or Y. Enabled only for lathes.

Define macro-driven axis feed rate in axis motion events

This option is not currently supported.

Axis label: absolute, incremental coordinates

Identifies the coordinate label that will be output in the NC code. Most Mastercam users should select the same coordinate as the Machine coordinate. Separate fields let you select different axis labels for absolute and incremental output.

Programmed axis motion = Physical axis motion

You can enter and save two different sets of axis motion settings. Most of the time, you only need to enter one set of values. In this case, select this option to disable the second set of settings. Deselect this option to activate both sets of settings. Then select Physical axis and Programmed axis in turn and enter the proper values for each. Use two sets of values when two linear axis components do not travel along orthogonal paths. The most common use is for Y-axis components on certain Mill-Turn machines. In this case, the Y-axis component is mounted on the X-axis component at an angle other than 90 degrees. Therefore, in order to create tool motion along the Y-axis that is properly perpendicular to tool motion along the X-axis, when a Y-axis move is programmed both X- and Y-axis components must move together so that the interpolated result is the programmed distance.

Physical axis

After selecting this option, enter the defined position, travel limits, and tilt angle of the axis. These numbers should reflect the position and orientation of the component as it is physically mounted on the machine - in other words, along the ball screw of the linear axis.

Programmed axis

After selecting this option, enter the defined position, travel limits, and tilt angle of the axis. These numbers should reflect the position and orientation of the component's programmed motion. In other words, these values should reflect the programmed positions and orientation of the tool, not the axis ball screw.

Positive axis direction (world coordinate system)

Click the graphic to select the linear axis positive direction in world coordinates. Allows you to set up a machine matrix to map the world coordinates to. For example, when creating a horizontal milling machine and defining the machine Y axis, you would select Y for the Machine coordinate, and Z+ for the Positive axis direction (world coordinate system).

Tilt angle

Select to apply a tilt angle to the axis, for example, the X axis on a slant-bed lathe.

Convert to mm, Convert to in

Use these buttons to automatically enter a millimeter equivalent for a inch value, or vice versa. For example, if you click Convert to mm, the inch value will be converted to millimeter and automatically copied to the metric field.

Minimum, Maximum limit

Enter the minimum and maximum travel limits of the axis. Use your machine tool documentation as a guide. The numbers entered here represent the location along the axis of a specific reference point when the component is at its limits. This should be the same reference point used for the Defined position. This information is typically only used by Mastercam Mill-Turn.

Defined position

Use this value to indicate where the solid model of the component has been drawn. This number indicates a position along its axis of travel, and should be chosen to represent a specific, known reference point on the component. The minimum and maximum travel limits can then be entered for this same point. For example, if you are entering values for an X-axis component, and the coordinates of the reference point on the component are (12,20,15), then you would enter 12 for the Defined position. This information is typically only used by Mastercam Mill-Turn.

From machine

Copies the value from the Axis feed rate limits tab of the General Machine Parameters dialog box.

Rapid traverse rate

Enter the speed at which rapid moves will take place along this axis. Use the left field for inches per minute, and the right field millimeters per minute. If you see the message Set from machine general parameters, then the values for the axis are set in the General Machine Properties dialog box and locked.
