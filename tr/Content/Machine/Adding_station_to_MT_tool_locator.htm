Adding a Tool Station to a Turret Tool Locator

Many tool locators have more than one tool station defined. Each tool station accommodates a specific tool type/orientation. For example, a tool locator might support both insert-up and insert-down tools.

It is important to understand that the different tool stations that are defined for an individual tool locator might or might not correspond to different physical features of the tool block. For example, a single slot might accommodate a tool in either insert-up or insert-down orientations. However, you need separate station definitions to tell Mastercam that the tool holder needs to be rotated 180 degrees and keyed off the opposite corner of the holder.

This picture shows an example of tool locators with multiple stations.

Follow these steps to create a new tool station on a turret tool locator.

  1. If necessary, display the properties of the tool locator. Double-click the component in the Mill-Turn Component Library dialog box to display its properties.

    The Turret Tool Locator dialog box opens with the Geometry tab displayed.

  2. Right-click on the tool locator base component in the Subcomponents window and select Add subcomponent, Add station. You can also click the Add station button.

  3. If you are creating an angled head for a milling tool locator that has a barrel subcomponent, add the station to the barrel, not the base.
  4. Enter a descriptive Name and Comment.

    These will be visible in the Tool Setup Manager and will help your users decide whether the stations on this tool locator are appropriate for their application.

  5. If this is a half-index tool locator, follow these steps:
    1. Select either Primary Minus or Primary Plus from the Uses index list.
    2. In the Offset angle field, enter the angular separation between the primary index position and the half-index position. For example, if the turret has 12 primary stations that are spaced 30 degrees apart, the Offset angle for the half-index position will be 15.

      Enter a positive value only—Mastercam knows the direction from the Plus or Minus indication.

  6. Many milling tool locators are geared to adjust or reverse the programmed spindle speed. If your tool station is for a milling tool locator, do the following:
    1. Enter the scaling factor in the Speed multiplier field. For example, if the locator doubles the programmed spindle speed, enter 2.0. Enter 1.0 if the locator does not scale the spindle speed.
    2. Enter the Speed multiplier as a positive number. Select the Reverse spindle direction option to change the direction.
  7. Select the Tool orientation. This is set separately for each tool station. For example, a single tool locator might have one tool station defined for tools oriented left, and another tool station for tools oriented right.
    Angular orientation

    For turning tool locators, enter the Tilt Angle. Use a positive angle for lower-turret tools that are tilted towards the right spindle, and a negative angle for the left spindle.

    For milling tool locators, the workflow is different depending on whether the tool locator has a barrel subcomponent.

    • If the station does not have a barrel, enter the Tilt Angle just like for turning tool locators.
    • If the station is located on a barrel, the only allowed orientation is Angular. Set the angle in either of two ways:

      • If you know the angle at which the tool will be mounted, select Manual and enter the angle. This field will also be available to end users, who can use the Tool Setup Manager to view the station properties and set this angle to the appropriate value for their part.
      • Select From operation to have Mastercam set the angle from the operation's tool plane. The angle field is grayed out, but once the locator has been selected for an operation this field will display its actual orientation angle.
    Cross tools

    Select Cross Tools Upper or Cross Tools Lower to indicate the desired turret. Then select Neutral, Left, or Right from the Cross orient drop-list.

    Mastercam uses this setting when it tries to select this tool station for tools that are pointing left or right. Neutral means that Mastercam can select this station for tools that are pointing in either direction.

    Note that this applies only to Mastercam's automatic tool-loading routines. You can override this manually in the Tool Setup Manager if you wish; for example, if you are cutting on the back of a part feature.

    Facing tools

    Select Face Tools Left Side or Face Tools Right Side depending on which way you want the tool to point. This setting is linked to the Face orient drop-list so the two settings will match.

    These tool stations are not limited to facing operations, but can be used for any operation where you want the tool axis to be perpendicular to cross-turning.

  8. Select the Allowable tool types. This setting is only used for turning tools.

    Do this after selecting the Tool orientation, since some tool types will be grayed out if they are not supported by the orientation.

  9. Tell Mastercam how to position the tool in the locator. This process is different depending on whether you are working on milling or turning locators.
    • For either type of locator, select the Tool position on locator or enter its coordinates. This identifies a reference point on the model that will be used for mating the tool.
    • For turning tool locators, select the corresponding Tool holder position to match. This identifies a reference location on the tool that will mate with the Tool position on locator. This is not required for milling tools since Mastercam assumes the center of the tool is the proper location.
    • Even if tools for different tool stations appear to mount in the same way, the Tool position on locator and the corresponding Tool holder position to match might be different. For example, insert-up and insert-down tool stations will usually have opposite corners selected for the Tool holder position to match, with the Tool position on locator coordinates shifted accordingly.
  10. Use the Projection adjustment field to control how much of the tool sticks out from the locator. Enter the adjustment amount in the field or click the Target button to return to the graphics window where you can adjust it dynamically.
For my application, do I need to create a new tool locator, or a new tool station?

For a specific tooling need, it can be confusing to decide whether you need to create a new tool station or a completely new tool locator component.

The key consideration is how the physical block is mounted on the turret. If the block needs to be mounted on the turret differently to orient the tool properly, you need to create a new tool locator component. For example, if the block needs to be flipped so that it points to the proper spindle, create separate tool locator components to be used for each spindle. Make a copy of the solid model and transform it so that it is properly oriented to the other spindle, and use it as the starting point for a new tool locator component.

In this example, the tool locator block as it is mounted on the turret can service either spindle. You need to create one tool locator component with separate stations for each spindle.

However, in this example, the tool locator block must be flipped to service the other spindle. You need to create two copies of the model and orient each model for the proper spindle, and then create two different tool locator components.

In most circumstances, you will also need to create separate components for upper turret vs. lower turret use.
