Geometry tab: Machine Component Manager (Chuck Jaws (Solid))

Use this tab to define your chuck jaws with a solid model. The model can be either a Mastercam solid on a level in your part file, or an external file in .STEP format. For most applications, only Mill-Turn users will need this tab. Most Lathe users will use the Parametric method to define their jaws.

  1. Select Solid entity on the Geometry tab for the chuck component.
    • In the picture below, click the left button to select a model from an external file.
    • Click the right button to select a model from the Solids Manager or the graphics window.

    Mastercam opens the Align Chuck Jaw function panel.

    What if more than one solid is available?

    Each component must be represented by only one single solid model. If your component model consists of more than one solid body, you must merge or combine them first.

    If your file contains models for more than one component—for example, a single file might contain models of both the chuck and jaws—select only the single model for your component and ignore the others.

    NOTE

    If your file contains models of each individual chuck jaw, select only one model for a single jaw. Mastercam automatically creates the proper number of jaws for you.

  2. Use the Align Chuck Jaw function panel to align the model so that it mates cleanly with the chuck. Click OK when you are done.
    • Any chuck jaw component can be loaded on either the left or right spindle. The Z-axis gnomon displayed in the graphics window should point to the positive Z direction. Mastercam flips the model as needed to maintain the proper Z-axis direction depending on which spindle you mount the jaws.
    • By convention, components are defined as if they are in the left spindle.
    • For most applications, click on the back face of the model so that the Z-axis arrow points to the front, and set the zero point along the inner diameter of the component. See Align Chuck/Jaw/Collet panel to learn more about using the function panel.

  3. Mastercam displays the model in the Preview window when you are done aligning it.
    • Right-click in the Preview window to see the chuck in different views to ensure that it is aligned properly. The following pictures show a properly oriented chuck jaw model in Top and Isometric view.

    • If the model is not correct, click the Align button to return to the graphics window, where you can use the Align Chuck Jaw function panel to change the orientation of the model.
  4. Complete the other settings on this page.
NOTE

If the model has been selected from your part file, Mastercam moves the model from its original level to the last available level in your file. If you import a model, Mastercam copies the selected solid model to your machine definition or component library when you have finished creating your chuck component. This means that there is no associativity between your chuck component and the original model, even if the model is on another level in your part file.

How many jaws will be created?

The number of jaws that will be created is determined by the Jaw configuration selection in the chuck settings.

If this option has not been set, the default is three jaws.

Parameters

Name

Enter the name that will be used for this component.

Geometry

Tell Mastercam how you want to model the component.

  • Select Solid entity to display the options described in this topic.
  • Select Chain to model the chuck jaws by selecting chained wireframe geometry. See Geometry tab: Machine Component Manager (Chuck Jaws (Chain)) to learn more. Most Mill-Turn users will select this option or Solid entity.
  • Select Parametric to model the jaws by entering dimensions directly in this dialog box. Most Lathe users will select the Parametric option. See Geometry tab: Machine Component Manager (Chuck Jaws (Parametric)) to learn more.

Color

Select the color for the component. This color will appear during simulation and when the component is previewed on the screen.

Chord tolerance

Use this setting to control how finely Mastercam will draw the model that represents this component. A smaller value will result in a finer, more detailed simulation; however, this could slow down simulation. If your workstation's performance during simulation is sluggish, consider increasing this value.

Translucency

Use this option to make the component appear more or less translucent when it is displayed on the screen. This will be applied both when the component is previewed and when it is used in simulation.

Set name from geometry

Click to populate the Name field with the name read from the model.

Set color from geometry

Click to populate the Color field with the color of the model.

NOTE

If you want to set the component's Name or Color from the component model, you must select these options before you select the model.

Buttons for working with solid models

Icon Description
Select a file that contains the solid model from your workstation.
Select a solid model from the graphics window or from the Solids Manager.
Open the Align Chuck function panel, which lets you change the alignment or orientation of the solid model. If more than one solid is available, you can also select a different solid.
Optimize the solid model.
Repair small faces.

Total height

The height of the model. Mastercam reads this dimension from the model but you can override this with your own value if necessary.

Total width

The width of the model. Mastercam reads this dimension from the model but you can override this with your own value if necessary.

Spin

Generates a boundary by spinning the model and selecting the furthest extents.

Slice

Generates a boundary from a cross-section of the model at the zero point.

Compare in preview

Select this option to display both Spin and Slice boundaries in the Preview window. Mastercam draws the two boundaries in blue and red. Select Display boundary in the Preview window to see the boundaries.

Preview window

Displays a live interactive 3D model of the component. Right-click in the window to change the view, dynamically rotate the component, or zoom in on component details. Depending on the type of component, other buttons or controls are available for additional preview options.

The Preview window also uses the Translucency setting found on the Geometry tab. Adjust the Translucency slider to examine hidden details of the component or model without leaving the dialog box.

This picture shows a chuck, but the Preview window is used by a number of different components so you might see a different component on your screen.

Display chuck jaw

Displays the component model in the Preview window.

Display boundary

Displays the Spin or Slice profile, whichever is selected. When Compare in preview is selected, both profiles are displayed in different colors.
