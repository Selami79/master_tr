Text page: Control Definition Manager

Use the individual sheets within the Text page to create, edit, and maintain post text entries in a number of categories. These are listed in the topic tree in the left side of the window.

You can edit the text data only when you open the disk copy of the control definition by choosing Control Definition from the Machine tab. Mastercam prevents you from editing the text data when you open the local copy of the control definition through a machine group in the Toolpaths Manager.

Once you open the disk copy of the control definition, choose a sheet from the topic tree window to see its text entries. They will be displayed in a spreadsheet-like interface where you can edit them as you wish. Double-click in a field to enter or edit a value.

  • Right-click in a spreadsheet data cell for more editing options and to import text from other sources.
  • Press [Tab] or [Shift+Tab] to move between data cells in data-entry mode.
  • The text in the gray Description column is the system default text. You cannot edit it. The various text entries that you enter will replace it in the interface.

Understanding the format of post text strings

Each cell in a post text sheet can have one of three types of strings. The picture shows you how these appear when you are working in the Text page.

The type of string controls how Mastercam displays the field that is associated with the post text entry.

Type of string What Mastercam does
Any text string

If the post text entry contains any text string—even just a single space—the associated data entry field in Mastercam is enabled. In addition:

  • Mastercam uses the string as the field label. This replaces the system default text label.
  • The string is written to the post text section of your post.
Empty cell

If the post text entry is blank, the associated data entry field in Mastercam is enabled. However:

  • Mastercam uses system default text—for example, Misc. Integer [5]—as the field label.
  • Nothing is written to the post text section of your post.

Pair of double quotes ""

If the post text entry is a pair of double-quotes, the data entry field in Mastercam is disabled.

  • "" is written to the post text section of your post.
How can I disable unused miscellaneous values?

Many posts have unused miscellaneous values that are visible on the Misc. Values page when you are creating toolpaths. If you wish, you can choose to disable these unused fields.

  1. Open a part that uses the post and control definition that you want to edit.
  2. Choose Machine Definition on the Machine tab. The Machine Definition Manager opens.
  3. Select the Control Definition button. The Control Definition Manager opens.

  4. Click the Existing Definitions button and select the desired control definition.
  5. Select the Text, Misc. Int/Real page. This page contains the field labels for each miscellaneous value.
  6. Double-click the text field of each miscellaneous value that you would like to disable and enter two double-quotes "" .

  7. Save the control definition.
  8. Close the Control Definition Manager and Machine Definition Manager. Follow the prompts to save the machine definition and reload it in the current part.
Right-click menu options

Click inside a data cell to see the right-click menu.

  • Import—Import an entire sheet, a category, or single cell of text. You can choose to import from another post or from another text section in the same post.
  • Default—Replace the current text with default values. Choose System to get the default values from the system defaults or Posts to read them from the DEFAULT section of your post.
  • Restore—Replace the current contents of the Text page with values that are read from the saved post on your workstation.
  • Export XML—Save the current set of post text entries as a stand-alone XML file. This includes everything inside the current <control> tag.
  • Text item display—Choose which category labels to use on the Text page:
    • User text to display the labels from the post text in your post.
    • System column text to use Mastercam's default text labels.
    • XML item tag to use the raw tag name from the XML data.
  • Save as default—Write the current contents of the Text page to the DEFAULT section of your post.
    • When you select Save as default from the right-click menu, Mastercam does not actually write the values at that time. Instead, it changes the mode of the Save command so that when you save the control definition, the post text that is currently displayed in the Text pages is saved to both the DEFAULT section of your post and the regular post text section.
    • When Save as default mode is active, a green check mark is displayed in the right-click menu.
    • Choose Save as default again to clear the check mark and restore the normal Save mode.
  • Select dialog bitmap—Select a custom bitmap (.bmp file) to replace the default picture used on the Cut parameters tab for drill toolpaths.
  • Remove text section from post—Deletes the current set of post text entries from your post. This includes everything inside the current <control> tag.
