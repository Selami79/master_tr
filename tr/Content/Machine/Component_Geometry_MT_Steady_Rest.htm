Geometry tab: Machine Component Manager (Mill-Turn Steady Rest)

Steady rest components in Mastercam Mill-Turn are defined as a base subcomponent with each arm and piston represented as additional subcomponents. Use the settings on this tab to select the component model and define other display properties for each part of the steady rest. This tab is used for both turret-mounted steady rests and standalone steady rests.

  • Use the Subcomponents window to create and organize the different parts of the steady rest. Right-click on the base component to add arms and pistons.

  • Use the settings in the Component group to select the model for each subcomponent and define other display properties.

The other settings on this tab are different depending on which type of subcomponent is selected in the Subcomponents window. Click on a drop-down below to learn about them. For detailed instructions about the specific tasks that you can accomplish in this tab, see the following topics:

  • Creating the Component Definition for a Turret-Mounted Steady Rest (Mill-Turn)
  • Adding an Arm to a Mill-Turn Steady Rest
  • Adding a Piston to a Mill-Turn Steady Rest

Creating a steady rest and adding it to the component library requires a number of separate tasks. These occur both before and after the work that you do in this tab. See Adding a Turret Mounted Steady Rest to a Component Library (Mill-Turn) for a higher-level overview of the process.

Parameters

Name

Enter the name that will be used for this component.

Subcomponents

Use this window to create and organize the base component and the subcomponents for arms and pistons. The fields in this tab change depending on which subcomponent is selected.

  • Right-click the base component to add arm or piston subcomponents.

  • Right-click on a subcomponent to rename or delete it.

The order of the subcomponents in the tree does not matter.

Buttons for working with subcomponents

Icon Description
Click to add an arm subcomponent. This is an alternative to using the right-click menu. This button is disabled if two arms are already present.
Click to add a piston subcomponent. This is an alternative to using the right-click menu.
Select a solid model from the graphics window or from the Solids Manager. The name is displayed in the Model field.

Type

Select the type of entity that will be used to model this component, for example, a solid or mesh entity.

Set name from geometry

Click to populate the Name field with the name read from the model.

Set color from geometry

Click to populate the Color field with the color of the model.

NOTE

If you want to set the component's Name or Color from the component model, you must select these options before you select the model.

Color

Select the color for the component. This color will appear during simulation and when the component is previewed on the screen.

Chord tolerance

Use this setting to control how finely Mastercam will draw the model that represents this component. A smaller value will result in a finer, more detailed simulation; however, this could slow down simulation. If your workstation's performance during simulation is sluggish, consider increasing this value.

Translucency

Use this option to make the component appear more or less translucent when it is displayed on the screen. This will be applied both when the component is previewed and when it is used in simulation.

Additional parameters for steady rest base components

Programmed center point (X, Y, Z)

Click the Select programmed point button to select the point that represents the steady rest in programmed commands. This is typically the center of the arc tangent to the two arms and piston when they are at their maximum open position. You can also enter coordinate positions directly in the X, Y, and Z fields.

Collision avoidance boundary

The collision avoidance boundary is a 2D profile that Mastercam uses when creating toolpaths so that the toolpath avoids the steady rest. The solid models are also used for collision checking but this occurs in simulation, after the toolpaths have been created. The collision avoidance boundary is applied while the toolpaths are being created. You can choose whether or not to apply this boundary during Job Setup.

Rectangular

Select this option to create a simple box around the steady rest. Do this when you do not have any wireframe geometry to use. Enter the dimensions in the Length and Height fields, then click the Preview boundary button to make sure it encloses the steady rest.

Chain

Click the Select button to select a wireframe profile to represent the collision avoidance boundary. This geometry must already have been merged into the component library before you are able to select it. Typically, this geometry would be drawn in the same file as the component models so that it is merged into the component library at the same time as the models.

Preview boundary

Click this button to preview the collision avoidance boundary in the graphics window.

Additional parameters for arm subcomponents

Center of rotation (X, Y, Z)

Click the Select center of rotation button to select the point around which the arm rotates as it opens and closes. For most applications, it does not matter whether you select the point on the front or back face of the model. You can also enter coordinate positions directly in the X, Y, and Z fields.

Maximum angle (degrees)

The number of degrees of movement between the open and closed positions of the arm. Enter this as a positive value regardless of direction.

Feed rate (degrees/minute)

The speed in degrees per minute at which the arm moves as it opens or closes. Mastercam uses this when it simulates the arms opening or closing.

Additional parameters for piston subcomponents

Maximum distance (inch)/(millimeter)

The total distance traveled by the piston as it moves from its open to closed positions. You must enter both inch and millimeter values. Click the Convert to inch or Convert to millimeters buttons to convert the value of one field to the other.

Feed rate (inch/minute)/(millimeter/minute)

The speed at which the piston moves as it opens or closes. Mastercam uses this when it simulates the arms opening or closing. You must enter both inch and millimeter values. Click the Convert to inch/minute or Convert to millimeter/minute buttons to convert the value of one field to the other.

Angle from +Y-axis (degrees)

The angle at which the vector along which the piston moves is offset from the world (machine coordinate system) Y-axis. For most pistons, this vector is parallel to the axis.

  • For a lower turret steady rest, this value is typically 0.

  • For an upper turret stead rest, this value is typically 180.

NOTE

This topic only applies to Mill-Turn steady rests. Steady rests for Mastercam Lathe are supported differently. See Adding a Steady Rest (Lathe) to learn more.
