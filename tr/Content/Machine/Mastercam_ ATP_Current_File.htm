Mastercam ATP dialog box: Current File

Use the Mastercam ATP dialog box to set the Mastercam ATP (Automatic Toolpathing) options to process a single, current part file. The purpose of this is to allow you to process a single part using the ATP process.

Parameters

New strategy

Opens the Strategy Explorer dialog box where you can create a new strategy.

Edit strategy

Opens the Strategies Explorer dialog box where you can edit the current strategy.

Machine definition

Displays the selected machine definition and its path. Use the Open button to the right to open a dialog box from which you can choose a pre-defined machine definition.

Strategy

Displays the current strategy and its path. Use the Open button to locate and choose a strategy for the current ATP session. Click Edit Strategy to change the current (active) strategy.

Conventional cutting direction

Applies the conventional cutting method, which cuts the closed contour with the tool rotating in the same direction as the tool motion. If this checkbox is not selected, Mastercam applies the climb cutting method, in which the tool rotates in a direction opposite the direction of travel along the cutting side of the tool.

Ignore empty levels

When checked, sets ATP to skip, rather than abort, a process if it encounters a level that has an operation mapped to it but contains no geometry.

Use active WCS

Select to use the current WCS setting when creating the ATP operations. When this option is deselected, ATP uses the WCS from the operation library specified in the active strategy.

Output NC

Select to save the NC file after ATP completes processing the file.

Edit

When checked, automatically launches the default text editor with the file displayed so that you can review or modify it. If multiple files are created, each will open in a separate window.

NOTE

This checkbox is only available if Output NC is selected.

Ignore blanked levels

When selected, the ATP strategy will not process entities that are blanked, hidden, or tagged as not visible in the Levels Manager.

NOTE

Fit all geometry to the graphics window [ALT + F1] to see all visible entities.

Use geometry depths

Select to use the Z depths from each part file's geometry during processing. If no Z depth is found, the saved operation depth will be used.

NOTE

All chains must be at Z0 to be toolpathed. However, ATP will search for any entity that lies in the top Cplane and use its Z depth if it is on the same level.

Part thickness

ATP uses this value with the Set Z zero top of part option to determine the correct depth of cut for all toolpaths. Choose the Select button and select a point on the geometry or enter a value in the field to the right of the Part Thickness text.

Set Z zero top of part

Sets the top of the part at Z0. When deselected, sets the table/spoilboard at Z0, which puts the bottom of the part at Z0.

Translate top of part to Z zero

Select to move the Z zero located at the bottom of a part to the Z zero at the top of the part.

Configuration

Displays the active ATP configuration. Click the arrow to display a drop-down list of other ATP configurations from which you can choose. These configurations are usually manufacturer-specific.
