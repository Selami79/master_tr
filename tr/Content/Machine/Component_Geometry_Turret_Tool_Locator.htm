Geometry tab: Machine Component Manager (Turret Tool Locator)

Use the settings in this tab to configure a tool locator for turret-mounted tools. Tool locators represent the blocks that are mounted on a turret into which the tool will be loaded. The tool locator determines the location and orientation of the tool that is loaded in it. For example, the same roughing tool can be mounted in any of the following ways:

  • parallel to either X or Z
  • oriented differently for upper or lower turret
  • pointed in different directions for face or back work

Turret-mounted tool locators can be created to support milling (live) tools or turning tools.

Each tool locator component includes a solid model. This represents the block or fixture which sits between the turret and the tool or holder. The model is displayed in Simulation and used for collision-checking.

Each tool locator component also includes at least one tool station. Each tool station identifies a group of settings that together tell Mastercam how to load a tool for a specific orientation or application. For example, a tool locator might be able to support both insert-up and insert-down tools. This usually requires that you create two stations, one for each orientation. Tool station definitions are also used to support half-index positions on the tool locator. As a general guideline, most tool locator components have between one and four tool stations.

Tool locators for milling tools support an optional barrel subcomponent. This represents the part of the locator that rotates, letting you create tool stations that can be used at angled orientations. The barrel is represented by an additional model, letting it be accurately simulated regardless of its programmed angle or orientation.

  • Use the Subcomponents window to create and organize the different parts of the tool locator. Right-click on the base component to add a barrel and stations.

  • Use the settings in the Component group to select the model for each subcomponent and define other display properties.

The other settings on this tab are different depending on which type of subcomponent is selected in the window. Click one of the drop-downs below to learn about them.

For detailed instructions about the specific tasks that you can accomplish in this tab, see the following topics:

  • Creating the Component Definition for a Turret-Mounted Tool Locator
  • Adding a Barrel to a Mill-Turn Tool Locator
  • Adding a Tool Station to a Turret Tool Locator

Creating a tool locator and adding it to the component library requires several separate tasks. These occur both before and after the work that you do in this tab. See Adding a Turret Tool Locator to a Component Library (Mill-Turn) for a higher-level overview of the process.

NOTE

Tool locators for tool spindles or B-axis heads are a different component type than tool locators for turrets. See Parameters tab: Machine Component Manager (Spindle Tool Locator).

NOTE

Tool locators are only used in Mastercam Mill-Turn.

Parameters

Name

Enter the name that will be used for this component.

See Naming Conventions for Tool Locators for guidelines on creating proper tool locator names.

Comment

Enter a comment that describes the tool locator. This will be visible in the Tool Setup Manager and should help the user decide if this tool locator is appropriate for their needs.

Subcomponents

Use this window to create and organize the base component and the subcomponents for barrels and stations. The fields in this tab change depending on which subcomponent is selected.

  • Right-click the base component to add a barrel or station subcomponent. Barrel subcomponents are only available for milling tool locators.

  • You can only add a single station to a barrel. If your locator does not have a barrel—meaning that the stations are attached directly to the base—you can add multiple stations.

  • Right-click on a subcomponent to rename or delete it.

The order of the stations in the tree does not matter.

Buttons for working with subcomponents

Icon Description
Click to add a barrel to the locator to support angled heads. This is an alternative to using the right-click menu.
Click to add a station. This is an alternative to using the right-click menu.

Select a solid or mesh model from the graphics window or the Solids Manager. The name is displayed in the Model field.

Select a model for the base and barrel only; you do not select a model for stations.

Type

Select the type of entity that will be used to model this component, for example, a solid or mesh entity.

Set name from geometry

Click to populate the Name field with the name read from the model.

Set color from geometry

Click to populate the Color field with the color of the model.

NOTE

If you want to set the component's Name or Color from the component model, you must select these options before you select the model.

Color

Select the color for the component. This color will appear during simulation and when the component is previewed on the screen.

Chord tolerance

Use this setting to control how finely Mastercam will draw the model that represents this component. A smaller value will result in a finer, more detailed simulation; however, this could slow down simulation. If your workstation's performance during simulation is sluggish, consider increasing this value.

Translucency

Use this option to make the component appear more or less translucent when it is displayed on the screen. This will be applied both when the component is previewed and when it is used in simulation.

Additional parameters for barrel subcomponents

Center of rotation (X, Y, Z)

Click the Select center of rotation button to select the point around which the barrel rotates. For most applications, it does not matter whether you select the point on the front or back face of the model. You can also enter coordinate positions directly in the X, Y, and Z fields.

Maximum, Minimum angle (degrees)

Enter the maximum and minimum angular positions of the barrel relative to its straight position. For most applications, this is the orientation in which the model is drawn. For example, a barrel might be positionable within the range of 90 degrees forward and –15 degrees backward.

Angle resolution

The smallest allowable increment (in degrees) by which the barrel can be repositioned.

Additional parameters for station subcomponents (milling tools)

Name

The name assigned to the tool station.

Comment

Enter a comment that describes the tool station. This comment will be visible to users when they are selecting a station.

Tool number

Once the locator has been selected for an operation, this specifies the tool number assigned to the station.

Uses index

Specifies whether the station uses the Primary index position, or a Primary Minus/Primary Plus half-index position.

Offset angle

If you select one of the half-index positions, enter the angular spacing between the primary index position and the half-index position. Enter a positive number only; Mastercam knows the direction from the Plus or Minus designation.

Speed multiplier

If your milling head is geared to increase or decrease the programmed spindle speed, enter the scaling factor here. For example, if your head is geared to double the spindle speed, enter 2.0 here.

Mastercam adjusts the spindle speed that is output in your NC file to account for this value. For example, if your toolpath settings specify a spindle speed of 1000 rpm, and the Speed multiplier is 2.0, your NC file will have an S500 spindle speed command.

Reverse spindle direction

Select this option if your milling head is geared to reverse the spindle direction.

Tool orientation

Tells Mastercam the desired orientation of the tools that will be mounted in this tool station. A single tool locator might have one tool station defined for tools oriented left, and another tool station for tools oriented right. See Adding a Tool Station to a Turret Tool Locator to learn more about the different options.

Tilt angle

If the selected Tool orientation is Angular, use this value to specify the angle at which the tool is mounted.

  • For milling tool locators, if the tool locator includes a barrel subcomponent, select From operation to have Mastercam set the angle from the operation's tool plane. Select Manual if you know the angle at which the tool will be mounted, then enter it here.

  • Selecting From operation disables the field where the angle is displayed; however, if a user examines the station settings after the locator has been used in an operation, this field will display the actual angle used in that operation.

Tool position on locator

Specifies a reference position on the tool locator model where the tool will be mounted; for example, the center of the rear of the collet. Click the Target button to return to the graphics window and select a point on the model, or enter the coordinate position directly in the fields.

  • For turning tools, this location will mate with the selected Tool holder position to match when the tool is mounted.

  • For milling tools, this location typically mates with the center of the rear face of the tool or tool holder, if one is selected.

Projection adjustment

Adjusts the stickout length of the tool when mounted in this tool locator. Select the checkbox to activate the field, and then enter the adjustment value. Use positive values to extend the tool from the locator and negative values to retract the tool.

Additional parameters for station subcomponents (turning tools)

Name

The name assigned to the tool station.

Comment

Enter a comment that describes the tool station. This comment will be visible to users when they are selecting a station.

Tool number

Once the locator has been selected for an operation, this specifies the tool number assigned to the station.

Uses index

Specifies whether the station uses the Primary index position, or a Primary Minus/Primary Plus half-index position.

Offset angle

If you select one of the half-index positions, enter the angular spacing between the primary index position and the half-index position. Enter a positive number only; Mastercam knows the direction from the Plus or Minus designation.

Allowable tool types

Controls which types of tools can be added to this station.

  • Fixed tools refer to all Lathe tools except drilling tools; select either or both insert positions (up or down) to establish which insert orientations are allowed in this locator.

  • Fixed drills refer to Lathe drills and center tools.

Tool holder position to match

Specifies the reference point on the tool that will be mapped to the tool locator. The Tool position on locator is the corresponding point on the tool locator model. These two points will line up when the tool is mounted. Position 5—the center of the shank—is generally used for round tool holders.

Tool orientation

Tells Mastercam the desired orientation of the tools that will be mounted in this tool station. A single tool locator might have one tool station defined for tools oriented left, and another tool station for tools oriented right. See Adding a Tool Station to a Turret Tool Locator to learn more about the different options.

Tilt angle

If the selected Tool orientation is Angular, use this value to specify the angle at which the tool is tilted.

Tool position on locator

Specifies a reference position on the tool locator model where the tool will be mounted; for example, the center of the rear of the collet. Click the Target button to return to graphics window and select a point on the model, or enter the coordinate position directly in the fields.

  • For turning tools, this location will mate with the selected Tool holder position to match when the tool is mounted.

  • For milling tools, this location typically mates with the center of the rear face of the tool or tool holder, if one is selected.

Projection adjustment

Adjusts the stickout length of the tool when mounted in this tool locator. Select the checkbox to activate the field, and then enter the adjustment value. Use positive values to extend the tool from the locator and negative values to retract the tool.
