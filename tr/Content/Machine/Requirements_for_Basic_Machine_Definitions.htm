Requirements for Basic Machine Definitions

The Machine Definition Manager offers a great many features and options that let you model a machine with high degree of sophistication. However, not all of these features are useful for all applications.

All Mastercam products except Mill-Turn— including all levels of Mill — only require a simple, parameter-based machine definition. This means that you do not need to create solid models of your machine components. However, these machine definitions do not support machine simulation.

Basic machine definitions let you get up and running quickly, without the need to create detailed component models. After you create your basic machine definition, you can easily add solid models later if your needs change.

All machine definitions need to meet minimum standards.

Click here to view these standards.

  • All linear and rotary axes must be represented in the component tree in the Machine Definition Manager. The axis orientation and labels must be set properly.
  • In addition, each component group that includes a programmable axis must include either a tool-holding component such as a tool spindle or work-holding component such as a table or chuck.
  • Aggregate tooling heads and blocks must also be defined in the component tree.

  • You must have at least one axis combination. Axis combinations tell Mastercam which axis components define the coordinate system for a given operation. For example, a twin-spindle lathe will have two possible Z axes, depending on which spindle is in use. Most 3-5 axis mills and routers will only have a single default axis combination:

  • Multi-turret, multi-spindle lathes will typically have several axis combinations that need to be individually defined:

  • You must select a control definition and post processor. All of your control definition pages should be completed and reviewed, since many post variables are initialized directly from their values.

For each component, you only need to complete the Parameters tab for the component properties:

The other tabs are used by the machine simulation function. They are also used by lathe stock and peripheral components.

Most of the General Machine Parameters are optional. Some of the tabs let you set useful defaults, such as tool libraries and construction planes, while other settings such as travel and motion limits are available to your post processor as parameter values.
