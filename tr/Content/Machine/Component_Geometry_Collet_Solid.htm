Geometry tab: Machine Component Manager (Collet (Solid))

Use this tab to define your collet with a solid model. The model can be either a Mastercam solid on a level in your part file, or an external file in .STEP format.

  1. Select Solid entity on the Geometry tab for the collet component.
    • In the picture below, click the left button to select a model from an external file.
    • Click the right button to select a model from the Solids Manager or the graphics window.

    Mastercam opens the Align Collet function panel.

    What if more than one solid is available?

    Each component must be represented by only one single solid model. If your component model consists of more than one solid body, you must merge or combine them first.

    If your file contains models for more than one component—for example, a single file might contain models of both the chuck and jaws—select only the single model for your component and ignore the others.

    NOTE

    If your file contains models of each individual chuck jaw, select only one model for a single jaw. Mastercam automatically creates the proper number of jaws for you.

  2. Use the Align Collet function panel to align the model so that it mates cleanly with the chuck. Click OK when you are done.
    • Any collet component can be loaded on either the left or right spindle. The Z-axis gnomon displayed in the graphics window should point to the positive Z direction. Mastercam flips the model as needed to maintain the proper Z-axis direction depending on which spindle you mount the jaws.
    • By convention, components are defined as if they are in the left spindle.
    • For most applications, click on the back face of the model so that the Z-axis arrow points to the front, and set the zero point along the inner diameter of the component.

    This picture shows the solid when you are about to select it. The location where you click when you select it establishes the initial axis orientation.

    This picture shows the model after the axis alignment has been set. You can use the dynamic gnomon to move the origin or rotate the X-axis orientation if desired—for example, in case you wanted to align the slots.

    See Align Chuck/Jaw/Collet panel to learn more about using the function panel.

  3. Mastercam displays the model in the Preview window when you are done aligning it.
    • Right-click in the Preview window to see the collet in different views to ensure that it is aligned properly. The following pictures show a properly oriented model in Top and Isometric view.

    • If the model is not correct, click the Align button to return to the graphics window, where you can use the Align Collet function panel to change the orientation of the model.
  4. Complete the other settings on this page.
NOTE

If the model has been selected from your part file, Mastercam moves the model from its original level to the last available level in your file. If you import a model, Mastercam copies the selected solid model to your machine definition or component library when you have finished creating your chuck component. This means that there is no associativity between your collet component and the original model, even if the model is on another level in your part file.

Parameters

Name

Enter the name that will be used for this component.

Geometry

Tell Mastercam how you want to model the collet.

  • Select Solid entity to display the options described in this topic.
  • Select Parametric to model the jaws by entering dimensions directly in this dialog box. Most Lathe users will select the Parametric option. See Geometry tab: Machine Component Manager (Collet (Parametric)) to learn more.

Color

Select the color for the component. This color will appear during simulation and when the component is previewed on the screen.

Chord tolerance

Use this setting to control how finely Mastercam will draw the model that represents this component. A smaller value will result in a finer, more detailed simulation; however, this could slow down simulation. If your workstation's performance during simulation is sluggish, consider increasing this value.

Translucency

Use this option to make the component appear more or less translucent when it is displayed on the screen. This will be applied both when the component is previewed and when it is used in simulation.

Set name from geometry

Click to populate the Name field with the name read from the model.

Set color from geometry

Click to populate the Color field with the color of the model.

NOTE

If you want to set the component's Name or Color from the component model, you must select these options before you select the model.

Buttons for working with solid models

Icon Description
Select a file that contains the solid model from your workstation.
Select a solid model from the graphics window or from the Solids Manager.
Open the Align Chuck function panel, which lets you change the alignment or orientation of the solid model. If more than one solid is available, you can also select a different solid.
Optimize the solid model.
Repair small faces.

Inside diameter (ID)

The size of the opening through which the stock is fed. For Mill-Turn users, this value determines the maximum diameter of the stock that you will be allowed to enter in Job Setup.

NOTE

The Allow through stock option on the Parameters tab for the chuck must be selected before you can use bar stock or feed stock through the chuck.

Outside diameter (OD)

The maximum diameter of the chuck. This is equal to the diameter of a spun profile of the chuck.

Length

The length of the model. Mastercam reads this dimension from the model but you can override this with your own value if necessary.

Preview window

Displays a live interactive 3D model of the component. Right-click in the window to change the view, dynamically rotate the component, or zoom in on component details. Depending on the type of component, other buttons or controls are available for additional preview options.

The Preview window also uses the Translucency setting found on the Geometry tab. Adjust the Translucency slider to examine hidden details of the component or model without leaving the dialog box.

This picture shows a chuck, but the Preview window is used by a number of different components so you might see a different component on your screen.

Display collet

Displays the component model in the Preview window.

Display boundary

Displays the 2D boundary that is created from the current settings.
