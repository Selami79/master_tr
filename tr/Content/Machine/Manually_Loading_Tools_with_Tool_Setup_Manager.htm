Manually Loading Tools with Tool Setup Manager

Perform the following procedure to manually load tools in the Tool Setup Manager.

  1. In the Toolpaths Manager, click the Setup tools on the machine button. The Tool Setup Manager displays.

  2. Use Active turret/spindle to filter the tools and tool locators based on the turret you want to use. You can see the available turrets in the Tool Setup Manager's right-hand pane.

    The image following shows the filter set to Lower Turret, so that Mastercam displays only the tools and tool locators that you can use on the lower turret.

  3. Hover your mouse over a tool to see a tooltip.

    It provides the information you need to match the tool with a tool locator. In the following image, notice that the selected tool is insert up and its orientation is cross lower insert right, which means this is a vertical tool (cross) for the lower turret, insert up and pointing to the right turret.

  4. In the lower pane of the Tool Setup Manager, look for a tool locator that is appropriate for the tool you want to use.

    In the following image, you can see a tool locator named LT – 20mm Cross Turning – LIU. Here, LT stands for lower turret and LIU stands for left spindle insert up. (This information is spelled out in the tool locator’s tooltip.) This tool locator, then, matches the tool selected in Step 3.

  5. Drag and drop the selected tool locator onto the turret.

    NOTE

    The tool number and the index number must match. In this case, the number of the tool to load is T02 (from Step 3), so the tool locator must be placed at Index 2. To load the tool at a different turret position, you must manually change the tool number.

  6. Right-click the turret, and choose View component and all children.

    Mastercam displays the turret and all the tools that have been loaded on it, including the tool locator that you have just selected.

     

  7. Press [Enter] to return to the Tool Setup Manager.

  8. Drag and drop the selected tool onto the tool locator shown in the right pane.

  9. Use the View component and all children command to see the results. Then press [Enter] to return to the Tool Setup Manager.

    NOTE

    Not all tool locators can be placed in all index positions. For example, some tool locators can be placed only at odd or even positions. A tool locator's attributes, summarized in its tooltip, show the positions at which the tool locator can be placed. Note also that there is no undo or cancel; all changes are immediately "live."
