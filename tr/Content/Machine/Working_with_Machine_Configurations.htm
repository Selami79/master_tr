Working with Machine Configurations

Machine configurations are sets of components that are used in Job Setup. Mastercam saves these in a separate file so that they can be reused. Mill-Turn users can save specific machine configurations to individual configuration files, then load them again to quickly use the same setup on additional parts.

Machine configurations can include any combination of work holding components, tools, and tool holding components. The tool holding components can include individual tools mounted in them. When you load a machine configuration, Mastercam preserves the tool number, index position, and tool orientation.

  • Access machine configurations from either Job Setup or the Tool Setup Manager.
  • You can add machine configurations to the .machine file, or save them anywhere on your workstation.

The Machine Configuration page in Job Setup includes Save Machine Configuration and Load Machine Configuration buttons. These are the starting point for working with machine configurations.

You can also work with machine configurations in the Tool Setup Manager. The same buttons are available to load or save configuration files.

NOTE

When you work with configuration files from Tool Setup Manager, Mastercam only loads the tools and locators. Components of other types that may be present in the configuration file are ignored.

Creating and Saving Machine Configuration Files

Clicking the Save Machine Configuration button opens the Save Machine Configuration function panel. Use it to select the components that will be included in the configuration file and to specify the name and location for the file.

To begin, click the Save file button to specify the name of the configuration file and its location.

Then, from the tree at the bottom, select the components to be included.

Click OK to save the file and exit the function panel. Mastercam creates the file with an .mtConfig extension in the folder that you specified. Existing configurations with the same name will be overwritten.

Loading Machine Configuration Files

To begin loading a machine configuration, click the Load Machine Configuration button from either Job Setup or the Tool Setup Manager.

This opens the Load Machine Configuration function panel.

Click the Select file button to select the configuration file that you wish to load.

You can only select .mtConfig files that were created from the same .machine file as the current part.

Once you select a configuration file, Mastercam displays its components in a tree. Select the components that you wish to import into your current part file.

  • Components that would replace a component in the current part are highlighted in orange. For example, if your part and the configuration file both have a tool mounted in turret position 2, that component is highlighted in orange. The components in the configuration file will replace the ones in your part if you select them.
  • Components that are used in an operation are highlighted in blue. Those components cannot be imported and you cannot select them.

Click OK when you are done selecting the desired components.

Saving Machine Configurations in Your Machine File

Mastercam gives you several ways to organize your machine configurations.

  • You can store the .mtConfig files in any convenient location on your workstation.
  • You can store them directly in your .machine file. This way they are always available to anyone who uses the .machine file.

To add them to your .machine file, open the .machine file in Code Expert. Then simply drag-and-drop the desired configuration files from Windows Explorer to the new Machine Configuration Files node in Machine Explorer.

You can also right-click on the Machine Configuration Files node and choose Select:

When you add the configuration files, Mastercam marks the files "dirty" with an asterisk ("*"). This means you must save the .machine file.

Right-click on a configuration file to see additional options.

Machine Configurations and Steady Rests

While steady rest components themselves are not saved in machine configuration files, Mastercam can save the mounted/unmounted state:

In other words, if the configuration file specifies an unmounted steady rest, and you load the configuration file into a part that includes a mounted steady rest, Mastercam will change the steady rest to unmounted. However, it does not replace the steady rest component itself. Similarly, if there are multiple steady rests in the machine, you can save mounted/unmounted states for each one individually.

Turret-mounted steady rests are not supported in machine configuration files.
