Nesting Options dialog box: ATP

Use the Nesting Options dialog box to define nesting parameters for the current ATP job. Select either Rectangular or TrueShape for the mode of operation, set the nesting sorting parameters, and choose the program control options.

Rectangular Nesting

Rectangular nesting places a bounding box around the part and uses the box as the part boundary for nesting. Rectangular nesting also places a bounding box around an irregular sheet, and nests parts within that boundary even if the bounding box extends beyond the sheet boundaries.

TrueShape Nesting

TrueShape nesting fits parts or toolpaths on a sheet based on shape and other parameters. It allows access to sheet and part libraries and extended functionality beyond Rectangular nesting. TrueShape nesting creates copies of the part or toolpath and fits them together using the part shape, a rotation angle, reversing the copy (mirror), and other parameters. It increases the yield of parts from a sheet of material. TrueShape nesting also allows nesting on irregularly shaped sheets by placing parts or toolpaths within the sheet boundary regardless of the shape of the sheet.

Parameters

Nest toolpaths

When selected, allows parts to be nested. This radio button is specific to Mastercam parts. Deselecting this checkbox disallows nesting but still applies toolpaths to each part.

Part-part distance

Controls the spacing between geometry parts or toolpaths in a nesting session. Enter the minimum amount of space between nested parts, allowing at least the diameter of the cutting tool you are planning to use as the distance between parts.

Sheet margin

Enter the amount of space between the edge of the sheet and the nested parts. When nesting geometry, be sure to enter a wide enough margin to accommodate toolpaths.

Use the sheet margin to allow enough space to keep the tool buried in the material when cutting parts close to the edge of the sheet, or to keep parts from contacting damaged edges of a sheet. Sheet margin can be used in addition to part margin to create adequate spacing (Nesting combines both values).

Accuracy

Determines the precision of the results. The finer the accuracy, the longer the results may take to generate. Each of the predefined accuracy settings corresponds to a resolution value, for example, Fine = .025. Use a larger setting such as Coarse for bigger parts. If available, choosing User Defined enables the resolution field where you can enter the resolution for the results.

NOTE

There may be cases where a finer accuracy setting may not produce optimum results, for example if the parts are big.

Nesting corner

Choose the corner of the sheet where Nesting will begin positioning parts. Changing the nesting corner can sometimes improve material usage.

Fill direction

Determines the direction in which the sheet is filled with nested parts. A sheet's fill direction runs perpendicular to the part's nesting direction. For example, if you select an Upper Left start point in the Nesting corner field and a Horizontal direction in the Fill Direction field, Mastercam does the following:

  • Places the first part in the upper left corner of the sheet
  • Nests the parts vertically down the left side.

When the nested parts reach the bottom of the sheet, the next column of parts is positioned to the immediate right of the parts already nested filling the sheet from left to right.

Use this parameter to control drop and improve material usage. Choose Calculate to have Mastercam apply the most efficient fill direction.

Guillotine

(TrueShape nesting only) Arranges the nested parts so that straight cuts across the material can be made.

Automatic sheet origins

Select to have Mastercam automatically place the sheets at different locations in the graphics window. When deselected, you need to set a unique origin for each sheet to keep Mastercam from placing them in the same position.

Sheet-sheet distance

Enter the distance between the sheets placed in the graphics window.

Sheet margin

Enter the amount of space between the edge of the sheet and the parts. If you enter 0 for the Sheet margin, Mastercam nests parts to the very edge of the sheet. Toolpaths may extend over the sheet boundary.

Common Geometry Nesting Controls

New

Starts a new nesting session. Mastercam reloads the nesting defaults file (DEFX) including the default sheet, if one exists.

Open

Opens a nesting session file (NSTX).

Save

Saves sheets, parts, and any parameter settings to a nesting session file (NSTX).

Save as

Saves the nesting session file (NSTX) to a new name.

Restore

Restores the defaults file and current nesting session to factory settings.

Config

Opens the Nesting Configuration dialog box.

Common Toolpath Nesting Controls

New

Starts a new nesting session. Mastercam reloads the nesting defaults file (DEFX) including the default sheet, if one exists.

Open

Select to open an .INI file.

Save

Saves sheets, parts, INI files, and any parameter settings to a nesting session file (NSTX).

Save as

Saves the nesting session file (NSTX) or .INI file to a new name.

Restore

Restores the defaults file and current nesting session to factory settings.

Config

Opens the Nesting Configuration dialog box.

Customize Controls

Delete

Allows the removal of one or more selected parts from a sheet.

Lock

Prevents changes to the position of selected parts on the sheet. When selected, the part color toggles from purple (unlocked) to grey (locked).

Rectangular

Nests parts as rectangles on a grid. TrueShape nests parts according to each part's actual shape and other parameters.

TrueShape

TrueShape nests parts according to each part's actual shape and other parameters.

Database order

Sorts the operations in the order they were placed on the sheet.

Maximum vacuum

Sorts the operations by size (smallest parts first), and then inside parts before outside parts to maintain as much vacuum coverage as possible.

NOTE

Drilling is always done first.

Tool number ascending

Sorts the operations starting with the operation that uses the lowest tool number.

Tool number descending

Sorts the operations starting with the operation that uses the highest tool number.

Minimize tool change

Sorts the operations by tool so that all parts using one tool are cut before changing to the next tool.

Off (separate file for each sheet)

When selected, outputs sheets to the NC file without program stops.

Stop

When selected, inserts a program stop (M00) between sheets.

Optional stop

When selected, inserts an optional stop (M01) between sheets.

Onion skin

Select to enable any onion skin settings in toolpaths that will be processed with ATP.

Common edge

When you enable Common Edge, the nesting function can position parts so that they share parallel edges. This allows a single pass to cut two edges simultaneously.

Create block drill operations

Creates a new block drill operation for each sheet generated when nesting block drill toolpaths. When you nest a block drill toolpath with this option chosen, the nesting operation posting is turned off in the Toolpaths Manager, and each new block drilling operation is inserted directly below the nesting result (as shown below). A new block drill operation will be created for each nesting result (sheet used).
