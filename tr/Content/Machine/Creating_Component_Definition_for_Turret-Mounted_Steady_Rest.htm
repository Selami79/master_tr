Creating the Component Definition for a Turret-Mounted Steady Rest (Mill-Turn)

Before you can use a turret-mounted steady rest, the component library (.mcam-gmd file) must have a component definition for it. Do this after you create the solid models for a steady rest component and import them into the component library. Follow these steps:

  1. Open the desired component library:
    1. Open the .machine file in Code Expert.
    2. Double-click the desired component library.

      The Mill-Turn Component Library dialog box opens in a new session of Mastercam, displaying the component library.

  2. Right-click on the turret component to which the steady rest will be mounted and select Add component, Steady rest.

    Mastercam opens the Geometry tab for the steady rest.

  3. If desired, edit the component Name. You can also choose to read the component name from the solid model, as described below.

  4. Follow these steps to select the solid model for the component.

    1. You can choose to read the name or color of the component from the model. To do this, select either of these options before selecting the model:

      • Set name from solid
      • Set color from solid

    2. Select the model by clicking the button next to the Model name field. Select only a single solid body; do not try to select any other geometry, such as construction geometry or other subcomponents.

    NOTE

    Before you can select the model, it must already have been imported into the component library. See Importing Solid Models into a Component Library to learn more.

  5. Click the Select programmed point button to select the location that will represent the steady rest's programmed position for NC commands. Mastercam returns you to the graphics window.

    This is typically the center of the arc tangent to the two arms and piston:

  6. Select or define the Collision Avoidance Boundary. This is a 2D boundary that you can choose to apply to selected operations. You can define the boundary in either of two ways:

    • Select Rectangular and enter the dimensions. Select this option if you have not imported any wireframe geometry to represent the boundary.
    • Select Chain to create a custom boundary from wireframe geometry. The geometry must already been imported into the component library before you can select it.

  7. Complete the settings on the Position tab. These settings tell Mastercam how the steady rest mounts to the turret.

    1. Select the proper Attachment Position. These are typically defined by the machine developer who configures the turret component in the component library. Attachment positions are used by both tool locator and steady rest components; see Working with Attachment Positions to learn more.

    2. Select the index position on the turret where the steady rest will be mounted from the Turret Index Mounting Position list.

    3. Some steady rests are large enough to span several index positions. If this is the case, use the Disabled Index Mounting Positions list to select any index positions that cannot be used when the steady rest is mounted. This ensures that Mastercam will not try to load a tool in those positions.

      • The position selected in the Turret Index Mounting Position is disabled automatically and does not appear in the list.

      • Your steady rest model should have been drawn so that the origin of the model is the same as the mounting location on the turret. The selected Turret Index Mounting Position should match this point.

      See Position tab: Machine Component Manager (Mill-Turn Turret Mounted Steady Rest) for more detailed information about these settings.

  8. Most turret mounted steady rests include arms and a piston. These are represented as subcomponents. To add them, right-click on the steady rest name in the Subcomponents window and select Add subcomponent and either Add arm or Add piston.

    • You can add the arms or piston in any order.
    • Each arm is a separate subcomponent.

    See the following topics for detailed instructions:

    • Adding an Arm to a Mill-Turn Steady Rest
    • Adding a Piston to a Mill-Turn Steady Rest
  9. Click OK when you have completed all the settings on both the Geometry and Position tabs.

Mastercam returns you to the Mill-Turn Component Library dialog box. See Previewing and Validating Components in a Library to check your work.
