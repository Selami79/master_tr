TOOL-M

[TOOL-M]

"ACTIVE-SPINDLE" Active spindle (Left/Right)
"ACTIVE-TURRET"

Active turret and spindle (Upper-Left/Upper-Right/Lower-Left/Lower-Right)
"AGGREGATE" Tool is aggregate (YES/NO)
”ALIAS” T0105 – Tool number and length offset number
”ANGLE” Shoulder angle
"ARBOR-DIAMETER" Diameter of tool arbor (same as shank)
"ARBOR-RADIUS" Radius of tool arbor (same as shank)
"ASSY-DESC" Assembly description
"ASSY-NAME" Assembly name
"BITMAP" Standard tool image file name
"CAPTION" Image caption (overrides only)
"CHIP-BREAK" Chip break amount (if drill)
”CHUCK” Reference string
"CODE" Manufacturer's tool code
"COMMENT" Comment
"COOLANT-OP" Coolant options set in operation
"COOLANT-TL" Coolant options set in tool
"COOLANT" Type of coolant (Flood/Mist/Spindle/Off)
"CORNER-RADIUS-TYPE" Tool corner radius type (Full/Etc)
"CORNER-RADIUS" Tool corner radius
"DESCRIPTION" Tool description
"DIAMETER" Tool diameter
"DIAMETER-OFFSET" Diameter offset number
"DWELL" Dwell time (if drill)
”FEEDRATE” Feed rate
"FEEDRATE-ABS" Absolute value of the feed rate
"FEEDRATE-TYPE" Type of feed rate. (M) per minute, (R) per revolution, (S) surface finish in microns
”FEEDRATE-VALUE” Feed rate (no prefix or suffix, just the numeric value)
”FINISH-X” Percentage of tool diameter for XY stepover in finishing
”FINISH-Z” Percentage of tool diameter for Z stepover in finishing
"FLUTE-LENGTH" Flute length of tool
"FLUTES" Number of flutes
"FPT" Feed per tooth
"HEAD-IDN" Aggregate head identification number
"HOLDER-DIAMETER" Diameter of tool holder
“HOLDER-DIAMETER-BOTTOM” Bottom diameter of tool holder
“HOLDER-DIAMETER-MAX” Maximum diameter of tool holder
“HOLDER-DIAMETER-MIN” Minimum diameter of tool holder
“HOLDER-DIAMETER-TOP” Top diameter of tool holder
"HOLDER-LENGTH" Length of tool holder
"HOLDER-LIB" Tool holder library
"HOLDER-NAME" Tool holder name
"IMAGE"  ”IMAGE2D”  ”IMAGE3D” Image file names (2D=Flat, 3D=Shaded with holder)
"LENGTH-OFFSET" Length offset number
"LOCATION" Tool crib location
"MACHINE-GROUP-COMMENT" Machine group comment
"MACHINE-GROUP-NAME" Machine group name
"MACHINE-NAME" Machine name from machine definition
"MACHINE-POST-EXT" Current post file extension
"MACHINE-POST-LONG" Current post filename (including path)
"MACHINE-POST-NAME" Current post filename (name only)
"MACHINE-POST-PATH" Current post file path
"MACHINE-POST-SHORT" Current post filename (name and extension)
"MACHINE-TYPE" Machine type from machine definition (“M”= Mill, “L”= Lathe, “R”= Router, “W”= Wire, “X”= Mill-Turn)
"MATERIAL" Material name
”MAX-X” ”MAX-Y” ”MAX-Z” Deprecated
”MAX-X-TPLANE” ”MAX-Y-TPLANE” ”MAX-Z-TPLANE” Maximum XYZ. Values output relative to common tool plane and its origin, otherwise NA.
"MAX-X-WCS” "MAX-Y-WCS” "MAX-Z-WCS” Maximum XYZ. Values output relative to common WCS and its origin, otherwise NA.
"MAX-X-WORLD” "MAX-Y-WORLD” "MAX-Z-WORLD” Maximum XYZ in world coordinates
"MC" Tool geometry file
"METRIC" Metric tool (YES/NO)
”MIN-X” ”MIN-Y” ”MIN-Z” Deprecated
”MIN-X-TPLANE” ”MIN-Y-TPLANE” ”MIN-Z-TPLANE” Minimum XYZ. Values output relative to common tool plane and its origin, otherwise NA.
"MIN-X-WCS” "MIN-Y-WCS” "MIN-Z-WCS” Minimum XYZ. Values output relative to common WCS and its origin, otherwise NA.
"MIN-X-WORLD” "MIN-Y-WORLD” "MIN-Z-WORLD” Minimum XYZ in world coordinates
”MINIMUM-LENGTH” Unused (blank)
"MNMXN-TPLANE" View name if all operations use a common tool plane
"MNMXV-TPLANE" View number if all operations use a common tool plane
"MNMXN-WCS" View name if all operations use a common WCS 
"MNMXV-WCS" View number if all operations use a common WCS 
"NAME" Tool name
"NECK-DIAMETER" Tool neck diameter
"NECK-LENGTH" Tool neck length
"NUMBER" Tool number (T1/T2/Etc)
"OOI" Raw Operation data
"OTI" Raw Tool data
”OUT-DIA” Tip diameter (Chamfer/Face/Engrave)
"OVERALL-DIA" Tool overall diameter
"OVERALL-DIAMETER" Tool overall diameter
"OVERALL-LENGTH" Overall length of tool (excluding holder)
"OVERALL-RADIUS" Tool overall radius
"PECK-1" "PECK-2" First and subsequent peck increments (if drill)
"PECK-CLR" Peck clearance (if drill)
"PILOT-DIAMETER" Pilot diameter of tool
"PILOT-RADIUS" Pilot radius of tool
”PLUNGE” Plunge rate
”PLUNGE-ABS” Absolute value of the plunge
”PLUNGE-TYPE” Type of plunge rate. (M) per minute, (R) per revolution, (S) surface finish in microns
”PLUNGE-VALUE” Plunge value for this tool
"PROBE" Probe: YES/NO
"RADIUS" Tool radius (diameter / 2.0)
”RETRACT” Retract rate
”RETRACT-ABS” Absolute value of the retract
”RETRACT-TYPE” Type of retract rate per minute (M)
”RETRACT-VALUE” Depth of retract plane
”ROOT-DIA” Root diameter (if tap)
”ROUGH-X” Percentage of tool diameter for XY stepover in roughing
”ROUGH-Z” Percentage of tool diameter for Z stepover in roughing
"SFM" Surface feet per minute
"SHANK-DIAMETER" Diameter of tool shank (same as arbor)
"SHANK-RADIUS" Radius of tool shank (same as arbor)
”SHIFT” Percentage of tool diameter shift value (for fine boring)
"SHOULDER-DIAMETER" Shoulder diameter of tool
"SHOULDER-LENGTH" Shoulder length of tool
[spAssembly] Assembly data from the tool manager
spHolder] Holder data from the tool manager
[spTool] Tool data from the tool manager
"SPINDLE-DIRECTION" Spindle direction (CW/CCW)
"SPINDLE-SPEED" Spindle speed
"SPINDLE-SPEED-MAX" Maximum spindle speed
"SPINDLE-SPEED-MAX-VALUE" Maximum spindle speed (no prefix or suffix, just the numeric value)
"SPINDLE-SPEED-VALUE" Spindle speed (no prefix or suffix, just the numeric value)
"STATION" Station number
"TAPER-ANGLE" Tool taper angle
"TAPER-LENGTH" Tool taper length
"THD-COUNT" Thread count
"THD-PITCH" Thread pitch
"THREADS" Threads per inch or pitch (if tap)
"TIME-LONG" Time (Hours, Minutes, Seconds)
"TIME-SHORT" Time (HH:MM:SS)
"TIP-ANGLE" Tip angle of tool (in degrees)
"TYPE" Tool type (Drill/Tap/Etc)
[OPERATION-M]or [OPERATION-R] Operations that use this tool (MULTIPLES POSSIBLE)
[OP] Tool list only (MULTIPLES POSSIBLE)
