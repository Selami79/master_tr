OPERATION-W

[OPERATION-W]

"AUTO-START-POS" Start wirepath at thread position (not start position)
"CAPTION" Image caption (overrides only)
"CHAIN-COUNT" Number of chains selected for operation
"COMMENT" Operation comment
"COOLANT" Coolant methods for both operation and tool. ON if both on and methods differ. OFF if both off. Method description displays if both methods match.
"COOLANT-OP" Coolant method for operation
"COOLANT-TL" Coolant method for tool
"CUT-POS" Cut position
"FILL-TANK" Empty(No), Fill(Yes)
"FLUSH" 0=on, 1=yes, 2=other
"IMAGE" Image file name
"LAND-HEIGHT" Land height
"LAND-HEIGHT-INCR" Land height coordinate is incremental
"MACHINE-NAME" Machine name from machine definition
"MACHINE-GROUP-COMMENT" Machine group comment
"MACHINE-GROUP-NAME" Machine group name
"MACHINE-OFFSET-ACTIVE" Machine offset being used
"MACHINE-POST-EXT" Current post file extension
"MACHINE-POST-LONG" Current post filename (including path)
"MACHINE-POST-NAME" Current post filename (name only)
"MACHINE-POST-PATH" Current post file path
"MACHINE-POST-SHORT" Current post filename (name and extension)
"MACHINE-TYPE" Machine type from machine definition (“M”= Mill, “L”= Lathe, “R”= Router, “W”= Wire, “X”= MillTurn)
”MAX-X” ”MAX-Y” ”MAX-Z” Deprecated
”MAX-X-TPLANE” ”MAX-Y-TPLANE” ”MAX-Z-TPLANE” Maximum XYZ. Values output relative to common tool plane and its origin, otherwise NA.
"MAX-X-WCS” "MAX-Y-WCS” "MAX-Z-WCS” Maximum XYZ. Values output relative to common WCS and its origin, otherwise NA.
"MAX-X-WORLD” "MAX-Y-WORLD” "MAX-Z-WORLD” Maximum XYZ in world coordinates
"METRIC" Metric (YES/NO)
”MIN-X” ”MIN-Y” ”MIN-Z” Deprecated
”MIN-X-TPLANE” ”MIN-Y-TPLANE” ”MIN-Z-TPLANE” Minimum XYZ. Values output relative to common tool plane and its origin, otherwise NA.
"MIN-X-WCS” "MIN-Y-WCS” "MIN-Z-WCS” Minimum XYZ. Values output relative to common WCS and its origin, otherwise NA.
"MIN-X-WORLD” "MIN-Y-WORLD” "MIN-Z-WORLD” Minimum XYZ in world coordinates
"MISC-INT" Miscellaneous integer (Post process customization values)
"MISC-REAL" Miscellaneous real (Post process customization values)
"MNMXN-TPLANE" View name if all operations use a common tool plane
"MNMXN-WCS" View name if all operations use a common WCS 
"MNMXV-TPLANE" View number if all operations use a common tool plane
"MNMXV-WCS" View number if all operations use a common WCS 
"NAME" Operation name (Contour/NoCore/Etc)
"OPCODE" Operation type (INTERNAL)
"OPIDN" Operation ID number (INTERNAL)
”OPNUM” Operation number based on OpMgr order
”PATH-LENGTH’ Length of wire path
"POWER" Power is on
"PROG-NUM" Program number
"RAPID-HEIGHT-ACTIVE" Rapid height is active
"RAPID-HEIGHT-INCR" Rapid height coordinate is incremental
"RAPID-HEIGHT" Rapid height coordinate (Z plane)
"SEQ-INCR" Sequence number increment
"SEQ-NUM" Starting sequence number
"START-POS" Start position
"SUPPRESS-CUT" Suppress cut position
"SUPPRESS-THREAD" Suppress thread position
”TAPER” Taper angle in degrees
"THREAD-POS" Thread position
”TOLERANCE” Linear tolerance
"TOOLPATH-GROUP-COMMENT" Toolpath group comment
"TOOLPATH-GROUP-NAME" Toolpath group name
"TOOLTYPE" Tool type: (M)ILL / (L)ATHE / (B)LOCK DRILL / (P)ROBE / (W)IRE
"UV-EXTENSION" UV extension coordinate (Z plane)
"UV-EXTENSION-ACTIVE" UV extension is active
"UV-EXTENSION-INCR" UV extension is incremental
"UV-HEIGHT" UV height coordinate (Z plane)
"UV-HEIGHT-INCR" UV height is incremental
"UV-THREADCUT-ACTIVE" UV common thread/cut is active
"UV-THREADCUT-MODE" UV mode (1=thread, 2=cut, 3=both)
"UV-THREADCUT-POINT" UV common thread/cut point
"UV-TRIM" UV trim coordinate (Z plane)
"UV-TRIM-ACTIVE" UV trim is active
"UV-TRIM-INCR" UV trim coordinate is incremental
"WIRE" Wire is on
"WIRE-DIAMETER" Wire diameter
"WIRE-RADIUS" Wire radius
"WORK-ORIGIN" Work origin (plane coordinates)
"XY-EXTENSION" XY extension coordinate (Z plane)
"XY-EXTENSION-ACTIVE" XY extension is active
"XY-EXTENSION-INCR" XY extension is incremental
"XY-HEIGHT" XY height coordinate (Z plane)
"XY-HEIGHT-INCR" XY height is incremental
"XY-TRIM" XY trim coordinate (Z plane)
"XY-TRIM-ACTIVE" XY trim is active
"XY-TRIM-INCR" XY trim coordinate is incremental
[BACKPLOT] Backplot information for this operation
[MAN-ENT-INFO] Point-specific entries for this operation
[TOOL] Power settings used in this operation
