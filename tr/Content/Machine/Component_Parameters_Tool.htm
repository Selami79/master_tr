Parameters tab: Machine Component Manager (Tool)

Use this tab to configure tool components. A tool component gives Mastercam the information it needs to associate a tool definition from a tool library with the machine definition so that the tool can be properly simulated. Tool components can be created in three ways:

  • Automatically, when you start the simulation function.
  • Using the Load tool on machine function. This automatically creates tool components from tools that have been selected for operations.
  • Right-click a tool-holding component in the Machine Definition Manager and select Add, Tool. For Mills, you typically add the tool to a tool spindle component, not a tool changer component.
NOTE

Tool components are only accessible in the local copy of the machine definition, not the master copy. This means that you should enter the Machine Definition Manager from the Toolpaths Manager, not the Job Setup group of the Machine tab.

Selecting the tool definition

Unlike most other components—where you create and store a solid model of the component—for tool components, Mastercam uses the information in the tool definition to display the tool during simulation. If a tool definition includes a custom profile, that profile will be rendered in the simulation.

You can select the tool definition from either a tool library, or from the active machine group. Click Select in the proper group to select the tool.

  • If you select a tool from a library, Mastercam loads a copy of the tool definition in the machine group, just like when you select a tool for an operation.
  • If the tool component was created automatically, the tool definition will have been automatically selected. If you are building the tool component manually, you need to select it yourself.
Editing the tool definition

Click Edit Tool to make changes to the tool properties. Any changes that you make affect the copy of the tool definition stored in the machine group. If the tool is already being used in a toolpath, your changes could require the toolpath to be regenerated.

Setting the tool in its turret or tool changer

Once the tool definition has been selected, use the bottom section to tell Mastercam how it is loaded in its tool changer or turret. Mastercam automatically enables the controls for a tool changer, turret, or gang tool depending on how the rest of your machine definition is set up. Enter the position number in the field or use the buttons to have Mastercam calculate it for you.

NOTE

Tool components are not required for all machine definitions. They are used only for machine simulation.

Parameters

Name

Enter the name that will be used for the component in the machine definition component tree.

Tool library

Click Select Mill Tool or Select Lathe Tool to load a tool from the active library. Loading the tool associates its tool definition and tool profile with the current tool component. Load tools from a different library by clicking the folder button and selecting a new folder of tool libraries, and then selecting the library from the list. These controls are disabled if a tool component is edited from anyplace except the Machine Definition Manager.

Machine group

Click Select Mill Tool or Select Lathe Tool to select from tools that are already loaded into the active machine group. These buttons are disabled if a tool component is edited from anyplace except the Machine Definition Manager.

Edit tool

Click to edit the tool definition associated with the current tool component. Any changes you make affect the copy of the tool definition stored in the machine group, and affect any operations for which the tool has already been selected.

Set component name to tool name

This button updates the component name with the tool name as stored in the tool definition. The component name is set to the tool name by default, if the tool component is generated automatically.

Automatic tool changer

Defines the ATC pocket with which this tool is associated. These controls are disabled if the tool is not loaded in a spindle, or if there is no ATC on the machine. The ATC pocket is set to the tool's station number by default if the tool component is generated automatically.

Turret

Defines the turret index position associated with the tool. Setting this properly is critical for locating the tool correctly on the turret. If the tool component is generated automatically, the turret index position is set to the tool locator index position or the tool station/tool number. These controls are disabled if the tool is not loaded in a turret.

Gang tool

Defines a unique identifier for this tool in a gang. These controls are disabled if the tool is not loaded in a gang. If the tool component is generated automatically, the gang tool identifier is set to the tool's station number.
