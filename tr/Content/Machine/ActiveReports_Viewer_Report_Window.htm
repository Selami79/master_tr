ActiveReports Viewer: Report Window

The report window automatically opens as a sub-window in the ActiveReports Viewer when you generate an ActiveReport from data in Mastercam. From this window you can view, search, and print labels and reports.

Define the default layout of any ActiveReport report type from the Reports property page in the System Configuration dialog box. Create and modify these templates with the ActiveReports Designer.

Parameters

Toggle sidebar

Toggles the navigation pane from hidden to displayed.

Print

Sends the current labels or report to the printer.

Copy

Copies the content from the current report or set of labels to your clipboard.

Find

Use this to search for text within the current report or labels by entering the text that you want to find.

Zoom out

Click to zoom out the view of the current report or label sheet(s). Every time you choose this button, the view zooms out incrementally by another 10% (i.e., 100%, 90%, 80%, etc.). You can zoom out to a minimum of 10% of the original view.

Zoom in

Click to zoom in the view of the current report or label sheet(s). Every time you choose this button, the view zooms in incrementally by another 50% (100%, 150%, 200%, etc.). You can zoom in to a maximum of 800% of the original view.

Current zoom

Displays current zoom level. Contains a drop-down list from which you can choose one of the pre-selected viewing percentages. You can also type in any number to zoom to a specific percentage of the current view.

Fit width

Click to fit the page width to the width of the window.

Fit page

Click to fit one full page to the window.

Single page view

Click to display a single sheet.

Continuous view

Click to scroll through all pages of reports or labels using your vertical scroll bar (similar to a Windows Print Layout view).

Multipage view

Click to display up to six sheets at once.

First page

Click to display the first page of the report.

Previous page

Click to display the sheet number previous to the one you are currently viewing. If you are viewing the first sheet of the report or labels, this button will be disabled.

Current page

Displays the current page and the number of pages in the report or label sheet. The format is displayed in an x/x format such as 4/10, designating that you are viewing the fourth page of ten pages. Type in a specific page number to jump to that page.

Next page

Click to display the sheet number following to the one you are currently viewing. If you are viewing the last sheet of the report or labels, this button is disabled.

Last page

Click to display the last page of the report.

Backward

Click to display the sheet number that you viewed previously by using the Previous or Next buttons. If you have not yet used the Previous or Next buttons, this button is disabled.

Forward

Click to display the sheet number that you viewed previously by using the Previous or Next buttons. If you have not yet used the Previous or Next buttons, this button is disabled.

Refresh

Click to reload the report.

Pan mode

Click to move the report in the window.

Selection mode

Click to select objects in the report. You can copy this content to the clipboard.

Snapshot mode

Click to select report content that you can paste as an image into any application that accepts pasted images.

Annotations

Add notes and comments to your report.

  1. Click the annotation button.
  2. Select the annotation type from the drop-down gallery. Drag and drop it to the desired location in the report.
  3. Right-click and select Properties to enter text and change the note's appearance.

Touch mode

Click to interact with the Viewer using simple touch gestures on any touch enabled Windows device.
