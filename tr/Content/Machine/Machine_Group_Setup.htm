Machine Group Setup panel: Mill/Router

The Machine Group Setup function panel (accessed from the Toolpaths Manager) manages machine groups for Mill and Router machines.

NOTE

  • While in the Machine Group Setup function panel, Mastercam creates a temporary viewsheet to show how your options affect the part, depending on the page you are currently on. Once the Machine Group Setup function panel is closed, the temporary viewsheet is deleted.

  • The Machine Group Setup panel displays a status indicator next to the page icons. This can be helpful when there is an error, but you are unable to locate which page contains it or to determine if a page is active or not.

What is the general workflow?

Machine Group Setup features a top-down workflow, represented by the icons on the left side of the panel. You begin your setup with the Machine page.

Setting up the machine

Separate sections in this page allow you to choose the files for the machine definition, default tool and operations libraries, and toolpath defaults. When you select a new machine, Mastercam loads a copy of the machine definition in your part file. When you select Configure machine to view or make changes to the machine definition, you are only changing the local copy stored in your part file, not the machine definition stored on your hard drive. Selecting a new machine definition updates the other paths and file names with the defaults in the new control definition.

NOTE

  • The control and post are displayed as read-only for information purposes only.

  • If there are already operations in the machine group, be careful when selecting a different machine to ensure the new machine supports all the toolpath features that have been programmed.

  • Refer to the Working with Toolpath Defaults Help topic for more information about toolpath defaults.

Setting up the master model

Use the Master Model page to define the final part after all of the machining is completed. This step is optional, but having a master model can be useful when using Mastercam Simulator. The selected master model is defined as the Workpiece when simulating, ensuring that you always have the correct finished part.

The master model can be made up of multiple entities. You can choose between a number of different selection options, located under the Selection list box.

Setting up the stock

Use the Stock Setup page to define the stock model. Stock models help you visualize your toolpaths more realistically in the graphics window. The stock model can also be displayed with the part geometry when viewing the file or toolpaths, or during Mastercam Simulator.

There are several methods to define your stock model. You can only have one selection as the stock model. The selection methods include the following:

  • Bounding Box function

  • From a file

  • A solid, a mesh or a stock model within the current part file

  • From two corners

  • From rectangular stock

  • From cylindrical stock

If necessary, you can also set the stock plane. For more information, refer to the How do I set the stock plane and when should I do this? drop-down.

Click on a solid face to use the Push-Pull function. Do this to extend the selected faces. You can set the mode to Absolute or Incremental. Additionally, you can select Both directions to modify both sides of the stock.

Setting up workholding components

Use the Workholding page to place and align fixtures and other workholding components. These components are used by Mastercam Simulator as Fixtures for collision checking. You can manipulate these components using the Align to Face and Dynamic Transform options.

You can group your selections, analyze them, or remove them by using the right-click menu in the Selection list.

Mastercam also allows you to select "Active" fixtures. By using the check boxes and the group creation capability you can have standard fixtures and activate ones you need. You can also leave non-active ones on the list. You are also able to rename Fixtures from the right-click menu.

Setting up your tooling

Use the Tool page to control how Mastercam assigns tools, tool offset numbers, and default values for feeds, speeds, coolant, and other toolpath parameters.

if you would like to number tools sequentially by default for all of your machine groups, refer to the Setting a Default Tool Numbering Method Help topic. You can also refer to the Working with Tool Numbers and Offsets for more information about how Mastercam uses tools.

Setting up simulation options

Use the Simulation page to modify your Mastercam machine definition (.mcam-mmd) for use with Mastercam Simulator. You can position the stock on the machine using this page, eliminating the need to re-enter Mastercam Simulator and reposition the stock each time there is an issue.

The default Machine comes from the Simulation tab of the General Machine Parameters dialog box. You can use the Post settings option to set post options before running Mastercam Simulator.

You can override the selected stock model when viewing operations in Mastercam Simulator. To do so, select your in-process stock model operation and subsequent toolpaths when entering Mastercam Simulator.

What is the purpose of each page?

The Machine Group Setup function panel is made up of different pages.

  1. Machine—Specifies the names and data paths of files needed by the operations in the machine group. This includes the machine definition and operation defaults.

  2. Master Model—Defines the final part after all machining is complete. This page also includes engineering information, such as the material.

    NOTE

    You do not need to define a master model. However, defining a master model provides workpiece data for Mastercam Simulator, ensuring that you always have a workpiece defined.

  3. Stock Setup—Defines the stock model. Stock models help you visualize your toolpaths more realistically in the graphics window. The stock model can also be displayed with the part geometry while viewing the file or toolpaths or during Mastercam Simulator.

  4. Workholding—Defines workholding components, such as fixtures or vices. You can also manipulate entities using Model Prep functions to place and align fixtures and other workholding components with your master model and stock.

  5. Tools—Controls how Mastercam assigns tool numbers, feeds, speeds, coolant, and other toolpath parameters.

  6. Simulation—Allows you to modify your Mastercam machine definition (*.mcam-mmd) for use with Mastercam Simulator.

Why am I getting a warning message when deleting geometry?

A warning message will appear if you attempt to remove geometry associated with a Toolpath, a Solid, or a Plane. Mastercam warns that if the geometry is deleted, it will permanently alter those operations. If you delete an entity, it will no longer be on the panel.

How do I...?

How do I replace a machine definition?

Mastercam lets you change the machine definition associated with the machine group. For example, if you need to run a job on a different machine, you can replace the machine definition instead of recreating your toolpaths. Mastercam validates the existing operations against the new machine definition. It also loads the post processor and control definition settings for the new machine. This ensures that when you post the operations, all of the tolerances and other NC settings are appropriate for the new machine.

For Mill or Router machines, select File under Properties in the Toolpaths Manager. Then choose Select machine definition.

How do I add group comments to operations?

To add a comment to every operation in the current machine group, select the Group comments option on the Machine page. The text entered in the Comment field, also located on the Machine page, is then added to all operations.

Mastercam also lets you record operation-specific comments. This is a comment that can be entered in every operation and can be different for each operation.

How do I create or edit toolpath default (*.mcam-defaults) files?

When you create a new toolpath or other operation, Mastercam populates the fields on each tab or properties page with default values. These values are stored in a file with a .mcam-defaults extension. These defaults are specific for each type of operation; for example, a contour toolpath, a facing toolpath, or a feature based drill operation.

You can create new toolpath default files (known as *.mcam-defaults) or edit existing ones. For more information about working with toolpath default files, refer to the Working with Toolpath Defaults Help topic.

How do I create or edit an operation defaults library (*.mcam-operations)?

Follow these steps to create a new operations library. This is a file with an .mcam-operations extension that contains commonly used toolpath operations that you can select and import into your current part file. Begin creating new operations files from a machine definition already loaded in the Toolpaths Manager.

  1. In the Toolpaths Manager, select Files under Properties.

  2. In the Machine page, select Edit operation library.

  3. Select Browse to open the Select Operation Defaults File dialog box.

  4. Right-click the operations file on which you want to base your new library, and choose Copy.

  5. Right-click in the window and choose Paste.

  6. Right-click the new copy of the operations file, choose Rename, enter the new file name, and press [Enter].

    NOTE

    When renaming your file, make sure not to change the file extension.

  7. Re-click the new file name. Make sure the new operations file name appears in the File name field.

  8. Click OK.

  9. If prompted to save changes to your operations library, click No.

  10. Edit the new operations library as necessary.

    NOTE

    You can also empty the new file and start with a blank file by choosing Delete all operations from the toolbar.

  11. When finished, click OK to save your changes and return to the Toolpaths Manager.

To edit an operations library, follow to step 4 and modify the operations in the list. You can do the following:

  • Edit the default parameters for a specific operation in the list by clicking Parameters.

  • Work in the operations list much like you can in the Toolpaths Manager. You can access many different options from the right-click menu, including adding, editing, and deleting operations. You can also perform other tasks similar to those in Toolpaths Manager such as selecting multiple operations, dragging and dropping operations within the list, and changing the names of operations.

Click OK when you are done. The operations library you just worked on is set as the current library for that machine group.

How do I set the stock plane and when should I do this?

You can select a stock plane to properly orient the stock model with respect to your part. This is usually done when creating toolpaths in a work coordinate system (WCS) other than Top and you want to align the stock model to your part. If you have several toolpaths in the machine group that use more than one WCS, Mastercam will use the stock plane to keep the stock model constant when the WCS for each operation changes. Mastercam stores the stock plane with the stock definition, independent of the WCS.

You can align the stock model to any plane saved in your part file. You can also create a new plane. When you choose a stock plane, the edges of the stock model will be parallel with the axes of the selected plane. These options are on the Stock Setup page, under the Stock Plane Transformation expander.

After selecting your plane, use Select origin to choose a point on the stock to use as the stock origin. The stock origin locates the stock relative to your part.

NOTE

If you change the stock plane, you may need to reset the stock origin.

How do I simulate my machine group?

If you wish to simulate your machine group and the operations within it, select Verify selected machine group from the Verify selected operations drop-down. Mastercam Simulator is then launched in Verify mode with the selected machine group.

How do I know what the current active stock is?

Mastercam lets you define multiple stock definitions, although only one at a time can be active. Initially, the first stock definition you create is active. To change the active stock definition, right-click a definition in the list, and choose Set as active stock. In the Stock Setup page, a green check indicates the active stock definition.

 

How do I know what stock I have on my screen?

When selected, the active stock defined on Stock panel will be displayed. If the Override is active it will be displayed instead.

How do I use the dynamic translation tool to move fixtures and stock to the proper location on the machine?

Mastercam lets you move fixtures and stock to the proper location on the machine. You must first have both Stock and Fixtures selected under Preview Settings.

You can move stock and/ or fixtures using the position location (X, Y, and Z) by manually entering values for X, Y, and Z which will update the new stock and/ or fixtures location.

Additionally, you can move stock and/ or fixtures by selecting the gnomon. When selecting the gnomon, first select the stock and/ or fixtures you want to move and then select the location of where you want it to move.

How do I set up my material in the Master Model or Stock Setup?
Mastercam has several ways to select material. Follow these steps when working with Mill or Router machine groups:

  1. Click Tool settings in the Toolpaths Manager.

  2. From the icons on the left side of the panel, click Master Model or Stock Setup. The Select material icon (see the following image) appears in the Engineering Information section under the currently selected material.

  3. Click Select material.
  4. If the panel displays the material you want to use, select the material, and click OK.
  5. If the panel does not display the material you want to use:
    1. Select a source from the Source list box.
    2. Choose the desired material, and click OK.
  6. If you want to define a new material:
    1. Click the Add a material button.
    2. Define the material in the fly-out panel, and click OK.
    3. Select the new material from the list, and click OK.

Use the Material fly-out to select materials and manage the current list or the library. Right-click on the material list box to see a menu of options. These will let you edit, create, or remove material.

Parameters

Add

Opens another material fly-out so you can add a new material definition. The new material will be saved in the current part file.

Edit

Opens another material fly-out so you can make modifications to the material.

Remove

Removes the selected material from the list.
