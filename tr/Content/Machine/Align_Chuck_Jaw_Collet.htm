Align Chuck/Jaw/Collet panel

This panel appears when you have selected a solid model while creating a chuck, collet, or chuck jaws component. Use this tab to align your model within the machine's coordinate system.

  • Follow the prompts on the screen to tell Mastercam how to orient the model and which point on the model maps to the origin.
  • Use the Defined and Opposite buttons on the function panel to flip the coordinate system. The initial orientations of Defined and Opposite are established by where you click when you select the model.
Working with chucks, collet chucks, and collets

After you select the solid model, Mastercam prompts you to select the solid model to establish the Z-axis direction. For most applications, click on the back of the chuck or collet. This sets the positive Z-axis direction moving from the back of the component towards the front.

Mastercam then prompts you to set the zero point for the Z-axis. In this example, the back of the model as imported is the end of the mounting pins. Use the dynamic gnomon to move the origin to the back of the chuck face.

The location that you establish here will be mated to the origin of your machine coordinate system; this is usually the center line of the spindle face. You can also use the gnomon to align the X-axis if necessary. For example, if you are working with a standard chuck, you might want to do this so that the channel for the jaws is aligned properly.

After you press [Enter] to accept the model's alignment, Mastercam displays the model in the Preview window of the Geometry tab, where you can define other properties. Right-click in the Preview window to look at the component in different views and make sure the alignment is correct.

Working with chuck jaws

The difference between working with chuck jaws and the other component types is that jaws are not radially symmetrical, so you need to align the X axis in addition to the Z axis.

Consider this example. The model is drawn in a typical orientation, with the ID clamping edge of the jaws aligned with the X axis.

After you select the solid model, Mastercam prompts you to select a face. This establishes the Z-axis direction. Select a face of the model that is parallel with the chuck or spindle face. For most applications, click on the back of the model. This sets the positive Z-axis direction moving from the back of the component towards the front.

Mastercam then prompts you to set the zero point for the component. In this example, the origin is on the wrong side of the model; it should be located along the ID clamping edge.

On the function panel, select Opposite for the Machine X Orientation.

Mastercam flips the model as shown here. Now the model is oriented properly.
