Adding a Turret Mounted Steady Rest to a Component Library (Mill-Turn)

Mastercam supports two types of steady rest component:

  • Standalone steady rests are permanently attached to your machine. These are defined in the machine definition (.mcam-lmd file) by a licensed machine developer. Only users with a Mill-Turn Developer license can create them.

  • Turret mounted steady rests are stored in component libraries (.mcam-gmd files). They can be selected and mounted during the Job Setup process, even if they are stored in a library outside your .machine file. Any Mill-Turn user can create them. This topic is about turret mounted steady rests, not standalone steady rests.

Creating a turret mounted steady rest and adding it to a component library requires a number of different tasks. This topic gives you a high-level description of the overall process with links to more detailed step-by-step procedures. Read and understand this topic before beginning any of the individual tasks.

  1. Acquire or create the component models. You will need separate models for the steady rest body, each arm, and piston. Save these as either Mastercam solids or mesh files. See Models for Steady Rest Components to learn more.

  2. Open the component library and import the models into it. You can add the steady rest to the component library in your .machine file or add it to a separate generic library that you can use with any .machine file. See Importing Solid Models into a Component Library to learn more.

  3. Create the component definition and add it to the desired turret. The steady rest component is added to the turret alongside the tool locators. This is the heart of the process: you will create the component definition, add the component models to it, and tell Mastercam how the steady rest mounts to the turret. See Creating the Component Definition for a Turret-Mounted Steady Rest (Mill-Turn) to learn more.

  4. Define the arm and piston subcomponents. See Adding an Arm to a Mill-Turn Steady Rest and Adding a Piston to a Mill-Turn Steady Rest.

  5. Save the completed steady rest back to the component library and validate that it mounts properly to the turret. See Previewing and Validating Components in a Library to learn more.

  6. Save the updated component library back to the .machine file, then save the .machine file. See Saving Updated Files to a .machine File to learn more.
