Machine Simulation Options dialog box

Use the Machine Simulation Options dialog box to customize Machine Simulation's appearance and functionality. To display the Machine Simulation Options dialog box, open Machine Simulation, click the File tab, and then choose Options.

Simulation Properties

Use these settings to configure Machine Simulation's general simulation properties.

Pop-up notifications

Use these options to enable or disable the display of pop-up notifications when the following events happen:

  • Collision & Gouge: When a collision happens between machine components or between a tool and stock, or when a gouge happens.
  • Proximity Alert: When the tool, stock, or machine components become very close to each other and could potentially collide.
  • Axis Limit Overrun: When a linear or rotational axis exceeds it physical limit.
  • Axis Value Mismatch: When the move values do not match the current move.
Miscellaneous

Use these options to show the edges/outlines of the machine components by a given thickness and to show the rotation direction arrows for the tool spindle and stock/workpiece spindle.

Toolpath backplot

Use these options to change the segments of Machine Simulation's backplot. The available options are as follows:

  • Toolpath line: Sets the size of the toolpath line.
  • Segment: Defines the amount of segments shown when Tool Path Rendering is set to Segment.
  • Axis vector length: Defines the length of the tool axis vector. This option can be based on the tool radius, or by a user defined value.
  • Toolpath points: Sets the size of the toolpath points.
Reset

Click Reset page settings to reset the settings on this page to their default values.

Verification Properties

Use these settings to configure Machine Simulation's general options for working with Simulation when it is set to Material Removal, such as collision alerts and engine settings.

Machine components collisions and alerts

Use this option to enable machine components collision and proximity alerts when the Simulation mode is set to Material removal.

Miscellaneous

Use this option to show edges/outlines of the stock material by a set line thickness.

Verification engine settings

Use this option to enable or disable true threads cutting for thread mills. To display an accurate helical thread, this option must be enabled and your Accuracy slider on the CutSim pane needs to be set to the maximum. The threads will then be true helical threads; otherwise, they are approximated cylindrical cuts.

Reset

Click Reset page settings to reset the settings on this page to their default values.

Graphics and Background

Use these settings to configure the background color, screen objects, animation speed, the Machine Simulation theme, and tips.

Screen Objects

Use these options to customize objects that display in the graphics window of the Machine Simulation. The options are as follows:

  • Coordinate System: Select to enable or disable the global coordinate system. Use the drop-down menu to set its position.
  • Show Rotation Cube: Select to display the rotation on the coordinate system.
  • Ruler: Select to enable or disable the ruler. Use the drop-down menu to set its position.
  • Show Machine Center Point: Select to enable or disable the machine center point.
  • Show Workpiece/Stock Center Point: Select to enable or disable the workpiece and stock center point.
  • Additional Info based on Simulation Mode: Select to enable or disable the display of additional information depending on the simulation mode selected.
Background

Use these options to set the background of the Machine Simulation. The background can be a solid color, a gradient, or selected from a file.

  • When using Gradient, you can set the orientation of the gradient and 2/3 of the screen will be the start color.
  • When using From File, you can set the position of the image.
Animation Speed

Use the slider bar to set the default animation speed of the simulation. This can also be adjusted outside of the Machine Simulation Options dialog box by using the Quick Access Toolbarr or the Simulation tab.

Machine Simulation Style

Use these options to set Machine Simulation's appearance.

  • Color Scheme: Sets the theme and color of Machine Simulation.
  • Office 2010 only: File tab: Sets the color of the File tab when the theme is set to Office 2010.
  • Office 2013 only: Accent Color: Sets the accent color. The accent color affects the File tab and bottom of the Machine Simulation window.
  • Ribbon Bar Background Image: Only applies when the Color Scheme is set to Office 2013. This option changes the image displayed in the upper-right corner of Machine Simulation.
Tips

Use these options to enable or disable tooltips for Machine Simulation's tabs. If enabled, you can then set the Screentip style which allows you to display the feature title and description, to not display the feature description, or to not display any information.

Reset

Click Reset Page Settings to reset the settings on this page to their default values.

Customize

Use this page to customize Machine Simulation's tabs, keyboard shortcuts, and mouse shortcuts

Tabs

Add and remove commands

To add a command to a tab:

  1. From the Choose commands from drop-down, select All Commands. The list populates with all the available functions:

  2. In the Customize the Ribbon list, expand the tab you wish to add a command to and select a group.
  3. Select a function from the Choose commands from list and click Add.
  4. The function has now been added to the selected group.

To remove a command from a tab:

  1. From the Customize the Ribbon list, select a function you wish to remove.
  2. Click Remove
  3. The function has now been removed.
Add a new tab

  1. Click New Tab .

  2. A new tab displays in the list, named New Tab (Custom).
  3. To rename the tab, right-click on the custom tab and choose Rename, or select the tab in the list and click Rename .
  4. You can now add groups and functions to the custom tab.
  5. Click OK in the Machine Simulation Options dialog box to accept the changes and display the new tab.
Edit the default tabs

You can do the following to the default tabs:

  • Rename the tabs.
  • Hide the tabs.
  • Add and remove groups.
  • Add commands, but only to a new group.

You cannot do the following to the default tabs:

  • Delete the tabs.
  • Remove commands, unless they are in a user-created group.
What are tabs and groups?

Tabs are the different ribbon bars located at the top of the Machine Simulation window.

Groups are sections within the tabs that are generally named after the type of functions within them.

Keyboard and mouse shortcuts

Customize keyboard shortcuts
  1. Click Customize next to Keyboard shortcuts.
  2. In the Customize Keyboard dialog box, select the command you wish to assign to a new shortcut key.
  3. Click Press new shortcut key and press a key to assign to the command. If the key has already been assigned to another command, it will display in the Currently assigned to parameter.
  4. Click Assign to assign the new shortcut key to the function.
Quick Access Toolbar

Use this page to customize the Quick Access Toolbar, located in the upper-left corner of the Machine Simulation window.

To add a command to the Quick Access Toolbar

  1. From the Choose commands from drop-down, select All commands. The list populates with all the available functions.

  2. Select a function in the Quick Access Toolbar list.

  3. Select a function from the Choose commands from list and click Add.

    The new function is added after the selected function.

To remove a command from the Quick Access Toolbar
  1. From the Quick Access Toolbar, select a function to remove.

  2. Click Remove.

    Mastercam removes the function.

You can also add and remove functions by using the drop-down at the end of the Quick Access Toolbar, shown below.

Capture

Use these settings to configure image and video capturing.

Image Capture

Use these options to customize image capturing in Machine Simulation:

  • Capture to File: Select to capture an image to a file.
  • Capture to Clipboard: Select to capture an image to your clipboard, so it can then be pasted into another program.
  • Image Size: Use the slider to set the size of the image.
  • Maintain Image Aspect Ratio: Select to maintain the image aspect ratio, if the image needs to be resized.
Video Capture

Use these options to set and configure the compression. Use the drop-down to change the compressor type. Use the Recording Frame Rate option to set the frame rate for the video.

General

Set the location of the image and video output. Click the button to choose a new location.
